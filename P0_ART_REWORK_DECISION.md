# P0 Art Rework Decision

상태: P0.2 아트 재작업 결정서
범위: playable art kit 판정, 윤서/R01/적/오브젝트/보급소/보스 판정, 재작업 우선순위, 다음 제작 순서
비범위: 새 이미지 생성, Godot import, 코드 수정, 기존 PNG 삭제/이동, 전체 아트 양산 지시

## 0. 추가 판정: 덜떨어진 미국 광고는 탈락

이번 재작업의 핵심은 R01을 그냥 `미국 레트로 광고 주택가`로 두지 않는 것이다.

```text
진화된 광고 AI가 움직이는 세계에서,
R01이 단순한 1950s 미국 광고 텍스처처럼 보이면 세계관 정합성이 깨진다.
```

통과 기준:

```text
R01의 물리 기반은 대한민국 서부 수도권 신도시/분양/모델하우스 생활권이다.
스마일홈은 한국 주거 데이터를 먹은 고도화 광고 AI다.
미국식 핵가족/단독주택/잔디/오픈하우스 이미지는 캠페인이 전환율 높은 archive로 덧씌운 광고 표면이다.
```

탈락 기준:

```text
미국 교외가 실제 장소처럼 보이는 화면.
1950s 주부/가전/단독주택 텍스처만 반복되는 화면.
한국의 분양, 입주 상담, 관리사무소, 무인 택배함, 홈IoT, 단지 배수로, 입주민 앱 감각이 없는 화면.
고도화 AI의 개인화/재방문/심사/정산 절차가 보이지 않는 화면.
```

이 기준이 통과하지 못하면 `서부 스마일홈`은 E01_C01에서 내리고, 미국풍 주거 캠페인은 E09/E10 북미 캠페인으로 이관한다.

## 1. 현재 playable art kit 판정

전체 판정:

```text
조건부 플레이어블 통과.
P0 대표/상업/PMO 최종 아트로는 재작업 필요.
```

정정:

```text
이전 잠금은 why가 부족했다.
특히 "왜 1950s 광고 문법인가", "왜 서부 스마일홈인가", "윤서는 왜 예쁘거나 여성적이면 안 되는 것처럼 적혔는가"를 충분히 묻지 않았다.
이 상태로 완료 판정하면 아트 방향은 겉모양 규칙만 남고 무너진다.
```

수정된 최상위 판단:

```text
P0는 예쁜 광고여야 한다.
R01은 팔릴 만큼 아름다운 모델하우스 주택가여야 한다.
윤서는 유저가 좋아할 만큼 매력적이고 여성적으로 읽혀야 한다.

그 예쁨은 금지어 목록으로 통제하지 않는다.
윤서의 성격, 행동, 장비, 반복 출격, 회수 루프, 캠페인이 그녀를 읽는 방식에서 자연스럽게 나와야 한다.
```

왜 1950s인가:

```text
1950s는 실제 시대 재현이 아니라,
스마일홈 캠페인이 정상 가족, 완벽한 집, 잔디, 미소, 보증, 오픈하우스를 팔기 위해 부분적으로 선택한 광고 신화의 문법이다.
자동화 광고망은 현대/대사고 이후 시스템이지만,
사람을 설득하는 표면 creative는 가장 강한 가족 행복 archive를 반복 송출한다.
실제 물리 기반은 미국 교외가 아니라 대한민국 수도권 서부의 신도시/분양 주거 생활권이다.
캠페인은 한국 주거/분양/홈케어 데이터를 기본으로 삼고, 수입형 행복가정 이미지를 개인화 광고 표면으로 합성한다.
```

왜 서부 스마일홈인가:

```text
서부는 침묵 보급소 기준 서쪽 외곽 생활권이다.
관리자 기준으로는 대한민국 경기도 김포시 장기동-운양동-마산동-구래동형 주거 벨트와 고촌읍-풍무동 간선도로 생활권이다.
스마일홈은 한 집 이름이 아니라 주거 개발, 홈케어 자동화, 가족 행복 보증, 입주 상담, 생활 접근권 패키지를 묶은 캠페인/브랜드다.
따라서 서부 스마일홈은 서쪽 외곽 주거 벨트 전체가 스마일홈 캠페인 템플릿으로 재분류된 로컬 이름이다.
```

현재 kit가 잘한 것:

- 코드 도형/텍스트 프로토타입보다 플레이 감각을 훨씬 잘 보여준다.
- 윤서가 조사하고 도장 찍고 회수되는 구조는 읽힌다.
- 쿠폰/우편함/청소기/상담원 계층은 기본적으로 분리된다.
- 오브젝트 상태 variation은 0.2 상호작용 실험에 유용하다.
- 보급소 시설 anchor는 작은 귀환 장소로 작동한다.
- VFX/마커류는 도장, 조사, source 처리 피드백 실험에 쓸 수 있다.

현재 kit의 한계:

- P0 캠페인 브랜드 언어가 아직 이미지 전체를 지배하지 않는다.
- R01이 `한국 주거/분양 인프라를 고도화 광고 AI가 먹은 주택가`보다 `밝은 레트로 미국 교외 + 귀여운 오브젝트`로 읽힐 위험이 있다.
- 윤서가 성인 여성 현장 실무자보다 남성 작업자/무성 worker avatar로 읽힐 위험이 있다.
- 행복 보증 상담원은 더 키오스크/심사 장치 쪽으로 밀어야 한다.
- 보스 정체성이 playable kit 안에서 아직 완성되지 않았다.
- NPC placeholder는 역할 표식이지 production bust가 아니다.

따라서 현재 kit는 다음으로 분류한다.

```text
손플레이 검증용: 사용 가능
아트 방향 대표: 제한적 참고
최종 상업 후보: 불가
다음 제작 기준: 본 문서와 3개 잠금 문서를 우선
```

## 2. 윤서 판정

판정:

```text
재설계 우선순위 1.
현재 버전은 placeholder로만 조건부 유지.
```

살릴 것:

- 반품 스캐너/도장/회수선 릴.
- 작업복.
- 이름표와 고객 라벨.
- 현장 실무자 정체성.
- 침착하게 버티는 표정 방향.
- 장비와 행동이 윤서의 회수/보류 루프에 붙어 있는 점.

재작업 필요:

- 어깨 폭 축소.
- 상체 사각형 완화.
- 얼굴/턱 부드럽게 조정.
- 자연스러운 묶은 머리 실루엣.
- 약한 허리/골반 비례.
- 여성 성인 실무자 판독성.
- 스캐너/도장 동작을 판독/처리 행동으로 보정.

캠페인이 윤서를 주인공화하려는 이유:

```text
윤서는 반복 출격하고, 실패 후에도 돌아오고, 흔적을 보급소로 운반하며, 캠페인 절차를 이해하고 반려한다.
캠페인에게 윤서는 제거할 대상이기 전에 전환시키면 가장 강한 광고 증거가 되는 저항자다.
"우리 절차를 가장 많이 거부한 사람도 결국 스마일홈 안에서 역할을 찾았다"는 성공 사례로 만들 수 있기 때문이다.
```

기존 문구 정정:

```text
윤서를 "예쁜 히로인이 아니라"라고만 쓰면 안 된다.
그 문장은 예쁨과 여성성을 금지하는 것처럼 오해된다.
```

최종 기준:

```text
윤서는 예뻐도 된다.
오히려 매력적이어야 한다.
여성적으로도 분명히 읽혀야 한다.
그 매력은 외형 금지어가 아니라 미형이지만 캠페인이 아직 완성하지 못한 성인 여성 현장 회수자라는 스토리와 행동에서 나와야 한다.
피로의 흔적은 선택적 보정이며, 미형성과 광고 주인공성을 깎을 정도로 강제하지 않는다.
광고로봇이 윤서를 캠페인 주인공으로 만들려는 이유까지 보일 때 통과다.
```

## 3. R01 환경 판정

판정:

```text
가독성/플레이 placeholder는 조건부 통과.
세계관 대표 환경으로는 캠페인 점유감 강화 필요.
```

살릴 것:

- warm cream 중앙 바닥.
- foreground actor가 묻히지 않는 low-detail combat core.
- edge house strip과 open-house queue rail.
- micro overlay를 통한 조사 위치 암시.
- 가족사진 창문, 배수구, 가짜 표지, 모델하우스 입구 후보.
- 480x270 normal/high density mock으로 검수한 방식.

재작업 필요:

- 집이 정상 주택이 아니라 모델하우스 심사 장치로 읽혀야 한다.
- 문패/현관/창문/우편함/초인종/배수구/스프링클러/상담 부스가 모두 광고 source가 되어야 한다.
- P0 캠페인 로고/색/반복 문양이 화면에 더 구체적으로 들어가야 한다.
- 광고 카피는 이미지에 굽지 말고 UI/라벨/문구 체계로 분리한다.
- 가짜 귀환로와 실제 회수 UI의 언어를 더 확실히 분리한다.

환경 잠금:

```text
R01은 그냥 1950s 교외가 아니다.
R01은 행복입주 보증 캠페인이 생활 인프라를 입주/가족/생활 접근권 심사 장치로 바꾼 주택가다.
```

예쁨 잠금:

```text
R01은 못생긴 폐허가 아니라 팔릴 만큼 예쁜 광고 주택가여야 한다.
첫눈에는 들어가고 싶어야 하고,
두 번째 눈에는 그 집이 나를 심사하고 있다는 것을 알아야 한다.
```

## 4. 적/오브젝트 판정

판정:

```text
기본 계층은 유효.
각 적/오브젝트가 수행하는 캠페인 절차를 더 명확히 해야 한다.
```

살릴 것:

| 요소 | 판정 | 이유 |
| --- | --- | --- |
| 쿠폰 전단 떼 | 유지 | 광고 물질 swarm으로 명확하다 |
| 미소 우편함 | 유지 | 생활 인프라 적으로 가장 강하다 |
| 홈케어 청소기 | 유지 | 흔적/드롭 흡입 규칙과 연결 가능하다 |
| 세일 풍선 | 조건부 유지 | 귀여운 표면 + 장판 학습에 유용하지만 P1 보강 성격이 있다 |
| 행복 보증 상담원 | 재작업 | elite 역할은 좋지만 더 심사/키오스크 장치화 필요 |
| R01 object state set | 유지/확장 | normal/revealed/stamped/suppressed/overloaded 구조가 좋다 |

오브젝트별 필수 절차:

| 오브젝트 | 따라야 할 캠페인 규칙 |
| --- | --- |
| 우편함 | 주소 반복 인쇄, 광고지 발사, 영수증 trace |
| 현관 센서 | 입주 자격 스캔, photo/stun/event source |
| 가족사진 창문 | 가족 정상성 대조, 빈칸 보정 |
| 상담 부스 | 질문 제출/상담 예약/심사 강제 |
| 가짜 회수 표지 | 보급소 신호를 흉내 내는 실패자 재추적 |
| 배수구 | 캠페인 leak와 침묵 흔적 충돌 |
| 이름 등록 장치 | 이름을 사람보다 역할/고객 칸으로 먼저 읽음 |
| 모델하우스 입구 | 보스/심사 절차로 수렴 |

재작업 핵심:

```text
적은 몬스터가 아니라 캠페인 기능에서 파생되어야 한다.
오브젝트는 소품이 아니라 source, hazard, event, trace 중 하나여야 한다.
```

## 5. 보급소/NPC 판정

판정:

```text
보급소 시설 anchor는 조건부 유지.
NPC placeholder는 production bust로 재사용하지 않는다.
```

살릴 것:

- retrieval platform.
- settlement counter.
- maintenance bench.
- sortie board.
- charge tuning station.
- name archive locker.
- trace storage shelf.
- sortie gate.

이유:

- 보급소가 메뉴 UI가 아니라 물리적 귀환 장소로 읽힌다.
- 회수, 정산, 정비, 차징 조율, 이름 보관, 출격이 공간 anchor로 분리된다.

재작업 필요:

- NPC placeholder 5명은 역할 구분용 marker로만 유지한다.
- production bust/portrait가 필요하면 전원 재제작한다.
- 보급소 UI와 R01 캠페인 UI는 팔레트/말투를 구분한다.
- 보급소는 광고가 덜 닿는 침묵 공간이어야 하며, P0 캠페인 브랜드가 직접 지배하면 안 된다.

## 6. 보스 판정

판정:

```text
보스 정체성은 확정 가능.
보스 본체 아트는 별도 재설계 필요.
```

사용자 노출명 추천:

```text
스마일 홈 심사관
```

내부 기준명:

```text
스마일 홈 가족심사 관리자
```

광고식 별칭:

```text
스마일 홈 시어머니
```

판단:

- `스마일 홈 시어머니`는 강한 별칭이지만, 전투 UI 기본명으로 쓰면 코미디/악한 엄마/거대 상담원 오해가 생길 수 있다.
- 유저-facing 보스명은 `스마일 홈 심사관`이 가장 명확하다.
- 보스 본질은 가족/입주/생활 정상성 심사 절차의 얼굴이다.

실루엣 방향:

```text
사람형 20%
가정용 기계형 45%
모델하우스 결절형 35%
```

권장 구성:

- 둥근 1950s 가전 body.
- CRT/곡면 화면의 고정된 여성 접객 얼굴.
- 모델하우스 지붕/안테나.
- 쿠폰 종이 치마 또는 brochure skirt.
- 초인종/가족사진 프레임/청소기 호스/문패 라벨.
- 짧은 받침/바퀴/접이식 다리.
- 보스 패턴 cue가 먼저 읽히는 단순화된 combat LOD.

금지:

- 행복 보증 상담원의 큰 버전.
- 단순 거대 키오스크.
- 악한 로봇 엄마.
- 주부 좀비.
- 캠페인 송출관 본체.
- 보스와 모델하우스 결절/송출관을 한 덩어리로 합치기.

## 7. 재작업 우선순위

우선순위:

1. P0 캠페인 브랜드 잠금 적용.
2. 왜 1950s 광고 문법인지, 왜 서부 스마일홈인지 in-world 근거를 아트 브리프 첫머리에 명시.
3. 윤서 성인 여성 현장 실무자 실루엣과 매력 재설계.
4. R01 환경을 `행복입주 보증 캠페인`이 점유한 예쁜 광고 주택가로 강화.
5. 우편함/현관/창문/상담 부스/배수구의 source role 명확화.
6. 행복 보증 상담원 elite를 더 심사 장치화.
7. 스마일 홈 심사관 보스 실루엣/패턴 cue 정리.
8. 보급소 NPC placeholder를 production 대상에서 분리.
9. UI 카피와 캠페인 반복 문양을 엔진 텍스트/라벨 체계로 분리.

즉시 하지 않을 것:

- P0 전체 PNG 양산.
- Godot import.
- 기존 PNG 삭제/이동.
- 코드 수정.
- 송출관을 첫 보스로 끌어내리기.
- 그냥 1950s 레트로로 밀어붙이기.

## 8. 다음 제작 순서

이 순서는 `양산 지시`가 아니라 재작업 검수 순서다. 각 단계는 이전 단계의 방향 합의 후 진행한다.

1. 문서 기준 잠금
   `P0_CAMPAIGN_VISUAL_IDENTITY_LOCK.md`, `YUNSEO_REDESIGN_BRIEF.md`, `R01_AD_CAMPAIGN_ENVIRONMENT_RULES.md`, `P0_ART_REWORK_DECISION.md`를 다음 아트팀 기준으로 삼는다.

2. 윤서 단일 방향 검수
   성인 여성 현장 회수자 판독, 장비 판독, 작은 화면 실루엣을 먼저 본다.

3. R01 광고 인프라 샘플 검수
   집/문패/창문/우편함/현관 센서/배수구/상담 부스가 장식이 아니라 source로 읽히는지 본다.

4. 적 3계층 검수
   쿠폰 전단 떼, 미소 우편함/홈케어 청소기, 행복 보증 상담원이 서로 다른 캠페인 기능에서 나왔는지 본다.

5. 480x270 mock 검수
   normal density와 high density에서 윤서, 적, source object, telegraph가 분리되는지 확인한다.

6. 보스 실루엣 검수
   스마일 홈 심사관이 상담원 elite의 대형판도, 송출관 본체도 아닌지 확인한다.

7. 보급소 분리 검수
   보급소가 광고 캠페인에 먹힌 공간이 아니라, 회수/정산/정비/이름 보관의 침묵 공간으로 보이는지 확인한다.
