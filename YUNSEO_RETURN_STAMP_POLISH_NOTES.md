# Yunseo Return Stamp Polish Notes

상태: P0-7A-2 차징 무기 실험 정리 메모

## PMO 결정

- 0.2 기준 차징 구조는 `캐릭터 1명 = 고유 차징 무기 1개`.
- 윤서의 고유 차징 무기는 `반품 도장`.
- 런타임 Tab 전환, 회수선, 긴급 방송은 0.2 윤서 기능에서 제외한다.
- 회수선/긴급 방송은 향후 다른 캐릭터 또는 operator 설계 후보로만 보관한다.

## 윤서 반품 도장에 남길 아이디어

- 반품 도장은 직선 검사/관통/표식 부여가 핵심이다.
- 표식이 붙은 적은 자동공격과 분열 광고탄의 우선 타겟이 되면 윤서의 직업성이 살아난다.
- 완벽 차징은 더 많은 적을 맞히기보다 표식 지속시간, 추적 우선도, 시각 피드백을 강화하는 방향이 좋다.
- 반품 도장 발사음은 짧고 명확한 스탬프/검수 사운드가 어울린다. 단, 사운드 생성 함수는 별도 polish 작업에서 다시 작게 넣는다.
- 미리보기는 넓은 부채꼴보다 얇고 긴 검사선이 맞다. 너무 넓으면 회수선/방송과 역할이 겹친다.
- 명중 피드백은 `반품 검사`, `반품 표식`, 붉은 도장 링, 짧은 종이 타격선을 중심으로 간다.

## 제외할 실험 요소

- Tab / Shift+Tab 무기 전환.
- 회수선 투척과 착탄지 끌어당김.
- 긴급 방송 광역 표식과 연쇄 폭발.
- 무기별 카드 해석을 윤서 0.2 기본 기능처럼 노출하는 구조.

## 다음 작업 후보

`윤서 반품 도장 polish`에서는 런타임 전환을 만들지 않고, 기존 차징 입력이 항상 반품 도장으로 작동하도록 정리한다.
