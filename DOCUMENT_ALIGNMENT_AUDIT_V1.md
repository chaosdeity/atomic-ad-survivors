# Atomic Ad Survivors Document Alignment Audit V1

## 0. 문서 목적

이 문서는 기존 문서들이 `PROJECT_DIRECTION_LOCK_V1.md`와 어디서 맞고, 어디서 충돌하는지 정리한다.

이 문서는 기존 문서를 즉시 삭제하기 위한 문서가 아니다. 어떤 문서를 살리고, 어떤 표현을 수정하고, 어떤 새 문서로 대체할지 결정하기 위한 감사표다.

---

## 1. 요약 판단

기존 문서의 강점:

- 보급소가 상점이 아니라 귀환 장소라는 방향은 유지 가능하다.
- R01이 클리어 후 버리는 스테이지가 아니라 지역 생태권이라는 방향은 유지 가능하다.
- 스마일 홈 시어머니가 단순 보스몹이 아니라 결절이라는 방향은 유지 가능하다.
- 윤서가 기본 캐릭터이지만 약하면 안 된다는 방향은 유지 가능하다.
- 캐릭터 애정 기반 수익화 원칙은 유지 가능하다.

가장 큰 충돌:

- 기존 문서는 0.2 최상위 보상을 `광고 전단 조각/캠페인 코어 파편/송출 기록` 중심으로 잡고 있다.
- 새 기준선은 플레이어가 첫 5분에 이해할 최상위 자원을 `밥표/전원표/신호표`로 둔다.
- 기존 문서는 첫 10분/첫 1시간/보스 접근 구조가 강하고, 새 기준선은 첫 5분 재미 검증을 더 앞에 둔다.
- 기존 카드 문서는 도구/밥벌이/약관 카드 구조가 아니다.
- 기존 R01 구현/문서는 오픈하우스 위험 구역의 물리적 긴장이 약하다.

정렬 원칙:

```text
밥표/전원표/신호표 = 플레이어가 보는 최상위 생존 자원
광고 전단 조각/캠페인 코어 파편 = 하위 성장/제작/결절 재료
송출 기록 = 스토리/해석 단서
흔적 = RPG 감정/지역 반응 물건
```

## 3. 문서별 처리 방향

### 3.1 `RPG_GROWTH_ARCHITECTURE_V1.md`

상태:

- 기존 0.2 성장 구조의 핵심 기준서.
- 윤서, 보급소, 첫 회수, 지역 반응, 보스 재도전 구조는 유용하다.
- 그러나 최상위 자원이 `광고 전단 조각/캠페인 코어 파편/송출 기록` 중심이다.

처리:

- 유지하되 새 기준선에 맞춰 v2로 재작성한다.
- `밥표/전원표/신호표`를 최상위 정산 자원으로 추가한다.
- `광고 전단 조각/코어 파편/송출 기록`은 하위 레이어로 내린다.

우선순위: P0

### 3.2 `META_PROGRESSION_ARCHITECTURE.md`

상태:

- 0.2 성장/보급소/지역 반응의 상세 구조가 많다.
- 기존 성장 재화 중심이라 새 기준선과 일부 충돌한다.

처리:

- `RPG_GROWTH_ARCHITECTURE_V2` 작성 시 흡수한다.
- 기존 파일은 참고 자료로 유지한다.

우선순위: P1

### 3.3 `NEXT_TASK_BACKLOG.md`

상태:

- 구현 작업 후보가 있으나 이전 구조 기준이다.

유지 가능:

- Run Result Evaluator
- main.gd 책임 분리
- Damage Router
- 보급소 UI 확장
- 보스 루프 체크리스트

수정 필요:

- Signal Clue Progression
- Meta Upgrade Tree
- Card Catalog
- 0.2 통합 개발 로드맵

처리:

- 새 `PROJECT_DIRECTION_WORKLIST_V1.md` 기준으로 v2 백로그를 만든다.

우선순위: P0

### 3.4 `story/04_content/run_cards.md`

상태:

- 카드 문체와 일부 효과 후보는 재사용 가능하다.
- 하지만 도구/밥벌이/약관 구조와 배급권 정산 연결이 없다.

처리:

- `CARD_AND_RULE_ENGINE_0_2.md` 작성 후 기존 카드 중 일부를 새 12장 구조로 흡수한다.

우선순위: P0

### 3.5 `story/03_regions/r01_suburb.md`

상태:

- R01이 클리어 후 버리는 스테이지가 아니라 로컬 생태권이라는 방향은 강하다.
- 구역 후보는 많지만 첫 5분 위험 구역 동선이 새 기준선만큼 선명하지 않다.

처리:

- `R01_REGION_REBUILD_0_2.md`에서 0.2 구역을 재정렬한다.
- 장기 구역 구조는 유지하되 첫 검증 구역은 줄인다.

우선순위: P0

### 3.6 `story/03_regions/r01_boss.md`

상태:

- 스마일 홈 시어머니의 결절 개념은 유지 가능하다.
- 기존 0.2 결과는 결절 파괴/기억 추출 중심이다.

처리:

- 장기 결과 5종을 반영하되, 0.2 구현은 2~3종으로 제한한다.
- `처치` 표현을 `처리`, `결절 결과`, `약관 파기` 쪽으로 바꾼다.

우선순위: P1

### 3.7 `story/02_hub/*`

상태:

- 보급소와 NPC 구조는 새 기준선과 잘 맞는다.

처리:

- 밥표/전원표/신호표 배분 반응을 추가한다.
- 미나/도윤/팝시/세븐/복희가 배급권 선택에 반응하도록 보강한다.

우선순위: P0

### 3.8 `story/90_implementation/*`

상태:

- 구현 지시 문서가 많고 이전 방향이 섞여 있다.
- 오래된 지시를 그대로 따르면 0.2 범위가 커진다.

처리:

- 새 기준선 이후 작성된 문서와 이전 문서를 분리한다.
- `90_implementation` 하위에 새 `0_2_direction_lock_readme` 또는 index 문서를 만들어 최신 기준 문서 목록을 고정한다.

우선순위: P1

## 6. 결론

기존 문서의 방향은 버릴 것이 아니다.

기존 문서가 잡은 핵심 강점:

- 보급소 RPG
- 지역 생태권
- 스마일 홈 결절
- 흔적과 이름
- 윤서 기본 캐릭터
- 캐릭터 애정 기반 BM

새 기준선이 보강한 것:

- 플레이어가 첫 5분에 이해할 밥표/전원표/신호표
- 카드/정산/위험 구역 중심 재미
- 액션 문턱 압박
- 모바일 F2P 장기 BM 금지선
- 장비 8슬롯/캠페인 카탈로그/노선 기록의 장기 방향

따라서 다음 작업은 기존 문서를 폐기하는 것이 아니라, 새 기준선에 맞게 레이어를 다시 나누는 것이다.
