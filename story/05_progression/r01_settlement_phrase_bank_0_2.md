# R01 Settlement Phrase Bank 0.2

상태: 0.2 문구 기준
목적: `R01 서부 스마일홈`의 출격 게시판, 지역 약관 예고, 런 정산, 보스 결과, 재방문 문구를 `campaign_experience_atlas_0_2.md` 기준으로 정리한다.
연결 문서: `story/05_progression/r01_campaign_experience_audit_0_2.md`, `story/01_bible/visible_terminology_rules_0_2.md`, `story/03_regions/r01_suburb.md`, `R01_REGION_REBUILD_0_2.md`

## 0. 사용 원칙

R01 문구는 세계관 설명이 아니라 행정/광고/정산 말투로 짧게 보여준다.

기준:

- UI 정식 자원명은 `식량태그`, `충전태그`, `수신태그`, `거주태그`, `통행태그`, `보급태그`를 쓴다.
- `밥표`, `전원표`, `신호표`는 NPC 속어, 낡은 표지, 손글씨 메모, 캠페인화 인간 생활어에만 쓴다.
- 출격 게시판과 정산 화면은 유저가 재화 종류를 오해하지 않도록 태그 기준명을 우선한다.
- 보스 결과는 `처치`, `사망`, `죽임`보다 `결절 처리`, `심사 중단`, `기억 추출`, `경로 우회`, `보류`를 쓴다.
- 사람과 로봇을 조롱하지 않는다. 얄미운 것은 광고 시스템, 약관, 고객센터 말투다.

권장 문장 길이:

| 위치 | 길이 |
|---|---|
| 출격 게시판 | 1줄, 20자 안팎 |
| 지역 약관 예고 | 1줄, 조건 + 결과 |
| 런 정산 | 1~2줄, 결과 + 후폭풍 |
| 보스 결과 | 2~3줄, 선택 결과 + 재방문 변화 |
| NPC 반응 | 1줄, 감정은 짧게 |
