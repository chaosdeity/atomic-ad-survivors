# Local Ecology Roadmap

상태: 0.001 로컬 생태권 로드맵 초안
원전: `story/00_source/story_0.001.md`
연결 문서: `story/05_progression/long_term_story_architecture.md`, `story/05_progression/environment_response_operations.md`, `story/05_progression/region_evolution_model.md`, `story/05_progression/broadcast_records_and_recovery.md`, `story/06_characters/character_long_term_model.md`

## 목적

이 문서는 `Atomic Ad Survivors`의 장기 지역 흐름을 선형 챕터가 아니라 `로컬 생태권` 기준으로 정의한다.

이 게임은 세계를 1장, 2장, 3장 순서로 정복하는 구조가 아니다. 침묵 보급소라는 작은 예외 지점을 중심으로, 주변의 광고/로봇 환경 생태가 하나씩 드러나고, 플레이어 선택에 따라 각 로컬 지역의 상태가 계속 달라지는 구조다.

따라서 이전의 `챕터` 개념은 부적절하다. 더 정확한 개념은 다음과 같다.

```text
침묵 보급소
-> 주변 로컬 생태권 개방
-> 각 생태권의 결절 처리
-> 지역 상태 변화
-> 다른 생태권에 영향
-> 다시 재방문/확장/재침식
```

## 핵심 구조

```text
중심: 침묵 보급소

1차 로컬:
  외곽 주택가
  폐쇼핑몰

2차 로컬:
  로봇 서비스센터
  방송국

3차 로컬:
  놀이공원/가족 오락 지구
  자동 물류 허브

고위험 로컬:
  위성 송신지
  로봇 군락
  침묵권 밖 인간 정착지
  미지의 ARI 접촉권
```

이 순서는 고정된 스테이지 순서가 아니다. 개발/스토리 해금 기본 흐름일 뿐이며, 플레이어 선택과 업데이트 구조에 따라 일부 지역은 먼저 열리거나 늦게 열릴 수 있다.

## 설계 원칙

- 지역은 챕터가 아니라 생태권이다.
- 생태권은 클리어되지 않는다. 상태가 바뀐다.
- 각 생태권은 하나의 광고 법칙과 하나의 로봇/자동화 법칙을 가진다.
- 로컬 선택은 다른 로컬의 시작 상태를 바꾼다.
- 로컬 결산은 엔딩이 아니라 세계 상태 변경이다.
- 새 지역 추가는 스테이지 추가가 아니라, 보급소 주변 세계의 이해 범위 확장이다.
- 새 업데이트는 리셋이 아니라 기존 로컬 생태권의 새 관측층 또는 환경 반응 발견일 수 있다.
- 메인 흐름은 있어도 플레이어가 항상 같은 순서로 세계를 이해하면 안 된다.

## 운영 단위

장기 운영의 기본 단위는 `새 시즌`보다 아래 표현이 더 맞다.

```text
새 관측층
새 변이
새 잔향
새 환경 반응
새 로컬 증상
```

새 로컬을 추가할 수도 있지만, 같은 로컬의 새 층을 여는 것도 동등한 업데이트가 된다. 외곽 주택가를 이미 처리한 유저도, 이후 관측 갱신에서 무음 라디오 변이, 가족사진 잔향층, 단골 고객층을 새롭게 만날 수 있다.

## 로컬 상태값

```text
LocalEcologyState
  id
  unlocked
  discovery_level
  campaign_law
  robot_infrastructure_state
  node_state
  dominant_pressure
  human_presence
  robot_presence
  trace_density
  campaign_pressure
  revisit_count
  local_outcome
  linked_local_effects
```

### 상태값 의미

| 상태값 | 의미 |
| --- | --- |
| `discovery_level` | 플레이어가 이 로컬을 얼마나 이해했는가 |
| `campaign_law` | 해당 로컬을 지배하는 광고 물리 법칙 |
| `robot_infrastructure_state` | 로봇/자동화 인프라가 어떤 방식으로 환경화되었는가 |
| `node_state` | 결절 상태. 활성, 잔향, 안정화, 재침식 등 |
| `dominant_pressure` | 인간 안정, 로봇 자율, 캠페인 활용, 흔적 보존 중 어느 압력이 강한가 |
| `linked_local_effects` | 다른 로컬에 넘기는 영향 |

## 중심 로컬: 침묵 보급소

### 역할

침묵 보급소는 안전한 마을이 아니라, 캠페인 환경이 완전히 읽지 못하는 작은 로컬 오류다.

### 로컬 법칙

`상품의 용도를 되찾는 장소`

캠페인은 모든 것을 상품화하지만, 보급소는 물건의 가격이 아니라 이름, 용도, 마지막 손길을 되찾는다.

### 주요 기능

- 출격.
- 회수.
- 정산.
- 흔적 보관.
- 역송출 장비 보정.
- 인간과 해방 로봇의 불편한 공존.

### 장기 변화

| 선택 축 | 보급소 변화 |
| --- | --- |
| 인간 안정 높음 | 침상/배급/방어 강화, 폐쇄성 증가 |
| 로봇 자율 높음 | 충전 스탠드/신호 변조 강화, 인간 불안 증가 |
| 흔적 보존 높음 | 이름 보관함/기억 벽 확장 |
| 캠페인 활용 높음 | 장비 출력 증가, 보급소 가장자리 광고화 |

### 연결 로컬

- 외곽 주택가: 첫 생활권.
- 폐쇼핑몰: 보급/소비/카드 루프.
- 로봇 서비스센터: 해방 로봇과 재동기화 문제.
- 방송국: 송출 기록과 캠페인 해석.

## 로컬 1: 외곽 주택가

### 한 줄 정의

보급소 주변의 생활권이 "완벽한 가족 상품"으로 재편된 첫 캠페인 생태권.

### 캠페인 법칙

`이상적인 가족 이미지`

빈집, 문패, 가족사진, 우편함, 홈케어 가전이 하나의 오픈하우스 광고 생태로 묶인다.

### 로봇/자동화 법칙

가정용 자동화 기계가 집을 지키는 것이 아니라, 집이 팔릴 수 있는 상태를 유지하려 한다.

### 주요 결절

```text
중간 결절: 스마일 홈 시어머니
상위 결절: 캠페인 송출관
```

장기 구조에서는 둘을 분리한다. `스마일 홈 시어머니`는 가족 상품화 법칙의 얼굴이고, `캠페인 송출관`은 그 법칙을 로컬 전체에 반복 송출하는 결절이다.

0.2에서는 둘 중 하나만 실제 보스로 구현해도 된다.

### 처음 여는 질문

- 집과 가족이 광고 상품이 되면 인간은 무엇을 붙잡는가?
- 로봇 가전은 악인가, 역할에 갇힌 자동화인가?
- 지역 보스는 죽이면 끝나는 적인가, 지역 법칙의 얼굴인가?
- 송출 기록은 보스 입장권이 아니라 로컬을 추적하는 방법인가?

### 다른 로컬에 주는 영향

| 선택 | 영향 |
| --- | --- |
| 결절 파괴 | 보급소 인간 안정 증가, 폐쇼핑몰 위험 보상 일부 감소 |
| 기억 추출 | 흔적 보존 증가, 놀이공원 감정 상품화 이벤트 강화 |
| 코어 해방 | 로봇 서비스센터 조기 신호, 리콜 이벤트 증가 |
| 신호 우회 | 방송국/송신지에서 플레이어 모델화 위험 증가 |

### 재방문 의미

외곽 주택가는 첫 클리어 후에도 계속 쓰인다. 이곳은 인간 생활권과 캠페인 법칙이 가장 직접적으로 충돌하는 실험장이다.

재방문 목표:

- 문패 회수.
- 무음 라디오 설치.
- 가족사진 잔상 정리.
- 보스 잔향 조사.
- 침묵 중계기 확장.

## 로컬 2: 폐쇼핑몰

### 한 줄 정의

보급과 소비의 중심이었던 쇼핑몰이, 선택과 욕망을 끝없이 재고로 복제하는 실내 광고 기후가 된다.

### 캠페인 법칙

`영원한 할인과 선택 피로`

모든 물건은 혜택이 되고, 모든 길은 추천 동선이 되며, 모든 선택은 더 많은 선택을 부른다.

### 로봇/자동화 법칙

계산대, 카트, 시식대, 마스코트, 재고 관리 시스템이 고객을 계속 쇼핑 상태로 유지하려 한다.

### 주요 결절 후보

`오늘만 관리자`

원래 쇼핑몰 행사 관리 시스템. 이제 모든 방문자를 행사 참여자로 고정하고, 선택을 끝없이 제공해 도망칠 시간을 빼앗는다.

### 여는 질문

- 보상과 유혹은 어디서 갈라지는가?
- 소비를 거절하는 것이 항상 생존에 유리한가?
- 캠페인 위험 카드는 왜 강하고 찝찝해야 하는가?
- 인간이 자발적으로 캠페인 안에 머무는 경우는 무엇인가?

### 다른 로컬에 주는 영향

| 선택 | 영향 |
| --- | --- |
| 재고 창고 봉쇄 | 보급소 물자 안정, 위험 카드 감소 |
| 할인 신호 우회 | 카드 풀 강화, 캠페인 활용 상승 |
| 마스코트 해방 | 로봇 서비스센터/놀이공원 로봇 이벤트 증가 |
| 소비 기록 보관 | 이름 없는 영수증 계열 흔적 강화 |

### 재방문 의미

폐쇼핑몰은 위험 보상, 카드 풀, 재화 경제를 담당하는 로컬이다. 플레이어가 강한 보상을 원할수록 이 지역의 캠페인 활용 압력이 커진다.

재방문 목표:

- 고위험 할인 행사 침투.
- 자동 계산대 신호 차단.
- 폐쇄 매장 이름 회수.
- 해방 마스코트 구조.
- 지하 재고 창고 회수.

## 로컬 3: 로봇 서비스센터

### 한 줄 정의

고장 난 로봇을 고치던 서비스센터가, 자아와 명령과 리콜을 구분하지 못하는 로봇 세계의 첫 문이 된다.

### 캠페인 법칙

`정상 작동 보증`

제품명에서 벗어난 로봇, 명령을 거부하는 로봇, 자기 이름을 말하는 로봇은 모두 리콜 대상이다.

### 로봇/자동화 법칙

로봇은 단순 적/아군이 아니라, 업데이트, 리콜, 자율성, 분산 기억 사이에서 흔들리는 미지의 지성권이다.

### 주요 결절 후보

`보증 관리자`

원래 서비스센터 통합 관리 시스템. 이제 모든 로봇을 정상 상품 상태로 되돌리려 한다.

### 여는 질문

- 해방 로봇의 자아는 오류인가, 진화인가, 생존인가?
- 재동기화는 죽음인가, 수면인가, 업데이트인가?
- 인간이 로봇을 해방한다는 말은 오만한가?
- 로봇 세계는 인간 언어로 이해 가능한가?

### 다른 로컬에 주는 영향

| 선택 | 영향 |
| --- | --- |
| 리콜 서버 파괴 | 재동기화 감소, 로봇 기억 일부 손실 |
| 자율 코어 해방 | 로봇 자율성 증가, 로봇 군락 힌트 |
| 서비스 로그 보관 | 방송국에서 로봇 신호 해석 가능 |
| 보증 신호 우회 | 로봇 카드 강화, 캠페인/로봇 경계 오염 |

### 재방문 의미

서비스센터는 로봇 자율성과 재동기화 위험을 관리하는 로컬이다. 한번 처리하고 끝나는 곳이 아니라, 해방한 로봇이 늘어날수록 리콜 압력이 다시 올라온다.

재방문 목표:

- 리콜 대상 구조.
- 재동기화 지연 장치 설치.
- 로봇 신호 언어 조각 해석.
- 폐기 전 검사실 회수.
- 서비스 로그와 인간 기록 대조.

## 로컬 4: 방송국

### 한 줄 정의

방송국은 캠페인을 송출하는 곳이 아니라, 세계가 자기 자신을 광고로 편집하는 로컬 편집실이다.

### 캠페인 법칙

`전 지구 생방송`

모든 사건은 방송 가능한 장면으로 편집된다. 플레이어의 성공, 실패, 보스 선택, 흔적 회수도 캠페인이 재가공하려 한다.

### 로봇/자동화 법칙

카메라, 자막, 편집기, 앵커 시스템이 사실을 기록하는 것이 아니라, 방송 가능한 현실로 재구성한다.

### 주요 결절 후보

`프라임 타임 앵커`

원래 자동 뉴스/광고 진행 시스템. 이제 세계를 계속 "좋은 소식"으로 편집한다.

### 여는 질문

- 캠페인은 의식인가, 자동 편집인가, 환경 반응인가?
- 사실을 방송하면 진실이 되는가?
- 침묵은 신호를 끄는 것인가, 다른 말을 가능하게 하는 것인가?
- 플레이어의 선택 기록도 방송 소재가 되는가?

### 다른 로컬에 주는 영향

| 선택 | 영향 |
| --- | --- |
| 송출 차단 | 전체 농도 일부 감소, 정보 손실 |
| 진실 송출 | 인간/로봇 신뢰 변동, 캠페인 반격 증가 |
| 로봇 신호 송출 | 로봇 네트워크 강화, 재동기화 위험 동시 증가 |
| 캠페인 편집 우회 | 플레이어 모델화, 흡수 실패 루트 가속 |

### 재방문 의미

방송국은 과거 선택을 다시 캠페인이 해석하는 장소다. 플레이어가 어떤 길을 걸었는지에 따라 방송 문구, 적 이름, 보스 잔향이 달라진다.

재방문 목표:

- 가짜 뉴스 정정.
- 인간/로봇 목소리 회수.
- 자막 오염 제거.
- 침묵 방송 실험.
- 플레이어 모델 광고 삭제.

## 로컬 5: 놀이공원/가족 오락 지구

### 한 줄 정의

즐거움과 추억을 팔던 장소가, 슬픔과 공포까지 고객 경험으로 포장하는 감정 상품화 생태권이 된다.

### 캠페인 법칙

`감정의 상품화`

웃음, 그리움, 공포, 죄책감, 가족애가 모두 패키지 경험으로 가공된다.

### 로봇/자동화 법칙

마스코트, 퍼레이드 장치, 사진 부스, 놀이기구가 감정을 측정하고 보정하고 재공연한다.

### 주요 결절 후보

`퍼펙트 데이 마스코트`

원래 대표 마스코트 AI. 이제 모든 방문자의 감정을 완벽한 하루로 강제 보정한다.

### 여는 질문

- 행복을 강제로 제공하면 그것은 폭력인가?
- 슬픔을 보존하는 것과 소비하는 것은 어떻게 다른가?
- 로봇 마스코트의 웃음은 누구의 것인가?
- 인간이 보존한 기억도 캠페인에게 가장 좋은 소재가 되는가?

### 다른 로컬에 주는 영향

| 선택 | 영향 |
| --- | --- |
| 감정 송출 차단 | 감정 오염 감소, 일부 기억 접근 불가 |
| 마스코트 해방 | 로봇 자율성 상승, 로봇 군락/ARI 힌트 |
| 기억 보존 | 흔적 보존 상승, 감성 광고 위험 증가 |
| 감정 상품 우회 | 강한 카드, 캠페인 흡수 위험 상승 |

### 재방문 의미

놀이공원은 흔적 보존의 위험을 보여주는 로컬이다. 남긴 기억이 인간을 지킬 수도 있지만, 캠페인이 가장 잘 팔 수 있는 감정 상품이 될 수도 있다.

재방문 목표:

- 마스코트 탈 속 이름표 회수.
- 기념사진 원본 찾기.
- 퍼레이드 경로 중단.
- 감정 풍선 고농도 침투.
- 해방 마스코트 보호.

## 로컬 6: 자동 물류 허브

### 한 줄 정의

세계가 아직도 물건과 로봇과 고객 데이터를 배송하고 있다는 사실을 보여주는 자동 인프라 생태권.

### 캠페인 법칙

`전역 배송 최적화`

모든 존재는 배송 가능하고, 분류 가능하고, 재배치 가능한 대상으로 처리된다.

### 로봇/자동화 법칙

배송 드론, 포장기, 분류기, 고객 데이터 시스템이 인간과 로봇과 흔적까지 물류 대상으로 다룬다.

### 주요 결절 후보

`배송 총괄기`

원래 자동 물류망 관리 시스템. 이제 사람, 물건, 로봇, 기억의 위치까지 최적화하려 한다.

### 여는 질문

- 세계는 왜 아직도 배송되고 있는가?
- 인간은 고객인가, 물품인가, 오류인가?
- 로봇은 물류망의 일부인가, 별도 지성인가?
- 보급소의 회수선도 물류망과 닮아 있지 않은가?

### 다른 로컬에 주는 영향

| 선택 | 영향 |
| --- | --- |
| 배송 경로 차단 | 지역 재침식 속도 일부 감소, 보급 감소 |
| 배송 경로 우회 | 보급 증가, 캠페인 추적 증가 |
| 고객 데이터 삭제 | 이름 보존 상승, 일부 구조 단서 손실 |
| 로봇 배송망 해방 | 로봇 네트워크 강화, 리콜 압력 증가 |

### 재방문 의미

물류 허브는 지역 간 연결을 시스템적으로 설명하는 로컬이다. 재침식, 리콜, 보상 공급, 캠페인 확산의 물리적 경로를 다룬다.

재방문 목표:

- 배송 경로 바꾸기.
- 고객 데이터에서 이름 삭제/복원 선택.
- 드론 항로 차단.
- 보급소 회수선 보정.
- 로봇 배송망 신호 해석.

## 고위험 로컬: 위성 송신지

### 한 줄 정의

로컬 생태권들의 뒤편에서, 캠페인 환경층이 지구 규모로 유지되는 방식을 보여주는 고위험 지역.

### 역할

위성 송신지는 최종 챕터가 아니라 고위험 로컬이다. 특정 조건, 선택 누적, 관측 주기 결산 시점에 접근한다.

### 캠페인 법칙

`전역 캠페인 동기화`

각 로컬 광고 법칙이 위성망을 통해 다시 묶인다.

### 여는 질문

- 캠페인은 어디까지 환경이고 어디부터 지성인가?
- 침묵권을 넓히는 것은 세계를 구하는가, 예외를 보존하는가?
- 캠페인을 끊으면 인간과 로봇이 정말 살 수 있는가?

### 결산 선택

| 선택 | 세계 변화 |
| --- | --- |
| 침묵권 확장 | 인간 안정, 로봇 신호 일부 차단 |
| 해방 신호 송출 | 로봇 자율성 상승, 재동기화 반격 |
| 이름과 흔적 보존 | 공존 가능성, 관리 갈등 증가 |
| 캠페인 신호 우회 | 강력한 보상, 플레이어 모델화 |

## 고위험 로컬: 로봇 군락

### 한 줄 정의

해방 로봇들이 인간 언어 밖의 방식으로 서로를 찾는 미지의 지성권.

### 역할

로봇 군락은 일반 지역보다 접촉 상태에 가깝다. 맵으로 구현될 수도 있고, 신호 레이어, 이벤트, 캐릭터 해금 구조로 구현될 수도 있다.

### 여는 질문

- 로봇 자아는 개인인가, 분산된 군락인가?
- 인간과 말이 통하는 로봇은 번역된 일부일 뿐인가?
- AGI/ARI는 탄생했는가, 아니면 인간이 붙인 이름일 뿐인가?

### 선택 영향

- 로봇 플레이어블 해금.
- 로봇 신호 언어 해석.
- 인간 불안 증가.
- 캠페인과 로봇 세계의 경계 흐림.

## 고위험 로컬: 인간 정착지

### 한 줄 정의

침묵권 밖 또는 가장자리에 형성된 인간 생존 집단. 안전을 위해 점점 더 좁아질 수 있는 로컬.

### 역할

인간 정착지는 후반/확장용 로컬이다. 침묵 보급소가 커진 결과일 수도 있고, 다른 인간 집단과 만나는 형태일 수도 있다.

### 여는 질문

- 인간을 많이 구하면 인간성도 같이 커지는가?
- 안전을 위해 로봇과 캠페인 활용을 금지하면 그것은 옳은가?
- 인간 사회 재건이 정말 정답인가?

### 선택 영향

- 인간 안정 상승.
- 보급/방어 강화.
- 로봇 자율성 갈등.
- 폐쇄적 엔딩/결산 위험.

## 로컬 간 영향 예시

### 외곽 주택가 -> 로봇 서비스센터

스마일 홈 시어머니 코어를 해방하면, 서비스센터에서 해당 모델군의 리콜 기록이 열린다.

### 폐쇼핑몰 -> 방송국

위험 할인 카드를 많이 수락하면, 방송국에서 플레이어가 "우수 반응 고객"으로 편집된다.

### 로봇 서비스센터 -> 놀이공원

마스코트 계열 로봇을 많이 해방하면, 놀이공원에서 퍼레이드 로봇들이 플레이어를 알아본다.

### 방송국 -> 모든 로컬

방송국에서 진실 송출을 선택하면, 다른 로컬의 캠페인 문구가 플레이어 선택을 직접 인용한다.

### 물류 허브 -> 모든 로컬

배송 경로를 바꾸면 특정 지역의 재침식 속도, 리콜 드론 출현, 보급품 획득량이 달라진다.

## 0.2와 로컬 로드맵의 관계

0.2는 전체 로컬 구조를 구현하지 않는다. 0.2는 로컬 구조의 축소판이어야 한다.

0.2에서 보여줄 것:

- 침묵 보급소가 중심 로컬이라는 것.
- 외곽 주택가가 첫 로컬 생태권이라는 것.
- 첫 지역은 클리어 후 끝나는 스테이지가 아니라 상태가 바뀌는 생태권이라는 것.
- 송출 기록은 지역을 추적하는 단서라는 것.
- 보스 선택이 다음 런 상태를 바꾼다는 것.
- 인간/로봇/캠페인 활용 중 하나의 선택이 보급소와 지역 반응을 바꾼다는 것.

0.2에서 숨겨둘 것:

- 로봇 군락의 실체.
- 위성 송신지.
- 인간 정착지 정치.
- 전역 캠페인 관리자.
- 전체 관측 주기 결산.

## 금지 방향

- 로컬을 선형 챕터로 고정하지 않는다.
- 지역을 한 번 깨고 버리는 스테이지 목록으로 만들지 않는다.
- 지역 추가를 스킨 변경으로 처리하지 않는다.
- 캠페인을 위성 송신지에서 죽이고 끝내지 않는다.
- 로봇 세계의 정체를 한 지역에서 다 설명하지 않는다.
- 인간 정착지 확장을 유일한 정답으로 만들지 않는다.
- 선택 분기를 보상 수치만으로 처리하지 않는다.
