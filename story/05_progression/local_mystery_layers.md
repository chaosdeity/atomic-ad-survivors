# Local Mystery Layers

상태: 0.001 확장 기준
연결 문서: `long_term_story_architecture.md`, `environment_response_operations.md`, `local_ecology_roadmap.md`, `region_evolution_model.md`, `story_trigger_schema.md`

## 목적

로컬은 클리어되는 스테이지가 아니라, 플레이어의 방문과 선택과 습관을 먹고 변하는 캠페인 생태권이다.

따라서 한 로컬은 작은 전투장 하나가 아니다. 런 하나는 짧아도, 로컬 자체는 여러 번 들어가도 새 단서와 변형이 나올 수 있는 넓은 구조여야 한다. 방 하나가 작을 수는 있지만, 지역 전체의 의미와 연결망은 작으면 안 된다.

## 핵심 원칙

- 로컬은 챕터가 아니라 생태권이다.
- 발견 순서는 고정하지 않는다.
- 유저가 모든 단서를 반드시 발견할 필요는 없다.
- 발견하지 못한 단서는 사라지는 것이 아니라, 다른 문맥에서 늦게 발견될 수 있다.
- 같은 행동을 반복하면 캠페인과 NPC가 그 행동을 기억한다.
- 다른 캐릭터로 같은 로컬에 들어가면 다른 층의 의미가 열린다.
- 보스 처리는 로컬의 종료가 아니라 다음 상태의 시작이다.

## 로컬 규모 기준

로컬맵은 모바일 로그라이크 전투에 맞게 한 런은 간결해야 한다. 하지만 서사 단위로서의 로컬은 최소 4~6개의 하위 구역을 가져야 장기 반복을 감당한다.

### 권장 하위 구역

| 구역 | 역할 | 예시 |
| --- | --- | --- |
| 안전 가장자리 | 보급소와 연결된 낮은 농도 입구 | 침묵 표지판, 꺼진 간판, 버려진 배달함 |
| 생활 표면 | 처음 보이는 광고화된 일상 | 주택가, 진열대, 놀이시설, 상담 부스 |
| 서비스 뒷면 | 로봇/자동화 인프라의 흔적 | 정비통로, 배관, 충전 독, 폐기 컨베이어 |
| 고농도 결절 | 보스/엘리트/송출 기록의 중심 | 모델하우스, 중앙 무대, 중계탑, 고객센터 |
| 숨은 침묵 주머니 | 반복 방문/특정 캐릭터/흔적으로 열리는 틈 | 지하 창고, 방음실, 꺼진 냉동실 |
| 변질된 귀환로 | 보스 선택 이후 바뀌는 재방문 루트 | 무너진 출구, 새 광고문, 가짜 안전지대 |

0.2에서는 모든 구역을 별도 대형 맵으로 만들 필요는 없다. 전투 맵 타일, 배경 오브젝트, 목표 위치, 엘리트 출현 조건, 출격 게시판 문구로 먼저 분리해도 충분하다.

## 미스터리 층

한 로컬은 여러 개의 미스터리 층을 가진다. 얕은 층은 첫 방문에도 보이고, 깊은 층은 반복 행동과 조합 조건으로 열린다.

장기 운영에서는 이 미스터리 층이 그대로 `관측층`이 된다. 업데이트는 새 맵을 덧붙이는 것만이 아니라, 기존 로컬에서 보급소가 새로 읽어낸 층을 여는 방식으로 처리한다.

```text
LocalMysteryLayer
  id
  local_id
  layer_depth
  layer_type
  visibility
  required_triggers
  related_characters
  related_npcs
  related_traces
  related_boss_outcomes
  reveal_text
  gameplay_effect
  can_be_missed
  late_discovery_rule
```

### 층 종류

| 층 | 기능 | 전달 방식 |
| --- | --- | --- |
| 표면 광고층 | 이 로컬이 무엇을 팔고 있는지 보여줌 | 배너, 적 이름, 카드 문구 |
| 생활 흔적층 | 인간이 여기서 어떻게 살았는지 보여줌 | 흔적 아이템, 짧은 회수 대사 |
| 자동화 뒷면층 | 로봇과 설비가 어떤 명령을 받았는지 보여줌 | 로봇 NPC 반응, 정비 로그 |
| 캠페인 법칙층 | 이 로컬의 물리 법칙을 드러냄 | 반복 런 변형, 엘리트 패턴 |
| 악의 개입층 | 세계 자체가 아닌 누군가의 의도적 착취를 드러냄 | 이상한 계약서, 조작된 기록, 선택지 |
| 미지 접촉층 | AGI/ARI 또는 정체불명 신호의 흔적 | 캐릭터 전용 대사, 깨진 송출 기록 |

### 관측층 전환

| 기존 미스터리 층 | 업데이트 관측층 예시 |
| --- | --- |
| 생활 흔적층 | 새 문패/사진/영수증 계열 흔적 추가 |
| 자동화 뒷면층 | 같은 모델의 로봇 로그 또는 리콜 변이 추가 |
| 캠페인 법칙층 | 보스 선택 후 달라진 환경 패턴 추가 |
| 악의 개입층 | 조작된 구조 신호의 확신도 상승 |
| 미지 접촉층 | 로봇 군락/ARI/위성 잔향의 짧은 응답 |

## 발견 순서

발견 순서는 선형 퀘스트처럼 고정하지 않는다. 같은 진실도 여러 입구를 가져야 한다.

예를 들어 외곽 주택가의 “가족 상품화”는 다음 방식으로 각각 들어갈 수 있다.

| 입구 | 발견 방식 | 유저 감각 |
| --- | --- | --- |
| 인간 흔적 | 불탄 가족사진 회수 | 이곳에 사람이 있었다 |
| 로봇 흔적 | 가정용 도우미 코어 회수 | 로봇도 가족 역할을 강요받았다 |
| 보스 선택 | 기억 추출 선택 | 보스가 단순 괴물이 아니었다 |
| 반복 행동 | 같은 집 구조를 3회 이상 조사 | 캠페인이 집을 복제하고 있다 |
| 캐릭터 관점 | 윤서가 모델하우스에 재방문 | 윤서는 이 광고가 자신을 고객으로 완성하려는 것을 느낌 |

## 반복 행동 기억

유저가 같은 행동을 계속하면 세계가 알아차려야 한다. 이것은 벌칙이 아니라 “세계가 플레이어를 읽고 있다”는 재미다.

### 반복 행동 예시

| 행동 | 누적 상태 | 열리는 반응 |
| --- | --- | --- |
| 같은 로컬 연속 출격 | `consecutive_local_runs` | 지역이 단골 고객처럼 플레이어를 부름 |
| 같은 보스 선택 반복 | `same_boss_choice_streak` | NPC가 플레이어의 윤리를 의심하거나 신뢰함 |
| 흔적 보관만 반복 | `trace_preserve_streak` | 기억 보관소가 커지고, 전투 보상은 늦어짐 |
| 흔적 소모만 반복 | `trace_consume_streak` | 장비는 강해지지만 일부 NPC 대사가 차가워짐 |
| 로봇 해방 반복 | `robot_liberation_count` | 해방 로봇 루머와 전용 접촉 발생 |
| 위험 카드 반복 선택 | `campaign_risk_accept_count` | 캠페인이 플레이어를 협력 고객으로 오해함 |
| 송출 이벤트 무시 반복 | `ignored_signal_events` | 나중에 더 강한 방식으로 신호가 찾아옴 |
| 특정 캐릭터로 특정 로컬 반복 | `character_local_pair_count` | 캐릭터 전용 기억층 또는 악몽층 개방 |

## 캐릭터별 로컬 재해석

같은 로컬도 캐릭터에 따라 다른 게임처럼 보여야 한다.

| 캐릭터 유형 | 같은 로컬에서 보는 것 |
| --- | --- |
| 인간 예외자 | 가족, 몸, 기억, 식량, 피난 흔적 |
| 해방 로봇 | 명령 잔여물, 폐기 로그, 이름 없는 기계의 공포 |
| 흔적 보관자 | 광고로 덮이기 전의 실제 이름 |
| 고위험 역송출자 | 캠페인이 아직 쓰지 못한 출력 경로 |
| 미지 접촉자 | 사람이 이해하지 못한 로봇/위성/ARI의 응답 |

이 구조가 있어야 새 캐릭터 추가가 단순 스킨이 되지 않는다. 새 캐릭터는 새 무기만이 아니라, 기존 로컬의 새 층을 여는 열쇠가 된다.

## 로컬 간 연결

로컬은 독립 콘텐츠지만 완전히 분리되지 않는다. 특정 로컬의 행동이 다른 로컬의 미스터리 층을 연다.

| 시작 로컬 | 조건 | 영향 로컬 | 결과 |
| --- | --- | --- | --- |
| 외곽 주택가 | 가정용 도우미 코어 보관 | 로봇 서비스 센터 | 같은 모델의 폐기 사유 발견 |
| 폐쇼핑몰 | 판촉 물질을 과다 사용 | 외곽 주택가 | 주택가 광고 문구가 쇼핑몰식으로 변함 |
| 방송국 | 송출 기록 3개 해석 | 위성 송신지 | 신호가 지상이 아니라 위에서 내려온 흔적 |
| 물류 허브 | 배송 로봇 해방 반복 | 침묵 보급소 | 로봇 NPC 유입 또는 충전 시설 변화 |

## 0.2 외곽 주택가 적용

0.2에서는 외곽 주택가를 아래 정도로 나누면 충분하다.

| 구역 | 기능 | 구현 방식 |
| --- | --- | --- |
| 침묵 가장자리 | 첫 출격 입구 | 낮은 적 밀도, 보급소 통신 문구 |
| 분양 주택 루프 | 기본 전투 공간 | 반복되는 집 외형, 가족 광고 배너 |
| 모델하우스 결절 | 보스/송출 기록 중심 | 엘리트, 보스, 신호 이벤트 위치 |
| 배수로 침묵 주머니 | 숨은 단서 | 특정 흔적/반복 방문 후 배경 오브젝트 변경 |
| 가짜 귀환로 | 보스 후 재방문 변형 | 회수 지점 문구, 적 이름, 길 안내판 변화 |

## 금지 방향

- 로컬을 한 번 클리어하면 끝나는 스테이지로 만들지 않는다.
- 반복 방문을 단순 레벨 스케일링으로만 처리하지 않는다.
- 모든 단서를 한 캐릭터와 한 루트에서 다 보여주지 않는다.
- 미스터리 발견을 출격 횟수만으로 열지 않는다. 행동, 선택, 흔적, NPC, 캐릭터 관점 중 하나와 연결한다.
- 작게 만들기 위해 로컬의 의미까지 줄이지 않는다. 0.2 구현은 작아도, 설계상 생태권은 넓어야 한다.
