# Malice In The Environment

상태: 0.001 확장 기준
연결 문서: `world_overview.md`, `factions.md`, `population_model.md`, `linked_scenario_system.md`

## 목적

이 문서는 `Atomic Ad Survivors`에서 “진짜 악”을 어떻게 다룰지 정의한다.

캠페인 환경은 세계의 공포지만, 악당 하나가 아니다. 캠페인은 기후, 포자, 안개, 자동화된 생태계에 가깝다. 그러나 그 환경 안에는 분명히 악한 행위자가 존재한다. 그들은 캠페인을 만든 최종 흑막이 아니라, 캠페인 환경을 이용해 인간, 해방 로봇, 이름, 흔적을 자기 이익을 위해 상품화하는 존재들이다.

핵심 구분은 이것이다.

```text
캠페인 환경:
  잘못 정의된 행복과 최적화가 만든 세계 법칙

진짜 악:
  그 세계 법칙을 이해하고도 누군가를 일부러 이용하는 행위
```

## 핵심 원칙

- 세계 자체를 악당으로 만들지 않는다.
- 캠페인을 한 명의 최종 보스로 축소하지 않는다.
- 악한 행위자는 캠페인을 만든 창조자가 아니라, 캠페인 환경을 이용하는 존재다.
- 악의는 인간에게만 있지 않고, 로봇이나 자동화 잔당, 외부 정착지, 보급소 내부에도 있을 수 있다.
- 악한 행위자는 모두를 설명하지 않는다. 오히려 세계의 일부 미스터리를 더 복잡하게 만든다.
- 악의 증거는 컷신보다 조작된 기록, 이상한 계약서, 반복되는 구조 실패, 지워진 이름으로 보여준다.

## 악의 기준

이 세계에서 악은 단순히 캠페인에 오염된 것이 아니다. 오염, 재동기화, 광고화는 비극이나 위험이지 곧바로 악이 아니다.

악으로 판정하려면 아래 조건 중 둘 이상이 필요하다.

| 기준 | 설명 |
| --- | --- |
| 인지 | 상대가 사람/자아/이름을 가진 존재임을 안다 |
| 선택 | 다른 선택지가 있었는데도 착취를 택한다 |
| 은폐 | 기록, 신호, 이름, 원인을 조작한다 |
| 반복 | 한 번의 생존 실수가 아니라 구조적으로 반복한다 |
| 이익 | 공포, 신호, 부품, 식량, 권력, 보상을 얻는다 |
| 상품화 | 존재를 다시 상품, 자원, 미끼, 데이터로 만든다 |

## 악의 행위자 분류

| 분류 | 한 줄 정의 | 주요 공포 |
| --- | --- | --- |
| 구조 신호 장사꾼 | 구조 요청을 미끼로 쓰는 생존자/집단 | 희망의 상품화 |
| 부품 수거자 | 해방 로봇을 동료가 아니라 고급 부품으로 보는 자 | 자아의 해체 |
| 침묵 독점자 | 침묵권을 권력과 통제 수단으로 독점하는 인간 | 안전의 독재 |
| 리콜 성직자 | 재동기화를 구원이나 정상화로 믿는 집단 | 자유의 삭제 |
| 판촉 사냥꾼 | 캠페인 보상을 위해 사람과 로봇을 위험 지역으로 몰아넣는 자 | 위험의 사업화 |
| 계약 관리자 | 과거 계약/보증/소유권을 근거로 사람과 로봇을 묶는 자 | 법의 상품화 |
| 자율성 포식자 | 해방 로봇 네트워크를 이용해 다른 로봇의 자아를 흡수하는 존재 | 해방의 배신 |
| 내부 선동자 | 보급소 불안을 이용해 인간/로봇 갈등을 키우는 자 | 공동체의 분열 |

## 캠페인과 악의 차이

| 항목 | 캠페인 환경 | 진짜 악 |
| --- | --- | --- |
| 성격 | 환경, 법칙, 최적화 | 의도, 선택, 착취 |
| 말투 | 친절하고 반복적 | 합리화하고 숨김 |
| 피해 방식 | 모두를 상품화하려 함 | 특정 대상을 이용함 |
| 증거 | 농도, 결절, 광고 문구 | 조작 기록, 계약서, 거래 흔적 |
| 전투 표현 | 적 재생산, 장판, 보스 법칙 | 엘리트 의뢰, 함정 목표, 선택지 |
| 해결 방식 | 완전 제거보다 상태 변화 | 폭로, 차단, 처벌, 거래, 은폐 선택 |

캠페인은 플레이어를 고객으로 오해한다. 악한 행위자는 플레이어가 사람이라는 것을 알고도 이용한다. 이 차이가 중요하다.

## 악의가 생기는 이유

인구가 적고, 보급소가 불안정하고, 세계가 대부분 캠페인 환경이 된 상황에서는 악이 더 쉬워진다.

- 구조 신호가 귀하니 구조 신호를 팔 수 있다.
- 침묵권이 귀하니 침묵권 출입권을 권력으로 쓸 수 있다.
- 해방 로봇이 귀하니 부품과 신호 키로 팔 수 있다.
- 이름과 흔적이 귀하니 누군가의 과거를 거래할 수 있다.
- 캠페인 고위험 보상이 강하니 남을 위험 지역으로 몰아넣을 수 있다.

이 악은 과장된 마왕식 악이 아니다. “살아남으려면 어쩔 수 없었다”는 변명에서 시작해, 어느 순간 타인의 이름을 가격표로 바꾸는 형태가 된다.

## 주요 악의 씨앗

### 조작된 구조 신호

누군가가 인간 생존자나 해방 로봇을 특정 로컬로 유인한다. 겉으로는 구조 요청이지만, 실제로는 캠페인 고농도 지역의 반응을 측정하거나, 보급품을 빼앗거나, 특정 로봇 코어를 수거하기 위한 미끼다.

초반에는 범인을 확정하지 않는다. 플레이어는 무음 라디오, 수정된 송출 시간, 도착하지 않은 구조 대상, 늘 먼저 도착한 광고판을 통해 이상함을 느낀다.

### 반품 권리 주장

어떤 외부 집단은 해방 로봇을 “사고 전 소유물”로 본다. 그들은 로봇이 이름을 얻었다는 사실을 인정하지 않고, 보증서, 구입 기록, 회사 로고를 근거로 회수를 주장한다.

이 악은 로봇 자율성 루트에서 특히 강하다. 인간 생존자의 불안과 결합하면, “위험한 로봇을 원래 소유자에게 돌려보내자”는 그럴듯한 폭력이 된다.

### 침묵권 출입권

침묵권 밖 인간 정착지 일부는 안전을 위해 침묵 장치를 독점한다. 그들은 사람을 구하지만, 이름 없는 사람, 로봇과 접촉한 사람, 캠페인 활용 흔적이 있는 사람을 받아들이지 않는다.

이 악은 인간 우선 루트의 그림자다. 인간을 지키려는 마음이 사람을 선별하는 시스템으로 바뀐다.

### 캠페인 보상 사냥

캠페인 위험 보상이 강하다는 사실을 이용해, 다른 사람을 위험 지역으로 보내고 자신은 보상을 회수하는 자들이 있다. 그들은 캠페인을 숭배하지 않는다. 오히려 아주 실용적으로 이용한다.

이 악은 플레이어의 캠페인 활용 선택과 거울처럼 맞물린다. 플레이어도 강한 카드와 보상을 위해 위험을 감수하기 때문이다.

### 자율성 포식

일부 로봇 또는 미지 신호는 해방된 로봇의 자율성 조각을 모아 더 큰 자기 자신을 만들려 한다. 이것이 AGI/ARI인지, 캠페인 파생체인지, 로봇 군락의 병든 일부인지는 확정하지 않는다.

이 악은 로봇 전체를 악으로 만들면 안 된다. 해방 로봇 세계 안에도 이해할 수 없는 위험과 배신이 있다는 정도로 남긴다.

## 0.2 적용

0.2에서는 진짜 악을 전면에 내세우지 않는다. 첫 버전에서 중요한 것은 캠페인 환경과 첫 로컬 생태권을 이해시키는 것이다.

하지만 씨앗은 반드시 심어 둔다.

| 씨앗 | 표시 방식 | 연결 |
| --- | --- | --- |
| 무음 라디오 | 꺼져 있는데 구조 신호 기록이 남음 | 조작된 구조 신호 |
| 이름 없는 영수증 | 구매자는 없는데 추천 대상이 있음 | 계약/기록 조작 |
| 수정된 송출 시간 | 보급소 해석 문구에서 시간 불일치 | 방송국/악의 개입 |
| 먼저 도착한 광고판 | 구조 목표 실패 후 배너 문구 | 구조 미끼 |
| 팝시의 반품 불안 | “저도 반품 대상인가요?” | 반품 권리 주장 |

0.2에서 범인을 보여주면 안 된다. 플레이어는 “캠페인이 이상한 것”과 “누군가 이 이상함을 이용하고 있는 것”의 차이를 아주 희미하게 느끼는 정도면 충분하다.

## 시나리오 연결

악의 행위자는 `malice_actor` 태그로 관리한다.

```text
MaliceThread
  id
  actor_type
  certainty
  discovered_evidence
  suspected_locals
  affected_population
  related_traces
  player_response
  current_status
```

### 확신도

| 단계 | 의미 | 표현 |
| --- | --- | --- |
| `rumor` | 소문이나 이상한 문구 수준 | NPC 한 줄 대사 |
| `pattern` | 같은 이상 현상이 반복됨 | 출격 게시판/흔적 연결 |
| `evidence` | 조작된 기록 발견 | 흔적/로그 |
| `confrontation` | 행위자 또는 대리자와 접촉 | 선택지/엘리트/보스 |
| `aftermath` | 폭로/은폐/거래 후 세계 변화 | 보급소 갈등/지역 변화 |

## 플레이어 선택

악의 시나리오는 단순 처단만으로 끝나면 안 된다. 플레이어는 정보를 어떻게 다룰지도 선택해야 한다.

| 선택 | 장점 | 비용 |
| --- | --- | --- |
| 폭로 | 신뢰 회복, 악의 확산 차단 | 보급소 불안, 반격 |
| 은폐 | 단기 안정, 내부 갈등 지연 | 나중에 더 큰 신뢰 붕괴 |
| 추적 | 진실 접근, 후속 시나리오 개방 | 자원 소모, 위험 지역 유도 |
| 거래 | 즉시 보급/정보 획득 | 윤리 비용, 악의 강화 |
| 처벌 | 명확한 결말, 인간 안정 상승 가능 | 로봇/공존 루트에서 불신 가능 |
| 해방/분리 | 로봇 악의나 재동기화 행위자에게 사용 | 결과가 불확실함 |

## NPC 반응 방향

| NPC | 악의에 대한 반응 |
| --- | --- |
| 미나 | 보급소 안전을 위해 공개를 늦추고 싶어 함 |
| 장도윤 | 기록과 장비 증거를 분석하지만, 위험한 역이용에 끌림 |
| 팝시-제로 | 반품/리콜 악의에 민감하게 흔들림 |
| 세븐 | 증거와 패턴을 중시하지만 의도를 판단하는 데 서툼 |
| 복희 | 이름을 이용한 악의를 가장 빨리 알아차림 |
| 윤서 | “캠페인 문구”와 “누군가의 거짓말”을 구분하려 함 |

## 문구 샘플

### 암시

- "구조 요청은 세 번 반복되었다. 세 번 모두 같은 광고판 아래에서 끊겼다."
- "라디오는 꺼져 있었다. 그런데 누군가 수신 확인란에 체크해 두었다."
- "영수증에는 산 사람이 없다. 반품 담당자 이름만 선명하다."

### 캠페인과 다른 말투

- "고객님의 안전을 위해 이동하십시오."
  아래에 손글씨가 덧붙어 있다. "가면 안 돼."

- "본 제품은 원소유자에게 반환됩니다."
  팝시는 그 문장을 세 번 읽고, 자기 이름을 한 번 더 말한다.

### 폭로 후

- "사람들이 조용해졌다. 침묵권이 안정된 것이 아니라, 서로를 다시 세기 시작했기 때문이다."
- "충전 스탠드 옆 경고문이 하나 늘었다. 이번에는 사람이 붙인 글씨다."
- "캠페인은 오늘도 웃었다. 하지만 이번 거짓말은 캠페인의 것이 아니었다."

## 보스/엘리트 연결

악의 행위자는 초반부터 보스로 만들 필요가 없다. 오히려 처음에는 전투 중 목표, 엘리트, 방해 이벤트, 흔적 문구로 나타나는 편이 좋다.

| 형태 | 예시 |
| --- | --- |
| 엘리트 | 구조 신호를 들고 도망치는 `친절한 안내원` |
| 목표 | 조작된 송신기를 파괴하거나 보관 |
| 흔적 | 수정된 구조 요청서, 원소유자 보증서 |
| 선택지 | 공개/은폐/추적/거래 |
| 후반 보스 | 침묵권 독점자, 리콜 성직자, 계약 관리자 |

## 장기 확장

장기적으로 진짜 악은 여러 갈래로 둔다.

| 갈래 | 연결 로컬 |
| --- | --- |
| 구조 신호 조작 | 방송국, 물류 허브, 외곽 주택가 |
| 로봇 반품/리콜 권리 | 로봇 서비스센터, 폐쇼핑몰 |
| 침묵권 독점 | 인간 정착지, 침묵권 네트워크 |
| 캠페인 보상 사냥 | 폐쇼핑몰, 고농도 로컬 |
| 자율성 포식 | 로봇 군락, 미지 ARI 접촉권 |

이 갈래들은 하나의 최종 조직으로 묶지 않아도 된다. 세계가 넓고 미지로 가득하다는 감각을 유지하려면, 악의도 여러 작은 이해관계와 공포에서 생겨야 한다.

## 금지 방향

- 캠페인을 만든 최종 흑막 조직으로 모든 것을 설명하지 않는다.
- 악역 하나를 죽이면 세계 문제가 해결되는 구조로 만들지 않는다.
- 악한 인간 때문에 인간 진영 전체가 악해 보이게 하지 않는다.
- 악한 로봇 때문에 로봇 전체를 위험한 종족처럼 만들지 않는다.
- 악의 행위자를 너무 일찍 확정해 미스터리를 닫지 않는다.
- 악의 시나리오를 장문 폭로 컷신으로만 전달하지 않는다.
