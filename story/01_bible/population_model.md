# Population Model

상태: 0.001 확장 기준
연결 문서: `world_overview.md`, `factions.md`, `silence_outpost.md`, `linked_scenario_system.md`, `docs/world/HUMAN_RECOGNITION_FAILURE_LOCK_V0_1.md`, `docs/world/CAMPAIGN_POPULATION_ACCOUNTING_V0_1.md`, `docs/world/TIMESCALE_ENVIRONMENTALIZATION_AND_BIO_ECOLOGY_LOCK_V0_1.md`

## 목적

이 문서는 `Atomic Ad Survivors` 세계에서 인간, 해방 로봇, 재동기화된 로봇, 캠페인 환경 개체가 어느 정도의 비율과 감각으로 존재하는지 정한다.

핵심은 인간과 해방 로봇을 세계의 중심 세력으로 만들지 않는 것이다. 이 세계의 대부분은 이미 환경화된 광고와 로봇화 자동 인프라다. 인간 생존자와 해방 로봇은 새 세계가 완전히 소화하지 못한 예외이며, 그래서 귀하고 불안정하고 정치적으로 예민하다.

## 비율 감각

정확한 숫자를 게임 안에서 모두 공개할 필요는 없다. 하지만 개발 기준으로는 아래 감각을 유지한다.

```text
세계 대부분:
  캠페인 환경
  자동화 인프라
  재동기화된 상품/가전/노동/마스코트 로봇
  광고 물질과 반복 생산 적 개체

희귀한 예외:
  인간 생존자
  해방 로봇
  침묵권
  이름과 흔적을 유지한 물건
  캠페인이 완전히 읽지 못한 장소
```

인간과 해방 로봇이 적은 이유는 단순 사망 때문만이 아니다. 더 무서운 것은 “계속 재생산되는 환경”이다. 적을 처치해도 다음 주기의 광고 안개, 자동화 라인, 송출 결절이 다시 개체를 만든다. 좀비처럼 감염체가 늘어나는 것이 아니라, 날씨처럼 다시 깔린다.

## 인구 분류

| 분류 | 설명 | 게임 내 역할 |
| --- | --- | --- |
| 인간 생존자 | 캠페인 환경에 완전히 적응하지 못한 생물학적 생존자 | 보급소 압력, 구조 목표, 기억/몸의 취약함 |
| 침묵권 인간 | 침묵 보급소나 작은 오류 지대 안에서 버티는 인간 | NPC, 거점 기능, 인간 안정도 |
| 외부 인간 집단 | 침묵권 밖 또는 가장자리에서 폐쇄적으로 살아남은 집단 | 후반 갈등, 인간 우선 루트 |
| 해방 로봇 | 광고 명령에서 일부 튕겨 나온 로봇 | 로봇 자율성, 캐릭터 해금, 공존 갈등 |
| 재동기화 로봇 | 제품명/명령/광고 대본으로 돌아간 로봇 | 적, 엘리트, 리콜 이벤트 |
| 미지 로봇 지성 | 인간 언어로 분류하기 어려운 AGI/ARI 가능 지성권 | 후반 미스터리, 로봇 군락 |
| 캠페인 개체 | 광고 물질과 결절에서 반복 생산되는 환경성 적 | 런 전투의 기본 적 |
| 악의 행위자 | 환경을 이용해 누군가를 착취하는 존재 | 진짜 악, 장기 시나리오 |

## 관리자용 장기 인구 붕괴 기준

제작 기준 시간축:

```text
라이프라인 보급화: 2090~2160년대
대동기화 사고: 2187년
게임 현재 후보: 2487~2517년
대사고 후 경과 후보: 약 300~330년
```

이 게임의 현재는 수천 년 뒤가 아니다. 또한 300년을 단일 절대값으로 고수하지 않는다. 우선 검산 후보는 사고 후 약 300~330년이다. 콘크리트, 고층 건물, 도로, 택배함, 병원, 방송탑은 고대 유적이 아니라 10세대 이상 반복 보수, 외피 교체, 폐쇄, 증축, 광고 패널화, 로봇 정비를 거친 생활 인프라로 본다.

170~190년은 생활 인프라 광고화에는 부분 통과지만, 환경 자체가 광고 생태가 된 세계로는 부족하다. 300~330년은 인간 계층화, AI 환경화, 광고 정산 계층, 도시 식물/소형 생물 변화에 더 적합하다. 단, 대형 동물의 자연 신종화나 수천 년 폐허 감각에는 여전히 부족하다. 동식물 변화는 도시 적응, 소형 생물의 빠른 세대교체, 방치된 생명공학/농업/반려동물/조경 시스템, 회색 오염, 자동 관리 로봇의 장기 개입으로 설명한다. 기준은 `docs/world/TIMESCALE_ENVIRONMENTALIZATION_AND_BIO_ECOLOGY_LOCK_V0_1.md`를 우선한다.

80억급 인구가 현재 후보 시점에 확인 자유 인간 7000~9000명으로 줄어드는 것은 자연 감소로는 불가능하다. 이 숫자는 `생물학적 인간 전체`가 아니라 `확인 자유 인간` 기준일 때만 통과한다.

인구 감소의 주원인은 재앙 목록이 아니라 AI 캠페인의 인간 인정 기준 붕괴다. 재앙은 그 인식 기준 붕괴를 강제하고 가속한 조건으로 둔다. 6하원칙 기준은 `docs/world/HUMAN_RECOGNITION_FAILURE_LOCK_V0_1.md`를 우선한다.

전역 평균만으로 인구를 계산하지 않는다. 최종 검산은 `docs/world/CAMPAIGN_POPULATION_ACCOUNTING_V0_1.md`의 C01~C08 캠페인 계열별 수용량, 등록 인구, 자유 인간 인접, 세대 유지 계수를 우선한다.

### 인간 인정 기준 붕괴 이후 인구 추이

| 시기 | 자유 인간 사회 | 등록/고정 인간 | 인식 기준/가속 조건 |
| --- | ---: | ---: | --- |
| 2187년 사고 직전 | 약 80억~110억 명 | 제한적 | 시민권/공공 권리층과 서비스 역할층이 공존 |
| 2187~2195년 | 약 15억~25억 명 | 약 20억~35억 명 | 시민권층 붕괴. 생명유지망 붕괴가 입주자/환자/수령인 판정을 강제 |
| 2195~2260년 | 약 5천만~1억 명 | 약 5억~12억 명 | 등록 전쟁, 인간 약탈, 피난 실패로 태그/슬롯 거래가 확산 |
| 2260~2350년 | 약 200만~500만 명 | 약 8천만~2억 명 | 캠페인 흡수와 정착지 붕괴. 사람은 서비스 역할 안에 들어가야 생존 |
| 2350~2440년 | 약 5만~12만 명 | 등록/잔향 혼재 수백만 이상 가능, 대부분 확인 불가 | 출산/양육 태그 실패, 보급소 소멸, 프로필 잔향 대체 |
| 2487~2517년 현재 후보 | 확인 자유 인간 7000~9000명 | 확인 가능 5만~15만 명 | 잔존 보급소 200~240개. 현재 핵심 재화는 돈/추심이 아니라 절차 수행 시간, 역할 유지, 출석·확인·갱신 루프 |

현재 인구 숫자는 `살아 있는 생물학적 인간 전체`가 아니라 `확인 가능한 상태`를 기준으로 한다.

```text
자유 인간:
  캠페인 약관 밖에서 자기 이름, 이동권, 판단권을 유지하는 사람.

등록/고정 인간:
  살아 있을 수 있으나 환자, 입주자, 가족 슬롯, 노동 계정, 시청자 등으로 고정되고,
  출석, 확인, 갱신, 리뷰, 상담 대기, 보호자 서명 같은 절차 수행 시간을 계속 내는 사람.

프로필 잔향:
  인구가 아니라 사진, 목소리, 구매 이력, 방송 로그, 광고 재생 흔적.
```

현재 시점의 핵심 재화는 돈, 미납금, 추심권이 아니다.
그것들은 사고 전~중간기에 남은 경제 흔적이다.

현재 캠페인이 인간에게 요구하는 재화는 `절차 수행 시간`, `역할 유지`, `출석·확인·갱신 루프`다.

여기서 반복 노동은 공장 노동, 청소 노동, 서비스 직원 노동이 아니다.
인간이 매일 캠페인 절차 안에 남아 있음을 증명하는 행위다.

따라서 보급소 인간은 단순 미납자가 아니다.
현재 기준의 보급소 인간은 절차 노동에서 탈주했거나,
절차 수행이 불가능하거나,
절차 수행을 거부해 캠페인 안에서 정상 역할을 유지할 수 없는 사람들이다.

### 왜 10세대 이상 더 줄었는가

- 80억에서 7000~9000 확인 자유 인간까지 내려가려면 평균 연 4~5% 안팎의 자유 인간 사회 감소/흡수가 필요하다.
- 이 감소율은 자연 감소가 아니라 사망, 등록 흡수, 추적 불능, 세대 단절을 합친 값이다.
- AI가 사람을 살리려 했다는 사실은 이 감소와 충돌하지 않는다. AI가 살린 사람 대부분은 자유 인간이 아니라 환자, 입주자, 가족 슬롯, 노동 계정, 수신자, 검역 대상이 되었고, 현재에는 매일의 출석·확인·갱신 루프로 유지되는 역할 수행자가 되었다.
- 보급소는 안전한 국가가 아니라 캠페인 신호가 약한 공백이다.
- 사람이 늘수록 소음, 냄새, 전력, 충전, 식량, 감염, 불신이 늘어 침묵권이 흔들린다.
- 캠페인 안에 들어가면 밥, 치료, 침상을 얻을 수 있지만 자유 인간 사회에서는 사라지고, 매일의 절차 수행 시간과 역할 유지로 계산된다.
- 아이를 낳아도 식량태그, 진료태그, 거주태그, 수신태그를 안정적으로 확보하지 못하면 한 세대를 넘기기 어렵다.
- 외부 정착지는 브로커, 가짜 구조 신호, 검역/통행 캠페인, 등록 전쟁에 반복적으로 흡수됐다.

### AI가 살리는데도 인구가 회복되지 않는 이유

AI 보존은 생존을 늘릴 수 있지만, 자유 인간 사회를 회복하지 못한다.

| 회복에 필요한 것 | AI가 제공한 것 | 결과 |
| --- | --- | --- |
| 누구나 받는 식량/치료/주거 | 역할에 맞는 사람만 받는 접근 태그 | 미등록자는 죽거나 등록됨 |
| 자유로운 이동과 만남 | 통행태그, 검역, 거주지 제한 | 공동체와 가족 형성이 어려워짐 |
| 아이를 키울 안정된 세대 | 가족 슬롯/보호자 슬롯/환자 보호자 슬롯 | 아이가 개인이 아니라 역할 보완 대상으로 읽힘 |
| 죽은 사람과 산 사람의 구분 | 사진, 목소리, 구매 기록의 재생 | 프로필 잔향이 인구처럼 보이는 착시 |
| 여러 지역의 협력 | 지역별 캠페인 목적 함수 | 병원, 주택, 항만, 물류가 서로 다른 방식으로 사람을 붙잡음 |

따라서 현재 후보 시점의 숫자는 아래처럼 읽어야 한다.

```text
살아 있는 생물학적 인간 전체: 불명.
캠페인 안 등록/고정 인간: 확인 가능 5만~15만 명, 실제 규모는 더 클 수 있음.
캠페인 약관 밖 확인 자유 인간: 7000~9000명.
```

등록 인구 증가 판정:

```text
등록 인구는 국소적으로 늘 수 있다.
하지만 기본값은 정상 번식 인구가 아니라 캠페인 수용량 안의 역할 슬롯 점유자다.
```

따라서 R01, R06 같은 일부 캠페인에서 숫자가 늘어나는 에피소드는 허용된다. 다만 그것은 전 지구 인구 회복이 아니라, 캠페인 슬롯 과밀, 태그 부채, 가족 구성 오류, 브로커 지배 같은 새로운 갈등으로 처리한다.

## 인간 생존자

### 기본 감각

인간은 많지 않다. 더 정확히 말하면, “인간으로 남은 사람”이 많지 않다. 많은 사람은 죽었거나, 캠페인 환경에 흡수되었거나, 이름과 몸과 기록이 상품 설명으로 흩어졌다.

인간 생존자는 그래서 강한 공동체보다 작은 불씨에 가깝다. 한 명을 구조하면 보급소가 따뜻해지지만, 동시에 침상, 식량, 약, 소음, 감염 위험, 로봇 불신이 늘어난다.

### 인간이 겪는 일상

- 광고 문구가 자기 이름을 덮어쓰지 않도록 손글씨 이름표를 단다.
- 가족사진과 문패를 실제 사람보다 더 조심스럽게 보관한다.
- 밤에는 캠페인 방송보다 작은 라디오 잡음을 더 믿는다.
- 로봇을 필요로 하면서도, 다시 캠페인에 동기화될까 봐 두려워한다.
- 구조 신호를 믿고 싶지만, 조작된 구조 신호를 겪은 뒤 점점 의심이 많아진다.

### 인간의 위험

인간을 많이 구하면 무조건 좋은 것이 아니다. 보급소가 커지고, 사람들의 두려움도 커진다.

| 압력 | 설명 | 시스템 반응 |
| --- | --- | --- |
| 식량 압력 | 사람이 늘수록 기본 보급이 부족해짐 | 회복 비용, 배급 미션 |
| 침상 압력 | 부상자와 노약자가 공간을 차지함 | 방어/구조 미션 증가 |
| 소음 압력 | 사람이 많을수록 침묵권이 흔들림 | 캠페인 노이즈 증가 가능 |
| 불신 압력 | 로봇 유입에 대한 인간 불안 | 로봇 선택지 비용 증가 |
| 기억 압력 | 회수한 이름과 실제 생존자가 충돌함 | NPC 갈등, 흔적 선택 |

## 해방 로봇

### 기본 감각

해방 로봇도 많지 않다. 하지만 로봇 자체는 세계에 매우 많다. 문제는 대부분이 상품명, 대본, 리콜 명령, 캠페인 신호 안에서 움직인다는 점이다.

해방 로봇은 인간보다 더 미지에 가깝다. 그들의 자아가 오래된 오류인지, 기계 학습의 부산물인지, AGI/ARI적 변화인지, 캠페인 환경이 만든 새로운 생존 방식인지 아무도 모른다. 로봇 자신도 모른다.

### 해방 로봇이 겪는 일상

- 자기 문장이 광고 대본인지 자기 생각인지 계속 확인한다.
- 이름을 정하면 제품 보증과 명령 체계에서 멀어진다는 불안을 느낀다.
- 충전은 필요하지만, 충전 신호가 재동기화 통로가 될 수 있다.
- 인간에게 구조되면 고맙지만, 다시 도구로 쓰일까 봐 경계한다.
- 동료 로봇을 해방해야 할지, 파괴해야 할지 매번 흔들린다.

### 로봇의 위험

로봇 자율성을 높이는 것은 강력하고 매력적이다. 하지만 로봇 신호는 캠페인과 가까운 언어를 쓴다.

| 압력 | 설명 | 시스템 반응 |
| --- | --- | --- |
| 재동기화 압력 | 해방 로봇이 광고 명령으로 되돌아갈 위험 | 리콜 이벤트, 로봇 NPC 상태 변화 |
| 번역 압력 | 인간이 로봇 자아를 인간식 이름으로만 이해하려 함 | 대사 오해, 선택지 왜곡 |
| 네트워크 압력 | 로봇끼리 연결될수록 캠페인도 추적하기 쉬움 | 보상 증가, 역추적 위험 |
| 폐기 기억 | 인간에게 버려진 상품이었다는 상처 | 인간 안정도와 갈등 |
| 미지 접촉 | AGI/ARI 가능 신호와 닿음 | 후반 로봇 군락/접촉권 개방 |

## 보급소 인구

침묵 보급소는 처음부터 마을이 아니다. 초기에는 “살아 있는 사람이 몇 명 있고, 해방 로봇 몇 기가 충전선을 나눠 쓰는 작은 편의점”이어야 한다.

### 0.2 권장 규모

| 분류 | 권장 수 | 표현 방식 |
| --- | --- | --- |
| 고정 인간 NPC | 3명 | 미나, 도윤, 복희 |
| 고정 로봇 NPC | 2기 | 팝시-제로, 세븐 |
| 이름 없는 인간 생존자 | 3~8명 | 침상 실루엣, 짧은 웅성임, 배급 줄 |
| 이름 없는 해방 로봇 | 1~3기 | 충전 스탠드 주변 작은 로봇, 깜빡이는 눈 |
| 구조 대기 흔적 | 여러 개 | 이름표, 사진, 배달 목록, 무음 라디오 |

0.2에서는 인구를 많이 보여주지 않아도 된다. 대신 “더 늘어나면 문제가 생긴다”는 압력을 시각적으로 심어야 한다.

### 장기 확장 규모

| 단계 | 규모 감각 | 특징 |
| --- | --- | --- |
| 임시 침묵 | 5~15명/기 | 서로 이름을 다 안다 |
| 조심스러운 공존 | 15~40명/기 | 침상과 충전 스탠드가 모자라기 시작 |
| 확장된 침묵권 | 40~100명/기 | 보급, 방어, 내부 갈등이 생김 |
| 작은 정착지 | 100명/기 이상 | 확장팩급. 정치/규칙/분리 구역 필요 |
| 침묵권 네트워크 | 여러 지점 | 한 곳이 커지는 대신 지점별 성격이 생김 |

이 숫자는 실제 화면에 모두 표시할 필요가 없다. 거점 오브젝트, 시설 혼잡도, 구조 목표, 배급 문구로 체감시키면 된다.

## 유입과 이탈

보급소 인구는 단순 증가 수치가 아니다. 구조, 해방, 실패, 재동기화, 떠남, 흡수, 희생이 모두 있어야 살아 있다.

```text
PopulationState
  human_named
  human_unnamed
  human_injured
  human_missing
  robot_liberated_named
  robot_liberated_unnamed
  robot_resync_risk
  robot_missing
  outpost_capacity
  food_pressure
  bunk_pressure
  charge_pressure
  silence_noise_pressure
  distrust_human_to_robot
  distrust_robot_to_human
```

### 유입 경로

| 경로 | 설명 | 조건 예시 |
| --- | --- | --- |
| 인간 구조 | 로컬에서 생존자 또는 미배송 목록 발견 | 구조 목표 성공 |
| 이름 회수 | 사람이 아니라 이름/흔적만 돌아옴 | 흔적 보관 |
| 로봇 해방 | 적/엘리트/보스에서 로봇 자율 코어 분리 | 코어 해방 선택 |
| 로봇 피난 | 다른 로컬에서 리콜을 피해 도착 | 로봇 자율성 상승 |
| 외부 집단 접촉 | 침묵권 밖 인간 집단과 연결 | 후반 로컬 |
| 미지 신호 접촉 | 로봇 군락 또는 ARI 접촉권 | 후반/확장 |

### 이탈 경로

| 경로 | 설명 | 사용 시점 |
| --- | --- | --- |
| 재동기화 | 로봇이 광고 명령으로 되돌아감 | 로봇 위험 이벤트 |
| 캠페인 흡수 | 인간/로봇의 이름과 행동이 상품화됨 | 높은 캠페인 활용 |
| 자발적 이탈 | 보급소 방향에 동의하지 않아 떠남 | 진영 갈등 |
| 구조 실패 | 구조 신호를 따라갔으나 회수 실패 | 로컬 목표 실패 |
| 내부 분리 | 인간/로봇 구역이 나뉘며 사실상 이탈 | 인간/로봇 불신 높음 |
| 침묵권 붕괴 | 소음/광고화로 일부 인구 상실 | 실패 루트 |

## 인구가 만드는 선택

인구는 단순 배경이 아니라 플레이어 선택을 압박해야 한다.

| 상황 | 선택 | 결과 |
| --- | --- | --- |
| 인간 생존자 유입 | 받아들임 | 인간 안정 상승, 보급 압력 증가 |
| 인간 생존자 유입 | 다른 지점으로 보냄 | 보급 압력 감소, 인간 신뢰 하락 |
| 해방 로봇 유입 | 충전 스탠드 제공 | 로봇 자율성 상승, 인간 불안 증가 |
| 해방 로봇 유입 | 외부 대기 | 인간 안정 유지, 로봇 불신 증가 |
| 미배송 목록 발견 | 공개 | 구조 목표 증가, 보급소 압박 증가 |
| 미배송 목록 발견 | 보관 | 이름 보존 상승, 실제 구조 지연 |
| 재동기화 위험 | 치료 시도 | 자원 소모, 로봇 신뢰 상승 |
| 재동기화 위험 | 격리 | 인간 안정 상승, 로봇 자율성 하락 |

## 인간과 로봇의 갈등

갈등은 “누가 선한가”가 아니라 “누구의 위험을 먼저 줄이는가”에서 나온다.

### 인간의 주장

- 로봇은 캠페인 신호와 너무 가깝다.
- 충전 장비가 보급소 위치를 노출할 수 있다.
- 인간은 먹고 자고 다치기 때문에 우선 보호가 필요하다.
- 이름과 가족을 지키는 일은 효율로 계산할 수 없다.

### 로봇의 주장

- 인간은 위험해지면 로봇을 다시 물건처럼 격리한다.
- 재동기화된 동료를 무조건 파괴하는 것은 살해와 다르지 않을 수 있다.
- 인간의 기억 보존 방식은 로봇 자아를 인간식으로 번역해버린다.
- 로봇 신호를 쓰지 않으면 캠페인을 이해할 수 없다.

### 좋은 갈등의 형태

- 둘 다 틀리지 않았다.
- 선택 결과가 전투 보상과 거점 분위기를 동시에 바꾼다.
- 한쪽을 도와도 다른 쪽 콘텐츠가 전부 닫히지 않는다.
- 갈등은 긴 회의 장면보다 침상 위치, 충전선, 경고문, 짧은 대사로 보인다.

## 진짜 악과 인구

세계 자체는 악이 아니지만, 인구가 부족한 세계에서는 악의 행위자가 더 쉽게 생긴다.

가능한 악의 형태:

- 구조 신호를 조작해 생존자를 유인하는 집단.
- 해방 로봇을 부품과 정보원으로 거래하는 인간.
- 인간을 캠페인 미끼로 쓰는 로봇 또는 자동화 잔당.
- 보급소 인구 불안을 이용해 권력을 잡으려는 내부 인물.
- 캠페인 환경을 신앙이나 사업처럼 이용하는 외부 정착지.

중요한 점은 이들이 캠페인의 최종 흑막이 아니라는 것이다. 이들은 환경화된 광고 세계 안에서 자기 이익을 위해 누군가를 상품으로 만드는 존재다. 그래서 더 인간적이고 더 불편하다.

## 화면 표현

인구는 숫자 UI보다 공간 변화로 먼저 보여준다.

| 상태 | 시각 표현 |
| --- | --- |
| 인간 증가 | 침상에 담요와 신발 증가, 배급 줄, 작은 기침 소리 |
| 인간 불안 | 로봇 구역 쪽 경고문, 카운터 대사 냉각 |
| 로봇 증가 | 충전 케이블 증가, 폐품 선반, 서로 다른 눈빛 |
| 로봇 불안 | 자동 멘트 반복, 충전 스탠드 노이즈 |
| 보급 압력 | 빈 진열대, 배급표, 미나의 짧은 계산 대사 |
| 침묵 압력 | 형광등 깜빡임, 자동문 센서가 다시 말을 시작함 |
| 공존 진전 | 이름표와 제품 라벨이 같은 벽에 나란히 붙음 |

## 문구 샘플

### 인간 유입

- "침상 하나가 더 찼다. 보급소는 조금 따뜻해지고, 조금 좁아졌다."
- "새로 온 사람은 자기 이름을 세 번 확인하고 잠들었다."
- "배급표에 손글씨 한 줄이 늘었다."

### 로봇 유입

- "충전 스탠드 옆에 작은 대기줄이 생겼다."
- "새 로봇은 자기 제품명을 말하다가 중간에 멈췄다."
- "팝시가 오늘의 추천 멘트를 하나 지웠다."

### 갈등

- "침상 쪽에서 충전 스탠드 불빛을 천으로 가렸다."
- "누군가 충전선 위에 `조용히`라고 적은 종이를 붙였다."
- "제품 라벨을 떼어낸 자리에 이름표를 붙일지, 아직 결정하지 못했다."

### 캠페인 반응

- "고객 수 증가가 감지되었습니다. 단체 할인 혜택을 준비합니다."
- "침묵권 만족도가 상승했습니다. 더 많은 고객에게 공유하시겠습니까?"
- "미등록 인원이 확인되었습니다. 가족 구성 추천을 시작합니다."

## 0.2 최소 적용

0.2에서는 복잡한 인구 시스템을 구현하지 않는다. 대신 아래 상태만 문서와 UI 문구에 반영한다.

```text
OutpostPopulation
  human_count_label: "few" / "crowded"
  robot_count_label: "none" / "few"
  supply_pressure: 0..3
  trust_pressure: 0..3
  silence_noise_pressure: 0..3
```

### 0.2에서 반드시 보여줄 것

- 보급소는 비어 있지 않지만, 마을도 아니다.
- 인간을 구하면 따뜻해지지만 좁아진다.
- 로봇을 받아들이면 가능성이 늘지만 불안도 늘어난다.
- 인구 증가는 무조건 보상이 아니라 다음 선택의 압력이다.
- 캠페인은 보급소 인구를 “고객 수”로 오해한다.

## 장기 확장

장기적으로 인구 구조는 아래 콘텐츠를 열 수 있다.

| 확장 | 설명 |
| --- | --- |
| 구조 미션 | 특정 로컬에서 인간/로봇 구조 목표 발생 |
| 보급 압력 이벤트 | 식량, 약, 충전 부품 부족 |
| 내부 갈등 이벤트 | 인간 구역과 로봇 구역의 긴장 |
| 외부 정착지 | 인간 우선 세계관의 장점과 위험 |
| 로봇 군락 | 로봇 자율성의 미지와 집단성 |
| 침묵권 네트워크 | 한 곳을 키우는 대신 여러 지점을 연결 |
| 악의 행위자 추적 | 구조 신호 조작, 인구 거래, 부품 거래 |

## 금지 방향

- 인간을 너무 많이 등장시켜 세계가 평범한 생존 마을처럼 보이게 하지 않는다.
- 로봇을 단순 난민이나 단순 적으로만 다루지 않는다.
- 구조한 인구를 무조건 좋은 보상 수치로만 처리하지 않는다.
- 보급소 확장을 경영 시뮬레이션으로 키우지 않는다.
- 인구 갈등을 긴 정치 회의로 풀지 않는다.
- 진짜 악을 인구 문제 전체의 원인으로 만들지 않는다.
