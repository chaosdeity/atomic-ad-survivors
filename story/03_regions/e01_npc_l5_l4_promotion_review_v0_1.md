# E01 NPC L5/L4 승격 후보 검토 v0.1

## 문서 상태

```text
상태:
작성 완료 v0.1

작업 성격:
E01 NPC 대량 후보 풀 120개 중 L5/L4 승격 후보 28개만 재검토한 상세화 컷 문서.
120개 전체 상세 프로필 작성 아님.
새 플레이어블 후보 작성 아님.
R01 확정 NPC, 보급소 5명, E01 보조 NPC 최소안을 대체하지 않음.

판정:
통과

기준 문서:
story/03_regions/e01_npc_mass_candidate_pool_v0_1.md
docs/world/NPC_MASS_PRODUCTION_TEAM_PROMPT_V0_1.md
docs/world/NPC_PRODUCTION_ROADMAP_V0_1.md
story/02_hub/outpost_core_npc_profiles_v1_0.md
story/03_regions/r01_npc_data_schema_application_v0_1.md
story/03_regions/e01_support_npc_minimum_plan_v0_1.md
story/06_characters/playable_20_name_registry_v0_1.md
```

핵심 판정:

```text
L5/L4는 "전부 직접 대면" 등급이 아니다.
L5도 플레이어블 기능을 침범하거나 후반 진실을 너무 빨리 열면 v0.1 또는 잔향으로 낮춘다.
L4도 상태값/선택 후폭풍이 강하면 짧은 프로필이나 대사 은행으로 보존한다.
```

이번 문서의 목적:

1. 프로필 v1.0으로 올릴 후보를 5명 이하로 제한한다.
2. v0.1 짧은 프로필은 지역 기능 검산용으로 남긴다.
3. 대사 은행 후보는 시설 기능보다 상태값 후폭풍을 말하게 한다.
4. R01/보급소/플레이어블과 겹치는 후보는 잔향 또는 배경으로 낮춘다.

## 0. 검토 요약

검토 대상:

```text
E01 NPC 대량 후보 풀의 L5/L4 승격 후보 28개
```

재분류 결과:

| 판정 | 수 | 처리 |
|---|---:|---|
| 프로필 v1.0 우선 후보 | 5 | 상위 3~5명만 1차 상세화 |
| 프로필 v0.1 후보 | 8 | 직접 대면 여부와 충돌 조건을 먼저 잠그는 짧은 프로필 |
| 대사 은행 후보 | 8 | 조건 대사/오브젝트 문구 중심 |
| 잔향/배경 하향 후보 | 7 | 사람 프로필 대신 로그, 음성, 표식, 배경 문구 |
| 합계 | 28 | 전체 후보 재분류 완료 |

우선 결론:

```text
프로필 v1.0 1차 후보는 r06_mass_001, r02_mass_001, r03_mass_001, r08_mass_001, r07_mass_001로 제한한다.

r04_mass_001은 원래 L5지만 PATCH, METER-06, RIO-08과 겹칠 위험이 커서 v0.1 조건부로 낮춘다.
r05_mass_001은 R05 세계 진실 조기 공개 위험이 커서 v0.1 잔향형으로 낮춘다.
```

직접 대면 총량 기준:

```text
이번 검토에서 직접 대면 우선으로 남길 수 있는 후보는 최대 8명이다.
그중 v1.0 우선 후보 5명도 실제 구현 때는 지역/시즌 오픈 순서에 따라 일부만 직접 대면한다.
```
