# 전역 NPC 확장 슬롯 레지스트리 v0.1

## 문서 상태

```text
상태:
작성 완료 v0.1

작업 성격:
420개 전역 씨앗을 768개 전역 제작 슬롯으로 확장하기 위한 실제 슬롯 등록 기준.
새 캠페인 작성 아님.
새 플레이어블 작성 아님.
E01 한국권 NPC 추가 생산 아님.
이름 있는 NPC 348명 추가 아님.

판정:
전역 수량 충족 슬롯 등록
```

이 문서는 `global_npc_quantity_satisfaction_matrix_v0_1.md`의 +348 증분을 실제 제작 슬롯으로 푼다.

핵심:

```text
E01은 +0으로 고정한다.
E02는 +24 슬롯만 추가한다.
E03~E12는 각 +30 슬롯, 총 +300 슬롯을 추가한다.
cross_region은 +24 슬롯만 추가한다.
증분 합계는 +348 슬롯이다.
420 + 348 = 768이다.
```


## 7. E02 이후 필수 질문 답변

1. 이 NPC가 왜 한국어 이름이면 안 되는가?

E02~E12 슬롯은 각 현실권의 등록 절차, 언어권, 이동권, 결제권, 항만권, 보급권을 보여준다.
한국어 임시명은 지역 질문을 흐리고 E01만 과밀하게 보이게 만든다.

2. 해당 현실권/언어권에 맞는가?

이번 단계에서는 본명보다 권역별 제도 앵커와 노출 방식을 먼저 맞췄다.
본명이 필요한 L5/L4만 시즌 오픈 시 언어권 검수를 다시 거친다.

3. 이름이 정말 필요한 계층인가?

대부분 필요 없다.
L3 이하는 역할명, 로그명, 잔향명, 오브젝트명, 목록명으로 충분하다.

4. 이름 없이 오브젝트/로그/잔향으로 처리해도 되는가?

대부분 그렇다.
특히 +348 슬롯 중 L3 이하 307개는 이름 없이 처리한다.

5. 플레이어블 기능을 빼앗고 있지 않은가?

빼앗지 않는다.
슬롯은 플레이어블의 능력 설명자가 아니라 캠페인 절차, 상태 반응, 선택 후폭풍, 작은 결말의 얼굴이다.

6. 이 NPC가 해당 캠페인의 작은 결말, 미스터리, 반전 중 무엇을 담당하는가?

L5/L4는 장기 미스터리와 반전 접점을 맡고, L3은 상태 후폭풍을 맡는다.
L2/L1/L0는 작은 결말의 증거물, 명단, 문구, 로그로 남는다.
