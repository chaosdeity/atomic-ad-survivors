```text

E02 첫 외부 슬라이스에서 NPC/잔향/오브젝트 후보를 실제 Vertical Slice 구현 범위로 자르는 문서.
NPC 소개장 아님.
```

## 0. 구현 요약

| 구현 등급 | 수 | 대상 |
|---|---:|---|
| 직접 대면 제한 | 2 | 마지막 보관함 수취인, 막차 환승 보류관 |
| 화면/음성 중심 | 2 | 방재 문구 승인자, 반복 안내 편집자 |
| 조건 대사 | 3 | 편의점 회수 카운터 잔류자, 반복 안내 편집자, 정기권 판독자 |
| 오브젝트/잔향 | 3 | 미세부품 검수자, 지하 대피실 조용한 관리자, 가족 정기권 보류자 |
| 보류 | 0 | 이 문서 대상 9개는 모두 낮은 구현 단위로 포함 |

직접 대면 제한:

```text
E02 첫 슬라이스는 직접 대면 NPC를 최대 2명으로 제한한다.
GATE-09와 BROAD-10이 플레이어블 트랙에 있으므로 E02 NPC가 해금 기능을 대신하면 안 된다.
```

## 2. 직접 대면 컷

### 2.1 포함

| 대상 | 이유 | 제한 |
|---|---|---|
| 마지막 보관함 수취인 | E02_C03의 미수령/수취인 질문을 가장 즉시 체감시킴 | 보관함 앞 1회 이하 |
| 막차 환승 보류관 | GATE-09 해금 전 E02_C07 압력을 세움 | 플랫폼 1회 이하, 해금 기능 금지 |

### 2.2 직접 대면 제외

| 대상 | 이유 |
|---|---|
| 방재 문구 승인자 | BROAD-10과 충돌 위험. 화면/음성/자막이 더 적합 |
| 미세부품 검수자 | PATCH/METER-06/RIO-08 기능 침범 위험 |
| 지하 대피실 조용한 관리자 | QUIET-07 튜토리얼화 위험 |
| 가족 정기권 보류자 | R01 가족칸 반복 위험 |

## 4. 플레이어블 충돌 방지

| 플레이어블 | 보호 기능 |
|---|---|
| GATE-09 | 환승 심사 전투, 개찰 클립, HP 0 플랫폼 잔류, 해금 장면 |
| BROAD-10 | 공영 안내 수신자, 청취 확인, 대피 문장 자동 승인, HP 0 반복 송출 |
| RETURN-05 | 반송 수취인 전투, 송장/포장끈/반송 스티커 정산 |
| PATCH | 보급소 수리 로봇, 자기부품화, 로봇 자율성 |
| METER-06 | 충전/계량/차단 전투 |
| QUIET-07 | 낮은 수신, 약한 인양, 침묵권 피난자 |
| OPEN-HOST | R01 가족/입주 안내와 동선 기능 |
