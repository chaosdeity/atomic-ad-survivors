## 0. 판정 요약

회원 등급 브로커는 R06 포인트 거리의 1차 핵심 NPC다.

이 인물은 캠페인의 원인이 아니다.
하지만 사고 후 남은 멤버십 약관과 등급 문구를 이용해, 사람에게 "선택했다"는 책임을 떠넘기는 인간 행위자다.

핵심 판정:

```text
프로필 v1.0 유지.
직접 대면 가능.
단, 거래 UI와 보상 상점 기능은 금지한다.
```

R06에서 이 NPC가 맡는 질문:

```text
선택권이 등급별 혜택으로 팔릴 때,
그 선택은 아직 선택인가?
```

## 3. 지역 질문

R06은 캠페인보다 인간이 더 불편하게 보일 수 있는 지역이다.

캠페인은 약관대로 움직인다.
회원 등급 브로커는 그 약관을 읽고, 사람이 덜 불리한 듯 보이는 선택지를 판다.

지역 질문:

```text
생활 접근권이 회원 등급과 리뷰 점수로 갈릴 때,
낮은 등급의 사람은 스스로 선택하지 않은 결과까지 책임져야 하는가?
```

이 NPC가 보여줘야 하는 것:

1. 멤버십 등급은 돈이 아니라 접근 허가의 언어가 되었다.
2. 인간 브로커는 캠페인을 만든 사람이 아니지만, 캠페인의 빈틈을 이용한다.
3. "할인", "승급", "우선 접근" 같은 말이 누군가의 지연, 박탈, 등록 압력을 숨긴다.

## 4. 핵심 오판

회원 등급 브로커의 핵심 오판:

```text
등급이 높으면 선택지도 높다.
선택지가 많아졌다면, 그 결과는 선택한 사람의 책임이다.
```

이 믿음이 위험한 이유:

```text
브로커가 늘리는 것은 자유가 아니라, 다른 사람의 접근권을 미리 당겨 쓰는 방식이다.
선택지는 늘어난 것처럼 보이지만, 낮은 등급의 줄 끝은 더 빨리 닫힌다.
```

대표 문장:

```text
"선택권은 무료가 아닙니다. 할인될 뿐이죠."
```

## 5. 보이는 역할과 실제 기능

### 보이는 역할

```text
멤버십 등급을 조정해 주는 사람.
부족한 태그를 빌려 주는 사람.
위험한 출입문을 열 수 있는 약관을 아는 사람.
```

### 실제 기능

```text
R06의 멤버십 압력과 인간 악의를 대표한다.
선택 후폭풍을 보급소와 다음 지역에 남긴다.
R02 보험/진료태그, R03 반품/보증태그, R07 통행태그가 R06에서 거래 언어로 바뀌는 지점을 보여준다.
```

이 NPC가 주면 안 되는 것:

```text
상점 목록
태그 교환 메뉴
할인 보상
안전한 우회권
전투 강화
```

반응 단계:

| 조건 | 반응 | 구현 메모 |
|---|---|---|
| `membership_pressure` 낮음 | 브로커는 농담처럼 약관을 말한다 | 직접 대면 전 약관 영수증 중심 |
| `membership_pressure` 높음 | 등급판의 낮은 등급 줄이 더 빨리 닫힌다 | 군중/대기열 문구 추가 |
| `supply_pressure` 높음 | 오늘의 접근권을 내일의 몫에서 당겨 쓰라고 압박한다 | 미나 대체 금지 |
| `human_stability` 낮음 | 선택지가 아니라 협박처럼 들리기 시작한다 | 대사 우선순위 상승 |
| `campaign_density` 높음 | 멤버십 안내 문구가 더 친절하고 정확해진다 | 캠페인이 브로커를 돕는 것이 아니라 약관이 선명해짐 |
| `trace_preservation` 높음 | 브로커가 지운 계약 조각이 남는다 | 복희/세븐 반응 가능 |

## 7. 선택 후폭풍

### 선택 A. 브로커의 승급 제안을 받아들인다

결과:

```text
즉시 접근 가능한 문이 하나 열린다.
대신 membership_pressure가 상승하고 낮은 등급 대기열 하나가 닫힌다.
보급소는 다음 귀환 때 "태그가 맞는데 몫이 맞지 않는" 상태를 본다.
```

브로커 대사:

```text
"축하합니다. 방금 선택지가 늘었습니다. 줄 끝이 줄어든 건 약관상 별도 항목입니다."
```

### 선택 B. 제안을 거절한다

결과:

```text
즉시 접근은 어려워진다.
대신 낮은 등급 대기열의 일부 흔적이 보존된다.
브로커는 유저를 비합리적인 사람으로 기록한다.
```

브로커 대사:

```text
"거절도 선택입니다. 다만 낮은 등급의 거절은 기록이 오래 남습니다."
```

### 선택 C. 가짜 태그를 폭로한다

결과:

```text
fake_tag_exposed 플래그가 켜진다.
일부 피해자의 접근권이 보류되지만, R06의 캠페인 농도와 인간 보복 위험이 함께 오른다.
```

브로커 대사:

```text
"진짜와 가짜를 구분하면 공정해질 것 같습니까? 공정은 늘 더 비싼 등급입니다."
```

### 선택 D. 보류하고 흔적만 보관한다

결과:

```text
즉시 보상 없음.
trace_preservation이 오르고, 후속 이름/현실 위치 검산에서 브로커의 본명 조각 후보가 열린다.
```

## 8. 플레이어블 충돌 방지

| 대상 | 충돌 위험 | 방지 기준 |
|---|---|---|
| 윤서 | 회수/정산 판단을 브로커가 대신 설명함 | 브로커는 윤서의 현장 회수 기능을 설명하지 않는다. 윤서는 선택 후폭풍을 회수하고 브로커는 선택지를 오염시킨다 |
| PATCH | 로봇 부품 거래나 수리 기능으로 이동 | 브로커는 로봇을 수리하거나 부품화하지 않는다. R04/R03 보증 흔적을 거래 언어로만 언급 |
| 하람 | 진료태그/보험 심사 기능 침범 | R02의 퇴원 보류와 의료 판정은 하람과 R02 NPC의 몫이다. 브로커는 보험 대기표가 거래되는 후폭풍만 담당 |
| 이루 | 반송/미수령 전투 기능 반복 | 브로커는 물류 대상을 분류하지 않는다. 반품 태그가 포인트 거래로 넘어오는 후폭풍만 보여준다 |
| 유건 | 충전태그/전력 사용량 거래 기능 침범 | 전력 접근권을 팔지 않는다. 충전태그가 멤버십 등급에 끌려 들어가는 문구만 허용 |
| 여울 | 침묵권/낮은 수신을 VIP 은신권처럼 팔 위험 | 침묵권은 상품화하지 않는다. 낮은 수신은 R08/여울의 리스크로 보존 |
| 리오 | 정품 복구/객실 정리 로봇 축 선점 | 리오의 체류/숙박 캠페인권과 정품 복구 공포는 언급하지 않는다 |
| 미카 / GATE-09 | 게이트 통과와 빠른 이동 기능 반복 | R06의 등급 게이트는 상업 접근 압박이며, 미카의 환승/개찰/막차 심사와 분리 |
| 향후 10~12번 후보 | Nora/BROAD-10, Mara/CLAUSE-11, Rafaela/LEDGER-12의 글로벌 방송/약관/태그 중개 축 선점 | R06 브로커는 E01 지역형 인간 악의로 제한한다. 마라의 대형마트 약관 회원, 라파의 항만시장 태그 중개자 축을 글로벌 확장으로 남긴다 |

## 9. 보급소/R01 NPC 충돌 방지

| 대상 | 충돌 위험 | 처리 |
|---|---|---|
| 미나 | 정산/배급 판단을 브로커가 대체 | 브로커는 정산하지 않는다. 보급소 귀환 후 미나가 "합계는 맞는데 몫이 틀린다"고 반응할 수 있는 흔적만 남긴다 |
| 장도윤 | 위험 출력/장비 개조를 상품화 | 강화 기능 금지. 약관과 등급판만 사용 |
| 팝시 | 충전 스탠드 리롤/마스코트 기능과 충돌 | 등급 리롤 같은 기능 금지 |
| 세븐 | 도감/정보 분석 역할 대체 | 세븐은 브로커 계약의 패턴을 분석할 수 있지만 정답을 주지 않는다 |
| 복희 | 이름 보관/기억 보존 기능 대체 | 브로커의 본명은 복희가 바로 보관하지 않는다. 계약 조각과 지워진 서명만 반응 대상으로 남김 |
| R01 OPEN-HOST | 내부 안내인 기능 반복 | 브로커는 길 안내자가 아니다. 그는 선택지를 비싸게 만드는 사람이다 |
| R01 WELCOME-DESK | 접수/상담 절차 비틀기 반복 | 브로커는 상담을 보류하지 않는다. 약관을 이용해 책임을 전가한다 |
