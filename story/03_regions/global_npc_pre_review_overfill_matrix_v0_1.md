# 전역 NPC 검수 전 추가 확장 매트릭스 v0.1

## 문서 상태

```text
상태:
작성 완료 v0.1

작업 성격:
검수팀 전달 전 전역 NPC/잔향/기록/배경 슬롯을 한 단계 더 확장하는 과충족 문서.
새 캠페인 작성 아님.
새 플레이어블 작성 아님.
E01 한국권 NPC 추가 생산 아님.
직접 대면 NPC 1,024명 작성 아님.

판정:
검수 전 수량 과충족
```

이 문서는 기존 `768개 제작 슬롯`을 폐기하지 않는다.
768개를 `전역 수량 충족 기준`으로 두고, 검수팀에 넘기기 전 여유분을 더해 `1,024개 제작 슬롯`까지 올린다.

핵심:

```text
1,024개는 전부 직접 대면 NPC가 아니다.
1,024개는 L5/L4/L3/L2/L1/L0를 합친 전역 제작 슬롯이다.
E01은 120개에서 더 늘리지 않는다.
증분 +256개는 E02~E12와 cross_region에만 배정한다.
L3 이하는 본명 없이 역할명, 잔향명, 로그명, 오브젝트명, 목록명으로 처리한다.
```

판정:

```text
전역 NPC 수량은 검수 전 기준 1,024개 제작 슬롯으로 과충족한다.
96개 캠페인에 전부 12명씩 직접 NPC를 붙이지 않는다.
캠페인 본편 슬롯은 총 976개이고, 나머지 48개는 cross_region 장기 축이다.
```

## 1. 왜 1,024까지 올리는가

768개는 전역 제작이 비어 보이지 않는 기준이다.
1,024개는 검수팀이 캠페인별 밀도, 권역별 언어권, 작은 결말, 상태 반응을 잘라낼 수 있는 여유 기준이다.

| 기준 | 의미 | 판정 |
|---|---|---|
| 420개 | 1차 게이트. 빈 캠페인 방지 | 유지 |
| 768개 | 전역 수량 충족. 시즌 오픈 대기열 확보 | 유지 |
| 1,024개 | 검수 전 과충족. 잘라내도 전역 밀도 유지 | 이번 목표 |
| 1,152개 | 96개 캠페인 x 12명 직접 NPC 방식 | 금지 |

1,024개가 안전한 이유:

```text
1,024개는 1,152개보다 낮다.
대부분 L3 이하 슬롯이라 이름과 대면 연출을 요구하지 않는다.
검수팀이 10~15%를 하향/폐기해도 768개 충족 기준 밑으로 떨어지지 않는다.
```

## 7. 검수팀 전달 기준

검수팀은 아래를 확인한다.

| 검수 항목 | 기준 | 판정 방식 |
|---|---|---|
| 수량 | 1,024 슬롯 합계 유지 | 숫자 검산 |
| E01 추가 생산 | 0 유지 | 통과/실패 |
| E02~E12 현실권 | 권역별 언어권/이주권/등록 절차 일치 | 통과/재검토 |
| 계층 | L5/L4 직접 대면 과밀 방지 | 계층 하향 |
| 이름 | L3 이하 본명 금지 | 이름 제거 |
| 플레이어블 충돌 | 기능/해금/전투/성장 침범 금지 | 보류/폐기 |
| R01/보급소 충돌 | 확정 NPC 기능 대체 금지 | 보류/폐기 |
| 작은 결말 | 캠페인별 미스터리/반전/후폭풍 연결 | 보강 요청 |
