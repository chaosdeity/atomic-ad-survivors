# 전역 NPC 2,048 1차 검수 배치 v0.1

## 문서 상태

```text
상태:
작성 완료 v0.1

작업 성격:
1,024개 검수 전 과충족 기준을 2,048개 1차 검수 배치로 확장하는 문서.
새 캠페인 작성 아님.
새 플레이어블 작성 아님.
E01 한국권 NPC 추가 생산 아님.
직접 대면 NPC 2,048명 작성 아님.

판정:
1차 검수 배치 수량 충족
```

이 문서는 `1,024개 제작 슬롯`을 `2,048개 제작 슬롯`으로 확장한다.
2,048개는 장기 풀스케일 10,240개의 20%에 해당하는 1차 검수 배치다.

핵심:

```text
E01은 120개에서 더 늘리지 않는다.
증분 +1,024개는 E02~E12와 cross_region에만 배정한다.
L5는 4개만 추가하고, 대부분은 L3 이하 상태 반응/잔향/로그/오브젝트/목록으로 처리한다.
2,048개는 직접 대면 NPC 수가 아니다.
```

판정:

```text

검수팀은 이 2,048개를 더 늘리지 않고, 하향/오브젝트화/이름 제거/폐기를 수행한다.
검수 후 최소 1,536개 이상을 유지하면 통과, 768개 아래로 떨어지면 실패다.
```

## 4. 계층별 2,048 목표

| 계층 | 1,024 기준 | 추가 증분 | 2,048 기준 | 이름 정책 |

| L5 핵심 | 36 | +4 | 40 | 이름 후보 가능. 시즌 오픈 후 검토 |
| L4 준핵심 | 108 | +44 | 152 | 이름 후보 조건부. 역할명 우선 |

| L1 배경/목록/호출 번호 | 186 | +260 | 446 | 본명 금지 |
| L0 제도 흔적/자동 문구 | 139 | +216 | 355 | 인물화 금지 |
| 합계 | 1,024 | +1,024 | 2,048 | L3 이하 이름 금지 기본 |

검수팀 절단 여유:

```text
L5/L4 합계는 192개로 제한한다.
L3 이하 1,856개가 세계 밀도를 담당한다.
검수팀이 25%인 512개를 하향/폐기해도 1,536개가 남는다.
이 수량은 768개 충족 기준의 2배다.
```
