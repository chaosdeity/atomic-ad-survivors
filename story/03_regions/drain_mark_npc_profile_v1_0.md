# DRAIN-MARK NPC 상세 프로필 v1.0

## 배수로 표식꾼 / P1 침묵권과 안전하지 않은 자유의 얼굴

## 문서 상태

```text
상태:
스토리팀 전달용 v1.0 후보

용도:
DRAIN-MARK의 NPC 기준, 첫 등장, 상태값 반응, 관계선, 대사, 금지 방향을 고정한다.

판정:
통과 추천

주의:
DRAIN-MARK는 자유로운 정찰꾼이 아니다.
집도 보급소도 믿지 못해서, 캠페인 신호가 약한 길만 기억하게 된 사람이다.
```

---

## 0. 캐릭터 핵심

DRAIN-MARK는 R01 배수로 침묵 주머니에 표식을 남기는 자유 인간이다.

그는 집 안으로 들어가지 않는다.
보급소에도 완전히 들어가지 않는다.
현관 줄에도 서지 않는다.

그는 신호가 약한 곳을 고른다.

```text
캠페인이 덜 듣는 곳.
광고음이 물소리에 묻히는 곳.
수신이 끊기는 곳.
비콘도 약해지는 곳.
```

핵심 문장:

```text
"집은 사람을 바꾸고,
보급소는 사람을 세죠.
나는 그냥 안 들리는 곳을 고릅니다."
```

DRAIN-MARK의 핵심은 이것이다.

```text
캠페인에게 덜 들키는 길은,
보급소에게도 덜 닿는 길이다.
```

## 1. 기본 프로필

```text
임시명:
DRAIN-MARK

유저용 표기:
DRAIN-MARK
또는 드레인마크

본명:
초기 비공개.
v1.0에서는 확정하지 않는다.
이 NPC는 개인 이름보다 "배수로에 표식을 남기는 사람"이라는 기능이 먼저 보여야 한다.

성별/표현:
확정 보류.
성별성보다 젖은 지도, 금속 표식, 낮은 목소리, 경계심이 먼저 보여야 한다.

나이감:
20대 후반~50대 후보.
너무 어린 생존자나 노련한 스카우트 고정은 피한다.

구분:
R01 자유 인간
배수로 은신자
침묵권 표식꾼
P1 숨은 루트 NPC

소속:
R01 배수로 침묵 주머니

첫 역할명:
배수로 표식꾼

내부 정체성:
캠페인 신호가 약한 길만 기억하는 자유 인간

캠페인이 등록하려는 역할:
통행자
검역 우회자
거주태그 미확인자
가족 구성 불응자
수신 불량 대상
주소 미확인자

핵심 믿음:
숨으면 산다.

핵심 결함:
숨는 것과 살아 있는 흔적을 남기는 것을 구분하지 못한다.

핵심 공포:
누군가 자신을 찾는 순간, 캠페인도 함께 찾을 것이라는 공포.

핵심 욕망:
아무에게도 등록되지 않고 살아남는 것.

담당 기능:
P1 침묵권
숨은 루트
배수로 우회
안전하지 않은 자유
통행태그 우회
인양 불안정 경고
캠페인 농도 상태값 반응
인간 안정도 상태값 반응
```

## 2. 유저용 첫 소개문

```text
DRAIN-MARK
배수로 표식꾼

R01 배수로 침묵 주머니에 표식을 남기는 사람.

집 안으로 들어가지 않는다.
보급소에도 완전히 들어가지 않는다.

캠페인 신호가 약한 길,
광고음이 물소리에 묻히는 길,
비콘도 흔들리는 길만 기억한다.

DRAIN-MARK는 숨으면 산다고 믿는다.

하지만 숨은 길은 캠페인에게서 멀다.
동시에 보급소에게서도 멀다.
```

```text
캠페인에게 덜 들키기 위해,

```

```text
숨으면 산다고 믿지만,
오래 숨으면 아무도 찾지 못한다는 것을 아직 받아들이지 못한 사람.
```

### 후보 C

```text
안전한 길은 아닙니다.
덜 들리는 길입니다.
```

권장:

```text

후보 A

후보 C
```

## 4. 담당 기능

DRAIN-MARK는 P1 침묵권과 숨은 루트의 감정 얼굴이다.

담당 기능:

```text
배수로 침묵 주머니
P1 숨은 루트
통행태그 우회
수신 불안정
긴급 인양 불안정
가짜 표식 구분
보급소 밖 자유 인간의 불안
윤서의 강제 구출 불가 감정 강화
```

DRAIN-MARK가 있어야 유저는 R01을 이렇게 이해한다.

```text
R01에서 집 안으로 들어가지 않는다고 자동으로 자유로운 것은 아니다.
숨은 사람도 다른 방식으로 갇힐 수 있다.
```

## 5. 잘못 믿는 것

DRAIN-MARK는 이렇게 믿는다.

```text
숨으면 산다.
```

이 믿음은 일부 맞다.

```text
캠페인 신호가 약한 곳에서는 등록 압력이 낮다.
가족/입주/상담 동선을 덜 듣는다.
통행태그 없이 우회할 수 있다.
```

하지만 이 믿음은 위험하다.

```text
보급소 회수선도 약해진다.
비콘 신호가 불안정해진다.
식량태그와 진료태그 접근이 어렵다.
흔적이 없으면 복희도 이름을 보관하기 어렵다.
아무도 못 찾으면 살아 있는지 사라진 건지 구분되지 않는다.
```

DRAIN-MARK는 이 차이를 배워야 한다.

## 6. 첫 등장

### 위치

```text
R01-L04 배수로 침묵 주머니
또는 P1 재방문 시 열리는 지하 배수로 입구
```

### 배경 오브젝트

```text
젖은 지도
금속 표식
물에 번진 화살표
끊긴 비콘 신호 조각
배수로 벽면 긁힘
녹슨 통행 안내판
캠페인 안내음이 끊기는 지점
```

### 첫 장면

윤서가 배수로 입구에 도착한다.

위쪽에서는 오픈하우스 안내음이 희미하게 들린다.
아래쪽에서는 물 흐르는 소리와 금속 울림이 들린다.

벽에 작은 금속 표식이 붙어 있다.

윤서가 표식을 만지려는 순간, 어둠 속에서 목소리가 들린다.

```text
"만지지 마요.
그건 광고가 못 읽는 글자입니다."
```

윤서:

```text
"누가 남긴 거야?"
```

DRAIN-MARK:

```text
"살아남은 사람이요.
살아남고 싶은 사람도 가끔 읽고요."
```

## 7. 상태값 반응

### 7-1. 통행태그

부족:

```text
배수로 우회로 가치가 커진다.
하지만 위험도도 커진다.
```

대사:

```text
"통행태그가 없으면 문은 못 열죠.
물길은 문이 아닙니다.
대신 물길은 사람을 안 봅니다."
```

안정:

```text
공식 통로와 배수로를 비교해 준다.
```

대사:

```text
"문으로 가도 됩니다.
오늘은요.
문이 덜 배고파 보이네요."
```

과의존:

```text
DRAIN-MARK가 경계한다.
```

대사:

```text
"태그가 열어준 길은,
태그가 닫을 수도 있습니다."
```

### 7-2. 캠페인 농도

낮음:

```text
DRAIN-MARK가 표식을 더 오래 남긴다.
대화가 조금 열린다.
```

대사:

```text
"오늘은 위쪽이 조용합니다.
그래도 오래 머물진 마요."
```

중간:

```text
배수로 입구에도 안내음이 새어 들어온다.
일부 표식을 지운다.
```

대사:

```text
"저쪽이 물소리까지 따라오기 시작했습니다."
```

높음:

```text
가짜 표식이 생긴다.
캠페인이 배수로 표식을 흉내 낸다.
```

대사:

```text
"저 표식은 제 것이 아닙니다.
너무 반듯하잖아요."
```

### 7-3. 인간 안정도

낮음:

```text
DRAIN-MARK는 길 공유를 거부한다.
사람들이 몰려오면 좁은 길에서 서로를 밀 것이라 본다.
```

대사:

```text
"사람들이 겁먹으면 좁은 길에서 서로를 밀어요.
위쪽이랑 다르지 않습니다."
```

중간:

```text
윤서에게 일부 표식만 보여준다.
```

대사:

```text
"한 명이 읽는 건 괜찮습니다.
열 명이 읽으면 길이 아닙니다."
```

높음:

```text
회수선 기준점 일부를 공유한다.
```

대사:

```text
"찾을 수 있게 남기는 건...
숨는 것과 다르겠죠."
```

### 7-4. 인양 안정도

안정:

```text
보급소 방식에 조금 덜 적대적이다.
```

대사:

```text
"여기까지 회수선이 닿는다면,
그건 길이 아니라 약속에 가깝겠네요."
```

불안정:

```text
자기 방식을 고집한다.
```

대사:

```text
"보세요.
저 줄도 여기선 끊깁니다."
```

고위험:

```text
윤서에게 깊이 들어오지 말라고 경고한다.
```

대사:

```text
"여기서 쓰러지면,
캠페인도 늦고 보급소도 늦습니다."
```

## 8. 관계선

### 8-1. 윤서

관계:

```text
데려오고 싶은 사람과 데려가고 싶지 않은 사람
```

대표 대화:

```text
윤서:
"보급소로 와.
거기엔 적어도 인양 장비가 있어."

DRAIN-MARK:
"알아요.
그래서 다들 돌아오죠.
그리고 이름표가 붙죠."

윤서:
"이름표가 붙는 게 나쁜 건 아니야."

DRAIN-MARK:
"집 문패도 처음엔 그랬습니다."
```

### 8-2. PORCH-LINE

관계:

```text
줄 안의 질서와 줄 밖의 은신
```

대표 대화:

```text
PORCH-LINE:
"줄에 서야 누가 밀렸는지 압니다."

DRAIN-MARK:
"줄에 서면 누가 데려갈지도 알죠."
```

### 8-3. ROOM-12

관계:

```text
너무 오래 앉은 사람과 너무 오래 움직인 사람
```

대표 대화:

```text
ROOM-12:
"저는 자리를 못 비워요."

DRAIN-MARK:
"저는 자리를 만들면 안 돼요."

윤서:
"둘 다 못 움직이는 거네."
```

### 8-4. OPEN-HOST

관계:

```text
알려진 길과 숨겨진 길의 충돌
```

대표 대화:

```text
OPEN-HOST:
"그 길을 알면 사람들을 살릴 수 있어요."

DRAIN-MARK:
"그 길을 알면 저쪽도 따라옵니다."
```

### 8-5. NAME-PAINTER

관계:

```text
읽히는 이름과 읽히지 않는 표식
```

대표 대화:

```text
NAME-PAINTER:
"이름이 없으면 어떻게 찾아요?"

DRAIN-MARK:
"이름이 있으면 저쪽도 찾아요."
```

### 8-6. MAIL-LOOP

관계:

```text
주소 호출과 비주소 표식
```

대표 대화:

```text
MAIL-LOOP:
"가장 가까운 가족 주소로 안내합니다."

DRAIN-MARK:
"주소가 있는 길은 이미 늦었습니다."
```

### 8-7. VAC-0

관계:

```text
사람용 표식과 청소 대상
```

대표 대화:

```text
VAC-0:
"벽면 오염 표식 감지."

DRAIN-MARK:
"그거 지우면 다음 사람은 물길로 갑니다."

VAC-0:
"위험 경로 표시입니까?"

DRAIN-MARK:
"살아 있는 사람용 지도입니다."
```

## 10. 퀘스트 / 이벤트 구조

### 10-1. 표식 읽기

```text
목표:
배수로 벽의 금속 표식을 읽는 법을 배운다.

갈등:
표식은 길이지만, 공개되는 순간 위험해질 수 있다.

선택:
1. 표식을 그대로 따라간다.
2. 세븐에게 분석시킨다.
3. 보급소 회수선 기준점으로 일부만 기록한다.
4. 표식 공개를 보류한다.
```

### 10-2. 회수선 기준점

```text
목표:
보급소가 배수로 깊은 곳까지 들어오지 않도록, 중간 회수선 기준점만 만든다.

의미:
DRAIN-MARK가 보급소에 완전히 들어가지 않으면서도 완전히 끊기지 않는 선택.
```

### 10-3. 가짜 표식

```text
목표:
캠페인이 흉내 낸 가짜 배수로 표식을 구분한다.

의미:
숨은 길도 오염될 수 있음을 보여준다.
```

## 13. 플레이어블 가능성

### 판정

```text
후속 가능
```

### 이유

DRAIN-MARK는 P1 침묵권과 숨은 루트의 감정 얼굴로 강하다.

다만 즉시 플레이어블보다 NPC 유지가 먼저다.

플레이어블화 가능성이 있으려면 전투/이동/인양 리스크가 명확해야 한다.

가능한 전투 축:

```text
신호 약화 구역 활용
통행태그 없이 우회
가짜 표식 구분
인양 위험을 감수한 고위험 루트
캠페인 호출 회피
```

위험:

```text
멋있는 은신 캐릭터가 되면 실패.
보급소 거부만 하는 반항 캐릭터가 되면 실패.
```

v1.0 기준 추천:

```text
P1 핵심 NPC 유지.
후속 게스트 동행 또는 플레이어블 후보로 보류.
```

## 14. 성장 방향

초기:

```text
보급소를 믿지 않는다.
집 안으로 들어가지 않는다.
표식을 숨긴다.
신호가 약한 길만 기억한다.
```

중반:

```text
윤서에게 일부 표식을 보여준다.

```

후반:

```text

하지만 일부 회수선 기준점은 남긴다.

```

최종 권장 상태:

```text
DRAIN-MARK는 R01 배수로에 남는다.
하지만 완전히 끊겨 있지는 않다.

숨는 표식은 숨긴다.
찾을 수 있는 표식은 남긴다.
```

최종 변화 문장:

```text
숨는 것만으로는 살 수 없다는 걸 배우고,

```

## 17. 최종 요약

DRAIN-MARK는 배수로 침묵 주머니에 표식을 남기는 사람이다.

그는 집을 믿지 않는다.
보급소도 완전히 믿지 않는다.

그는 신호가 약한 길을 고른다.

그 길은 캠페인에게서 멀다.
하지만 보급소에게서도 멀다.

DRAIN-MARK의 드라마는 숨어 사는 자유를 찬양하는 이야기가 아니다.

```text
숨는 것만으로는 살 수 없다는 걸,
그리고 발견될 수 있는 흔적을 스스로 남기는 법을 배우는 이야기다.
```
