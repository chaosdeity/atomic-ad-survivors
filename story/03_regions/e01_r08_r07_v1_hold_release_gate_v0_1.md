## 1. `r08_mass_001` 가짜 구조 신호 보관자

### 1.1 해제 조건

| 조건 | 기준 |
|---|---|
| R08 시즌 순서 | R08이 단순 보급소 확장이 아니라 `서부 비상보급망` 시즌 슬라이스로 열린다 |
| QUIET-07 기능 보호 | 여울의 낮은 수신, 약한 인양, 침묵권 피난자 기능이 먼저 유저에게 읽혔다 |
| R08-L05 잠금 | 가짜 구조 신호탑이 보스 전 내부 위기로 쓰일 시점이 확정되었다 |
| 보급소 압력 | 보급소가 구조 신호를 크게 키우는 선택의 위험을 이미 경험했다 |

### 1.2 v1.0에서 잠글 핵심

```text
잘못된 믿음:
크게 부르면 더 빨리 산다.

실제 기능:
구조 신호가 클수록 캠페인과 외부 위험도 더 잘 듣는다.

직접 대면:
가능하더라도 신호탑 내부의 짧은 대면 1회.
장기 안내 NPC 금지.
```

### 1.3 금지

```text
여울의 은신/정찰/낮은 수신 전투 대체
침묵권 지도자화
구조 신호 정답 설명자화
보급소 확장 시설 담당화
```

## 2. `r07_mass_001` 출항 심사 보류관

### 2.1 해제 조건

| 조건 | 기준 |
|---|---|
| R07 시즌 순서 | R07이 항만/공항 검역 축으로 실제 오픈된다 |
| GATE-09 기능 보호 | 미카의 막차 환승 심사, 개찰 미통과, 플랫폼 잔류가 먼저 읽혔다 |
| 출항 질문 잠금 | "나가면 산다"가 정답이 아니라는 지역 질문이 시즌 목표에 포함되었다 |
| 검역/통행 분리 | 의료 판정, 환승 판정, 출항 판정이 서로 다른 절차로 구분되었다 |

### 2.2 v1.0에서 잠글 핵심

```text
잘못된 믿음:
밖이라는 말은 도장이 찍힌 뒤에만 쓸 수 있다.

실제 기능:
출항 심사는 탈출 가능성이 아니라, 통과 가능한 사람을 다시 보류하는 절차다.

직접 대면:
출항 관리탑 또는 검역 게이트에서 1회 제한.
빠른 이동, 선장, 브로커, 탈출 안내자 금지.
```

### 2.3 금지

```text
GATE-09 환승/개찰/빠른 이동 기능 반복
탈출 정답 제공자
항만 브로커 영웅화
검역문 열화상 판독자와 기능 중복
```
