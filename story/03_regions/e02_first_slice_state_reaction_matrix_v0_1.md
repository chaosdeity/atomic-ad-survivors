```text
직접 대면 반응은 마지막 보관함 수취인과 막차 환승 보류관에만 둔다.

그 외 6개 대상은 조건 대사, 로그, 장치 음성, 잔향 오브젝트로만 처리한다.
```

## 1. 상태 키 기준

| 상태 키 | E02 첫 슬라이스에서의 의미 | 주요 사용 대상 |
|---|---|---|
| `delivery_state` | 보관, 회수, 미수령, 만료 사이의 흔들림 | 마지막 보관함 수취인, 편의점 회수 카운터 잔류자 |
| `passage_tag` | 막차, 개찰, 환승, 정기권 판독 절차 | 막차 환승 보류관, 정기권 판독자, 가족 정기권 보류자 |
| `signal_tag` | 보관함 알림, 막차 안내, 방재 문구 수신의 어긋남 | E02_C03, E02_C05, E02_C07 |
| `signal_stability` | 안내/자막/방재 방송이 안정적으로 읽히는 정도 | 방재 문구 승인자, 반복 안내 편집자, 지하 대피실 조용한 관리자 |
| `campaign_density` | 절차가 사람을 더 강하게 캠페인 상태로 고정하는 정도 | E02_C05, E02_C07 |
| `human_stability` | 안내, 미통과, 보류 문구가 사람을 흔드는 정도 | 방재 문구 승인자, 막차 환승 보류관, 정기권 판독자 |
| `trace_preservation` | 물건, 자막, 명단, 라벨이 사람의 흔적을 대신 보존하는 정도 | E02_C03, E02_C04, E02_C05 |
| `robot_autonomy` | 정품/리콜/미세부품 절차가 로봇 자율성을 묶는 정도 | 미세부품 검수자 |
| `charge_tag` | 충전/정품/검수 절차 접근권 | 미세부품 검수자 |
| `extraction_stability` | 보급소가 외부 신호를 붙잡는 안정도 | 지하 대피실 조용한 관리자 |
| `residence_tag` | 세대, 가족권, 통근권이 사람을 묶는 정도 | 가족 정기권 보류자 |
| `supply_pressure` | 대피소, 회수실, 보급 절차의 압력 | 지하 대피실 조용한 관리자 |
