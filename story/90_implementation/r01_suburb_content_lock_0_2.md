# R01 Suburb Content Lock 0.2

상태: 0.2 외곽 주택가 콘텐츠 확정표  
연결 문서: `story/03_regions/r01_suburb.md`, `story/03_regions/r01_boss.md`, `story/05_progression/region_evolution_model.md`, `story/90_implementation/implementation_0_2_story_directive.md`

## 목적

이 문서는 0.2에서 외곽 주택가에 실제로 넣을 콘텐츠를 확정한다.

기준:

```text
첫 지역 하나만으로도 이 게임의 재미가 보여야 한다.
```

외곽 주택가는 짧은 튜토리얼 맵이 아니다. 0.2에서는 작게 구현하지만, 유저가 반복 방문할 이유와 지역이 반응한다는 느낌을 반드시 남겨야 한다.

## 한 줄 정의

```text
외곽 주택가는 완벽한 가족과 완벽한 집을 팔던 동네가, 끝나지 않는 오픈하우스 광고 생태권으로 변한 첫 로컬이다.
```

유저가 첫 30분 안에 느껴야 하는 것:

- 밝고 귀엽지만 어딘가 잘못됐다.
- 광고 문구가 실제 공격과 장판이 된다.
- 집, 가족, 이름이 상품으로 바뀌고 있다.
- 윤서의 반품 스캐너/반품 도장이 이 세계와 잘 맞는다.
- 같은 지역을 다시 가면 조금 다르게 보인다.

## 0.2 포함 범위

반드시 포함:

| 범주 | 콘텐츠 |
| --- | --- |
| 기본 지역 | 외곽 주택가 1맵 |
| 하위 구역 표현 | 침묵 가장자리, 분양 주택 루프, 모델하우스 결절 |
| 적 | 쿠폰 전단 떼, 미소 우편함, 세일 풍선 |
| 엘리트 | 행복 보증 상담원 |
| 보스 | 스마일 홈 시어머니 |
| 보스 선택 | 결절 파괴, 기억 추출 |
| 흔적 | 지워지지 않은 문패, 반쯤 탄 가족사진, 이름 없는 영수증 |
| 송출 기록 | 120초, 180초, 240초 이벤트 3개 |
| 반복 반응 | 재방문 1회, 재방문 3회, 윤서 반복 방문 |
| 게시판 변화 | 기본, 결절 파괴 후, 기억 추출 후, 단골 고객 주간 |

높은 우선순위로 가능하면 포함:

| 범주 | 콘텐츠 |
| --- | --- |
| 적 | 홈케어 청소기 |
| 흔적 | 고장 난 마스코트 배지, 무음 라디오, 납 차폐 배터리 |
| 하위 구역 표현 | 배수로 침묵 주머니 |
| 반복 변형 | 침묵 배수로, 고객 맞춤형 추적 |
| 윤서 반응 | 같은 집 반복, 이름 없는 영수증 |

0.2에서 보류:

| 콘텐츠 | 이유 |
| --- | --- |
| 코어 해방 선택 | 팝시/로봇 감정선이 더 필요 |
| 신호 우회 선택 | 캠페인 활용 루트가 아직 이르다 |
| 캠페인 송출관 보스전 | 상위 접근 목표로만 둔다 |
| 조작된 구조 신호 본격 추적 | 진짜 악의 씨앗만 둔다 |
| 완전한 다구역 맵 | 0.2 구현 범위 초과 |

## 하위 구역 0.2 표현

0.2에서는 실제 맵을 크게 나누지 않아도 된다. 같은 전투 맵 안에서 배경, 목표 위치, 배너 문구, 스폰 구성을 바꿔 구역처럼 느끼게 한다.

| 구역 | 표현 | 기능 |
| --- | --- | --- |
| 침묵 가장자리 | 보급소 무전, 낮은 광고 소음 | 시작/첫 회수 |
| 분양 주택 루프 | 반복 집, 웃는 우편함, 쿠폰 전단 | 기본 전투 |
| 모델하우스 결절 | 오픈하우스 표지판, 상담원, 가족 포스터 | 엘리트/보스 전조 |
| 배수로 침묵 주머니 | 형광 슬라임 배수구, 라디오, 배터리 | 선택 확장 |
| 가짜 귀환로 | 방향 표지판, 수상한 회수 문구 | 보스 후 확장 |

최소 구현:

```text
침묵 가장자리
분양 주택 루프
모델하우스 결절
```

## 전투 흐름

5분 기준 첫 런/반복 런 공통 흐름:

| 시간 | 내용 | 유저 학습 |
| --- | --- | --- |
| 00:00~00:45 | 쿠폰 전단 떼 | 이동 + 자동공격 |
| 00:45~01:30 | 미소 우편함 추가 | 직선 탄 회피 |
| 01:30~02:15 | 세일 풍선 추가 | 장판 회피 |
| 02:00 전후 | 송출 기록 1 이벤트 | 깊이 들어가면 단서가 나온다 |
| 03:00 전후 | 행복 보증 상담원 | 엘리트와 목표 보상 |
| 03:00 전후 | 송출 기록 2 이벤트 | 신호가 방향을 가진다 |
| 04:00 전후 | 송출 기록 3 이벤트 | 보스 접근 절차 |
| 04:30 이후 | 보스 전조 또는 보스 접근 | 지역 결절 대면 |

첫 전투는 100초 전후 `signal_overload`로 강제 회수한다. 따라서 첫 전투에서는 180초/240초 이벤트를 보지 못해도 된다.

## 적 확정

### 쿠폰 전단 떼

0.2 필수.

역할:

- 기본 잡몹.
- 대량 처치감.
- 광고 전단 조각의 세계관 개연성 제공.

패턴:

```text
낮은 체력
빠른 접근
대량 스폰
```

문구:

```text
놓치면 손해입니다. 맞아도 손해입니다.
```

### 미소 우편함

0.2 필수.

역할:

- 원거리 압박.
- 플레이어 이동 경로 흔들기.
- 개인 편지가 광고지로 바뀌는 첫 이미지.

패턴:

```text
짧은 예고
직선 광고지 탄 발사
느린 이동 또는 고정
```

문구:

```text
고객님께 도착한 특별 제안입니다.
```

### 세일 풍선

0.2 필수.

역할:

- 귀여운 표면 + 위험 장판.
- 모바일에서 읽기 쉬운 폭발 학습.

패턴:

```text
느린 접근
근접 또는 파괴 시 폭발
짧은 슬라임 장판 생성
```

문구:

```text
부풀수록 혜택도 커집니다.
```

### 홈케어 청소기

0.2 선택, 가능하면 포함.

역할:

- 로봇이 악이 아니라 역할에 묶였다는 암시.
- XP/드롭 회수를 방해하는 재미.

패턴:

```text
약한 흡입 영역
플레이어와 드롭을 조금 끌어당김
```

주의:

- 드롭을 완전히 빼앗으면 불쾌하다.
- 해방 로봇이 아니라 재동기화된 자동화 기계로 보이게 한다.

## 엘리트 확정

### 행복 보증 상담원

0.2 필수.

원래 정체:

```text
무인 부동산 안내 키오스크
```

전투 역할:

- 플레이어 동선을 광고식으로 바꾸는 첫 엘리트.
- 송출 기록 이벤트와 함께 써도 된다.

패턴:

| 패턴 | 효과 |
| --- | --- |
| 추천 매물 구역 | 바닥 위험 구역 표시 후 쿠폰/슬라임 발생 |
| 상담 예약 | 플레이어 위치에 지연 폭발 표식 |
| 고객 동선 최적화 | 주변 잡몹 이동속도 잠시 증가 |

문구:

```text
고객님의 도주 동선에 맞는 매물을 찾았습니다.
```

0.2 합격 기준:

- 위험 범위가 모바일에서 명확히 보인다.
- 맞으면 짜증나는 것이 아니라 “아, 광고가 내 길을 팔고 있구나”가 느껴진다.

## 송출 기록 이벤트

송출 기록은 시간 보상이 아니라 목표 보상이다.

### 송출 기록 1

시점:

```text
120초 전후
```

이벤트:

```text
전단 소용돌이 안에서 신호 표식 회수
```

전투 문구:

```text
축하합니다! 고객님의 위치가 일부 확인되었습니다.
```

보급소 해석:

```text
세븐: 이건 좌표가 아니라 영수증입니다. 누군가를 산 기록이 아니라, 누군가를 찾으려다 실패한 기록입니다.
```

### 송출 기록 2

시점:

```text
180초 전후
```

이벤트:

```text
움직이는 모델하우스 표지판을 일정 시간 추적
```

전투 문구:

```text
가까운 행복 매장을 안내합니다. 이 길은 항상 처음입니다.
```

보급소 해석:

```text
도윤: 광고가 방향을 갖고 있어. 길 안내가 아니라, 길을 만드는 쪽에 가까워.
```

### 송출 기록 3

시점:

```text
240초 전후
```

이벤트:

```text
고농도 신호 결절 주변 엘리트 처치 또는 생존
```

전투 문구:

```text
개장 준비 완료. 가족 단위 고객은 정문으로 입장해 주세요.
```

보급소 해석:

```text
세븐: 세 조각이 같은 문을 가리킵니다. 위치라기보다 입장 절차입니다.
```

확정 문구:

```text
송출관 접근 절차 확정
```

## 흔적 확정

### 지워지지 않은 문패

0.2 필수.

획득:

```text
엘리트 처치 후 또는 오래된 현관 목표 근처 생존
```

효과:

- 이름 보관함 반응.
- 미나/복희 대사.
- 같은 집의 이름 시나리오 씨앗.

문구:

```text
페인트 밑에 아직 사람 이름이 남아 있습니다.
```

### 반쯤 탄 가족사진

0.2 필수.

획득:

```text
모델하우스 결절 근처 또는 보스 기억 추출 후
```

효과:

- 기억 추출 루트 강화.
- 복희 대사.
- 가족사진 잔향층 씨앗.

문구:

```text
얼굴은 탔지만, 손을 잡은 모양은 남아 있습니다.
```

### 이름 없는 영수증

0.2 필수.

획득:

```text
미소 우편함 드롭 또는 송출 기록 이벤트 보조 보상
```

효과:

- 윤서의 빈 고객란 시나리오 씨앗.
- 미나/윤서 반응.
- 캠페인 기록 조작 암시.

문구:

```text
구매 항목은 남아 있는데, 구매자 이름만 지워졌습니다.
```

### 무음 라디오

0.2 선택.

획득:

```text
배수로 침묵 주머니
```

효과:

- 조작된 구조 신호 암시.
- 도윤/세븐 반응.

문구:

```text
아무 소리도 나지 않는데, 안테나는 같은 방향을 가리킵니다.
```

### 고장 난 마스코트 배지

0.2 선택.

획득:

```text
세일 풍선 무리, 어린이 행사장 잔해, 또는 행복 보증 상담원 보조 드롭
```

효과:

- 팝시-제로 출시 씨앗.
- 팝시/세븐 반응.
- 로봇이 단순 적이 아니라 반품된 상품이었음을 암시.

문구:

```text
웃는 얼굴 배지는 고장 났지만, 뒤쪽 리콜 번호는 아직 따뜻합니다.
```

### 납 차폐 배터리

0.2 선택.

획득:

```text
차고 또는 배수로 침묵 주머니
```

효과:

- 회수선/차징 조율대 개조.
- 보급소 확장 개연성.

문구:

```text
무거운 배터리 표면에 광고 문구가 붙지 못하고 벗겨져 있습니다.
```

## 보스 확정

보스:

```text
스마일 홈 시어머니
```

0.2 역할:

- 외곽 주택가의 중간 결절.
- 캠페인 송출관과 분리.
- 가족 상품화 법칙의 첫 얼굴.

필수 패턴:

| 패턴 | 구현 |
| --- | --- |
| 오늘의 가족 특가 | 쿠폰 탄막 |
| 깨끗한 거실 | 홈케어 청소기/흡입 장판 |
| 기념사진 시간 | 플래시 둔화 |

0.2 선택지:

| 선택 | 지역 변화 | 유저 체감 |
| --- | --- | --- |
| 결절 파괴 | 캠페인 농도 감소, 조용한 집 증가 | 이겼고 조금 안전해졌다 |
| 기억 추출 | 흔적/가족 잔향 증가, 미스터리 심화 | 이겼지만 더 찜찜해졌다 |

보스 후 문구:

```text
가족 구성이 완료되지 않았습니다.
행복 보증을... 갱신하시겠습니까?
```

## 반복 방문 변화

### 재방문 1회

표현:

```text
게시판 부제 변경
NPC 한 줄 반응
같은 집 구조 1회 강조
```

문구:

```text
단골 고객님, 같은 행복을 또 찾으셨군요!
```

### 재방문 3회

표현:

```text
단골 고객 주간
윤서 전용 문구
행복 보증 상담원 등장 확률 증가
```

문구:

```text
캠페인: 윤서 고객님께 어울리는 가족 구성을 준비했습니다.
윤서: 내 가족을 네가 정하지 마.
```

### 보스 후 재방문

결절 파괴:

```text
게시판: 조용한 오픈하우스
미나: 오늘은 저쪽 집들이 조금 덜 웃고 있어.
```

기억 추출:

```text
게시판: 보정된 가족사진
복희: 사진이 사람을 기억하는 건지, 사람이 사진에 갇힌 건지 모르겠구나.
```

```text
8.0 / 10
```

판정:

첫 지역으로 매우 좋다. 단, 실제 플레이에서 밝고 재미있는 전투가 먼저 살아야 한다. 가족/기억/이름의 슬픔은 흔적과 짧은 대사로만 묻혀야지, 초반부터 장문으로 누르면 유저가 피곤해진다.

10점으로 가는 개선 조건:

- 같은 지역을 다시 갔을 때 10초 안에 변화가 보인다.
- 세일 풍선과 우편함이 귀엽지만 위협적으로 기억된다.
- 윤서의 반품 도장이 이 지역 적에게 특히 잘 어울린다.
- 보스 선택 후 다음 런의 공기가 확실히 달라진다.
- 유저가 “이 집 또 나왔네?”를 리소스 재탕이 아니라 의도된 기괴함으로 받아들인다.

## 테스트 항목

반드시 확인:

- 첫 전투 100초 회수가 억지 패배처럼 느껴지지 않는가.
- 쿠폰 전단 떼가 처치 쾌감을 주는가.
- 미소 우편함 탄이 모바일에서 읽히는가.
- 세일 풍선 폭발 범위가 공정한가.
- 행복 보증 상담원이 불쾌한 방해가 아니라 재미있는 엘리트인가.
- 송출 기록 이벤트가 시간 보상이 아니라 목표 보상으로 느껴지는가.
- 보스 선택 후 게시판/대사/적 구성 중 하나 이상이 바뀌는가.
- 재방문 3회 문구가 유저를 기분 나쁘게만 하지 않고 재미있게 찝찝한가.
- 흔적 문구가 너무 우울하거나 장황하지 않은가.

## 금지 방향

- 외곽 주택가를 일회성 튜토리얼 맵으로 만들지 않는다.
- 첫 지역부터 길찾기/퍼즐을 복잡하게 만들지 않는다.
- 가족 상실을 긴 컷신으로 설명하지 않는다.
- 캠페인 송출관을 첫 보스로 뭉개지 않는다.
- 스마일 홈 시어머니를 악한 로봇 엄마로 단순화하지 않는다.
- 반복 집 구조가 단순 리소스 재사용처럼 보이게 두지 않는다.
- 보스 선택 후 변화가 게시판 문구 하나로만 끝나지 않게 한다.
- 내부 상태값을 유저에게 점수 UI로 보여주지 않는다.
