# First Boss Directive 0.2

상태: 0.2 첫 보스 구현 작업 지시서  
대상: `스마일 홈 시어머니`  
목적: 첫 보스가 전투, 보상, 지역 진화, 캐릭터 애정을 한 번에 연결하게 만든다.  
기준: 첫 보스는 클리어 체크포인트가 아니라, 외곽 주택가가 선택을 먹고 변한다는 첫 증거여야 한다.

## 핵심 목표

0.2 첫 보스는 아래 감정을 만들어야 한다.

```text
드디어 큰 결절을 건드렸다.
이겼지만 지역이 끝난 것은 아니다.
내 선택 때문에 다음 외곽 주택가가 달라진다.
윤서의 반품 도장이 보스전에서도 의미 있었다.
```

## 보스 고정 정보

```text
id = "boss_smile_home_mother_in_law_001"
name = "스마일 홈 시어머니"
region_id = "r01_suburb"
boss_type = "local_mid_node"
original_form = "가족 상담형 홈 어시스턴트"
campaign_role = "외곽 주택가를 완벽한 가족 상품 생태권으로 묶는 결절"
```

주의:

- 스마일 홈 시어머니는 0.2 실제 보스다.
- 캠페인 송출관은 상위 접근 목표다.
- 두 존재를 같은 보스로 합치지 않는다.

## 전투 진입 조건

0.2 권장 조건:

```text
broadcast_records_decoded >= 3
transmitter_access_state = "procedure_confirmed"
r01_suburb_state.node_state = "active"
```

유저 표현:

```text
송출관 접근 절차 확정
외곽 주택가 중심 결절이 노출되었습니다.
```

주의:

- `좌표 조각` 같은 별도 재화를 만들지 않는다.
- 보스 조우는 출격 횟수 보상이 아니라 송출 기록 이벤트 성과로 열린다.

## 보스 아레나

장소:

```text
외곽 주택가 중앙 모델하우스 앞 원형 교차로
```

필수 오브젝트:

- 오픈하우스 표지판.
- 같은 집을 가리키는 화살표.
- 가족사진 플래시 장치.
- 쿠폰 전단.
- 홈케어 청소기 소환 지점.

모바일 기준:

- 위험 범위는 원, 부채꼴, 사각형 중심으로 명확하게 표시한다.
- 패턴 예고는 짧고 선명해야 한다.
- 가족/광고 문구는 배경이 아니라 전투 판독에 방해되지 않는 위치에 둔다.

## 패턴 P0

### 1. 오늘의 가족 특가

역할:

- 기본 탄막.
- 쿠폰 전단 떼와 보스 테마 연결.

패턴:

```text
화면 가장자리 또는 보스 주변에 쿠폰 탄막을 배치한다.
짧은 예고 후 쿠폰이 안쪽으로 접혀 들어온다.
```

문구:

```text
오늘 계약하시면 가족 구성원이 하나 더 추가됩니다.
```

### 2. 깨끗한 거실

역할:

- 위치 선정 방해.
- 홈케어 자동화가 역할에 갇힌 존재라는 느낌 제공.

패턴:

```text
홈케어 청소기를 소환하고 원형 흡입 장판을 만든다.
장판은 플레이어를 약하게 끌어당기며 드롭 아이템도 흔든다.
```

문구:

```text
어제의 먼지와 슬픔을 한 번에 정리해드립니다.
```

### 3. 기념사진 시간

역할:

- 보스 정체성 핵심.
- 기억 추출 선택과 연결.

패턴:

```text
플레이어 위치에 큰 플래시 예고 범위를 만든다.
맞으면 짧은 둔화 또는 포즈 고정 상태가 걸린다.
```

문구:

```text
웃어주세요. 빠진 가족은 나중에 합성됩니다.
```

## 패턴 P1

### 추천 평면도

역할:

- 후반 압박.
- 캠페인이 도주 동선을 상품처럼 수정하는 감각.

패턴:

```text
바닥에 모델하우스 평면도 선이 그어지고 일부 구역이 잠깐 막힌다.
```

문구:

```text
고객님의 도주 동선을 가족 친화적으로 수정했습니다.
```

0.2 일정이 빠듯하면 생략한다. 단, 보스 잔향이나 0.3 강화 패턴으로 남길 수 있게 데이터 슬롯은 둔다.

## 페이즈

### 1페이즈: 입주 상담

- 오늘의 가족 특가 중심.
- 쿠폰 전단 떼 소환.
- 패턴 간격은 넉넉하게 둔다.

### 2페이즈: 집안 정리

- 깨끗한 거실 추가.
- 홈케어 청소기 소환.
- 안전 공간을 조금씩 흔든다.

### 3페이즈: 가족 완성

- 기념사진 시간 빈도 증가.
- 추천 평면도는 P1일 경우 여기서 추가.
- 캠페인이 윤서를 가족 구성원으로 부르려 한다.

## 윤서 연결

보스전에서 윤서의 차징 스킬은 반드시 의미 있어야 한다.

필수:

- 반품 도장이 보스 외피에 명확한 피해 피드백을 준다.
- 차징 명중 시 보스 화면/광고 문구가 깨지는 VFX가 있다.
- 차징이 무조건 정답은 아니지만, 잘 쓰면 페이즈 전환을 빠르게 만든다.

윤서 대사:

```text
전투 전: 집을 팔려고 가족을 만든 거야?
차징 명중: 반품.
결절 파괴 선택: 이제 적어도 저 집들은 누굴 흉내 내진 않겠지.
기억 추출 선택: 부서뜨리기 전에, 이름 하나라도 꺼내자.
```

## 보스 HP 성과 보상

보스 조우만으로 캠페인 코어 파편을 주지 않는다.

권장:

| 조건 | 보상 |
| --- | --- |
| 보스 조우만 함 | 코어 파편 0 |
| HP 65% 이하 첫 도달 | 캠페인 코어 파편 +1 |
| HP 25% 이하 첫 도달 | 캠페인 코어 파편 +1 |
| 보스 처리 성공 | 캠페인 코어 파편 +2 |

정산 문구:

```text
캠페인 코어 파편 +1
세븐: 표면이 아닙니다. 이번에는 안쪽을 건드렸습니다.
```

실패 문구:

```text
보스 외피 손상 기록이 남았습니다.
다음 송출 주기 분석이 가능해졌습니다.
```

## 0.2 보스 선택

0.2 선택지는 2개만 구현한다.

```text
destroy_node
extract_memory
```

보류:

```text
liberate_core -> 0.3 이후
reroute_signal -> 위험 보상 루트 준비 후
```

### 선택 A: 결절 파괴

표시명:

```text
결절 파괴
```

선택 설명:

```text
모델하우스 신호망을 끊어 외곽 주택가의 광고 농도를 낮춥니다.
```

상태 변화:

```text
r01_suburb_state.boss_outcome = "destroy_node"
r01_suburb_state.node_state = "aftershock"
r01_suburb_state.campaign_pressure -= 20
active_events += ["quiet_open_house", "hunter_elite_hint"]
```

보상:

```text
trace_nameplate_001 candidate or guaranteed
card_premium_silence_001 candidate
campaign_core_fragments +2
```

출격 게시판:

```text
외곽 주택가: 조용한 오픈하우스
```

NPC 반응:

```text
미나: 잘했어. 저 집들, 적어도 이제 팔리진 않겠지.
복희: 깨진 것도 기억이란다. 다만, 못 듣게 된 이름도 있겠지.
```

다음 런 변화:

- 적 밀도 소폭 감소.
- 추적형 엘리트 또는 조용한 감시 이벤트 증가.
- 문패 회수 목표 증가.

### 선택 B: 기억 추출

표시명:

```text
기억 추출
```

선택 설명:

```text
보스가 붙잡은 가족사진과 문패 로그를 흔적으로 꺼냅니다.
```

상태 변화:

```text
r01_suburb_state.boss_outcome = "extract_memory"
r01_suburb_state.node_state = "memory_afterimage"
r01_suburb_state.campaign_pressure -= 8
active_events += ["family_photo_afterimage", "fake_memory_wave"]
```

보상:

```text
trace_burnt_family_photo_001 guaranteed
trace_nameless_receipt_001 candidate
card_family_bundle_001 candidate
campaign_core_fragments +2
```

출격 게시판:

```text
외곽 주택가: 보정된 가족사진
```

NPC 반응:

```text
복희: 이건 버리면 안 된다. 아픈 이름도 이름이야.
세븐: 기억 추출 완료. 일부 데이터는 가족이 아니라 광고입니다.
```

다음 런 변화:

- 가족사진 잔상 이벤트 증가.
- 흔적 회수 목표 증가.
- 가짜 기억/환각형 장판 위험 증가.

## 전투 전후 문구

### 등장

```text
어서 오세요. 혼자 오셨군요. 걱정 마세요, 가족은 추가 가능합니다.
이 동네는 비어 있지 않습니다. 아직 팔리지 않았을 뿐입니다.
```

### 미나 무전

```text
저 목소리... 모델하우스에서 들리던 거야. 듣지 마. 친절한 척하는 가격표야.
```

### 팝시 씨앗 반응

```text
저 말투, 저장된 홍보 음성과 비슷해요. 그런데... 너무 오래 켜져 있었어요.
```

### 처치 직후

```text
가족 구성이 완료되지 않았습니다.
행복 보증을... 갱신하시겠습니까?
```

## 보급소 연결

보스 처리 후 필수:

- 정산 카운터에 코어 파편 표시.
- 출격 게시판 부제 변경.
- NPC 반응 1~2개 출력.
- 이름 보관함 또는 코어 파편 유리병 변화.
- 다음 재방문 목표 1개 제시.

선택 후 NPC 반응은 2개 이하로 제한한다. 대사가 많으면 전투 승리의 속도가 죽는다.

## 재방문 연결

결절 파괴 후:

```text
재방문 목표: 문패 회수
지역 문구: 처리된 결절 주변에 잔향이 남았습니다.
위험: 추적형 엘리트
```

기억 추출 후:

```text
재방문 목표: 가족사진 잔상 정리
지역 문구: 꺼진 TV가 아직 가족을 부릅니다.
위험: 가짜 기억 이벤트
```

공통:

```text
송출관 잔향 배너 낮은 확률 등장
보스 패턴 일부가 환경 이벤트로 재사용 가능
```

## 유저 선호 점수

예상 점수:

| 기준 | 점수 | 이유 |
| --- | ---: | --- |
| 첫 보스 인상 | 8 | 가족/광고/집 테마가 명확함 |
| 전투 판독성 | 8 | 쿠폰/흡입/플래시 패턴이 읽기 쉬움 |
| 윤서 손맛 | 8 | 반품 도장이 보스 외피 파괴와 연결됨 |
| 선택 후 만족 | 8 | 결절 파괴/기억 추출 차이가 분명함 |
| 재방문 동기 | 8 | 보스 후에도 지역이 끝나지 않음 |
| 구현 현실성 | 7 | 패턴 3개와 선택 2개는 가능하지만 연출 부담 있음 |

종합:

```text
8.0 / 10
```

10점으로 올리기 위한 다음 개선:

- 보스 전용 사운드/광고 징글 추가.
- 차징 명중 시 화면 문구가 깨지는 전용 피드백.
- 보스 선택 후 다음 런 첫 30초 안에 지역 변화가 보이게 조정.
- 0.3에서 코어 해방을 팝시-제로 서사와 함께 추가.

## 검증 항목

| 관찰 | 합격 기준 | 실패 시 수정 |
| --- | --- | --- |
| 보스 정체 이해 | 유저가 가족 상품화 결절로 인식 | 등장 문구/아레나 오브젝트 강화 |
| 패턴 판독 | 모바일에서도 피할 수 있음 | 예고 범위/속도 조정 |
| 차징 만족 | 반품 도장이 보스전에서 시원함 | 외피 파괴 피드백 강화 |
| 실패 보상 납득 | HP 65/25 성과 보상이 이해됨 | 정산 문구 개선 |
| 선택 차이 | 다음 런에서 변화가 보임 | 게시판/적/흔적 목표 강화 |

## 금지 방향

- 보스를 악한 로봇 엄마로 단순화하지 않는다.
- 가족 상실을 긴 컷신으로 울리지 않는다.
- 보스 조우만으로 코어 파편을 주지 않는다.
- 첫 보스 선택지를 3개 이상 열지 않는다.
- 보스 처치 후 외곽 주택가를 완전히 정화하지 않는다.
- 차징 방해나 보스 패턴이 윤서의 핵심 재미를 빼앗게 하지 않는다.
- 캠페인 송출관을 캠페인 전체의 최종 정체처럼 확정하지 않는다.
