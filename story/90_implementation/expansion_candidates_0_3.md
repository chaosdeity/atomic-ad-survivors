# Expansion Candidates 0.3

상태: 0.3 캐릭터/로컬 확장 후보 정리  
연결 문서: `story/05_progression/local_ecology_roadmap.md`, `story/06_characters/player_characters.md`, `story/90_implementation/character_score_upgrade_backlog.md`, `story/90_implementation/playtest_observation_0_2.md`

## 목적

이 문서는 0.2가 핵심 루프 검증에 성공했다는 전제에서, 0.3에 무엇을 추가하면 유저가 더 오래 남고 캐릭터에 더 강하게 애착을 갖는지 정리한다.

기준:

```text
0.3은 더 크게 만드는 버전이 아니라, 더 오래 하고 싶게 만드는 버전이어야 한다.
```

0.3의 핵심은 세 가지다.

- 윤서를 낡게 두지 않는다.
- 팝시-제로를 첫 확장 캐릭터 후보로 구체화한다.
- 두 번째 로컬은 폐쇼핑몰을 우선 검토한다.

## 결론

0.3 권장 조합:

```text
윤서 8.5 개선
팝시-제로 출시 준비 또는 출시
폐쇼핑몰 관측층/프로토타입 추가
외곽 주택가 P1 흔적과 재방문 변형 보강
```

0.3에서 피할 것:

```text
로컬 2개 동시 추가
플레이어블 2명 동시 추가
송출관 실제 보스전
신호 우회 본격 루트
최종 엔딩
```

## 후보 평가

| 후보 | 유저 매력 | 수익성 | 구현 위험 | 세계 확장 | 추천 |
| --- | ---: | ---: | ---: | ---: | --- |
| 윤서 8.5 개선 | 9 | 6 | 3 | 7 | 필수 |
| 팝시-제로 | 9 | 9 | 6 | 8 | 최우선 |
| 폐쇼핑몰 | 8 | 8 | 6 | 8 | 최우선 로컬 |
| 로봇 서비스센터 | 8 | 7 | 7 | 9 | 0.4 이후 |
| 나리 | 7 | 6 | 5 | 8 | 0.4 이후 |
| 고위험 역송출형 | 8 | 8 | 8 | 7 | 후순위 |
| 방송국 | 7 | 6 | 8 | 9 | 후반 |

종합 추천:

```text
0.3 = 윤서 보강 + 팝시-제로 + 폐쇼핑몰
```

## 왜 폐쇼핑몰인가

폐쇼핑몰은 0.3에 가장 적합한 두 번째 로컬이다.

이유:

- 광고 디스토피아의 재미가 즉시 보인다.
- 카드/보상/소비 유혹과 직접 연결된다.
- 팝시-제로의 마스코트/리콜/상품 자아와 잘 맞는다.
- 외곽 주택가와 톤이 다르다.
- 유저가 “더 많은 보상”을 원할 때 자연스럽게 들어간다.

한 줄:

```text
폐쇼핑몰은 선택과 혜택을 끝없이 복제하는 실내 광고 기후다.
```

0.3에서 보여줄 질문:

- 보상과 유혹은 어디서 갈라지는가?
- 좋은 카드를 얻는 것이 항상 좋은 선택인가?
- 상품으로 만들어진 로봇이 자기 이름을 찾을 수 있는가?
- 유저는 강한 혜택 앞에서 얼마나 찝찝함을 감수하는가?

## 왜 서비스센터는 0.4 이후인가

로봇 서비스센터는 강력하지만, 0.3 첫 확장으로는 조금 이르다.

이유:

- 로봇 자율성 이야기를 정면으로 다루기 때문에 설명량이 많다.
- 팝시-제로가 먼저 유저에게 사랑받아야 서비스센터의 리콜 공포가 강해진다.
- 서비스센터를 먼저 열면 로봇 세계의 미스터리를 너무 빨리 설명하게 될 위험이 있다.

권장:

```text
0.3에서 팝시-제로와 마스코트 흔적을 심는다.
0.4에서 로봇 서비스센터를 연다.
```

## 0.3 캐릭터 후보

### 1순위: 팝시-제로

현재 점수:

```text
8.3 / 10
```

0.3 목표:

```text
8.8 / 10
```

유저가 좋아할 이유:

- 귀엽다.
- 윤서와 전투 리듬이 다르다.
- 로봇 자아와 리콜 불안이 감정적으로 강하다.
- 스킨/대사/이펙트 확장이 매우 좋다.
- 폐쇼핑몰과 테마 궁합이 좋다.

0.3 전투 판타지:

```text
광고 명령을 읽고, 귀엽게 고장 내고, 자기가 한 말이 광고인지 계속 의심하는 해방 마스코트.
```

차징 후보:

```text
상쾌한 오작동
주변 적 명령을 잠깐 꼬이게 하고, 일부 드롭 또는 카드 선택지를 변조한다.
```

0.3 필수 작업:

| 작업 | 이유 |
| --- | --- |
| 전투 판타지 확정 | 윤서와 차별화 |
| 차징 손맛 프로토타입 | 귀여움만으로는 부족 |
| 리콜 불안 대사 5개 | 장기 애정 |
| 마스코트 배지 반응 | 0.2 흔적 회수 |
| 성능 필수화 방지 | 수익 신뢰 |

금지:

- 리롤/드롭 변환을 필수 성능으로 만들지 않는다.
- 팝시를 로봇 진영의 정답처럼 만들지 않는다.
- 귀여움만 남기고 상처를 장식으로 쓰지 않는다.

### 2순위: 나리

현재 점수:

```text
7.2 / 10
```

0.3 권장:

```text
출시하지 말고 전투 손맛 프로토타입만 준비
```

이유:

- 흔적/이름 시스템이 더 쌓인 뒤 강해진다.
- 지금 출시하면 조용하고 안전한 캐릭터로 오해될 수 있다.
- 0.3에서는 나리의 자리를 이름 보관함, 가족사진, 폐쇼핑몰 영수증 흔적으로 예고한다.

### 3순위: 고위험 역송출형

현재 점수:

```text
7.5 / 10
```

0.3 권장:

```text
출시하지 않는다.
도윤의 위험 개조와 캠페인 위험 카드로 씨앗만 남긴다.
```

이유:

- 너무 빨리 내면 성능 과금 캐릭터로 오해될 수 있다.
- 신호 우회 루트가 아직 본격화되지 않았다.
- 0.3에서는 폐쇼핑몰 위험 카드가 이 캐릭터의 미래를 예고하면 충분하다.

## 0.3 로컬 후보

### 1순위: 폐쇼핑몰

0.3 목표:

```text
새 전체 지역 완성 또는 관측층 프로토타입
```

권장 범위:

| 범주 | 0.3 권장 |
| --- | --- |
| 지역 | 폐쇼핑몰 일부 구역 |
| 법칙 | 영원한 할인과 선택 피로 |
| 적 | 쇼핑 카트, 쿠폰 계산대, 시식 풍선 |
| 엘리트 | 오늘만 매니저 |
| 흔적 | 이름 없는 회원카드, 반품 영수증, 고장 난 마스코트 손 |
| 보스 | 0.3에서는 전조만 가능 |
| 팝시 연결 | 마스코트/리콜/상품명 대사 |

0.3에서 꼭 필요한 감정:

```text
보상이 좋아 보이는데, 너무 친절해서 불안하다.
```

폐쇼핑몰이 0.3에 주는 시스템 이점:

- 카드 풀 확장.
- 위험 보상 실험.
- 팝시-제로 해금/출시 근거.
- 광고 전단 조각 사용처 확장.
- 신호 우회 루트의 전조.

### 2순위: 외곽 주택가 새 관측층

0.3 목표:

```text
0.2 지역을 버리지 않고 계속 살아있게 만든다.
```

추가 후보:

- 무음 라디오 관측층.
- 가족사진 잔향층.
- 단골 고객층.
- 고장 난 마스코트 배지 반응.
- 행복 보증 상담원 변종.

이점:

- 새 지역 없이도 반복 재미 증가.
- 0.2 유저 선택을 0.3에서 다시 사용.
- 윤서와 팝시의 반응을 모두 붙일 수 있다.

권장:

```text
폐쇼핑몰이 너무 크면, 0.3은 외곽 주택가 관측층 + 팝시-제로 출시 준비로 줄인다.
```

### 3순위: 로봇 서비스센터

0.3 권장:

```text
직접 지역 구현은 보류.
팝시 대사와 리콜 문구로 예고.
```

예고 문구:

```text
팝시-제로: 제 리콜 번호가 아직 살아 있어요. 이상하죠? 저는 여기 있는데.
세븐: 서비스센터 신호가 대답했습니다. 좋지 않은 방식입니다.
```

## 0.3 추천 패키지

### 패키지 A: 안정 추천

```text
윤서 8.5 개선
외곽 주택가 P1 흔적 3개 추가
팝시-제로 NPC/해금 씨앗 강화
폐쇼핑몰 전조/관측층만 추가
```

장점:

- 구현 위험 낮음.
- 0.2를 버리지 않는다.
- 팝시-제로 출시 전 반응을 볼 수 있다.

단점:

- 새 캐릭터/새 지역을 기대한 유저에게 약할 수 있다.

추천 점수:

```text
8.2 / 10
```

### 패키지 B: 수익 추천

```text
윤서 8.5 개선
팝시-제로 플레이어블 출시
폐쇼핑몰 일부 구역 추가
팝시 전용 스킨/대사팩 후보 설계
```

장점:

- 캐릭터 수익성과 신선함이 강하다.
- 윤서와 팝시의 대비가 좋다.
- 폐쇼핑몰과 팝시 테마 궁합이 좋다.

단점:

- 팝시 성능 필수화 위험.
- 캐릭터+지역 동시 추가라 구현 부담이 있다.

추천 점수:

```text
8.6 / 10
```

### 패키지 C: 세계관 추천

```text
윤서 8.5 개선
외곽 주택가 새 관측층
로봇 서비스센터 전조
팝시-제로 출시 준비
```

장점:

- 세계관 깊이가 좋아진다.
- 로봇 미스터리를 서서히 연다.
- 0.4 서비스센터로 자연스럽게 이어진다.

단점:

- 새로움이 문구/전조에 치우칠 수 있다.
- 액션 유저에게는 약할 수 있다.

추천 점수:

```text
8.0 / 10
```

최종 추천:

```text
패키지 B를 목표로 잡되, 일정이 위험하면 패키지 A로 축소한다.
```

## 0.3 필수 검증

0.3을 만들기 전에 0.2 플레이테스트에서 반드시 확인해야 할 것:

| 0.2 결과 | 0.3 결정 |
| --- | --- |
| 윤서 8점 이상 | 팝시-제로 출시 검토 가능 |
| 윤서 7점 이하 | 새 캐릭터보다 윤서 보강 우선 |
| 보상 루프 8점 이상 | 폐쇼핑몰 위험 보상 실험 가능 |
| 보상 루프 7점 이하 | 새 로컬보다 성장/정산 수정 |
| 외곽 주택가 재방문률 높음 | 새 관측층 추가 가치 높음 |
| 재방문률 낮음 | 두 번째 로컬보다 재방문 이유 수정 |
| 유저가 팝시 씨앗에 반응 | 팝시-제로 출시 우선 |
| 팝시 반응 약함 | 마스코트 배지/로봇 대사 보강 후 출시 |

## 금지 방향

- 0.2가 검증되기 전에 팝시-제로를 성급히 출시하지 않는다.
- 윤서를 낡은 무료 기본캐로 두지 않는다.
- 폐쇼핑몰을 단순 아이템 상점 지역으로 만들지 않는다.
- 로봇 서비스센터에서 로봇 세계의 정체를 한 번에 설명하지 않는다.
- 0.3에서 신호 우회 루트를 본격 해금하지 않는다.
- 새 캐릭터가 없으면 손해라는 압박을 만들지 않는다.
- 캐릭터 점수를 유저 UI에 보여주지 않는다.
