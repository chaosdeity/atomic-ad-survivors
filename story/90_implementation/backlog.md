# Story Backlog

이 문서는 `Atomic Ad Survivors` 스토리 작업의 실제 진행표다. 체크가 끝난 항목은 해당 문서에 반영된 상태를 뜻한다.

## 진행 원칙

- 한 번에 한 문서씩 완성한다.
- 완성 기준은 "바로 구현 작업 지시서로 바꿀 수 있는가"다.
- 장문 설정보다 UI 문구, NPC 한 줄 대사, 런 목표, 보스 선택처럼 게임 안에 들어갈 단위를 우선한다.
- 큰 방향이 바뀌면 `story/00_source/`에 새 버전을 만들고, 작은 확장은 각 문서에 반영한다.
- 현재 단계에서는 Godot 코드 구현을 하지 않는다. 설정, 스토리, 지역, 아이템, 카드, 엔딩 문서를 먼저 마무리한다.
- 구현 작업 지시서는 모든 핵심 스토리 문서가 닫힌 뒤 마지막에 작성한다.
