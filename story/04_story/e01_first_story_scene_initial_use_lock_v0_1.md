## 5. 사용 가능 범위

| 사용 가능 범위 | 허용 방식 | 제한 |
|---|---|---|
| 내부 스토리 연결 검토 | 6개 장면 사이의 의문, 흐름, 연결감을 검토 | 새 장면 자동 확장 금지 |
| E01 장면 순서 조정 | `scene_e01_001`~`scene_e01_006`의 순서와 기능 배치를 검토 | 장면 목적 훼손 금지 |
| 장면별 플레이 감각 검토 | 윤서 단독 U5의 이동감, 불안, 발견 순서를 검토 | 새 플레이어블 추가 금지 |
| 오브젝트/잔향 배치 후보 검토 | 패널, 표지, 서류함, 게이트, 방송, 잔향의 후보 맥락을 검토 | 실제 좌표, 실제 씬 파일, 실제 오디오 파일 확정 금지 |
| 후속 E01 장면 설계 참조 | 후속 E01 장면의 톤과 정보량 기준으로 참조 | 새 NPC 이름, 직접 대면 NPC 확정 금지 |
| 전역 240개 항목 참조 | 비대면 오브젝트/문구/방송/잔향 후보로만 참조 | 1,856개 전체 문구 작성 또는 직접 대면 전환 금지 |

사용 가능한 수량 경계:

| 수량 | 사용 해석 |
|---|---|
| 120 | E01 NPC 120 고정 구조 안에서만 참조 |
| 2,048 | 전역 NPC 2,048 슬롯 구조는 변경하지 않음 |
| 192 | L5/L4 192개는 직접 대면 후보 참조선만 유지 |
| 1,856 | L3 이하 1,856개 전체 문구 작성 금지 |
| 240 | 현재 실제 사용 가능한 전역 JSON/핸드오프 항목 범위 |
| 10,240 | 장기 목표이며 현재 전체 착수 대상 아님 |

## 6. 아직 금지된 확장

| 금지 항목 | 잠금 상태 | 이유 |
|---|---|---|
| 새 장면 자동 확장 | 금지 | 초기 사용 가능 범위는 6개 장면으로 고정 |
| 새 캐릭터 이름 추가 | 금지 | 새 인물명 없음 상태 유지 |
| 새 플레이어블 추가 | 금지 | 윤서 단독 U5 시작 구조 유지 |
| 직접 대면 NPC 확정 | 금지 | 전역 240개 항목은 비대면 오브젝트/문구/방송/잔향으로만 사용 |
| L5/L4 승격 | 금지 | L5/L4 192개 상한과 직접 대면 후보 참조선 유지 |
| 미개방 캠페인 상세 결재 | 금지 | E02 이후는 연결 준비 또는 암시 범위만 허용 |
| 1,856개 전체 문구 작성 | 금지 | 현재 실제 범위는 240개 항목으로 제한 |
| 10,240 전체 착수 | 금지 | 장기 풀스케일 목표이며 현재 제작 대상 아님 |
| 구현 파일 생성 | 금지 | 실제 엔진/레벨/UI/오디오 구현 착수 전 단계 |
| 실제 좌표 확정 | 금지 | 오브젝트/잔향 배치 후보 검토까지만 허용 |
| 실제 씬 파일 생성 | 금지 | 레벨 구현 산출물 생성 금지 |
| 실제 UI 파일 생성 | 금지 | UI 구현 산출물 생성 금지 |
| 실제 오디오 파일 생성 | 금지 | 오디오 구현 산출물 생성 금지 |
| Godot 리소스 생성 | 금지 | 엔진 리소스 변환 전 단계 |
| GDScript/로더 코드 생성 | 금지 | 런타임 직결 코드 작성 금지 |

금지선 판정: 초기 사용 가능은 스토리 검토 기준 본문 잠금이며, 제작 범위 확장이나 구현 착수를 의미하지 않는다.
