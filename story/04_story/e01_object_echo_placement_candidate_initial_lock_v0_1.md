## 5. 사용 가능 범위

| 사용 가능 범위 | 허용 방식 | 제한 |
|---|---|---|
| E01 장면별 오브젝트/잔향 후보 참고 | 6개 장면별 후보 수와 story_function을 참조 | 실제 레벨 배치 확정 금지 |
| 후속 레벨/서사 연결 검토 기준 | 패널, 표지, 호출번호, 방송, 잔향의 후보 역할을 검토 | 실제 좌표, 씬 파일, UI 파일, 오디오 파일 확정 금지 |
| 비대면 문구/방송/잔향 배치 후보 기준 | 전역 240개 중 E01 16개를 비대면 후보로만 사용 | 직접 대면 NPC 확정 금지 |
| 장면 정보량 조절 참고 | 장면별 후보 수와 정보량을 비교해 과밀 여부를 검토 | 새 본문 확장, 새 장면 추가 금지 |
| 후속 QA 기준 | E01 16개 ticket/sample과 `candidate_only` 상태를 기준으로 재검산 | 240개 외 항목 추가 금지 |

사용 가능 판정: 후보는 장면 검토와 후속 연결 검토에 사용할 수 있으나, 실제 구현 산출물로 전환하려면 별도 승인과 QA가 필요하다.

## 6. 아직 금지된 확정

| 금지 항목 | 잠금 상태 | 이유 |
|---|---|---|
| 실제 레벨 배치 확정 금지 | 금지 | 이번 문서는 후보 잠금이며 실제 배치 문서가 아님 |
| 실제 좌표 확정 금지 | 금지 | placement_hint만 유지, 좌표 없음 |
| 새 오브젝트 생성 확정 금지 | 금지 | suggested_surface는 후보 표면일 뿐 생성 확정 아님 |
| 새 NPC 이름 추가 금지 | 금지 | 윤서 외 신규 고유 인명 없음 |
| 직접 대면 이벤트 추가 금지 | 금지 | 모든 후보는 비대면 오브젝트/문구/방송/잔향 범위 |
| 새 플레이어블 추가 금지 | 금지 | 윤서 단독 U5 유지 |
| L5/L4 승격 금지 | 금지 | E01 16개는 L3 이하 후보로만 유지 |
| 구현 파일 생성 금지 | 금지 | 엔진/레벨/UI/오디오 파일 생성 없음 |
| 실제 씬 파일 생성 금지 | 금지 | `.tscn` 등 레벨 산출물 생성 없음 |
| 실제 UI 파일 생성 금지 | 금지 | 호출번호는 UI 파일이 아니라 후보 표면 |
| 실제 오디오 파일 생성 금지 | 금지 | 잔향은 오디오 파일이 아니라 후보 표면 |
| Godot 리소스 생성 금지 | 금지 | 리소스 변환 전 단계 유지 |
| GDScript/로더 코드 생성 금지 | 금지 | 런타임 직결 코드 없음 |
| 240개 외 항목 추가 금지 | 금지 | E01 16개만 초기 후보로 잠금 |
| 1,856개 전체 문구 작성 금지 | 금지 | 전체 작성으로 확장하지 않음 |

금지선 판정: 초기 후보 잠금은 후보 참조 가능 상태의 확정이며, 실제 배치/구현/신규 제작 확정이 아니다.
