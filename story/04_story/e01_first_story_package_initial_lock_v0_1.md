## 7. 사용 가능 범위

| 사용 가능 범위 | 허용 방식 | 제한 |
|---|---|---|
| E01 내부 스토리 연결 기준 | 6개 장면의 의문, 단서, 보류점을 연결하는 기준으로 사용 | 새 장면 자동 확장 금지 |
| 후속 E01 장면 설계 기준 | 후속 장면의 위치, 톤, 정보량 판단 기준으로 참조 | 새 본문 확장 금지 |
| 장면별 플레이 감각 검토 기준 | 윤서 단독 U5의 관찰, 확인, 지연, 되짚기 감각 검토 | 새 플레이어블 추가 금지 |
| 오브젝트/잔향 후보 배치 검토 기준 | E01 16개 후보의 장면별 적합성 참고 | 실제 배치/좌표 확정 금지 |
| 후속 제작팀 전달용 기준 패키지 | 본문 v0.2, 순서 잠금, 후보 잠금을 함께 전달 | 구현 파일 생성 금지 |

패키지 사용 판정: 후속 팀은 이 문서를 기준 패키지로 참조할 수 있으나, 실제 구현이나 신규 제작 확정에는 별도 승인과 QA가 필요하다.
