## 0. 판정 요약

판정:

```text
통과로 승격 가능
```

MODEL-15의 조건부 사유는 이번 문서에서 해소 가능하다고 본다.

마야는 리뷰어, 인플루언서, 해커, 방송인, 알고리즘 천재가 아니다. 마야는 E10_C05 알고리즘 추천권에서 추천 모델이 자신을 `학습 완료 샘플`과 `광고 타깃`으로 고정하기 전, 입력 데이터와 반응 판정을 늦추는 플레이어블 후보다.

핵심 질문은 유지한다.

```text
사람이 추천되지 않으면 자유로운가,
아니면 학습되지 않은 데이터로 다시 붙잡히는가?
```

핵심 문장은 아래로 고정한다.

```text
나를 맞힌 추천은,
나를 이해한 것이 아닙니다.
```

이번 문서에서 잠근 것:

1. REVIEW-15 legacy를 폐기하고 MODEL-15로 코드와 역할을 고정한다.
2. 리뷰어/인플루언서/해커/방송인화를 차단한다.
3. E10_C05-L01~L02만 직접 노출하고 E10 전체 확장을 금지한다.
4. BROAD-10, CLAUSE-11, LEDGER-12, OBS-20과의 절차 차이를 고정한다.
5. HP 0을 `모델 학습 완료`, `추천 프로필 고정`, `광고 타깃 확정`으로 고정한다.
6. 정산 루트를 `미학습 로그`, `추천 프로필 조각`, `광고 타깃 보류표` 중심으로 잠근다.
7. Maya Chen 이름은 유지하되 원어/현지 표기는 별도 검수로 보류한다.

## 2. 리뷰어/인플루언서/해커화 차단

마야의 핵심은 추천을 평가하는 능력이 아니다.

마야는 추천을 잘 맞히거나, 추천을 비판하거나, 피드를 장악하는 사람이 아니다. 마야는 추천 모델이 자기 반응을 완성된 프로필로 닫기 전, 모델이 원하는 확정값을 일부러 주지 않으면서도 보급소가 회수할 수 있는 불완전 로그를 남기는 후보다.


## 5. 전투 리듬 고정

기본 리듬:

```text
추천 / 노출 / 반응 / 미학습 / 타깃 확정 판정을 밀고 당긴다.
```

마야의 능력은 빠른 이동, 방송, 해킹, 정보 공개가 아니다. 마야의 능력은 추천 모델이 원하는 확정 반응을 주지 않는 대신, 보급소가 인양할 수 있을 만큼의 불완전 로그를 남기는 것이다.

고정 기능:

| 기능 | 의미 | 실패 위험 |
|---|---|---|
| 추천 숨김 | 특정 추천 노출과 캠페인 호출을 한 박자 늦춘다 | 숨김 행동 자체가 반응 데이터가 된다 |
| 미학습 로그 남기기 | 보급소가 마야의 좌표를 잡을 불완전 로그를 남긴다 | 로그가 누적되면 모델이 더 정확히 추론한다 |
| 오답 추천 유지 | 모델의 확정값을 일부러 틀린 상태로 둔다 | 오답도 교정 데이터로 흡수된다 |
| 반응률 흐림 | 웨이브 호출과 광고 타깃 우선순위를 흔든다 | 보급소도 마야를 놓칠 수 있다 |
| 타깃 확정 지연 | 광고 타깃/관심사 클러스터 고정을 늦춘다 | 지연이 길면 캠페인 농도가 상승한다 |

전투 감각:

```text
클릭하지 않는 것이 안전한 선택이 아니다.
무반응도 반응으로 읽히기 때문에,
마야는 모델과 보급소 사이에 일부러 불완전한 흔적을 남긴다.
```

HP 0 상태명:

```text

```


HP 0 흐름:

| 단계 | 상태 | 의미 |
|---:|---|---|
| 1 | 추천 프로필 병합 | 흩어진 추천 프로필 조각이 하나로 붙는다 |
| 2 | 반응률 확정 | 무반응과 숨김 행동까지 반응률로 계산된다 |

| 4 | 모델 학습 완료 | 마야가 학습 완료 샘플로 닫힌다 |
| 5 | 긴급 인양 | 보급소가 미학습 로그와 광고 타깃 보류표의 불일치를 잡아 되당긴다 |

