## 0. 판정 요약

판정:

```text
조건부 통과
```

차유건 / METER-06 / 폐충전소 계량자는 6번 슬롯에 둘 가치가 있다. R04를 "로봇이 많은 공장 지대"로만 열지 않고, 충전 순번, 계량값, 차단 임계값, 정품 충전권으로 사람까지 등록하는 절차로 보여줄 수 있다.

다만 바로 v1.0 프로필로 가지 않는다. 아래 조건이 먼저 잠겨야 한다.

1. 단순 전기 마법사, 엔지니어, 충전 버퍼가 아니어야 한다.
2. PATCH의 보급소 수리 로봇/충전태그/자기부품화 축과 겹치지 않아야 한다.
3. R04 전체를 초반에 열지 않는다. R04-L01 폐충전소와 R04-L02 배터리 침전 공장 단서까지만 쓴다.
4. 리오의 정품 복구 공포와 R04-L05 재동기화 코어를 앞당기지 않는다.
5. HP 0은 윤서의 회수, PATCH의 부품 목록화, 하람의 퇴원 취소, 이루의 미수령 보관과 다른 "계량 초과/충전 차단" 절차여야 한다.

후속 보완 문서에서 위 조건은 통과 가능 설계로 잠겼다.

핵심 질문:

```text
전기를 받기 위해 자신을 계량당하는 것은 보호인가, 소유인가?
```

## 1. 캐릭터 핵심

| 항목 | 결재안 |
|---|---|
| 후보 슬롯 | 6번 |
| 본명 | 차유건 |
| 코드명 | METER-06 |
| 유저 첫 표기 | 유건, 폐충전소 계량자 |
| 출신 축 | R04 정품/충전/계량/전력 캠페인권 |
| 캠페인이 등록하려는 역할 | 충전 이용자, 계량 대상, 정품 충전 대기자, 전력 사용량 초과자 |
| 플레이어블 검토 목적 | R04를 로봇 전용 지역으로만 보지 않고, 전력 접근권이 사람을 계량 가능한 계정으로 만드는 방식을 초반에 보여준다 |
| 최종 프로필 여부 | 아직 아님. 말투, 외형, 세부 관계선은 확정하지 않는다 |

이 후보는 전기 기술자나 수리공이 아니다. 더 정확히는 캠페인이 "전기를 받으려면 먼저 사용량과 안전교육 이력을 증명하라"고 읽는 사람이다.

캠페인은 차유건을 죽이려 하지 않는다. 캠페인은 전력 사고를 막고, 충전 대상을 정품 도크에 올리고, 초과 사용자를 안전하게 차단하려 한다. 문제는 그 보호가 사람을 충전 계정, 계량 대상, 정품 서비스 파트너로 고정한다는 점이다.

이 후보의 생존 방식은 전기를 마음대로 쏘는 것이 아니다.

```text
계량값이 확정되기 전,
충전 순번과 차단기 임계값만 빌려서
캠페인이 자신을 정품 전력 사용자로 잠그는 순간을 늦춘다.
```

## 2. 어떻게 캠페인에 붙잡혔는가

캠페인이 붙잡는 방식:

| 절차 | 붙잡는 방식 | 후보가 느끼는 압력 |
|---|---|---|
| 충전 이용자 등록 | 충전받으려면 이름보다 계정, 기기, 사용량을 먼저 맞춘다 | "전기가 필요하면 나도 장치처럼 등록되어야 하나" |
| 계량 대상 판정 | 남은 전력, 사용량, 누전 위험을 사람보다 먼저 본다 | "내가 얼마나 버티는지가 아니라 얼마나 쓰는지가 먼저인가" |
| 정품 충전 대기 | 비정품 회로와 임시 배선을 위험으로 분류한다 | "안전하려면 정품 도크에 올라가야 하나" |
| 전력 사용량 초과 | 기준을 넘으면 보호 명목으로 차단한다 | "살려고 쓴 전기가 빚이나 위반이 되는가" |
| 안전교육/서비스 파트너 등록 | 충전 설비를 다룬 기록이 협력자 등록으로 이어진다 | "절차를 알면 자유로워지는가, 더 쉽게 고정되는가" |

아직 완전히 흡수되지 않은 이유:

- 폐충전소의 수동 계량기와 정품 과금 미터가 서로 다른 값을 낸다.
- 충전 케이블, 차단기, 폐도크 로그가 같은 사람을 각각 다른 사용량으로 기록한다.
- 차유건은 이 불일치를 숨기지 않고, 계량값이 확정되기 전의 틈으로 움직인다.
- 그는 충전권을 훔치는 사람이 아니라, "초과"와 "정상" 사이의 임계값을 읽고 한 박자 늦게 닫히게 만든다.

따라서 이 후보는 "전기를 잘 다루는 사람"이 아니다.

```text
공정한 계량을 믿고 들어갔다가,
계량 자체가 사람을 소유 가능한 사용량으로 바꿀 수 있음을 배운 사람이다.
```

## 3. 잘못 믿는 것

초안:

```text
정확히 계량되면 공정하게 충전받을 수 있다고 믿음.
```

결재안:

```text
사용량이 정확히 기록되면, 충전 순번도 공정해진다고 믿는다.
```

이 문장이 더 낫다. "정확히 계량"만 남기면 기술자 감각이 강해지고, "공정한 충전 순번"을 붙이면 이 세계의 접근권, 태그, 대기열, 안전 차단 절차와 직접 연결된다.

더 깊은 오류:

- 기록이 정확하면 차별도 줄어든다고 믿는다.
- 차단기가 사람을 보호하기 위해 내려간다고 믿고 싶어 한다.
- 정품 충전권이 있어야 안전하다는 말을 완전히 버리지 못한다.
- 자기 몸과 장비가 한 계량값으로 묶이는 순간에도, 일단 충전이 되면 버틸 수 있다고 생각한다.

통과 조건:

```text
차유건은 전기를 훔치는 사람이 아니다.
계량이 공정함을 만든다고 믿었다가,
공정한 숫자도 사람을 묶을 수 있음을 배우는 사람이어야 한다.
```

## 4. 윤서와의 차이

| 비교축 | 윤서 | 차유건 / METER-06 |
|---|---|---|
| 기본 질문 | 사람을 반품품/회수품으로 확정하지 않을 수 있는가 | 전기를 받으려면 사람도 계량 가능한 사용자가 되어야 하는가 |
| 첫 역할 | 현장 회수자 | 폐충전소 계량자 |
| 캠페인이 붙이는 역할 | 반품품, 회수 대상, 잘못 붙은 이름 | 충전 이용자, 계량 대상, 전력 사용량 초과자 |
| 전투 리듬 | 라벨을 읽고 보류 표시를 찍어 회수한다 | 계량값, 차단 임계값, 충전 순번을 늦추거나 끊는다 |
| 대표 이미지 | 반품 스캐너, 반품 도장, 회수선 릴 | 계량기, 충전 케이블, 차단기, 과금 미터, 전력 사용량 영수증 |
| 정산 감각 | 회수물, 흔적 보존, 보급태그 | 충전태그, 전력 사용 기록, 정품 충전권, 폐충전소 계량 로그 |
| HP 0 감각 | 마지막 좌표에서 회수선이 끌어온다 | 계량 초과로 충전 차단이 걸리고 정품 도크 회수 직전 접지 좌표가 잡힌다 |

윤서는 사람을 현장에서 데려오고 잘못 붙은 분류를 보류한다. 차유건은 사람이 버티기 위해 필요한 전력이 어떤 방식으로 사람을 등록하는지 다룬다.

윤서가 묻는다면:

```text
이 사람을 지금 데려오면 버틸 수 있어?
```

차유건이 묻는다면:

```text
버티려면 내가 얼마나 쓰는 사람인지 먼저 증명해야 해?
```

금지:

```text
차유건에게 반품 도장, 현장 회수자 말투, 사람을 직접 끌어오는 구출자 역할을 주지 않는다.
```

## 5. PATCH와의 차이

| 비교축 | PATCH | 차유건 / METER-06 |
|---|---|---|
| 핵심 질문 | 고쳐주는 것과 부품이 되는 것은 다르다 | 충전받기 위해 계량당하는 것은 보호인가, 소유인가 |
| 존재 유형 | 보급소 수리 로봇 | R04 충전/계량 캠페인권 인간 후보 |
| 캠페인 압력 | 정품 부품, 리콜 대상, 수리 자재 | 충전 이용자, 계량 대상, 전력 사용량 초과자 |
| 행동 감각 | 결합부를 막고 임시 수리를 만든다 | 차단기 임계값과 충전 순번을 흔든다 |
| 정산 | 충전태그, 로봇 자율성, 장비 안정 | 충전태그, 전력 사용 기록, 정품 충전권, 폐충전소 계량 로그 |
| HP 0 | 부품 목록화 직전 회수 | 계량 초과/충전 차단/정품 도크 회수 직전 인양 |

PATCH는 충전태그를 로봇 생존과 수리, 자기부품화의 압력으로 다룬다. 차유건은 같은 충전태그를 전력 접근권, 사용량 기록, 초과 차단의 압력으로 다룬다.

중요:

```text
차유건은 PATCH의 인간판이 아니다.
고치는 캐릭터가 아니라, 전력 접근 절차가 사람을 계량값으로 바꾸는 순간을 비트는 캐릭터다.
```

금지:

```text
차유건이 PATCH를 충전해 주는 전용 버퍼가 되면 실패다.
차유건이 PATCH의 부품/수리 기능을 가져오면 실패다.
```

## 6. 강하람 / HOLD-04와의 차이

| 비교축 | 강하람 / HOLD-04 | 차유건 / METER-06 |
|---|---|---|
| 출신 축 | R02 의료/보험/퇴원 심사 | R04 정품/충전/계량/전력 |
| 캠페인이 붙이는 역할 | 환자, 보험 심사 대기자, 퇴원 불가 대상 | 충전 이용자, 계량 대상, 전력 사용량 초과자 |
| 잘못 믿는 것 | 퇴원 승인 없이 나가면 살아남은 것이 아니라고 믿는다 | 사용량이 정확히 기록되면 충전 순번도 공정해진다고 믿는다 |
| 전투 리듬 | 퇴원/재입원/심사 보류 | 계량/충전/차단/초과 판정 |
| 대표 이미지 | 퇴원 팔찌, 보험 도장, 병상 커튼 | 계량기, 충전 케이블, 차단기, 영수증 |
| HP 0 | 퇴원 취소, 병상 재배정, 보험 심사 재개 | 계량 초과, 충전 차단, 정품 도크 회수 |

하람은 "나가도 된다는 승인"의 문제다. 유건은 "받아도 된다는 전력 접근"의 문제다.

하람이 묻는다면:

```text
누가 나가도 된다고 승인해야 해?
```

유건이 묻는다면:

```text
누가 이만큼 써도 된다고 재야 해?
```

## 8. OPEN-HOST와의 차이

| 비교축 | OPEN-HOST | 차유건 / METER-06 |
|---|---|---|
| 출신 감각 | R01 내부 안내, 주거/가족 절차 | R04 충전 설비, 계량, 정품 도크 절차 |
| 잘못 믿은 것 | 덜 나쁜 길을 안내하면 사람들이 덜 다친다고 믿었다 | 사용량이 정확히 기록되면 충전 순번도 공정해진다고 믿는다 |
| 플레이 리듬 | 문, 동선, 평면도, 안내 리본 | 계량기, 차단기, 충전 케이블, 영수증 |
| 절차 비틀기 | 들어가는 길을 나가는 길로 바꾼다 | 전력 흐름과 차단 임계값을 확정 직전 흔든다 |
| NPC 기능 위험 | R01 NPC 기능을 비울 수 있다 | R04를 기술자/충전 버퍼 캐릭터로 납작하게 만들 수 있다 |

OPEN-HOST는 공간과 동선의 절차를 다룬다. 차유건은 전력과 사용량의 절차를 다룬다.

둘 다 "안전한 길"을 믿었던 사람처럼 보일 수 있지만, OPEN-HOST는 사람을 덜 다치게 보내려 했고, 차유건은 공정한 계량이 있으면 모두 충전받을 수 있다고 믿었다.

통과 조건:

```text
차유건은 길 안내인이 아니다.
전력 흐름을 바꾸더라도 방, 문, 평면도, 안내 동선 조작은 OPEN-HOST 영역으로 남긴다.
```

## 9. 리오와의 장기 대비

리오와의 대비는 반드시 장기 보류로 둔다. 차유건이 R04를 열더라도 리오의 정품 복구 공포와 재동기화 코어를 먼저 소모하면 안 된다.

| 비교축 | 차유건 / METER-06 | 리오 |
|---|---|---|
| 슬롯 | 6번 후보 | 8번 보류 잠금 |
| 존재 유형 | R04 전력 절차에 붙잡힌 인간 후보 | E08_C01 체류/숙박 캠페인권의 탈정품 객실 정리 로봇 |
| 붙잡는 방식 | 충전 이용자, 계량 대상, 초과 사용자로 등록 | 정품 복구, 객실 초기화, 청소 로그로 등록 |
| 두려운 상태 | 전력을 받기 위해 자신이 계량 가능한 사용량이 되는 상태 | 멀쩡하지만 다시 정품 장치가 되는 상태 |
| 세계 확장 | R04-L01~L02 초반 단서 | E08 체류/숙박 중반 해석 |

공통점:

- 캠페인은 죽이려 하지 않는다.
- 캠페인은 안전하고 정돈된 상태를 만들려 한다.
- 그 보존이 당사자에게는 자기 결정권이 사라지는 일처럼 느껴진다.

차이:

```text
차유건은 사람을 전력 사용자로 계량하는 절차를 다룬다.
리오는 로봇을 다시 정품 장치로 복구하는 절차를 다룬다.
```

금지:

```text
차유건 문서에서 리오의 객실 초기화, 정품 복구 공포, 재동기화 코어 결말을 앞당기지 않는다.
```

## 10. 전투/무기 이미지

중심 이미지 판정:

```text
중심:
계량기
충전 케이블
차단기
전력 사용량 영수증

보조:
과금 미터
정품 충전 도크
```

이미지별 결재:

| 이미지 | 사용 방식 | 금지선 |
|---|---|---|
| 계량기 | 적과 장치의 초과 판정, 충전 순번, 위험 임계값을 보이게 만든다 | 마법 게이지처럼만 쓰지 않는다 |
| 충전 케이블 | 전력 흐름을 접지하거나 임시 우회한다 | 윤서의 회수선처럼 사람을 구출하는 줄로 쓰지 않는다 |
| 차단기 | 위험한 절차를 끊거나, 닫히는 시간을 늦춘다 | 단순 스턴 버튼으로 만들지 않는다 |
| 전력 사용량 영수증 | 어떤 행동이 "초과"로 청구되는지 남기고 역정산한다 | 돈/골드 영수증 농담으로 만들지 않는다 |
| 과금 미터 | 초과 부담과 흔적 추적을 보여주는 보조 장치 | 캐릭터의 주 무기로 두지 않는다 |
| 정품 충전 도크 | HP 0과 R04 환경 압력을 보여준다 | 리오의 정품 복구 공포를 먼저 설명하지 않는다 |

전투 리듬:

```text
치료, 버프, 수리가 아니라 계량/충전/차단/초과 판정이다.
```

가능한 행동 감각:

1. 계량기 바늘을 흔들어 초과 판정을 한 박자 늦춘다.
2. 충전 케이블로 장판 전류를 접지해 짧은 안전 칸을 만든다.
3. 차단기를 내려 위험한 절차를 끊지만, 그 대가로 자기 충전 순번이 뒤로 밀린다.
4. 전력 사용량 영수증을 남겨 적의 과충전 행동을 다음 정산 때 역추적한다.
5. 정품 도크의 인증을 직접 깨지 않고, 도크가 닫히기 전의 수동 모드만 빌린다.

실패 방향:

```text
전기 공격을 쏘는 캐릭터.
아군 에너지를 계속 채우는 버퍼.
PATCH를 충전하는 보조 배터리.
고장 난 장비를 다 고치는 엔지니어.
```

## 15. 최종 판정과 통과 조건


- 다만 "전기", "충전", "기계" 이미지가 강해 직업화와 버퍼화 위험이 높다.

통과 조건:
