## 0. 작성 이유

14~20번 전역 후보 교정은 한국어권 과밀을 낮추는 방향으로는 맞았다.
그러나 외국인 후보로 바꾸었다고 해서 캠페인 오픈 순서를 건너뛰어도 되는 것은 아니다.

플레이어블은 좋은 캐릭터 아이디어가 아니라, 열린 캠페인을 다르게 읽는 관점이다.
따라서 각 후보는 원점 캠페인의 NPC, 오브젝트, 상태 반응, 작은 결말이 먼저 깔린 뒤 상세 결재로 이동한다.

이번 문서는 아래 과속을 막기 위해 만든다.

1. 미개방 캠페인 후보를 상세 결재까지 밀기.
2. 상세 결재 뒤 바로 조건부 해소와 프로필 v1.0으로 고정하기.
3. 산출물 번호, 코드명 숫자, 최종 해금 순번을 같은 것으로 읽기.
4. 한국어권 과밀을 피했다는 이유로 전역 캠페인 개방 순서를 무시하기.

---

## 1. 보류 게이트 원칙

미개방 캠페인 후보는 아래 조건이 확인되기 전까지 상세 결재, 조건부 해소, 프로필 v1.0 작성으로 넘어가지 않는다.

1. 원점 캠페인이 실제 시즌 슬라이스에서 열렸거나 별도 승인되었다.
2. 해당 캠페인의 NPC, 오브젝트 흔적, 상태값 반응, 작은 결말이 먼저 존재한다.
3. 후보가 기존 플레이어블 또는 NPC 기능을 빼앗지 않는다.
4. 현실권/언어권/이름 기준이 해당 광역권과 맞는다.
5. HP 0은 전투 사망이 아니라 해당 캠페인의 제도적 판정으로 잡힌다.
6. 프로필 산출 번호, 코드명 숫자, 최종 해금 순번을 분리한다.

## 3. MODEL-15 정정 판정

Maya Chen / MODEL-15는 방향이 비교적 맞다.
E10_C05 알고리즘 추천권 후보로서 한국어권 과밀 문제를 피했고, 추천 모델 학습과 광고 타깃 확정 지연이라는 절차도 BROAD-10, CLAUSE-11과 분리된다.

단, 판정은 아래처럼 고정한다.

```text
통과 가능 / E10_C05 개방 후 후보
출시 초기 해금 금지
E10_C05 개방 전 최종 해금 고정 금지
프로필 산출물은 기능 초안으로 보존
```

MODEL-15는 `통과 가능`이지만, 그것은 지금 즉시 합류하거나 출시 초기에 등장한다는 뜻이 아니다.
E10_C05의 추천 피드, 모델 학습 큐, 광고 타깃 보류표가 실제 시즌 슬라이스에 깔린 뒤 직접 해금 후보로 다시 확인한다.

## 4. LOCKOUT-17 정정 판정

Omar Al-Hadid / LOCKOUT-17은 과속이다.
14~20 글로벌 이름/원점 재검토에서 이미 `캠페인 미개방으로 대기`였는데 상세 결재, 조건부 해소, 프로필 v1.0까지 밀렸다.

따라서 기존 산출물은 폐기하지 않고 아래처럼 재라벨링한다.

```text
기능 초안 / 보류 산출물
최종 프로필 v1.0 고정 아님
캠페인 미개방으로 대기
E06_C04 개방 확인 전 추가 상세화 금지
```

LOCKOUT-17의 안전잠금, 작업자 ID, 충전 접근 차단 이미지는 후보 기능으로 보존한다.
그러나 E06_C04 에너지 허브 로컬 슬라이스가 열리기 전에는 플레이어블 통과/프로필 확정으로 읽지 않는다.

## 5. NAME-18 정정 판정

Leila Haddad / NAME-18은 다음 상세 결재로 이동하지 않는다.
E08_C07 피난검역 통행권 역시 캠페인 미개방 상태이므로, 이름 대조/검역명 보류/피난 대기열 축은 후보 메모로만 남긴다.

```text
캠페인 미개방으로 대기
상세 결재 금지
조건부 해소 금지
프로필 v1.0 작성 금지
```

E08_C07이 열리려면 먼저 피난검역 통행권의 NPC, 대기열 오브젝트, 이름 대조 실패 상태값, 작은 결말이 필요하다.
그 전에는 NAME-18을 윤서/복희/R01 이름 축의 외국어판으로 밀지 않는다.

## 6. 금지 방향

금지:

- 한국어권 과밀을 고쳤다는 이유로 미개방 캠페인을 바로 상세 결재한다.
- 외국인 이름을 붙였다는 이유로 전역 후보를 즉시 프로필 v1.0으로 고정한다.
- 14번 이후 기존 번호를 해금 순번처럼 읽는다.
- `NAME-18 상세 결재`를 다음 작업으로 둔다.
- LOCKOUT-17 프로필 산출물을 최종 고정 프로필처럼 참조한다.
- 캠페인 개방 전 후보를 출시 초기 합류로 오해하게 쓴다.
