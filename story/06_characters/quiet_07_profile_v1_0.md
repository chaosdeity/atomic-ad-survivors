## 0. 캐릭터 핵심

백여울은 R08 침묵권에서 "피난자"와 "구조 대기자" 사이에 오래 걸려 있던 사람이다.

여울은 숨어 다니는 사람이 아니다. 더 정확히는 캠페인 신호를 줄이면 적의 등록과 호출도 흐려지지만, 보급소가 자신을 찾는 회수선도 같이 약해진다는 사실을 배운 사람이다.

핵심 문장:

```text
침묵권에서는 덜 들키지만, 덜 구해진다.
```

R08은 여울을 죽이려 하지 않는다. R08은 여울을 피난자, 배급 대상, 대피 명단, 구조 대기자, 수신 불안정 대상 중 하나로 보존하려 한다. 문제는 그 보존이 이동 순서, 배급 순서, 구조 순서를 고정한다는 점이다.

여울은 침묵권을 안전지대로 믿고 싶었다. 하지만 침묵은 보호막이 아니라 거래였다. 캠페인에게 덜 들키는 만큼 보급소에게도 덜 닿는다. 여울은 그 비용을 완전히 없애지 못한다. 대신 언제 줄이고, 언제 남기고, 언제 보급소가 겨우 들을 만큼만 신호를 올릴지 판단한다.

## 4. 플레이어블 핵심

캠페인이 등록하려는 역할:

| 역할 | 의미 | 여울이 비트는 방식 |
| --- | --- | --- |
| 피난자 | 안전을 위해 대피소와 명단에 묶이는 사람 | 명단에 완전히 들어가기 전에 이동 신호를 낮춘다 |
| 배급 대상 | 식량태그와 물자 접근 순서에 묶이는 사람 | 배급 접근권은 필요하지만 배급 줄이 곧 위치 고정이 되지 않게 흔든다 |
| 대피 명단 | 구조 순서와 이동 통제가 합쳐진 등록 | 이름이 불리기 전에 저소음 비콘으로 자기 좌표를 일부만 남긴다 |
| 구조 대기자 | 반복 구조 방송과 비콘 확인을 기다리는 사람 | 구조 호출을 무조건 받지 않고, 호출이 캠페인 추적으로 바뀌는 순간을 늦춘다 |
| 수신 불안정 대상 | 낮은 신호 지역의 위험/미확인 대상으로 표시되는 사람 | 낮은 수신을 완전 회피가 아니라 위험한 선택값으로 운용한다 |

잘못 믿는 것:

```text
덜 들리면 덜 붙잡힌다고 믿는다.
```

이 믿음은 절반만 맞다. 덜 들리면 캠페인의 등록과 호출은 흐려진다. 하지만 보급소의 회수선, 긴급 인양 보정, 지원 신호도 같이 약해진다. 여울의 성장은 "더 조용해지는 것"이 아니라 "끝내 아무에게도 닿지 않는 상태를 피하는 것"이다.

플레이어블 가치:

```text
여울은 E01 초반 한국인 인간 슬롯의 마지막 후보로,
보급소가 안전한 왕국이 아니라 낮은 신호와 약한 인양 사이에서 버티는 장소임을 보여준다.
```

## 5. 전투/탐색 리듬

여울의 전투는 암살이나 은신 처치가 아니다. 여울은 전장과 탐색 구간에서 수신 강도를 낮추고, 캠페인 호출을 늦추며, 보급소 회수선이 끊기지 않을 만큼의 신호를 남긴다.

기본 리듬:

```text
1. 접이식 방호천으로 캠페인 신호 반사를 낮춘다.
2. 낮은 수신 상태에서 적의 등록/호출/추적을 늦춘다.
3. 너무 조용해지면 인양 안정도가 내려간다.
4. 저소음 휘슬이나 수동 비콘 끈으로 최소 신호를 남긴다.
5. 캠페인이 다시 듣기 전에 위치를 옮기거나, 보급소가 겨우 들을 만큼만 좌표를 보정한다.
```

전투/탐색 이미지:

| 이미지 | 사용 방식 | 금지 방향 |
| --- | --- | --- |
| 접이식 방호천 | 시야 방패가 아니라 신호 반사와 수신 강도를 낮추는 임시 차폐 | 탱커 방패, 투명 망토 |
| 저소음 휘슬 | 큰 구조 요청이 아니라 짧은 수동 비콘 | 정찰꾼 신호탄, 탐지 스킬 |
| 낡은 표식봉 | 대피 명단의 줄과 이동 기준점을 흔드는 도구 | DRAIN-MARK의 배수로 표식 대체 |
| 손으로 접은 대피 지도 | 덜 들리는 길과 인양 가능한 길의 차이를 표시 | 만능 지도, 맵 공개 |
| 수동 비콘 끈 | 보급소가 겨우 듣는 최소 좌표를 남김 | 무료 탈출 버튼 |

플레이 감각:

```text
조용해질수록 안전해지는 것이 아니다.
조용해질수록 적도 늦게 듣고, 보급소도 늦게 듣는다.
```

## 7. HP 0 / 인양 차별화

HP 0 핵심:

```text
수신 공백
저소음 비콘 지연
인양 지연
```

여울의 HP 0은 사망/부활이 아니다. 캠페인 호출도 흐려지고 보급소 비콘도 흐려지는 수신 공백이 먼저 온다. 이후 대피 명단이 여울을 구조 대기자로 묶으려 하고, 여울은 큰 구조 신호 대신 저소음 비콘만 남긴다.

절차:

| 단계 | 상태 | 연출 |
| --- | --- | --- |
| 1 | 수신 공백 | 캠페인 호출과 보급소 비콘이 동시에 흐려진다 |
| 2 | 대피 명단 고정 | 주변 절차가 여울을 피난자/구조 대기자로 묶으려 한다 |
| 3 | 저소음 비콘 지연 | 큰 구조 신호를 내면 캠페인도 듣기 때문에 짧은 비콘만 남는다 |
| 4 | 인양 지연 | 보급소가 좌표를 늦게 잡고 일부 태그/지도 조각/비콘 로그 손실이 커진다 |
| 5 | 긴급 인양 | 보급소가 겨우 들은 수동 비콘 끈을 따라 끌어오지만, 회수 보정은 낮다 |

비교:

| 비교 대상 | 차이 |
| --- | --- |
| 윤서 | 마지막 좌표 즉시 회수가 아니라, 좌표가 일부러 약해진 상태다 |
| PATCH | 부품 목록화나 정품 복구 직전 회수가 아니라, 비콘이 너무 약해지는 위험이다 |
| 강하람 | 퇴원 취소/병상 재배정이 아니라, 구조 대기 명단과 수신 공백이 겹친다 |
| 한이루 | 미수령 보관/재배송 대기가 아니라, 저소음 비콘 로그가 늦게 잡힌다 |
| 차유건 | 계량 초과/충전 차단이 아니라, 회수선이 신호를 놓치는 문제다 |

여울은 쓰러질 때 더 잘 숨는 것이 아니라,

## 8. DRAIN-MARK/PATCH/윤서/R08 보급소 관계

### DRAIN-MARK

```text
DRAIN-MARK:
R01 배수로 침묵 주머니에 남는 NPC 표식.

백여울:
R08 침묵권에서 온 플레이어블 후보.
```

여울은 DRAIN-MARK의 후계자나 상위호환이 아니다. DRAIN-MARK는 R01에서 "덜 들키는 길은 덜 구조되는 길일 수 있다"는 경고를 남기는 NPC 기능이고, 여울은 그 위험을 플레이어블의 전투/정산/HP 0으로 직접 운용한다.

금지:

```text
여울이 R01 배수로 표식을 해설하지 않는다.
여울이 DRAIN-MARK의 숨은 루트를 대체하지 않는다.
여울이 DRAIN-MARK를 스승처럼 호출하지 않는다.
```

### PATCH

PATCH는 보급소 안쪽에서 자기 몸을 부품처럼 써온 수리 로봇이다. 여울은 보급소 바깥의 낮은 수신과 약한 인양을 다루는 인간 후보다.

관계 감각:

```text
PATCH는 여울의 장비를 고쳐 줄 수 있지만,
여울의 침묵을 안전하다고 착각하지 않는다.
여울은 PATCH를 충전/수리 담당으로 부르지 않고,
보급소가 자신을 놓치는 순간 PATCH도 늦게 움직일 수 있음을 안다.
```

차이:

| PATCH | 여울 |
| --- | --- |
| 보급소 수리 로봇 | R08 침묵권 인간 피난자 |
| 자기부품화 | 자기수신 약화 |
| 부품 목록화 위험 | 비콘 약화/인양 지연 위험 |
| 로봇 자율성 축 | 수신 안정도/인양 안정도 축 |

### 윤서

윤서는 현장에서 마지막 좌표를 붙잡고 위험물을 회수하는 기본 주인공이다. 여울은 좌표를 약하게 만드는 사람이다.

관계 감각:

```text
윤서는 "어디에 있었는지"를 묻는다.
여울은 "얼마나 들렸는지"를 먼저 답한다.
```

윤서는 여울을 단순히 숨은 사람으로 보지 않는다. 윤서에게 여울은 회수해야 할 대상인 동시에, 회수선 자체가 얼마나 불안정한지 보여주는 사람이다.

### R08 보급소

R08 보급소는 안전한 왕국이 아니다. 비상보급, 대피 명단, 민방위 저장고, 무음 플랫폼이 모두 남아 있지만, 그만큼 유지 비용과 인양 불안도 크다.

여울은 R08 보급소의 운영자가 아니다. 여울은 그 바깥에서 침묵권의 비용을 먼저 몸으로 겪은 사람이다.

R08 노출 범위:

```text
허용:
R08-L03 폐역 무음 플랫폼
R08-L04 민방위 저장고 단서

보류:
R08-L05 가짜 구조 신호탑
보급소 내부 배신
리오의 체류/숙박 캠페인권
```

### 리오 전 위치

여울은 8번 리오 전의 마지막 E01 초반 한국인 인간 후보로 둔다. 여울은 리오를 설명하지 않는다. 여울은 리오가 열리기 전, 보급소 회수선과 외부 캠페인 신호가 완전히 분리되어 있지 않다는 감각만 남긴다.

```text
7번 여울:
낮은 수신과 약한 인양의 인간 후보.

8번 리오:
중반 체류/숙박 캠페인권의 로봇 후보. 보류 잠금 유지.
```

## 10. 성장 방향

초기 상태:

```text
덜 들리면 덜 붙잡힌다고 믿는다.
```

초기 여울은 조용한 길을 안전한 길로 착각하고 싶어 한다. 대피 명단에 올라가면 구조될 수 있다는 사실도 알고 있지만, 명단이 자신을 움직이지 못하게 할 수 있다는 것도 안다. 그래서 여울은 명단 밖과 명단 안, 침묵과 호출 사이를 계속 오간다.

성장 방향:

```text
덜 들리는 것과,
끝내 아무에게도 닿지 않는 것을 구분한다.
```

성장 후에도 여울은 큰 소리로 구조를 부르는 사람이 되지 않는다. 대신 언제 보급소가 들어야 하는지, 언제 캠페인 호출을 피해야 하는지, 언제 수동 비콘 끈을 당겨야 하는지 판단한다.

좋은 성장 장면:

| 단계 | 장면 |
| --- | --- |
| 초반 | 구조 방송을 피하려고 비콘을 낮추지만, 보급소 회수선도 늦어진다 |
| 중반 | 윤서나 PATCH에게 "나를 못 찾은 게 네 잘못이 아니다"라고 말한다 |
| 전환 | 대피 명단에 완전히 들어가지 않고도, 보급소가 들을 만큼의 신호를 남긴다 |
| 안정 | 침묵권을 안전지대로 팔지 않고, 침묵의 비용을 다른 출격자에게 숨기지 않는다 |

## 11. 최종 판정

판정:

```text
통과 / 7번 플레이어블 후보 프로필 v1.0 고정
```

통과 사유:

1. 백여울 / QUIET-07은 단순 은신자, 정찰꾼, 생존주의자가 아니라 침묵권 피난자로 고정되었다.
2. 핵심 문장은 "침묵권에서는 덜 들키지만, 덜 구해진다"로 잠겼다.
3. DRAIN-MARK는 R01 배수로 표식 NPC로 분리했고, 여울은 R08 침묵권 출신 플레이어블 후보로 분리했다.
4. 플레이어블 능력은 암살/스텔스가 아니라 신호 약화, 캠페인 호출 회피, 회수선 리스크 관리다.
5. HP 0은 수신 공백, 저소음 비콘 지연, 인양 지연으로 기존 1~6번 후보와 구분된다.
6. 여울은 8번 리오 전, E01 초반 한국인 인간 슬롯의 마지막 후보로 적합하다.
7. R08 노출은 R08-L03 폐역 무음 플랫폼과 R08-L04 민방위 저장고 단서까지만 허용하고, R08-L05 가짜 구조 신호탑은 보류한다.

7번 플레이어블 후보

판정:
통과 / 프로필 v1.0 고정
