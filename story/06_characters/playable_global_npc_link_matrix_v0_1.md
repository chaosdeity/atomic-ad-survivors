# 플레이어블 20인 전역 NPC 연결 매트릭스 v0.1

## 문서 상태

```text
상태:
작성 완료 v0.1

작업 성격:
플레이어블 20인과 전역 NPC/잔향/기록 씨앗의 연결을 잠그는 문서.
새 플레이어블 후보 생성 아님.
플레이어블 프로필 v1.0 작성 아님.

판정:
통과
```

---

## 0. 연결 원칙

1. 플레이어블은 NPC 기능을 빼앗지 않는다.
2. NPC 씨앗은 플레이어블을 설명하는 장식이 아니라, 원점/심화/교차 캠페인에서 다른 각도의 압력을 준다.
3. 11~20번은 아직 상세 전이므로 이름/국적보다 캠페인 등록 방식과 연결 씨앗을 먼저 고정한다.
4. 로봇 플레이어블에게 억지 사람 이름을 붙이지 않는다.

---
