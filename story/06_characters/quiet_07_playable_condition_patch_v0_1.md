## 0. 조건부 통과 사유 요약

기존 상세 결재 문서는 7번 침묵권 피난자를 조건부 통과로 판정했다.

조건부 판정의 이유:

1. 낮은 수신과 조용한 이동 규칙이 단순 은신, 정찰, 생존주의 캐릭터처럼 보일 위험이 있다.
2. R01 배수로 침묵 주머니 NPC인 DRAIN-MARK와 숨은 길/표식/침묵 기능이 겹칠 수 있다.
3. "침묵권은 안전하다"로 읽히면 보급소 회수선이 약해지는 핵심 리스크가 사라진다.
4. HP 0이 일반 긴급 인양으로 처리되면 윤서, PATCH, 강하람, 한이루, 차유건과 차이가 약해진다.
5. R08 전체를 초반에 열면 리오 8번 보류 전부터 세계가 너무 넓어진다.

보완의 목표:

```text
이 후보를 숨는 사람이 아니라,
덜 들키는 대신 덜 구해지는 값을 관리하는 침묵권 피난자로 잠근다.
```

## 1. 해결해야 할 위험

| 위험 | 문제 | 이번 문서의 잠금 |
|---|---|---|
| DRAIN-MARK와 역할 중복 | R01 배수로 표식, 숨은 루트, 침묵 경고를 플레이어블이 가져갈 수 있다 | DRAIN-MARK는 R01 NPC, 여울은 R08 침묵권 플레이어블 후보로 분리한다 |
| 단순 은신자화 | 낮은 수신이 투명화나 암살 스텔스처럼 보인다 | 낮은 수신은 적 추적과 보급소 인양을 동시에 낮추는 위험값으로 둔다 |
| 정찰꾼화 | 지도, 표식봉, 휘슬이 탐색 전담 직업처럼 보인다 | 지도는 안전 경로가 아니라 인양 가능한 길과 덜 들리는 길의 차이를 표시한다 |
| 생존주의자화 | 혼자 잘 버티는 캐릭터가 되면 보급소와 배급 압력이 사라진다 | 식량태그, 대피 명단, 보급 압력, 인양 안정도를 계속 묶는다 |
| 침묵권 안전지대화 | 침묵권이 피난처 왕국처럼 보이면 초반 보급소의 취약성이 흐려진다 | 침묵권은 안전한 곳이 아니라 신호와 구조가 같이 약해지는 공백으로 둔다 |
| HP 0 중복 | 일반 긴급 인양으로 처리되면 기존 후보들과 차이가 없다 | 수신 공백, 저소음 비콘 지연, 인양 지연을 핵심 절차로 잠근다 |
| 리오 8번 조기 침범 | 수신/체류/로봇 장기 서사를 앞당길 수 있다 | 여울은 인간 피난자이고, 리오의 체류/숙박/정품 복구 공포는 보류한다 |

## 2. DRAIN-MARK와 역할 분리

핵심 분리:

```text
DRAIN-MARK:
R01 배수로 침묵 주머니에 남는 NPC 표식.
캠페인에게 덜 들키는 길은 보급소에게도 덜 닿는다는 경고를 담당한다.

백여울 / QUIET-07:
R08 침묵권에서 온 7번 플레이어블 후보.
캠페인 호출을 낮추는 대신 보급소 회수선도 약해지는 값을 전투/정산에서 직접 관리한다.
```

비교 잠금:

| 비교축 | DRAIN-MARK | 백여울 / QUIET-07 |
|---|---|---|
| 문서상 기능 | R01 P1 NPC/표식 기능 | 7번 플레이어블 후보 |
| 위치 | R01 배수로 침묵 주머니 | R08-L03 폐역 무음 플랫폼, R08-L04 민방위 저장고 단서 |
| 플레이 기능 | 직접 전투 기능 없음 | 낮은 수신, 호출 회피, 회수선 리스크 관리 |
| 핵심 메시지 | 이 길은 덜 들리지만 덜 구조된다 | 그 위험을 몸값과 실패 조건으로 운용한다 |
| 금지 침범 | 플레이어블 상위호환 금지 | R01 숨은 루트 안내와 표식 기능 탈취 금지 |

운영 잠금:

```text
여울은 DRAIN-MARK의 제자, 후계자, 강화판이 아니다.
여울은 R01 배수로 표식을 읽어 주는 캐릭터가 아니다.
여울은 R08 침묵권에서 온 플레이어블 후보이며,
DRAIN-MARK는 R01에서 침묵의 위험을 먼저 보여주는 NPC 표식으로 남는다.
```

## 3. 클리셰 차단

금지 유형:

| 금지 유형 | 왜 금지하는가 | 고정 대안 |
|---|---|---|
| 은신자 | 낮은 수신이 곧 무조건 회피가 된다 | 낮은 수신은 캠페인 호출과 보급소 인양을 같이 약화한다 |
| 정찰꾼 | 지도와 표식봉이 안전한 길 안내로 납작해진다 | 지도는 덜 들리는 길과 인양 가능한 길의 차이를 기록한다 |
| 생존주의자 | 혼자 사는 능력이 보급소 관계와 배급 압력을 지운다 | 여울은 대피 명단과 배급 절차를 피하면서도 완전히 벗어나지 못한다 |
| 암살자 | 조용한 이동이 후방 처치/치명타로 바뀐다 | 전투 중심은 피해량이 아니라 호출, 수신, 회수선 리스크다 |
| 저항군 연락책 | 침묵권을 선한 반군 기지로 만든다 | 침묵권은 유지 비용이 큰 공백이며 누구에게나 안전하지 않다 |
| 보급 담당자 | 미나의 배급/정산 기능을 빼앗는다 | 여울은 보급 압력을 드러내지만 배분 결정을 맡지 않는다 |

허용되는 캐릭터 방향:

```text
여울은 조용히 싸우는 사람이 아니라,
얼마나 들릴지, 얼마나 구해질지를 매번 계산하는 사람이다.
```

## 4. 핵심 명제 고정

버릴 문장:

```text
침묵권에 있으면 안전하다.
```

고정 문장:

```text
침묵권에서는 덜 들키지만, 덜 구해진다.
```

이 문장이 7번 후보의 중심이다.

- 캠페인 신호가 약해지면 적의 등록, 추적, 호출도 약해진다.
- 하지만 보급소의 회수선, 긴급 인양 보정, 지원 신호도 같이 약해진다.
- 여울은 안전지대를 얻은 것이 아니라, 양쪽 신호가 모두 약한 곳에서 선택 비용을 배운 사람이다.
- 그래서 플레이어블 능력은 숨기, 암살, 탐지가 아니라 신호 약화와 인양 위험 관리다.

성장 방향:

```text
더 완벽히 숨는 것이 아니라,
끝내 아무에게도 닿지 않는 상태를 피하는 쪽이다.
```

## 5. 플레이어블 능력 잠금

능력 축:

| 축 | 허용 | 금지 |
|---|---|---|
| 신호 약화 | 일정 시간 캠페인 호출, 등록 압력, 추적 신호를 낮춘다 | 완전 은신, 무적, 타겟 해제 |
| 호출 회피 | 캠페인 방송/구조 호출/대피 명단 고정을 지연한다 | 적 AI를 꺼 버리는 스텔스 |
| 수동 비콘 | 보급소가 겨우 잡을 만큼만 신호를 남긴다 | 무료 순간 탈출, 안전 귀환 버튼 |
| 회수선 리스크 관리 | 낮은 수신 상태에서 인양 안정도 손실을 예측하고 줄인다 | HP 0 패널티 제거 |
| 조용한 이동 규칙 | 높은 소음/높은 수신 경로 대신 느리고 불안정한 경로를 고른다 | 정찰꾼의 맵 공개 |

전투 리듬:

```text
1. 수신을 낮춰 캠페인 호출을 피한다.
2. 낮아진 수신 때문에 보급소 회수선도 약해진다.
3. 수동 비콘이나 표식봉으로 최소 신호를 남길지 결정한다.
4. 너무 조용해지면 HP 0 때 인양이 늦어진다.
5. 너무 들리면 캠페인이 대피 명단/구조 호출로 다시 묶는다.
```

중심 이미지는 유지한다.

| 중심 이미지 | 쓰임 |
|---|---|
| 접이식 방호천 | 시야 방패가 아니라 신호 반사와 수신 강도를 낮추는 임시 차폐 |
| 저소음 휘슬 | 큰 구조 요청이 아니라 아주 짧은 수동 비콘 |
| 낡은 표식봉 | 대피 명단의 줄을 세우는 물건을 조용한 이동 기준점으로 비튼다 |
| 손으로 접은 대피 지도 | 안전 지도 대신 인양 가능한 길과 덜 들리는 길의 차이를 표시한다 |
| 수동 비콘 끈 | 잡아당기면 보급소가 듣지만 캠페인도 조금 더 듣는다 |

## 6. 정산 루트와 상태값 반응

정산 루트:

| 정산 항목 | 고정 의미 |
|---|---|
| 식량태그 | 대피 명단과 배급 접근의 생활 압력 |
| 통행태그 | 조용한 이동 경로와 봉쇄 우회 접근 |
| 수신태그 | 구조 방송, 비콘, 회수선, 지도 신호의 불안정성 |
| 인양 안정도 | 보급소가 출격자를 찾고 끌어올 수 있는 정도 |
| 보급 압력 | 침묵권 유지비와 실제 물자 부족 |
| 저소음 비콘 로그 | HP 0과 긴급 인양의 흔적 |

상태값 반응:

| 상태값 | 반응 |
|---|---|
| 캠페인 농도 높음 | 낮은 수신의 이득이 커지지만, 한 번 들키면 대피 명단 고정이 거칠어진다 |
| 보급소 노출도 높음 | 여울은 수신을 낮추려 하지만, 보급소가 자신을 찾기 어려워지는 위험도 커진다 |
| 인간 안정도 낮음 | 조용한 이동이 배신, 이탈, 배급 회피로 오해받는다 |
| 로봇 자율성 높음 | PATCH/리오 축을 해설하지 않고, 비콘 장비와 수신 장치가 로봇에게도 부담임을 짧게 비춘다 |
| 수신 안정도 낮음 | 적 추적은 약해지지만 보급소 지원과 긴급 인양 보정도 흔들린다 |
| 인양 안정도 낮음 | HP 0 때 저소음 비콘 로그가 늦게 잡혀 손실이 커진다 |
| 보급 압력 높음 | 식량태그와 대피 명단의 현실 비용이 여울의 선택을 압박한다 |

정산 감각:

```text
조용함은 무료 방어력이 아니다.
조용함은 적과 보급소 양쪽의 귀를 동시에 멀게 하는 거래다.
```

## 7. HP 0 / 긴급 인양 차별화

HP 0 핵심:

```text
수신 공백
저소음 비콘 지연
인양 지연
```

절차:

| 단계 | 상태 | 연출 |
|---|---|---|
| 1 | 수신 공백 | 캠페인 호출도 보급소 비콘도 동시에 흐려진다 |
| 2 | 대피 명단 고정 시도 | 주변 절차가 여울을 피난자/구조 대기자로 묶으려 한다 |
| 3 | 저소음 비콘 지연 | 큰 구조 신호를 내면 캠페인도 듣기 때문에 짧은 비콘만 남는다 |
| 4 | 인양 지연 | 보급소가 좌표를 늦게 잡고, 일부 태그/지도 조각/비콘 로그 손실이 커진다 |
| 5 | 긴급 인양 | 보급소가 겨우 들은 수동 비콘 끈을 따라 끌어오지만, 회수 보정은 낮다 |

기존 후보와의 차이:

| 비교 대상 | HP 0 차이 |
|---|---|
| 윤서 | 마지막 좌표 즉시 회수선이 아니라, 좌표가 일부러 약해진 상태다 |
| PATCH | 부품 목록화/정품 복구 직전 회수가 아니라, 비콘이 너무 약해지는 위험이다 |
| 강하람 | 퇴원 취소/병상 재배정이 아니라, 구조 대기 명단과 수신 공백이 겹친다 |
| 한이루 | 미수령 보관/재배송 대기가 아니라, 저소음 비콘 로그가 늦게 잡힌다 |
| 차유건 | 계량 초과/충전 차단이 아니라, 회수선이 신호를 놓치는 문제다 |
| DRAIN-MARK | NPC 경고가 아니라 플레이어블 실패 조건으로 직접 정산된다 |

고정 문장:

```text
여울은 쓰러질 때 더 잘 숨는 것이 아니라,
보급소가 겨우 들을 만큼만 남는 상태로 밀린다.
```

## 8. 해금 범위 제한

허용 범위:

```text
R08-L03 폐역 무음 플랫폼
R08-L04 민방위 저장고 단서
```

선행 단서:

```text
R01-L04 배수로 침묵 주머니에서 침묵권 재료와 수신태그 단서를 본 뒤,
R08-L03 폐역 무음 플랫폼에서 보급소 인양 반경 보정 문제가 열린다.
```

범위 잠금:

| 범위 | 허용 | 금지 |
|---|---|---|
| R08-L01 침묵 보급소 | 기준점으로 언급 가능 | 여울을 보급소 운영 핵심으로 만들지 않는다 |
| R08-L02 역송출 주유소 | 인양 장비 배경으로만 언급 가능 | 여울을 장비 담당자나 엔지니어로 만들지 않는다 |
| R08-L03 폐역 무음 플랫폼 | 직접 해금 후보 | 만능 은신 허브로 만들지 않는다 |
| R08-L04 민방위 저장고 | 보급 압력 단서 | 미나의 배급 정산 기능을 대체하지 않는다 |

## 9. E01 초반 7번 슬롯과 리오 연결

판정:

```text
E01 초반 한국인 슬롯의 마지막 후보로 적합하다.
단, 8번 리오를 앞당기지 않는 연결 역할로만 쓴다.
```

7번에 두는 이유:

- 4번 강하람은 의료/보험의 퇴원 판정을 연다.
- 5번 한이루는 물류/반품의 도착/반송 판정을 연다.
- 6번 차유건은 충전/계량의 사용량 판정을 연다.
- 7번 백여울은 보급소 바깥의 낮은 수신과 약한 인양을 연다.
- 따라서 3~7번 초반 한국인 인간 후보의 마지막 칸은 "보급소가 안전 왕국이 아니다"라는 감각으로 닫을 수 있다.

리오 전 연결:

```text
여울은 리오를 설명하지 않는다.
여울은 리오가 열기 전,
보급소 회수선과 외부 캠페인 신호가 완전히 분리되어 있지 않다는 감각만 남긴다.
```

리오와의 거리:

| 비교축 | 백여울 / QUIET-07 | 리오 / RIO-08 |
|---|---|---|
| 존재 유형 | 한국인 인간 피난자 | 체류/숙박 캠페인권의 객실 정리 로봇 |
| 초반 기능 | 낮은 수신과 약한 인양의 리스크 | 중반 보류, 정품 복구 공포 |
| 지역 감각 | R08-L03~L04 제한 | E08_C01 체류/숙박권 |
| 금지 | 로봇 자율성 해설, 정품 복구 공포 선공개 | 8번 잠금 앞당기기 |
