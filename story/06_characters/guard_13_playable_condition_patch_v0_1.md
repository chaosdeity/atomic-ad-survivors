## 1. 조건부 사유 목록

상세 결재에서 남은 조건부 사유:

```text
1. 전태주 이름은 최종 확정 금지 상태다.
2. E12_C02 이동진료 승인권을 원점으로 삼으면 이름/국적/언어권 재검토가 필요하다.
3. E12_C02 원점은 우선 후보지만 아직 프로필 v1.0 잠금 전이다.
4. E04_C02 열대격리 진료권은 보조 후보로만 제한해야 한다.
5. E01/R02 의료 후속 직접 원점은 보류해야 한다.
6. HOLD-04와 환자/퇴원 보류/보험 심사 축이 겹치면 안 된다.
7. R02 의료 NPC와 병원 행정 얼굴 기능이 겹치면 안 된다.
8. 전투 리듬이 힐러/탱커/보디가드/법률 보호자 이미지로 새면 안 된다.
9. HP 0은 보호자 서명 확정 / 치료 접근권 담보 고정으로 고정해야 한다.
```

## 2. 이름/국적/언어권 해소

판정:

```text
전태주 유지 가능 / 프로필 v1.0에서 최종 고정 검토
```

전태주 이름은 프로필 v1.0로 넘겨도 된다. 단, 이 이름은 E12_C02 전체 현실권을 대표하는 이름이 아니다. GUARD-13은 E12_C02 이동진료 승인권에서 보호자/보증인 등록 압박을 겪은 사람으로 배치되며, 이름은 전역 이동진료망에 섞인 개인의 이름으로 취급한다.

유지 조건:

1. 프로필 v1.0에서도 `전태주`를 무검수 확정처럼 쓰지 않고, 문서 상태에서 이름 검수 조건을 명시한다.
2. 유저 첫 표기는 `태주, 보호자 등록 거부자`로 둘 수 있으나, E12_C02 전체 권역의 대표 이름으로 설명하지 않는다.
3. 현실 도시, 병원, 구호단체 실명과 결합하지 않는다.
4. 국적/언어권은 캐릭터 본문에서 과하게 설명하지 않고, 제작 관리 필드로만 최소 유지한다.
5. 로컬라이즈 단계에서 현지명/다국적 이름으로 바꿀 필요가 생기면 `GUARD-13` 코드명과 역할명은 유지한다.

해소 이유:

```text
이름 문제는 캐릭터 축의 결함이 아니라 노출/검수 문제다.
GUARD-13의 플레이어블 핵심은 보호자/보증인 등록 절차이며,
전태주 이름은 이 핵심을 망치지 않는다.
따라서 조건부를 유지할 필요는 없고, 프로필 v1.0에서 최종 고정 검토로 넘길 수 있다.
```

금지:

```text
전태주를 E12_C02 대표 인물처럼 설명하지 않는다.
이름을 이유로 E12_C02를 한국어권처럼 바꾸지 않는다.
이름을 임의 폐기하고 새 이름을 확정하지 않는다.
```

## 3. E12_C02 원점 잠금

판정:

```text
E12_C02 이동진료 승인권 원점 잠금
```

E12_C02는 GUARD-13의 원점으로 적합하다. 이동진료 트럭, 승인 환자표, 임시 약품 천막, 승인 데스크는 보호자/보증인 등록 절차를 병원 내부 신파가 아니라 이동진료 접근권의 문제로 바꾼다.

잠금 기준:

| 요소 | GUARD-13 적용 |
|---|---|
| 이동진료 트럭 | 접근권이 이동 경로와 승인 순번에 묶이는 장면 |
| 승인 환자표 | 치료 대상이 보호자/보증인 칸으로 닫히는 절차 |
| 임시 약품 천막 | 약품 접근이 서명과 임시 서약에 담보 잡히는 공간 |
| 승인 데스크 | 누가 책임자로 등록되는지 확정하는 판정 장소 |

해소 판정:

```text
해소.
E12_C02는 GUARD-13의 보호자/보증인 등록 압박과 충분히 붙는다.
```

금지 확장:

```text
E12 전체 광역권 설명
E12_C06 모바일머니 회원권 선점
E12_C08 우물/창고 보급권 선점
실제 도시/병원/구호단체 실명 노출
```

## 4. E04_C02 및 E01/R02 제한선

E04_C02 판정:

```text
보조 후보로 제한
```

E04_C02 열대격리 진료권은 격리 동의, 보호자 확인, 임시 약품선 단서로만 쓴다. 원점으로 전환하지 않는다. E04_C02는 격리 환자와 진료 대상 이미지가 강해 HOLD-04의 환자 고정 축과 가까워질 위험이 있다.

E01/R02 판정:

```text
직접 원점 보류
```

R02에서는 보호자/보증인 압력의 암시만 허용한다. 예를 들어 미서명 보호자 칸의 잔향, 호출벨 기록의 짧은 흔적, HOLD-04 이후 비교용 후속 단서는 가능하다. 하지만 GUARD-13의 직접 해금, 상세 설명, 원점화는 금지한다.

제한표:

| 축 | 허용 | 금지 |
|---|---|---|
| E04_C02 | 격리 동의, 보호자 확인, 임시 약품선 단서 | 원점 전환, 전체 상세화 |
| E01/R02 | 보호자/보증인 압력의 암시 | 직접 해금, 퇴원 심사 설명, R02 의료 NPC 대체 |
| E12_C02 | 원점, 직접 대면 후보, 승인 환자표 | E12 전체 광역권 설명 |
