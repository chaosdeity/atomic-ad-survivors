## 0. 판정 요약

판정:

```text
조건부 통과
```

백여울 / QUIET-07 / 침묵권 피난자는 7번 슬롯에 둘 가치가 있다. 4번 강하람이 의료/보험, 5번 한이루가 물류/반품, 6번 차유건이 충전/계량을 열었다면, 7번은 보급소가 완전한 안전지대가 아니라 "신호를 낮춘 대신 회수선도 약해지는 예외 공간"임을 플레이어블 관점으로 보여줄 수 있다.

다만 바로 v1.0 프로필로 가지 않는다. 아래 조건이 먼저 잠겨야 한다.

1. 단순 은신자, 정찰꾼, 생존주의자가 아니어야 한다.
2. DRAIN-MARK의 R01 배수로 표식/숨은 루트 NPC 기능을 빼앗지 않아야 한다.
3. 전투 능력보다 캠페인 신호를 덜 받는 대신 보급소 회수선도 약해지는 리스크가 중심이어야 한다.
4. R08 전체를 초반에 열지 않는다. R08-L03 폐역 무음 플랫폼과 R08-L04 민방위 저장고 단서까지만 쓴다.
5. HP 0은 윤서의 현장 회수, PATCH의 부품 회수, 하람의 퇴원 취소, 이루의 미수령 보관, 유건의 계량 초과와 다른 "비콘 약화/인양 지연" 절차여야 한다.

핵심 질문:

```text
덜 들리는 곳에 있으면 안전한가,
아니면 아무도 찾지 못하는가?
```

| 플레이어블 검토 목적 | 보급소와 침묵권이 완전한 안전지대가 아니라, 낮은 신호와 약한 인양 사이의 위험한 균형임을 보여준다 |
| 최종 프로필 여부 | 아직 아님. 말투, 외형, 세부 관계선은 확정하지 않는다 |

이 후보는 숨어 다니는 정찰꾼이 아니다. 더 정확히는 캠페인 신호를 덜 받는 곳에서 살아남는 법을 배웠지만, 그 때문에 보급소 회수선도 자신을 늦게 찾는다는 사실을 몸으로 아는 사람이다.

캠페인은 백여울을 죽이려 하지 않는다. 캠페인은 여울을 피난자, 배급 대상, 구조 대기자, 대피소 인원으로 등록해 안전하게 묶으려 한다. 문제는 그 등록이 "안전"이라는 이름으로 이동, 배급, 구조 순서를 고정한다는 점이다.

여울의 생존 방식은 적에게 들키지 않는 것이 아니다.

```text
캠페인 신호가 닿는 정도와
보급소 회수선이 닿는 정도가
같은 축에 묶여 있음을 이용한다.
```

## 2. 어떻게 캠페인에 붙잡혔는가

캠페인이 붙잡는 방식:

| 절차 | 붙잡는 방식 | 후보가 느끼는 압력 |
|---|---|---|
| 피난자 등록 | 대피소 입장과 이동을 명단 순서로 묶는다 | "명단에 없으면 밀려나고, 명단에 있으면 움직일 수 없다" |
| 배급 대상 판정 | 식량/물자 접근을 대피 명단과 연결한다 | "먹으려면 어디에 있는 사람인지 먼저 증명해야 한다" |
| 구조 대기자 호출 | 반복 구조 방송과 비콘 확인으로 위치를 확정한다 | "구조 신호가 내 위치를 살리는지, 팔아넘기는지 모르겠다" |
| 수신 불안정 대상 | 낮은 신호 지역을 위험/미확인으로 표시한다 | "덜 들리면 덜 잡히지만, 덜 구조된다" |
| 인양 위험 판정 | 비콘 약화와 회수선 흔들림이 누적된다 | "보급소가 나를 잡으려면, 나도 조금은 들려야 한다" |

아직 완전히 흡수되지 않은 이유:

- R08 침묵권은 완전 차단 구역이 아니라, 금속 차폐 구조와 수동 설비가 만든 약한 공백이다.
- 여울은 캠페인 신호가 약한 길과 보급소 회수선이 잡을 수 있는 길을 동시에 읽는다.
- 대피 명단에 완전히 들어가지 않았지만, 명단 바깥에 있으면 배급과 구조에서도 밀린다.
- 여울은 "숨은 사람"이 아니라 "수신을 낮춘 채 인양 가능성을 잃지 않으려는 사람"이다.

## 3. 잘못 믿는 것

초안:

```text
조용히 있으면 안전하다고 믿음.
```

결재안:

```text
덜 들리면 덜 붙잡힌다고 믿는다.
```

이 문장이 더 낫다. "조용히 있음"만 남기면 은신자나 정찰꾼처럼 보인다. "덜 들림"으로 바꾸면 캠페인 수신, 비콘, 보급소 회수선, 인양 안정도를 모두 건드릴 수 있다.

더 깊은 오류:

- 캠페인에게 덜 들키는 길이 곧 안전한 길이라고 믿고 싶어 한다.
- 보급소가 자신을 못 찾는 상황도 자유의 일부라고 착각할 때가 있다.
- 대피 명단 밖에 있으면 누군가 자신을 밀어 넣지 못한다고 믿는다.
- 하지만 명단 밖에 오래 있으면 식량, 통행, 구조 순서에서도 빠진다.

따라서 이 후보의 성장은 "더 잘 숨기"가 아니다.

```text
덜 들리는 것과,
끝내 아무에게도 닿지 않는 것을 구분하는 쪽이다.
```

## 4. E01 초반 7번 슬롯 적합성

판정:

```text
적합 / 단, 조건부 통과
```

이유:

- 7번은 E01 초반 3~7번 한국인 후보의 마지막 칸으로 적합하다.
- 4번 R02, 5번 R03, 6번 R04가 외부 캠페인권의 절차를 보여준 뒤, 7번은 보급소와 R08 자체의 취약성을 되돌아보게 한다.
- 여울은 새로운 대륙이나 후반 광역권을 열지 않는다. 첫 시즌의 보급소/침묵권 유지비를 깊게 만든다.
- 리오 8번 보류 잠금을 앞당기지 않는다.
- 3~7번에 새 로봇을 넣지 않는 원칙을 지킨다.

주의:

```text
7번이 강한 이유는 "침묵권 기술자"라서가 아니다.
보급소가 안전한 왕국이 아니라는 사실을,
플레이어블의 몸값과 실패 조건으로 보여줄 수 있기 때문이다.
```

## 5. DRAIN-MARK와의 차이

핵심 차이:

```text
DRAIN-MARK:
R01 배수로 침묵 주머니에 표식을 남기는 NPC.
캠페인에게 덜 들키는 길은 보급소에게도 덜 닿는 길이라는 경고를 담당한다.

백여울 / QUIET-07:
R08 침묵권에서 온 플레이어블 후보.
캠페인 신호를 낮춰 움직이는 대신, 자기 인양 안정도와 보급소 회수선도 약해지는 리스크를 전투/정산으로 운용한다.
```

비교 잠금:

| 비교축 | DRAIN-MARK | 백여울 / QUIET-07 |
|---|---|---|
| 역할 | R01 P1 NPC | 7번 플레이어블 후보 |
| 출신 감각 | R01 배수로 침묵 주머니 | R08 침묵권 군락, 폐역/민방위 잔여권 |
| 담당 기능 | 숨은 루트, 배수로 표식, 경고 | 낮은 수신 이동 규칙, 인양 위험, 보급 압력 |
| 잘못 믿는 것 | 숨으면 산다 | 덜 들리면 덜 붙잡힌다 |
| 전투화 가능성 | 플레이어블 기능을 맡기지 않는다 | 캠페인 신호 수신과 회수선 안정도를 함께 흔든다 |
| 금지 침범 | 플레이어블이 아니다 | R01 배수로 표식과 숨은 루트 안내를 빼앗지 않는다 |

금지:

```text
여울이 R01 배수로 표식을 찍어 주는 캐릭터가 되지 않는다.
여울이 DRAIN-MARK의 숨은 루트를 대체하지 않는다.
여울이 DRAIN-MARK를 스승/상위호환처럼 만들지 않는다.
여울은 R08 침묵권에서 온 플레이어블 후보로 둔다.
```

## 6. 윤서/PATCH/강하람/한이루/차유건/OPEN-HOST/리오와의 차이

| 비교 대상 | 상대의 핵심 | 여울과의 차이 |

| PATCH | 자기 몸을 부품처럼 써온 보급소 수리 로봇 | PATCH는 보급소 안쪽의 수리/부품/로봇 자율성 축이다. 여울은 보급소 바깥의 낮은 수신/인양 위험 축이다. |

| 한이루 / RETURN-05 | 자기 자신이 반송/미수령/재배송 대상으로 읽힌 사람 | 이루는 주소와 배송 상태를 흔든다. 여울은 수신 강도와 인양 가능성을 흔든다. |
| 차유건 / METER-06 | 충전량/사용량/차단 기준을 비트는 사람 | 유건은 계량 초과와 차단을 다룬다. 여울은 수신 불안정과 회수선 약화를 다룬다. |
| OPEN-HOST | R01 내부 동선과 방의 배치를 뒤집는 후보 | OPEN-HOST는 길과 방의 선택지를 연다. 여울은 조용한 이동이 회수 안정도를 떨어뜨리는 조건을 연다. |
| 리오 | 중반 체류/숙박 캠페인권의 탈정품 객실 정리 로봇 | 리오는 8번 보류 잠금이다. 여울은 E01 초반 마지막 한국인 인간 후보이며 리오의 외부 체류/로봇 축을 당기지 않는다. |

## 7. 전투/무기 이미지

| 이미지 | 사용 방식 | 금지 방향 |
|---|---|---|
| 접이식 방호천 | 시야를 막는 은신 장비가 아니라, 신호 반사를 낮춰 캠페인 수신을 흐린다 | 탱커 방패나 투명 망토로 만들지 않는다 |
| 저소음 휘슬 | 큰 호출이 아니라 짧은 수동 비콘 신호로 쓴다 | 정찰꾼의 신호탄이나 탐지 스킬로 만들지 않는다 |
| 낡은 표식봉 | 대피 명단의 줄을 세우는 물건을, 조용한 이동 규칙의 기준점으로 비튼다 | DRAIN-MARK의 배수로 표식 기능을 빼앗지 않는다 |
| 손으로 접은 대피 지도 | 안전한 길이 아니라 인양 가능한 길과 덜 들리는 길의 차이를 표시한다 | 만능 지도/스카우트 기능으로 만들지 않는다 |
| 수동 비콘 끈 | 잡아당기면 보급소에 닿지만, 캠페인에도 조금 더 들킨다 | 무료 탈출 버튼으로 만들지 않는다 |

전투 리듬:

```text
전투 능력보다 수신 상태 조절이 중심이다.
캠페인 신호를 덜 받으면 적의 등록/추적/호출은 약해진다.
대신 보급소 회수선, 긴급 인양 보정, 일부 지원 신호도 약해진다.
```

## 8. 정산 루트

| 정산 항목 | 의미 | 사용 제한 |
|---|---|---|
| 식량태그 | 대피 명단/배급 접근과 연결되는 생활 압력 | 미나의 배급 정산 기능을 빼앗지 않는다 |
| 통행태그 | 대피로, 봉쇄 우회, 낮은 수신 이동의 접근 조건 | 정찰/잠입 만능키로 쓰지 않는다 |
| 수신태그 | 구조 방송, 비콘, 회수선, 지도 신호의 불안정성 | 방송/추천 캠페인 전체로 확장하지 않는다 |
| 인양 안정도 | 보급소가 출격자를 찾을 수 있는 정도 | 강점과 약점이 동시에 되도록 유지한다 |
| 보급 압력 | 침묵권 유지비와 실제 물자 부족 | 보급소가 큰 왕국처럼 보이지 않게 한다 |
| 저소음 비콘 로그 | HP 0/긴급 인양의 단서 | DRAIN-MARK 표식 로그와 구분한다 |

정산 감각은 "더 조용해져서 강해진다"가 아니다.

```text
조용해질수록 캠페인에게 덜 읽히지만,
보급소에게도 덜 읽힌다.
```

## 9. 상태값 반응

| 캠페인 농도 높음 | 저소음 이동의 이득이 커지지만, 실패 시 추적 신호가 더 거칠게 튄다 | 덜 들리는 것이 완전한 안전은 아니다 |
| 보급소 노출도 높음 | 여울이 수신을 낮추려 하지만, 보급소 위치 노출과 인양 지연 사이에서 갈등한다 | 보급소는 보호소이면서 표적이다 |
| 인간 안정도 낮음 | 대피 명단과 배급 줄이 무너지며, 여울의 조용한 이동 규칙이 오해받는다 | 조용함은 신뢰 부족으로 읽힐 수 있다 |
| 로봇 자율성 높음 | PATCH와 리오 축을 직접 설명하지 않되, 비콘/충전/수신 장비가 로봇에게도 부담임을 짧게 비춘다 | 로봇 서사를 앞당기지 않는 보조 반응 |
| 수신 안정도 낮음 | 적의 추적은 줄지만 보급소 지원과 회수선도 흔들린다 | 여울의 핵심 리스크 |
| 인양 안정도 낮음 | HP 0이 늦게 잡히고 손실이 커진다 | 은신이 아니라 회수 위험의 캐릭터임을 보여준다 |
| 보급 압력 높음 | 식량태그/배급 물자 정산에서 여울의 대피 명단 거부가 문제로 드러난다 | 미나 기능을 빼앗지 않고 보급소 압박을 키운다 |

상태값 반응의 목표:

```text
여울은 안전한 은신 플레이어블이 아니라,
낮은 수신과 약한 인양 사이에서 값을 치르는 플레이어블 후보다.
```

## 10. HP 0 / 긴급 인양 차별화

여울의 HP 0은 사망/부활이 아니다. "대피 명단 고정"과 "비콘 약화"가 동시에 진행되며, 보급소 회수선이 늦게 좌표를 잡는 긴급 인양이다.

| 단계 | 상태 | 연출 |
|---|---|---|
| 1 | 수신 공백 | 캠페인 신호도 약하지만, 보급소 비콘도 흔들린다 |
| 2 | 대피 명단 고정 | 주변 절차가 여울을 피난자/구조 대기자로 묶으려 한다 |
| 3 | 저소음 비콘 지연 | 큰 구조 신호를 내면 캠페인도 듣기 때문에, 짧은 수동 비콘만 남는다 |
| 4 | 방호천 접힘 | 접이식 방호천이 위치를 숨기던 기능을 잃고, 회수선이 잡을 만큼만 흔적을 남긴다 |
| 5 | 긴급 인양 | 보급소가 늦게 잡힌 저소음 비콘 로그를 따라 끌어오지만, 미정산 태그와 일부 지도 조각 손실이 커진다 |

고정 문장:

```text
여울은 쓰러지는 순간 더 조용해지는 것이 아니라,
보급소가 겨우 들을 만큼만 들리는 상태로 밀린다.
```

차별점:

- 윤서처럼 마지막 좌표를 즉시 회수선이 잡지 않는다.
- PATCH처럼 정품 복구/부품 회수 직전에 끊기지 않는다.
- 하람처럼 퇴원 취소/병상 재배정으로 돌아가지 않는다.
- 이루처럼 수취인 확인 실패/미수령 보관이 아니다.
- 유건처럼 계량 초과/충전 차단이 아니다.
- DRAIN-MARK처럼 숨은 루트 NPC의 경고로 끝나지 않는다.
