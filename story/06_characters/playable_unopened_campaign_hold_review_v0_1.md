## 0. 결론

이번 재검수는 후보를 더 만드는 작업이 아니다.

결론:

1. MODEL-15는 방향을 보존하되 E10_C05 개방 전 직접 해금과 최종 고정을 금지한다.
2. LOCKOUT-17은 E06_C04 개방 전 프로필 v1.0 고정을 금지하고, 기존 산출물을 기능 초안으로 강등한다.
3. NAME-18은 상세 결재를 시작하지 않는다.
4. RATION, LINE, OBS 계열도 캠페인 개방 전 상세 결재를 금지한다.
5. 13~19 한국어 임시명은 legacy placeholder로만 보존한다.
6. 출시 초기 한국인 플레이어블은 기본 주인공과 추가 1~2명 제한을 유지한다.

## 3. MODEL-15 재검수

MODEL-15는 폐기하지 않는다.
Maya Chen / MODEL-15는 E10_C05 알고리즘 추천권 후보로서 한국어권 과밀을 줄이고, 추천 모델 학습과 광고 타깃 확정이라는 절차를 비교적 선명하게 잡았다.

하지만 아래는 금지한다.

```text
E10_C05 개방 전 직접 해금.
출시 초기 합류.
프로필 v1.0 최종 고정.
E10 전체 추천/빅테크/앱스토어 안내자화.
```

MODEL-15 관련 산출물은 `기능 초안`과 `개방 후 재검토 후보`로만 읽는다.

## 4. LOCKOUT-17 재검수

LOCKOUT-17은 과속 산출물이다.
E06_C04가 열리기 전 상세 결재, 조건부 해소, 프로필 v1.0 산출물까지 이어졌으므로 최종 고정으로 취급하면 안 된다.

재분류:

```text
lockout_17_playable_decision_v0_1.md:
기능 초안 / 보류 산출물

lockout_17_playable_condition_patch_v0_1.md:
조건부 해소 초안 / 보류 산출물

lockout_17_profile_v1_0.md:
프로필 v1.0 최종 고정 아님 / 기능 초안
```

오마르의 안전잠금, 작업자 ID, 비상정지줄, 충전 접근 차단 이미지는 유지한다.
그러나 E06_C04 캠페인 개방 전에는 통과 후보로 고정하지 않는다.

```text
한국어 이름이 편하다는 이유로 복구하지 않는다.
```

## 7. 출시 초기 한국인 제한

출시 초기에는 한국인 기본 주인공과 추가 1~2명까지만 허용한다.

유지:

```text
윤서:
기본 주인공.

추가 한국인:
최대 1~2명. 별도 승인 필요.
```

금지:

```text
3~7번 한국인 인간 후보를 모두 출시 초기 플레이어블처럼 배치.
E01 한반도 권역 캐릭터를 계속 증식.
14번 이후 한국어 임시명을 최종명으로 복구.
전역 캠페인이 열리기 전에 한국인 후보로 빈 슬롯 채우기.
```

이 제한은 기존 산출물을 지우는 뜻이 아니다.
프로필 산출물과 최종 해금 순번을 분리한다는 뜻이다.
