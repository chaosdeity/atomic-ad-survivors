## 0. 조건부 통과 사유 요약

기존 상세 결재 문서는 6번 폐충전소 계량자를 조건부 통과로 판정했다.

조건부 판정의 이유:

1. R04 정품/충전/산업 캠페인을 로봇 없이 여는 장점은 강하지만, 단순 전기 마법사처럼 보일 위험이 크다.
2. 충전 케이블, 차단기, 과금 미터 이미지가 엔지니어, 수리공, 충전 버퍼 직업으로 납작해질 수 있다.
3. 충전태그를 쓰기 때문에 PATCH의 보급소 수리 로봇/자기부품화/수리 축과 겹칠 수 있다.
4. 정품 충전 도크와 재동기화 위험을 앞세우면 리오의 정품 복구 공포를 너무 빨리 침범한다.
5. R04-L03~L05까지 열면 정품/충전권 전체와 해방 로봇 장기 분기가 초반에 과확장된다.

보완의 목표:

```text
이 후보를 전기를 쓰는 사람이 아니라,
전기를 받기 위해 계량당하는 인간으로 잠근다.
```

## 1. 해결해야 할 위험

| 위험 | 문제 | 이번 문서의 잠금 |
|---|---|---|
| 단순 전기 마법사화 | 계량/차단 절차가 사라지고 번개 피해, 감전 공격, 전기 속성 캐릭터가 된다 | 전투 리듬은 피해 속성이 아니라 계량 초과, 차단, 누전, 과금 기준을 흔드는 방식으로 고정한다 |
| 엔지니어/충전 버퍼화 | 장비를 고치거나 아군을 충전해 주는 지원 직업으로 보일 수 있다 | 수리와 보급소 장비 안정은 PATCH 영역으로 남긴다. 유건은 충전량/사용량/차단 기준을 비트는 사람이다 |
| PATCH와 충전태그/수리 축 겹침 | 두 캐릭터가 모두 충전태그와 장비 안정에 연결되면 역할이 흐려진다 | PATCH는 자기 몸을 부품처럼 써온 보급소 수리 로봇, 유건은 계량당하는 인간으로 분리한다 |
| 리오의 정품 복구 공포 조기 침범 | 정품 도크와 재동기화 공포를 깊게 다루면 리오 8번 보류 축이 약해진다 | 정품 충전 도크는 HP 0 위험 이미지로 제한하고, 정품 복구 공포와 객실 초기화는 리오 영역으로 남긴다 |
| R04 정품/충전권 과확장 | R04-L03 리콜 펌웨어실, R04-L05 재동기화 코어까지 당길 수 있다 | R04-L01 폐충전소와 R04-L02 배터리 침전 공장 단서까지만 허용한다 |

## 2. PATCH와의 차이 고정

핵심 차이:

```text
PATCH:
자기 몸을 부품처럼 써온 보급소 수리 로봇.

유건:
계량당하는 인간, 충전량/사용량/차단 기준을 비트는 사람.
```

비교 잠금:

| 비교축 | PATCH | 차유건 / METER-06 |
|---|---|---|
| 존재 유형 | 보급소 수리 로봇 | R04 폐충전소의 인간 후보 |
| 핵심 상처 | 자기부품화 | 자기계량화 |
| 캠페인이 붙이는 역할 | 정품 부품, 리콜 대상, 수리 자재, 서비스 부품 묶음 | 충전 이용자, 계량 대상, 정품 충전 대기자, 전력 사용량 초과자 |
| 전투 출발점 | 결합부, 임시 수리, 리벳, 패치 테이프 | 계량기, 충전 케이블, 차단기, 과금 미터, 전력 사용량 영수증 |
| 충전태그 감각 | 로봇이 버티기 위한 생존 접근권 | 사람이 전력 사용자로 등록되는 접근권 |
| 위험 | 쓸모 있게 분해된다 | 정확하게 기록되어 소유 가능한 사용량이 된다 |
| HP 0 | 부품 목록화/정품 복구 직전 회수 | 계량 초과/충전 차단/정품 도크 회수 직전 인양 |

PATCH의 문장:

```text
고쳐주는 것과, 부품이 되는 것은 다르다.
```

유건의 문장:

```text
충전받는 것과, 사용량이 되는 것은 다르다.
```

금지:

```text
유건이 PATCH를 충전해 주는 전용 배터리가 되지 않는다.
유건이 PATCH 대신 보급소 장비를 고치지 않는다.
유건이 로봇 자율성 문제를 해설하지 않는다.
유건의 충전태그는 수리재가 아니라 전력 접근권으로만 쓴다.
```

## 3. 리오와의 차이 고정

핵심 차이:

```text
리오:
체류/숙박 캠페인권의 객실 정리 로봇, 정품 복구 공포.

유건:
R04 폐충전소의 전력 계량/충전 차단/정품 도크 회수 직전 인간.
```

비교 잠금:

| 비교축 | 리오 | 차유건 / METER-06 |
|---|---|---|
| 슬롯 | 8번 보류 잠금 | 6번 후보 |
| 존재 유형 | 탈정품 객실 정리 로봇 | 계량당하는 인간 |
| 출신 축 | 체류/숙박 캠페인권 | R04 정품/충전/계량/전력 |
| 핵심 공포 | 멀쩡하지만 다시 정품 장치로 복구되는 것 | 전기를 받기 위해 계량 가능한 사용량으로 고정되는 것 |
| 대표 이미지 | 객실 카드, 청소 브러시, 초기화 램프, 객실 로그 | 계량기, 충전 케이블, 차단기, 과금 미터, 전력 사용량 영수증 |
| 금지 침범 | 리오의 정품 복구 공포를 앞당기지 않는다 | 유건의 정품 도크는 위험 이미지로만 제한한다 |

리오의 문장:

```text
비어 있지 않은 방을 비어 있다고 정리하지 않기로 한 로봇.
```

유건의 문장:

```text
남은 전력을 받기 위해 자기 사용량이 되지 않기로 한 사람.
```

금지:

```text
유건 문서에서 객실 초기화, 청소 로그, 흔적 삭제, 체류/숙박 캠페인을 다루지 않는다.
유건의 정품 도크를 리오의 정품 복구 공포처럼 깊게 설명하지 않는다.
R04-L05 재동기화 코어를 유건 해금 조건으로 쓰지 않는다.
```

## 4. 중심 이미지 고정

중심 이미지는 다섯 가지로 고정한다.

```text
계량기
충전 케이블
차단기
과금 미터
전력 사용량 영수증
```

위험 이미지:

```text
정품 충전 도크
```

세부 판정:

| 이미지 | 채택 | 역할 | 이유 |
|---|---|---|---|
| 계량기 | 중심 | 사용량, 초과 임계값, 수동 계량 로그를 보여준다 | 유건이 전기를 쏘는 사람이 아니라 계량당하는 사람임을 가장 빨리 보여준다 |
| 충전 케이블 | 중심 | 접지, 임시 우회, 누전 경로, 짧은 충전 창을 만든다 | 회수선이 아니라 위험한 전력 흐름의 물리 이미지다 |
| 차단기 | 중심 | 절차를 끊거나, 닫히는 시간을 늦추거나, 일부러 위험하게 올린다 | 전투 리듬의 켜고 끄는 박자를 담당한다 |
| 과금 미터 | 중심 | 초과 사용, 위험 보상, 추적 부담을 표시한다 | 충전이 공짜 회복이 아니라 기록과 부담을 남긴다는 감각을 만든다 |
| 전력 사용량 영수증 | 중심 | 어떤 행동이 초과로 청구되었는지 남기고 역정산한다 | 숫자가 사람을 묶는다는 주제를 전투 후에도 남긴다 |
| 정품 충전 도크 | 위험 제한 | HP 0 때 회수하려는 환경 장치 | 리오의 정품 복구 공포와 재동기화 코어를 침범하지 않기 위해 주 무기로 쓰지 않는다 |

이미지 결론:

```text
유건의 손에 들리는 것은 번개가 아니다.
유건의 주변에 따라붙는 것은 계량값, 케이블, 차단기, 과금 기록이다.
```

금지:

- 손에서 전기 구체를 쏘는 연출.
- 충전 케이블을 윤서의 회수선처럼 사람 구출용 줄로 쓰는 연출.
- 정품 충전 도크를 개인 장비나 성장 장비처럼 들고 다니는 구조.
- 과금 미터를 골드, 돈, 점수 보너스 장치로 소비하는 구조.

## 5. 전투 리듬

핵심 판정:

```text
충전해주는 버퍼가 아니라,
계량 초과/차단/누전/과금 기준을 비트는 리듬.
```

전투 리듬:

| 리듬 | 의미 | 플레이 감각 |
|---|---|---|
| 계량 초과 흔들기 | 초과 판정이 확정되기 전 값을 위험하게 튼다 | 적의 등록/추적/공격 타이밍을 한 박자 늦춘다 |
| 충전 차단 역이용 | 차단기가 내려가는 순간을 공격/회피 템포로 쓴다 | 안전을 잃는 대신 위험 절차도 같이 끊는다 |
| 누전 접지 | 새는 전류를 짧게 접지해 위험 칸을 바꾼다 | 장판을 지우는 것이 아니라 흐름을 바꾼다 |
| 과금 기준 전환 | 누가 초과 사용자인지 임시로 바꾼다 | 적의 과충전 행동을 다음 정산 부담으로 돌린다 |
| 사용량 자해성 조정 | 자기 전력 사용량을 일부러 위험하게 흔든다 | 강한 행동일수록 자기 계량 초과와 과방전 위험이 커진다 |

자기 전력 사용량을 흔드는 방식:

- 차단기가 내려가기 직전까지 충전 케이블을 붙잡는다.
- 계량기 바늘이 초과선을 넘나들게 만들어 짧은 행동 창을 만든다.
- 과금 미터에 일부 사용량을 남겨 다음 정산 부담을 감수한다.
- 누전 경로를 자신 쪽으로 끌어 짧게 접지하지만, 과방전 위험을 남긴다.

전투 결론:

```text
유건은 아군을 안정적으로 충전하지 않는다.
유건은 위험한 충전 절차가 닫히는 순간을 비틀어,
지금 한 번 더 움직일 수 있는 불안정한 틈을 만든다.
```

허용되는 보조:

- 아주 짧은 충전 창 제공.
- 차단 직전 한 번의 행동 기회 제공.
- 누전 장판의 위험 방향 변경.
- 초과 판정 지연.

금지:

- 아군 에너지를 지속적으로 채우는 버퍼.
- 보급소 충전 스탠드를 대체하는 캐릭터.
- 장비 고장과 수리를 해결하는 엔지니어.
- 전기 피해량이 정체성의 대부분인 속성 딜러.

## 6. 정산 루트

정산 루트는 아래 네 가지로 고정한다.

| 정산 루트 | 역할 | 사용 방식 | 위험 |
|---|---|---|---|
| 충전태그 | 전력 접근 승인권 | 전기를 받아도 되는 조건, 초과 사용 여부, 차단 기준을 연다 | PATCH의 로봇 생존/수리 태그와 겹치지 않게 전력 사용자 접근권으로 제한한다 |
| 전력 사용 기록 | 이 후보의 핵심 기록 | 누가 얼마나 썼는지, 어떤 행동이 초과였는지 남긴다 | 정확한 기록이 공정함과 소유를 동시에 만든다는 압력을 유지한다 |
| 폐충전소 계량 로그 | 흡수되지 않은 근거 | 수동 계량기, 과금 미터, 차단기 로그가 서로 어긋난 기록을 남긴다 | 단순 던전 로그나 수집품으로 만들지 않는다 |
| 정품 충전권 | 위험 보상 | 안전한 충전 접근을 약속하지만 정품 도크 회수 위험도 함께 남긴다 | 주 정산 루트가 되면 R04 정품권 전체를 너무 빨리 연다 |

정산 결론:

```text
유건의 정산은 충전을 많이 받는 일이 아니다.
어떤 충전이 누구의 사용량으로 기록되었고,
그 기록이 다음 출격에서 무엇을 차단할지 결정하는 일이다.
```

정품 충전권 제한:

- 정품 충전권은 희귀 위험 보상이다.
- 정품 충전권은 안전을 주지만 정품 도크 회수 위험을 남긴다.
- 정품 충전권은 R04-L03~L05 해금 열쇠가 아니다.
- 정품 충전권을 많이 모아도 리오의 정품 복구 공포를 설명하지 않는다.

정산 금지:

```text
충전태그를 마나나 골드처럼 쓰지 않는다.
전력 사용 기록을 단순 점수판으로 만들지 않는다.
폐충전소 계량 로그를 장비 제작 재료로만 쓰지 않는다.
정품 충전권을 무조건 좋은 보상으로 만들지 않는다.
```

## 7. HP 0 / 긴급 인양 차별화

핵심 판정:

```text
HP 0의 중심은 계량 초과다.
시각적 고정은 충전 차단이다.
신체/장비 상태는 과방전이다.
위험 결말은 정품 도크 회수 직전 인양이다.
```

단계별 잠금:

| 단계 | 핵심 상태 | 연출/의미 |
|---|---|---|
| HP 0 직전 | 계량 초과 | 계량기 바늘과 과금 미터가 동시에 초과선을 넘는다 |
| HP 0 | 충전 차단 | 폐충전소 차단기가 내려가고 케이블이 잠긴다 |
| 전투 불능 상태 | 과방전 | 몸과 장비가 같은 사용량으로 묶이며 움직임이 둔해진다 |
| 인양 직전 위험 | 정품 도크 회수 | 정품 충전 도크가 "안전 회수" 명목으로 닫히기 시작한다 |
| 긴급 인양 성공 | 접지 좌표 가로채기 | 보급소가 수동 계량 로그의 불일치 접지 좌표를 잡아당긴다 |
| 복귀 후유증 | 전력 사용 기록 오염 | 다음 출격에서 초과 부담, 차단 임계값 흔들림, 정품 추적 잔류가 남을 수 있다 |

윤서/PATCH/하람/이루와의 차이:

| 캐릭터 | HP 0 위기 | 긴급 인양 감각 |
|---|---|---|
| 윤서 | 마지막 좌표에서 회수 대상처럼 끌려올 위기 | 회수선이 사람/물건 판정을 보류하며 당긴다 |
| PATCH | 부품 목록화/정품 복구로 회수될 위기 | 보급소 수리선이 부품화 직전 끊어 낸다 |
| 강하람 / HOLD-04 | 퇴원 취소와 병상 재배정으로 다시 환자화될 위기 | 팔찌 절단 신호와 들것 회수선이 재입원 절차 직전 당긴다 |
| 한이루 / RETURN-05 | 수취인 확인 실패와 미수령 보관으로 배송 상태화될 위기 | 반송 로그의 역배송 좌표를 보급소가 가로챈다 |
| 차유건 / METER-06 | 계량 초과와 충전 차단으로 정품 도크에 회수될 위기 | 수동 계량 로그의 불일치 접지 좌표를 보급소가 당긴다 |

한 줄 정의:

```text
계량 초과로 충전 차단이 걸리고 정품 도크가 닫히기 직전,
보급소가 폐충전소 계량 로그의 불일치 접지 좌표를 잡아 긴급 인양한다.
```

금지:

- 감전 사망.
- 충전 후 부활.
- 배터리 교체로 즉시 회복.
- PATCH가 와서 수리해 주는 구조.
- 리오식 정품 복구 공포를 직접 재현하는 구조.

## 8. R04 노출 범위 제한

R04 전체를 열지 않는다.

허용 범위:

| 노드 | 사용 여부 | 사용 목적 |
|---|---|---|
| R04-L01 폐충전소 | 사용 | 계량기, 충전 케이블, 차단기, 과금 미터, 정품 도크 위험을 처음 보여준다 |
| R04-L02 배터리 침전 공장 | 제한 사용 | 과방전, 누전, 회색 오염 산업형 장판 단서만 보여준다 |
| R04-L03 리콜 펌웨어실 | 금지 | 해방 로봇 서사 핵심이므로 PATCH/리오 장기축을 위해 보류한다 |
| R04-L04 폐기 프레스 라인 | 금지 | P2 압착/폐기 선택으로 남긴다 |
| R04-L05 재동기화 코어 | 금지 | 리오의 정품 복구 공포와 로봇 자율성 분기까지 앞당긴다 |

노출 원칙:

```text
유건은 R04 전체를 설명하는 캐릭터가 아니다.
유건은 폐충전소의 수동 계량과 충전 차단 절차가 사람을 어떻게 읽는지만 보여준다.
```

허용 정보:

- 충전태그는 전력 사용 승인권이다.
- 정품 충전권은 안전하지만 위험 보상을 남긴다.
- 폐충전소에는 수동 계량기와 과금 미터가 충돌하는 로그가 있다.
- 배터리 침전 공장 쪽에서는 누전/과방전/회색 오염 장판 단서만 보인다.

금지 정보:

- R04 정품/충전권 전체 구조 해설.
- 리콜 펌웨어실 세부.
- 재동기화 코어의 규칙.
- 해방 로봇 장기 분기 결말.
- 리오의 정품 복구 공포를 설명하는 대사.
