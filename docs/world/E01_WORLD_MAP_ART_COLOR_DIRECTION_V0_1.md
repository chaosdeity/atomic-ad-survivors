## 0. 판정 요약

- 월드맵은 기존 전투 HUD의 베이지/노랑 계열을 그대로 쓰지 않는다.
- 월드맵의 기본 정체성은 `침묵 보급소의 수신 지도`다.
- 핵심 색은 어두운 청흑 배경, 낮은 채도의 지도 회색, 청록 인양선, 코랄 광고 위험색이다.
- 노란색은 보스/주의 신호의 작은 점멸에만 쓴다. 넓은 배경, 지도 표면, 버튼 바탕에는 쓰지 않는다.
- 황혼/노을/불타는 지도처럼 보이면 실패다.
- R08/R01 P0는 선명하게, R02/R03/R04 P1은 흐린 잡음으로 보인다.
- 색은 기능을 먼저 구분하고, 장식은 나중에 붙인다.

가장 중요한 잠금:

```text
월드맵은 따뜻한 포스트아포칼립스 풍경이 아니라,
끊긴 생활권 신호를 수동으로 붙잡는 차가운 출격 장비다.
```

## 1. 화면 재료감

| 요소 | 재료감 | 색 방향 |
|---|---|---|
| 지도 표면 | 오래된 e-ink 수신판, 정비소 모니터, 종이 위에 덧댄 신호 레이어 | 청흑, 먹색, 낮은 회청 |
| 침묵권 | 전파가 비어 있는 공백, 약한 차폐, 보급소 장비의 안정 신호 | 청록, 민트, 희미한 백색 |
| 캠페인 신호 | 광고가 사람을 읽는 압력, 심사/등록/감사 경고 | 코랄, 적색, 아주 작은 노랑 |
| P1/P2 원격 신호 | 아직 해석되지 않은 권역 신호 | 저채도 회색, 백색 잡음, 흐린 라벨색 |
| 선택 패널 | 정비판, 라벨 프린터, 출격 게시판 | 어두운 패널 + 얇은 밝은 선 |

```text
지도 배경, 노드 기본색, 패널 바탕에 쓰지 않는다.
```

## 7. P0와 P1 대비

P0는 선명해야 하고, P1은 존재만 느껴져야 한다.

| 구분 | P0 | P1 |
|---|---|---|
| 불투명도 | 85~100% | 18~38% |
| 라벨 | 노드명 표시 | 대부분 라벨 없음 |
| 선 | 실선/점선 구분 명확 | 흐린 잡음/짧은 끊김 |
| 클릭 | 클릭 가능 | 클릭 가능해도 정보만 표시 |
| 버튼 | 출격/정비/장비 확인 | 신호 해석 필요 |

P1 신호가 너무 예쁘거나 선명하면 유저가 첫 목표를 잃는다. P1은 유혹이 아니라 배경의 압력이어야 한다.

## 11. 금지 팔레트

| 금지 조합 | 이유 |
|---|---|
| 노랑+주황+갈색이 화면 대부분을 차지 | 황혼/불탄 도시/서부극처럼 보인다. |
| 베이지 패널이 화면 대부분을 차지 | 기존 보급소 UI와 겹치고 지도 장비감이 약하다. |
| 보라/남색 그라데이션이 지배 | 흔한 SF 지도처럼 보이고 광고 생활권 색이 사라진다. |
| 빨강을 넓은 면으로 깔기 | 모든 것이 위험으로 보여 정보 위계가 무너진다. |
| 순백 의료 신호를 크게 깔기 | R02가 첫 화면 주인공처럼 보인다. |
| 형광 녹색을 넓게 쓰기 | 오염/해석/성공의 구분이 사라진다. |

허용:

- 노랑은 작은 점멸/주의 마크.
- 베이지는 종이 태그 조각과 회수물 미리보기의 소량 배경.
- 빨강은 보스 링, 위험 호, 선택 경고.

## 14. 적용 우선순위

## 15. 검증 체크리스트

| 항목 | 기준 |
|---|---|
| 황혼/노을/불색처럼 보이지 않는가? | 노랑/주황/갈색 면적이 작다. |
| R08/R01 P0가 가장 먼저 보이는가? | 청록 실선과 선명한 라벨이 P0에만 있다. |
| R01-L03 보스 신호가 목표로 보이는가? | 코랄 링과 작은 노랑 점멸이 있다. |
| P1이 너무 선명하지 않은가? | 라벨 없는 낮은 알파 잡음 중심이다. |
| 광고색과 인양색이 분리되는가? | 광고는 코랄/적색, 인양은 청록/민트다. |
| 태그가 돈/포인트처럼 보이지 않는가? | 상단 숫자보다 노드 회수물 맥락에 둔다. |
| 기존 HUD와 충돌하지 않는가? | 기존 고채도색은 강조로만 제한한다. |
