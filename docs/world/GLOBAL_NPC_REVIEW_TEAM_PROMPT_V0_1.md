# 전역 NPC 검수팀 프롬프트 v0.1

이 문서는 NPC 제작 결과를 검수팀에게 넘기기 위한 복붙용 프롬프트다.

검수팀의 역할은 NPC를 더 만드는 것이 아니다.
검수팀의 역할은 `1,024개 전역 제작 슬롯`을 캠페인, 현실권, 계층, 이름, 플레이어블 충돌 기준으로 잘라내고 통과/조건부/하향/폐기를 판정하는 것이다.

---

## 0. 검수 대상

```text
게임:
Atomic Ad Survivors

검수 범위:
E01~E12 / 96개 정식 캠페인 / cross_region

검수 기준 수량:
1차 게이트 420개 씨앗
전역 충족 기준 768개 제작 슬롯
검수 전 과충족 기준 1,024개 제작 슬롯

주의:
1,024개는 직접 대면 NPC 1,024명이 아니다.
1,024개는 L5/L4/L3/L2/L1/L0 전체 슬롯이다.
```

## 1. 절대 원칙

```text
새 NPC를 만들지 않는다.
새 플레이어블을 만들지 않는다.
E01 한국권 NPC를 추가하지 않는다.
한국어 임시명으로 E02~E12 슬롯을 채우지 않는다.
1,024개를 전부 직접 대면 NPC로 해석하지 않는다.
L3 이하에 본명과 과거사를 붙이지 않는다.
R01 확정 NPC, 보급소 5명, 플레이어블 기능을 대체하지 않는다.
실제 도시/기관/회사 실명을 유저 노출용으로 쓰지 않는다.
캠페인 미개방 지역의 v1.0 장문 프로필을 작성하지 않는다.
```

## 2. 검수 목표

검수팀은 아래를 판정한다.

1. 1,024개 슬롯 수량이 맞는가.
2. 1,024개 중 직접 대면 후보가 과밀하지 않은가.
3. E01이 추가 생산되지 않았는가.
4. E02~E12가 각 현실권/언어권/등록 절차에 맞는가.
5. L5/L4만 이름 후보를 검토하고 있는가.
6. L3 이하가 역할명/로그명/잔향명/오브젝트명/목록명으로 처리되는가.
7. 플레이어블의 기능, 해금, 전투, 성장, 정체성을 침범하지 않는가.
8. 보급소 5명과 R01 확정 NPC의 역할을 대체하지 않는가.
9. 각 슬롯이 캠페인의 제도, 오브젝트, 상태 반응, 작은 결말 중 하나를 담당하는가.
10. 검수 후에도 768개 충족 기준을 밑돌지 않는가.

## 3. 검수 판정값

판정 값은 아래 중 하나만 쓴다.

| 판정 | 의미 |
|---|---|
| 통과 | 현재 계층/권역/이름 정책 유지 |
| 조건부 통과 | 수량 유지 가능. 이름, 노출, 기능 문구만 수정 |
| 이름 재검토 | L5/L4 이름 후보가 현실권과 맞지 않음 |
| 현지권 재검토 | 캠페인 현실권/언어권/등록 절차와 맞지 않음 |
| 계층 하향 | 직접/제한 대면이 과함. L3 이하로 내림 |
| 오브젝트화 | 인물 슬롯보다 로그/문구/오브젝트가 적합 |
| 보류 | 시즌 오픈 전 상세화 금지 |
| 폐기 | 기능 중복, 플레이어블 침범, 현실권 오류가 큼 |

## 4. 필수 검수표

아래 표를 반드시 작성한다.

| npc_id | 캠페인 | 현실권/언어권 | 계층 | 이름 필요 여부 | 이름/표기 | 기능 | 플레이어블 침범 위험 | 판정 |
|---|---|---|---|---|---|---|---|---|

작성 규칙:

```text
L5/L4는 슬롯 단위 또는 묶음 단위로 검수한다.
L3 이하는 개별 본명 검수가 아니라 역할/오브젝트/로그 묶음 단위로 검수한다.
E03~E12 반복 패턴은 권역별 대표 행과 위험 행을 나누어 검수한다.
cross_region은 스포일러 등급과 장기 미스터리 침범 여부를 별도 확인한다.
```

## 5. 수량 검산표

아래 숫자가 맞는지 검산한다.

| 범위 | 기준 수량 | 검수팀 허용 처리 |
|---|---:|---|
| E01 | 120 | 증가 금지. 하향/오브젝트화만 가능 |
| E02 | 96 | 직접 대면 과밀 시 L3/L2로 하향 |
| E03~E12 | 760 | 권역별 76 유지. 이름 과밀 시 역할명화 |
| cross_region | 48 | 스포일러 과밀 시 보류 |
| 합계 | 1,024 | 검수 후 768 아래로 떨어지면 실패 |

계층 목표:

| 계층 | 검수 전 기준 | 검수 방향 |
|---|---:|---|
| L5 | 36 | 늘리지 않음 |
| L4 | 108 | 과밀하면 L3/L2로 하향 |
| L3 | 233 | 상태 반응 중심 |
| L2 | 322 | 잔향/로그/오브젝트 중심 |
| L1 | 186 | 목록/호출 번호 중심 |
| L0 | 139 | 제도 흔적/자동 문구 중심 |
| 합계 | 1,024 | 통과 |

## 6. E02 이후 필수 질문

E02 이후 모든 검수 묶음에 아래 질문을 답한다.

1. 이 슬롯이 왜 한국어 이름이면 안 되는가?
2. 해당 현실권/언어권/등록 절차에 맞는가?
3. 이름이 정말 필요한 계층인가?
4. 이름 없이 오브젝트/로그/잔향으로 처리해도 되는가?
5. 플레이어블 기능을 빼앗고 있지 않은가?
6. 보급소 5명 또는 R01 확정 NPC 기능을 대체하지 않는가?
7. 이 슬롯은 작은 결말, 미스터리, 반전, 상태 후폭풍 중 무엇을 담당하는가?
8. 시즌 오픈 전 공개하면 안 되는 스포일러인가?

```text

```
