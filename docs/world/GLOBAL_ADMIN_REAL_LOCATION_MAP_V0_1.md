# Atomic Ad Survivors 전 지구 관리자용 현실 위치 매트릭스 v0.1

상태: v0.1 내부 제작 기준
목적: 유저에게 직접 노출하지 않을 전 지구 캠페인의 현실 위치, 도시권, 인프라 앵커를 고정한다.
연결 문서: `docs/world/GLOBAL_CAMPAIGN_CATALOG_V0_1.md`, `docs/world/GLOBAL_CAMPAIGN_MAP_PLACEMENT_V0_1.md`, `docs/world/E01_ADMIN_REAL_LOCATION_MAP_V0_1.md`, `docs/world/E01_ADMIN_CITY_DONG_LOCATION_LOCK_V0_1.md`, `docs/world/E02_PLUS_REGION_EXPANSION_PRINCIPLES_V0_1.md`

## 0. 판정 요약

전 지구 12개 광역권과 96개 정식 캠페인은 모두 관리자용 현실 위치를 가진다.

유저용 지도에는 실명 지명을 직접 노출하지 않는다. 그러나 제작자, 시나리오 팀, 레벨 팀, 월드맵 팀은 각 캠페인이 실제 지구 어디의 어떤 인프라에서 파생됐는지 알아야 한다.

잠금:

```text
광역권 12개 = 관리자용 현실 권역 고정.
정식 캠페인 96개 = 관리자용 도시권/인프라 앵커 고정.
소형 로컬 노드 = 부모 정식 캠페인의 관리자 앵커 안에서만 생성.
유저용 지도 = 실루엣 + 가명 + 10% 인식 밀도 유지.
```

주의:

```text
이 문서는 유저 공개용 세계지도 문서가 아니다.
실명 지명, 도시권, 항만, 공항, 철도, 산업축은 내부 제작자가 길을 잃지 않기 위한 기준이다.
```

## 1. 노출 원칙

| 구분 | 관리자 문서 | 유저용 지도 |
|---|---|---|
| 현실 국가/도시 | 명시 가능 | 직접 노출 금지 |
| 항만/공항/철도/산업축 | 명시 가능 | 형태와 거리감으로 암시 |
| 로컬 노드 | 현실 인프라 앵커 필요 | 캠페인식 가명으로 표시 |
| 좌표/GPS | 필요 시 내부 필드로만 보관 | 후반 기록 조각에서만 제한 암시 |
| 실제 사건 패러디 | 금지 | 금지 |

## 2. 소형 로컬 노드 배정 규칙

정식 캠페인 아래의 소형 로컬 노드도 공중에 뜨면 안 된다.

E01은 이미 `docs/world/E01_ADMIN_REAL_LOCATION_MAP_V0_1.md`에서 40개 로컬 노드의 관리자 위치를 따로 가지며, `docs/world/E01_ADMIN_CITY_DONG_LOCATION_LOCK_V0_1.md`에서 국가-시/구-읍면동급 제작 앵커까지 별도로 잠근다. E02~E12의 소형 로컬 노드는 시즌 슬라이스가 열릴 때 같은 깊이로 세부화하되, 아래 96개 캠페인 앵커 밖으로 벗어나면 안 된다.

필수 필드:

```text
local_node_id:
parent_campaign_id:
admin_real_region:
admin_city_band:
admin_infra_anchor:
admin_local_anchor_status:
user_display_name:
expose_admin_geo_to_user: false
```

로컬 노드 최소 배치:

```text
도입 노드:
캠페인을 처음 체감하는 안전하지 않은 입구.

절차 노드:
등록/심사/수령/충전/통행 같은 핵심 규칙이 작동하는 곳.

갈등 노드:
사람, 로봇, 태그, 보급소, 캠페인 규칙이 충돌하는 곳.

결절 노드:
보스, 핵심 장치, 기억 추출, 권한 박탈이 일어나는 곳.

후폭풍 노드:
P1 재방문, 숨은 루트, 잔향, 회수선 기준점이 남는 곳.
```

## 5. 제작 잠금

### 5.1 큰 캠페인

큰 캠페인은 `E##_C##` 정식 캠페인이다. 이 문서의 96개를 기준으로 한다.

금지:

```text
정식 캠페인 추가 후 위치 미지정.
실제 도시권 없이 가상 권역만으로 배치.
유저용 텍스트에 실명 위치 직접 노출.
```

### 5.2 작은 캠페인

작은 캠페인은 정식 캠페인 아래의 로컬 노드, 잔향, 파편, 보스 전후 이벤트다.

규칙:

```text
작은 캠페인은 반드시 부모 정식 캠페인 ID를 가진다.
작은 캠페인은 반드시 관리자용 현실 인프라 앵커를 가진다.
작은 캠페인은 유저에게 캠페인식 가명으로만 노출한다.
작은 캠페인은 정식 캠페인 밖에서 즉흥 생성하지 않는다.
```

예시:

```text
R01-L03 모델하우스 결절
parent_campaign_id: E01_C01
admin_real_region: 대한민국 서부 수도권 외곽 신도시/모델하우스 축
user_display_name: 모델하우스 결절
expose_admin_geo_to_user: false
```
