# Atomic Ad Survivors E01 관리자용 현실 위치 매트릭스 v0.1

상태: v0.1 내부 제작/관리자 기준
목적: 유저에게는 숨기는 E01의 현실 지리 앵커를 관리자와 제작팀이 명확히 공유한다.
연결 문서: `docs/world/WORLD_MAP_CAMPAIGN_ECOLOGY_V0_1.md`, `docs/world/GLOBAL_CAMPAIGN_MAP_PLACEMENT_V0_1.md`, `docs/world/E01_FIRST_SEASON_LOCAL_NODES_V0_1.md`, `docs/world/E01_WORLD_MAP_BLOCKOUT_V0_1.md`, `docs/world/E01_WORLD_MAP_PRODUCTION_TABLE_V0_1.md`, `story/06_characters/playable_20_roster_blueprint_v0_1.md`

## 0. 판정 요약

- E01의 관리자용 현실 위치는 `대한민국 / 한반도 서부 수도권-항만-산업 벨트`로 잠근다.
- 유저용 월드맵에는 실명 지명과 실제 도시명을 직접 노출하지 않는다.
- 관리자/제작자는 실제 도시권, 항만, 공항, 병원, 물류, 산업단지, 방송/상업 축을 알아야 한다.
- 캐릭터 원점과 해금 캠페인은 이 관리자용 현실 권역을 기준으로 배치한다.
- 정확 GPS 좌표와 실제 주소는 v0.1에서 확정하지 않는다. 이 문서는 `현실 권역/도시권/인프라 축`을 잠그는 문서다.

핵심 원칙:

```text
유저는 캠페인 이름과 실루엣을 본다.
관리자는 현실 권역과 인프라 축을 본다.
런타임 UI는 관리자용 실명 필드를 노출하지 않는다.
```

## 1. 노출 레이어 규칙

| 레이어 | 보는 사람 | 사용 내용 | 노출 여부 |
|---|---|---|---|
| 유저 표시명 | 유저 | `서부 스마일홈`, `반품 회수 벨트`, `폐충전소` 같은 캠페인식 이름 | 노출 |
| 월드맵 실루엣 | 유저 | 해안선, 도시 밀도, 항만/공항 거리, 도로축, 방송탑 실루엣 | 노출 |
| 관리자 현실 권역 | 기획/아트/레벨/QA | 실제 도시권과 인프라 축 | 내부 전용 |
| 제작 세부 앵커 | 기획/아트/레벨 | 병원군, 물류 허브, 신도시, 산업단지, 검역 터미널 같은 실제 기능 | 내부 전용 |
| 정확 좌표/주소 | 추후 지도 제작 단계 | 좌표화가 필요한 경우 별도 잠금 | v0.1 미확정 |

금지:

```text
초반 UI에 서울/인천/김포/부천/시흥/안산/영종 같은 실명 도시를 그대로 쓰지 않는다.
관리자용 현실 권역 필드를 클라이언트 유저 노출 JSON으로 그대로 내려보내지 않는다.
실존 장소를 직접 풍자하거나 특정 재난 피해지처럼 보이게 하지 않는다.
```

허용:

```text
내부 문서와 관리자 툴에서는 현실 권역명을 쓴다.
아트/레벨 제작자는 실제 인프라 배치를 참고한다.
후반 기록, 깨진 좌표, 옛 지도 조각에서는 실명 일부를 암시할 수 있다.
```

## 2. E01 광역 현실 기준

E01은 아래 현실 축을 모델로 삼는다.

```text
국가/광역:
대한민국 / 한반도 서부 수도권-항만-산업 벨트

관리자용 도시권 힌트:
서울 서부/서남권
인천/영종/송도/인천항
김포/고양/파주 외곽권
부천/광명/시흥/안산 산업-물류권
수원/군포/이천형 내륙 물류/의료 확장축

유저용 표현:
E01 서부권
닫힌 항로
반품 벨트
마지막 수신권
서부 비상보급망
```

주의:

```text
이 문서의 현실 도시권 힌트는 관리자용 제작 기준이다.
유저에게는 캠페인식 이름, 신호, 태그, 실루엣, 노이즈로 변환한다.
```

| OPEN-HOST | R01 | 외곽 신도시 모델하우스/상담 동선 | R01 NPC 선등장 후 전환 후보 |
| 강하람/HOLD-04 | R02 | 대형 병원/보험 심사권 | E01 의료권 한국인 후보 유지 |
| 한이루/RETURN-05 | R03 | 수도권 물류/반품 회수축 | E01 물류권 한국인 후보 유지 |
| 차유건/METER-06 | R04 | 산업단지/폐충전소 | E01 산업권 한국인 후보 유지 |
| 백여울/QUIET-07 | R08 | 침묵권/비상보급권 | E01 피난권 한국인 후보 유지 |

중요:

```text
E01 초반에 8명 주인공이 모두 존재하는 것이 아니다.
P0는 윤서 중심, PATCH 초반 합류, R01 NPC 선등장이 핵심이다.
4~7번은 P1/P1+ 확장 후보이며, 8번 리오는 중반 외부 권역 후보로 잠근다.
9~20번은 각자 열리는 실제 광역권에 맞춰 현지명/다국적 이름으로 다시 판정한다.
```

## 7. 최종 요약

E01은 가상 대륙이 아니다.

관리자 기준으로는 대한민국 서부 수도권-항만-산업 벨트다.

하지만 유저는 처음부터 그 이름을 읽지 않는다.

유저는 해안선, 도시 밀도, 항만/공항 거리, 도로축, 모델하우스, 병원, 물류창고, 충전소, 방송탑을 통해 어렴풋이 알아차린다.

따라서 앞으로 모든 맵/캐릭터/캠페인 배치는 아래 기준을 따른다.

```text
유저에게는 캠페인명.
관리자에게는 현실 권역.
제작자에게는 인프라 앵커.
```
