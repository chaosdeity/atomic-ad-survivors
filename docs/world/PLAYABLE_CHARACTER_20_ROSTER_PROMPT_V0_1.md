# Atomic Ad Survivors 플레이어블 20인 로스터 프롬프트 v0.1

아래 프롬프트는 윤서 한 명만 다루는 용도가 아니라, `Atomic Ad Survivors`의 장기 플레이어블 캐릭터 20명을 만들고 한 명씩 결재하기 위한 용도다.
