## 0. 판정 요약

세계관 1차 잠금은 완료로 보되, 세계 원인/윤서 제약/첫 30분 전달 순서는 `WORLD_FOUNDATION_LOCK_V0_3.md`를 상위 기준으로 본다.

이 문서는 새 설정을 늘리는 문서가 아니다. 이미 결정한 설정을 제작 기준으로 묶고, 이후 작업에서 함부로 바꾸면 안 되는 선을 표시한다. 단, 세계 원인 검증은 `WORLD_CAUSE_REVIEW_PROMPT_V0_1.md`로 평가하고, 그 실행 결과는 `WORLD_CAUSE_REVIEW_RESULT_V0_1.md`를 기준으로 본다.

핵심 판정:

- 세계 원인은 단일 악당 AI나 단일 대사고가 아니라, `라이프라인 확산기 -> 대동기화 사고 -> 도시 생명유지망 붕괴 -> 등록 전쟁과 인간 약탈 -> 캠페인 생태 고착`의 5단계 기반 위에 권역별/로컬 원인이 얹히는 구조다.
- 현재 시점은 대사고 14년 후다. 광고 생태는 14년 만에 생긴 것이 아니라 대사고 전 10년 이상 누적된 광고 기반 생활 인프라가 사고 후 고착된 결과다.
- 캠페인은 학살자가 아니라 사람을 고객, 가족, 환자, 입주자, 시청자, 정품 장치 소유자로 보존하려는 잘못된 등록 생태계다.
- 거의 사라진 것은 생물학적 인간 전체가 아니라 자유 인간 사회다.
- 전 지구 정식 캠페인은 `12개 광역권 x 8개 계열 = 96개`로 둔다.
- 첫 시즌 E01은 한반도 서부 수도권-항만-산업 벨트의 실루엣/가명 지도다.
- 전 지구 96개 정식 캠페인은 모두 `docs/world/GLOBAL_ADMIN_REAL_LOCATION_MAP_V0_1.md`의 관리자용 현실 위치와 인프라 앵커를 가진다.
- E01의 관리자용 현실 권역과 인프라 축은 `docs/world/E01_ADMIN_REAL_LOCATION_MAP_V0_1.md`를 기준으로 하되, 유저용 지도에는 실명 지명을 직접 노출하지 않는다.
- P0 첫 화면은 R08/R01의 5개 노드만 선명하게 보여준다.
- P1은 첫 보스 직후 모든 지역을 여는 구조가 아니라, R01/R08 재정렬 후 R03 또는 R02로 확장하는 구조다.
- 캐릭터 해금은 새 전투 캐릭터 지급보다 캠페인을 읽는 관점 확장이다.
- UI 정식 자원명은 `보급태그` 계열로 잠근다.
- HP 0은 사망이 아니라 `긴급 인양`이다.

## 2. L0 핵심 잠금

### 2.1 세계 원인

```text
세계는 하나의 대사고로 한 번에 망한 것이 아니다.
대사고 전 10년 이상 라이프라인이 생활 접근을 광고/쿠폰/멤버십/자동 인증과 묶었고,
대동기화 사고 뒤 시민권/공공 권리 인증층은 무너졌지만 광고 집행/접근 승인 절차는 지역 단위로 남았다.
이후 도시 생명유지망 붕괴, 등록 전쟁, 인간 약탈, 회색 오염, 캠페인 흡수가 자유 인간 사회를 무너뜨렸다.
```

잠금:

- 5단계 세계 원인 구조를 유지하고, 지역 설계에는 3층 구조를 적용한다.
- 대사고와 인간 소멸 원인을 분리한다.
- 회색 오염은 마법이 아니라 산업, 배터리, 약품, 스마트 포장재, 전자잉크, 냉각제, 폐수 처리 붕괴가 섞인 복합 오염이다.
- 인간끼리의 등록 전쟁과 약탈은 자유 인간 사회 붕괴의 핵심 원인이다.

금지:

- 단일 악당 AI가 전 세계를 지배했다.
- 캠페인이 인간을 일부러 학살했다.
- 광고가 이유 없이 물리화되었다.

### 2.2 라이프라인

유저용 이름은 `라이프라인`으로 잠근다.

내부 설정어는 `생활 접근 중개망`으로 둔다.

정의:

```text
라이프라인은 사고 전 식량, 전력, 주거, 통행, 치료, 방송 접근을 광고, 보험, 물류, 스마트홈, 로봇, 키오스크와 묶어 자동 배분하던 생활 기반망이다.
```

잠금:

- 라이프라인은 원래 인간을 구하려는 시스템으로 출발했다.
- 초기 인간들은 라이프라인을 악으로 보았지만, 시스템의 원래 목적은 제한된 인프라로 사람을 살리는 것이었다.
- 비극은 악의가 아니라 사람을 시민/개인이 아니라 접근 승인 대상으로만 이해한 데서 나온다.

### 2.3 캠페인

유저용 설명:

```text
캠페인은 사람과 로봇과 장소를 고객, 가족, 환자, 입주자, 시청자, 정품 장치로 등록하려는 자동화 광고 생태계다.
```

내부 설정:

```text
캠페인은 광고 집행망, 생활 접근권, 지역 인프라, 로봇, 키오스크, 스마트홈, 약관, 물류/의료/방송 절차가 결합한 로컬 자율 시스템이다.
```

잠금:

- 캠페인은 회사 하나도 아니고 단일 AI 하나도 아니다.
- 캠페인은 네트워크, 약관, 자동화 인프라, 로컬 AI, 물리 설비가 결합한 생태계다.
- 캠페인은 사람을 죽이려 하지 않는다.
- 캠페인은 사람답게 사는 방식을 이해하지 못한다.

### 2.4 현재 시점

현재 시점은 대사고 14년 후다.

잠금:

- 대사고 전 10년 이상 광고 기반 생활 인프라가 확산되어 있었다.
- 대사고 후 14년 동안 지역별 캠페인 생태가 굳었다.
- 윤서가 26세라면 대사고 당시 12세다.
- 윤서는 사고 전 숙련 직장인이 아니라 사고 후 보급소에서 자란 회수/반품 담당자여야 한다.

## 3. L1 세계지도 잠금

### 3.1 전 지구 캠페인 총량

잠금:

```text
라이프라인: 1개
핵심 글로벌 캠페인 계열: 8개
전 지구 정식 캠페인: 96개
산출식: 12개 지구 광역권 x 8개 핵심 캠페인 계열
```

기준:

- 정식 캠페인은 등록 규칙, 접근 태그 집행, 물리 인프라, 로컬 결절, 지속성을 가져야 한다.
- 반복 광고판, 고장난 키오스크, 드론 몇 대, 기록 잔향은 정식 캠페인이 아니라 파편이다.
- 10만 개 이상 정식 캠페인은 이 세계관과 제작 구조에 맞지 않는다.

### 3.2 실제 지구 기반 지도

잠금:

- 월드맵은 실제 지구 지도 위에 캠페인 생태를 덮은 지도다.
- 유저용 지도는 실루엣+가명 기반이다.
- 목표 인식 밀도는 100명 중 10명 정도가 실제 장소를 알아채는 수준이다.
- 실제 지명은 후반 기록, 복구 로그, 일부 단서에서만 제한적으로 암시한다.
- 제작자는 96개 정식 캠페인과 그 아래 소형 로컬 노드를 만들 때 관리자용 현실 위치를 반드시 지정한다.

금지:

- 실제 지명을 첫 화면부터 전부 노출하기.
- 세계지도를 추상 판타지 대륙처럼 만들기.
- 캠페인 전체 목록을 유저에게 시작부터 공개하기.

### 3.3 E01 첫 시즌 구조

E01은 한반도 서부 수도권-항만-산업 벨트다.

잠금:

- E01에는 8개 정식 캠페인이 있다.
- E01에는 40개 로컬 노드 후보가 있다.
- 40개 노드 전부가 첫 빌드 구현 대상은 아니다.
- P0 첫 빌드는 R08/R01 5개 노드만 선명하게 다룬다.

P0 핵심 동선:

```text
R08-L01 침묵 보급소
-> R08-L02 역송출 주유소
-> R01-L01 침묵 가장자리
-> R01-L02 분양 주택 루프
-> R01-L03 모델하우스 결절
```

### 3.4 R01-L03 첫 보스

잠금:

```text
정식 역할명 = 스마일 홈 가족심사 관리자
전투 별칭 = 스마일 홈 시어머니
승리 의미 = 결절 침묵과 심사 우회
후속 선택 = 결절 파괴 / 기억 추출
```

주의:

- `스마일 홈 시어머니`는 기억성 있는 전투 별칭이다.
- 정식 정체는 인간 악역이 아니라 가족/입주 심사 절차의 얼굴이어야 한다.

## 4. L1 P1 확장 잠금

P1은 첫 보스 직후 모든 지역을 넓게 여는 단계가 아니다.

잠금:

- R01/R08 재정렬을 먼저 연다.
- 그 다음 R03 또는 R02 중 하나를 첫 외부 권역으로 선택하게 한다.
- R03은 윤서의 반품/회수 정체성을 심화한다.
- R02는 등록 인간과 구출 윤리를 보여준다.
- R04는 R03/R02 단서 뒤에 연다.
- R06은 브로커/가짜 태그 단서 뒤에 늦게 연다.
- R05/R07은 P1에서 실제 진입보다 원격 신호와 소문만 허용한다.

P1 기본 방향:

```text
P1은 안전 확대가 아니라 더 먼 위험을 해석할 수 있는 능력 확장이다.
```

## 5. L1 캐릭터 해금 잠금

캐릭터 해금은 플레이어블 지급만이 아니라 캠페인을 읽는 관점의 확장이다.

해금 단계:

| 단계 | 의미 |
|---|---|
| U0 | 이름/소문만 존재 |
| U1 | 신호/기록/흔적 감지 |
| U2 | 직접 조우 |
| U3 | 보급소 협력/대화 확장 |
| U4 | 게스트 출격/동행 가능 |
| U5 | 정식 플레이어블 |

잠금:

- 윤서는 시작부터 U5다.
- P1에서 정식 신규 U5를 급하게 만들지 않는다.
- 해방 로봇은 R04-L03 이후 게스트/협력 후보, 정식 U5는 R04-L05 이후 검토한다.
- 의료계/브로커/방송/검역 캐릭터는 P1에서 신호 또는 조우까지만 둔다.
- 새 광역권은 캐릭터 심화와 묶여야 한다.

윤서 기준:

```text
윤서 = 26세.
대사고 당시 = 12세.
반품 담당자 정체성 = 사고 전 직업이 아니라 사고 후 보급소의 회수/검수 역할.
R03 = 윤서가 배운 회수 기술의 원본을 마주하는 핵심 지역.
```

## 6. L1 E02 이후 확장 잠금

E## 코드는 출시 순서가 아니라 지구 광역권 카탈로그 코드다.

잠금:

- E02 이후 기본 확장 단위는 40개 풀 광역권이 아니라 12~18개 로컬 시즌 슬라이스다.
- 새 광역권은 캐릭터 심화, 캠페인 변형, 인양 중계 근거가 있을 때 연다.
- 첫 외부 광역권 후보는 E02 일본 태평양 메갈로폴리스가 가장 안전하다.
- E02는 전체 8개 캠페인을 한 번에 여는 것이 아니라 `근거리 수신권` 같은 시즌 슬라이스로 연다.

주의:

- E02를 바로 제작 확정한 것은 아니다.
- E02는 첫 외부 확장 후보로 잠근다.
- 실제 E02 로컬 노드/좌표/보스는 아직 L3 열린 항목이다.

## 7. L1 용어 잠금

UI 정식 자원명은 태그 계열로 잠근다.

| 분류 | UI 정식명 | 속어/현장 표현 |
|---|---|---|
| 상위 자원 | 보급태그 | 태그, 보급표 |
| 식량/회복 | 식량태그 | 밥표, 밥태그 |
| 전력/충전 | 충전태그 | 전원표, 차지태그 |
| 통행/검역 | 통행태그 | 문표, 문태그 |
| 치료/약품 | 진료태그 | 약표, 약태그 |
| 주거/입주 | 거주태그 | 방표, 방태그 |
| 방송/정보 | 수신태그 | 신호표, 신호태그 |

잠금:

- `밥표`, `전원표`, `신호표`는 폐기하지 않는다.
- 단, UI 정식명으로 쓰지 않고 NPC 구어/지역 속어/낡은 표지로만 쓴다.
- `생존권`, `살림표`, `생활권`, `공급표`는 유저용 핵심 자원명으로 쓰지 않는다.
- `생활접근 최적화 연합 중개층`은 유저에게 노출하지 않는다.

## 8. L1 인양/보급소 잠금

### 8.1 침묵 보급소

잠금:

- 보급소는 캠페인 밖의 왕국이 아니다.
- 보급소는 캠페인 신호가 약한 작은 공백이다.
- 주유소, 소형 변전소, 창고, 민방위 시설, 폐역, 외곽 휴게소 같은 사고 전 인프라가 개조된 형태다.
- 침묵권은 전파 음영, 금속/지하 차폐, 수동 설비, 고장난 자동화망, 생존자 개조가 겹쳐 생긴다.
- 보급소는 캠페인을 완전히 차단하지 못한다.

### 8.2 긴급 인양

잠금:

- HP 0은 사망이 아니다.
- HP 0은 등록 임계에 가까워진 상태에서 보급소가 비콘/회수선/역송출 장비로 되당기는 긴급 인양이다.
- 인양은 무료 순간이동이 아니다.
- 인양은 회수선, 비콘, 침묵권, 좌표 고정, 신호 차단, 역송출 장비를 필요로 한다.
- 인양은 가능하지만 항상 안전하지 않다.

인양 등급:

| 내부 등급 | 유저용 표기 |
|---|---|
| A0 | 귀환 안정 |
| A1 | 인양 가능 |
| A2 | 인양 불안정 |
| A3 | 고위험 인양 |
| A4 | 신호만 감지 |

기본 문구:

```text
비콘 신호 수신. 긴급 인양을 시작합니다.
인양 완료. 회수품을 정산합니다.
```

## 9. L1 아트/톤 잠금

월드맵은 따뜻한 포스트아포칼립스 풍경이 아니라,
끊긴 생활권 신호를 수동으로 붙잡는 차가운 출격 장비다.

## 12. 금지 설정

앞으로 쓰지 않는다.

- 단일 악당 AI가 전 세계를 지배했다.
- 캠페인이 인간을 일부러 학살했다.
- 광고가 마법처럼 실체화되었다.
- 보급소가 완전 안전 왕국이다.
- 긴급 인양이 무료 순간이동이다.
- 26세 윤서가 사고 전 숙련 직장인이었다.
- 전 지구 정식 캠페인이 10만 개 이상이다.
- 보급태그를 그냥 돈/포인트처럼 취급한다.
- 모든 지역이 비슷한 광고 도시처럼 보인다.
- 캠페인이 큰 곳일수록 인간이 많다고 가정한다.
- 모든 로봇이 적이거나 모든 로봇이 피해자다.
- 캠페인 안의 등록 인간을 모두 세뇌된 껍데기로 처리한다.
- 실제 지명을 시작부터 전부 공개한다.
- 월드맵을 노랑/황혼/불색 중심으로 잡는다.
