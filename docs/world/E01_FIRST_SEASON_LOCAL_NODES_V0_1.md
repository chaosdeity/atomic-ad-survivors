# Atomic Ad Survivors E01 첫 시즌 로컬 노드 지정 v0.1

상태: v0.1 지도 제작 전 로컬 노드 기준  
목적: E01 한반도 서부 수도권-항만-산업 벨트 안의 실제 플레이 노드를 먼저 고정한다.  
연결 문서: `docs/world/WORLD_MAP_CAMPAIGN_ECOLOGY_V0_1.md`, `docs/world/GLOBAL_CAMPAIGN_CATALOG_V0_1.md`, `docs/world/GLOBAL_CAMPAIGN_MAP_PLACEMENT_V0_1.md`, `docs/world/E01_WORLD_MAP_BLOCKOUT_V0_1.md`, `docs/world/WORLD_TERMINOLOGY_LOCK_V0_1.md`

## 0. 판정 요약

- E01은 첫 시즌의 실제 플레이 광역권이다.
- E01에는 8개 정식 캠페인이 있고, 각 캠페인 아래 5개 주요 로컬 노드를 둔다.
- 따라서 E01 v0.1 로컬 노드는 총 40개다.
- 40개는 모두 처음부터 노출하지 않는다. P0 5개, P1 16개, P2 19개로 나눠 연다.
- 월드맵은 실제 지구 실루엣 위에 놓되, 초반에는 실명 지명을 숨긴다.
- 유저는 가명, 해안선, 도로축, 항만/공항 거리, 도시 밀도, 산지/전파 음영으로 실제 장소를 어렴풋이 알아챈다.
- 침묵권은 캠페인이 아니다. R08은 E01_C08 비상보급/대피 캠페인의 잔해와 전파 음영을 생존자들이 개조한 예외 공간이다.

## 1. 로컬 노드 사용 원칙

### 1.1 ID 규칙

```text
E01_C03 = E01 광역권의 3번 정식 캠페인
R03-L02 = R03 권역 안의 2번 로컬 노드
```

주의:

- `E01_Cxx`는 전 지구 96개 정식 캠페인 중 하나다.
- `Rxx-Lyy`는 실제 출격, 회수, 보스, 이벤트가 걸리는 지도 노드다.
- R08 로컬은 침묵권 노드지만, 정식 캠페인으로는 E01_C08 비상보급/대피 잔여권에 연결한다.
- 정식 캠페인명에 붙는 `권`, `망`, `거리`는 고유명사다. 유저가 회수하거나 소비하는 자원명은 `식량태그`, `충전태그`, `통행태그`, `진료태그`, `거주태그`, `수신태그`로 통일한다.

### 1.2 노출 우선순위

| 우선순위 | 노드 수 | 의미 |
|---|---:|---|
| P0 | 5 | 첫 빌드, 튜토리얼, 첫 보스까지 필요한 핵심 동선 |
| P1 | 16 | 1장 확장, 윤서 과거, 해방 로봇, 의료/브로커 서사 진입 |
| P2 | 19 | 후반 미스터리, 권역 보스, 항만/방송/세계 진실 확장 |

### 1.3 지도 노출 원칙

| 항목 | 원칙 |
|---|---|
| 실명 지명 | 초반 유저용 지도에는 직접 쓰지 않는다. |
| 내부 지리 앵커 | 실제 지도 제작 기준으로만 쓴다. |
| 유저 노출 단서 | 해안선, 간선도로, 강, 산지, 항만/공항 거리, 도시 밀도, 방송탑 실루엣으로 암시한다. |
| 인식 밀도 | 100명 중 10명 정도가 실제 장소를 알아채는 수준을 목표로 한다. |
| 정확 좌표 | v0.1에서는 확정하지 않는다. 맵 아트/월드맵 UI 단계에서 좌표화한다. |

## 2. E01 광역 지도 기준

E01의 내부 실제 지리 기준은 `한반도 서부 수도권-항만-산업 벨트`다. 유저에게는 실명 대신 `서부권`, `닫힌 항로`, `반품 벨트`, `마지막 수신권` 같은 캠페인식 이름으로 보인다.

| 방향 | 권역 | 정식 캠페인 | 지도 의미 |
|---|---|---|---|
| 중심 외곽 | R08 침묵권 군락 | E01_C08 서부 비상보급망 | 시작 보급소, 인양 중심, 침묵권 과학 근거 |
| 동/남동 | R01 외곽 주택가 | E01_C01 서부 스마일홈 | 첫 출격, 가족/입주 등록, 첫 보스 |
| 동/도심 | R02 의료 격리 구역 | E01_C02 백색팔찌 격리권 | 환자 등록, 구출 윤리, 진료태그 |
| 남/남서 | R03 물류 회수 벨트 | E01_C03 반품 회수 벨트 | 윤서 정체성, 반품/정산/회수 |
| 남 | R04 산업 오염 지대 | E01_C04 정품충전 공단 | 해방 로봇, 충전태그, 회색 오염 |
| 북/도심 고지점 | R05 방송 중추 지구 | E01_C05 마지막 수신탑 | 대사고 기록, 추천 알고리즘, 후반 진실 |
| 도심 고밀도 | R06 상업/멤버십 지구 | E01_C06 포인트 거리 | 브로커, 태그 거래, 인간 악역 |
| 서/남서 | R07 항만/이송 봉쇄 구역 | E01_C07 마지막 출항권 | 탈출 미스터리, 검역/통행/항만 |

## 4. P0 첫 플레이 동선

P0는 첫 빌드와 첫 보스까지의 최소 동선이다.

```text
R08-L01 침묵 보급소
-> R08-L02 역송출 주유소
-> R01-L01 침묵 가장자리
-> R01-L02 분양 주택 루프
-> R01-L03 모델하우스 결절
```
