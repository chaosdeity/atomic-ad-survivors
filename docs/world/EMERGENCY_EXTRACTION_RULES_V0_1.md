# Atomic Ad Survivors 긴급 인양 규칙/문구 v0.1

상태: v0.1 시스템/문구 기준  
목적: HP 0, 귀환, 실패, 손실, 인양 등급, 유저용 문구와 내부 설정어를 분리해 정리한다.  
연결 문서: `docs/world/E01_P0_WORLD_MAP_FIRST_SCREEN_FLOW_V0_1.md`, `docs/world/E01_WORLD_MAP_BLOCKOUT_V0_1.md`, `docs/world/WORLD_TERMINOLOGY_LOCK_V0_1.md`

## 0. 판정 요약

- HP 0은 사망이 아니다.
- HP 0은 `등록 압력 임계` 또는 `작전 지속 불가` 상태다.
- 긴급 인양은 순간이동이 아니다.
- 긴급 인양은 비콘, 회수선, 역송출 주유소, 침묵권 차폐, 좌표 고정이 맞물려 가능한 제한적 구조다.
- 인양은 대체로 가능해야 한다. 그래야 모바일 액션 RPG의 반복 출격 구조가 성립한다.
- 하지만 인양은 항상 안전하지 않다. 손실, 장비 부담, 캠페인 추적, 보급소 노출 위험이 남는다.
- P0에서는 인양 규칙을 짧고 명확하게 보여주고, 세부 손실은 과하게 벌주지 않는다.

## 1. 유저용 한 문장

```text
쓰러지면 죽는 것이 아니라, 비콘 신호를 따라 보급소가 긴급 회수합니다. 다만 회수품과 신호 일부는 잃을 수 있습니다.
```

더 짧은 UI 설명:

```text
HP 0: 긴급 인양
```

## 2. 내부 설정용 핵심 문장

```text
출격자는 보급소에서 고정한 비콘 태그를 들고 나가며, HP 0 또는 등록 압력 임계에 도달하면 R08-L02 역송출 주유소의 회수 장비가 마지막 좌표를 잡아 침묵권 안으로 되당긴다. 이 과정은 순간이동이 아니라 짧은 시간의 회수, 신호 차단, 위치 재고정이며, 거리와 캠페인 간섭에 따라 손실과 실패 조건이 달라진다.
```

물리적 기준:

```text
게임 화면은 암전으로 넘어가지만, 설정상 출격자의 몸은 사라지는 것이 아니다. 비콘 좌표를 따라 회수 드론, 소형 차량, 와이어 장비, 유선 릴이 가장 가까운 저신호 통로까지 접근하고, 역송출 장비가 캠페인 신호를 잠깐 밀어낸 사이 보급소가 출격자를 끌어낸다.
```

## 3. HP 0의 의미

| 구분 | 의미 | 유저에게 보이는 말 |
|---|---|---|
| HP 감소 | 부상, 피로, 광고 등록 압력, 장비 손상, 신호 침식이 합쳐진 전투 지속력 | HP |
| HP 0 | 사망이 아니라 작전 지속 불가/등록 임계 | 긴급 인양 |
| 사망 | 일반 반복 플레이에서 쓰지 않는다 | 사용 금지 |
| 등록 | 캠페인이 사람을 고객/입주자/환자/가족 등으로 고정하려는 상태 | 등록 압력 |

주의:

- `사망`, `부활`, `리스폰` 같은 단어는 기본 UI에서 쓰지 않는다.
- 내부 문서에서는 `등록 임계`를 쓸 수 있지만, P0 유저 문구에서는 `긴급 인양`을 우선한다.
- A0~A4 유저 표기와 HP 0/인양 완료 기본 문구는 `docs/world/WORLD_TERMINOLOGY_LOCK_V0_1.md`를 기준으로 한다.

## 4. 인양 원리

### 4.1 필요한 장치

| 장치 | 역할 | 유저용 표현 |
|---|---|---|
| 비콘 태그 | 출격자의 마지막 좌표와 상태를 보급소에 보낸다 | 비콘 |
| 회수선 | 인양 신호를 따라 접근하는 보급소 회수 장비. 드론/소형 차량/유선 장비의 묶음 | 회수선 |
| 역송출 장비 | 캠페인 신호를 거꾸로 밀어내 짧은 귀환 창을 만든다 | 역송출 |
| 침묵권 차폐 | 보급소 주변의 낮은 신호 공백 | 침묵권 |
| 좌표 고정 | 출격 전 기록한 귀환 기준점 | 귀환 좌표 |

### 4.2 작동 순서

```text
1. 출격 전 비콘 태그 등록.
2. R08-L02에서 귀환 좌표와 회수선 대기.
3. 출격 중 HP 0 또는 등록 압력 임계 도달.
4. 비콘이 마지막 좌표와 신호 상태를 송신.
5. 회수 드론/소형 차량/와이어 장비가 가장 가까운 저신호 통로까지 접근.
6. 역송출 장비가 캠페인 신호를 밀어내는 동안 회수선이 출격자를 끌어냄.
7. 출격자는 침묵권 안으로 인양됨.
8. 회수품, 태그, 장비 손실을 정산함.
```

화면 연출 기준:

- 즉시 순간이동처럼 번쩍 사라지지 않는다.
- 짧은 신호 끊김, 잡음, 회수선 접근, 흰 노이즈/암전, 정산 화면으로 이어진다.
- P0에서는 연출을 길게 끌지 않는다. 처음 한 번만 조금 더 보여주고 이후에는 빠르게 처리한다.

## 5. 인양 등급

| 내부 등급 | 유저용 표기 | 성공 기준 | 손실 기준 | 대표 노드 |
|---|---|---|---|---|
| A0 | 귀환 안정 | 항상 안전 | 손실 없음 또는 정비 비용만 | R08-L01, R08-L02 |
| A1 | 인양 가능 | 비콘만 살아 있으면 가능 | 미정산 회수품 일부 손실 | R01-L01~R01-L03 |
| A2 | 인양 불안정 | 비콘+회수선 상태 필요 | 회수품 손실, 태그 일부 훼손, 보급소 전력 소모 | R03/R02/R06 일부 |
| A3 | 고위험 인양 | 중계 또는 추가 장비 필요 | 큰 손실, 캠페인 추적도 상승, 장비 손상 | R04/R07/R03 보스권 |
| A4 | 신호만 감지 | 기본 인양 불가 | 진입 전 중계기/침묵권 확장 필요 | R05, R07 깊은 구역 |

P0 적용:

- R01-L01~R01-L03은 A1이다.
- P0 첫 실패는 가혹하게 벌주지 않는다.
- 첫 인양은 규칙 학습이 목적이므로, 핵심 회수품을 전부 잃지 않게 한다.

## 6. 손실 규칙

### 6.1 손실 대상

| 손실 대상 | 설명 | P0 적용 |
|---|---|---|
| 미정산 회수품 | 출격 중 주웠지만 보급소에서 정산하지 않은 물품 | 일부만 손실 |
| 보급태그 조각 | 안정화 전 태그 조각 | 소량 손실 가능 |
| 장비 내구 | 회수선/비콘/방어 장비 부담 | P0에서는 문구만 |
| 캠페인 주목도 | 같은 경로 반복 실패 시 캠페인이 학습 | P0에서는 낮게 |
| 보급소 전력 | 역송출 장비 사용 비용 | P0에서는 직접 비용 표시하지 않음 |

### 6.2 등급별 손실 감각

정확한 수치는 밸런스 단계에서 정한다. v0.1에서는 감각만 잠근다.

| 인양 등급 | 손실 감각 | 추천 수치 방향 |
|---|---|---|
| A0 | 없음 | 0% |
| A1 | 작음 | 미정산 회수품 일부 |
| A2 | 중간 | 미정산 회수품 상당량, 태그 일부 |
| A3 | 큼 | 전리품 대부분, 장비 부담, 추적도 |
| A4 | 출격 제한 | 기본적으로 인양 불가. 먼저 중계 필요 |

주의:

- 이미 정산한 성장 재화는 잃지 않는다.
- 핵심 스토리 아이템은 첫 획득 후 완전히 사라지지 않게 한다.
- 손실은 벌이라기보다 `위험한 곳에 깊이 들어갔다`는 비용이어야 한다.

## 7. 실패 조건

인양 실패는 플레이어 캐릭터 영구 사망이 아니다. 실패는 `인양 지연`, `회수품 손실`, `캠페인 등록 흔적 증가`, `보급소 위치 위험`으로 표현한다.

| 실패 조건 | 결과 | 유저용 표현 |
|---|---|---|
| 비콘 파손 | 회수 지연, 회수품 큰 손실 | 비콘 신호 손실 |
| 회수선 노출 | 장비 손상, 다음 출격 비용 증가 | 회수선 손상 |
| 강신호 간섭 | 인양 불가 또는 중계 필요 | 신호 간섭 과다 |
| A4 지역 무리 진입 | 출격 자체 제한 | 중계 필요 |
| 캠페인 추적 성공 | 보급소 주변 노이즈 증가 | 추적 신호 잔류 |

P0에서는 아래 실패만 사용한다.

```text
비콘 신호 약화
회수품 일부 손실
주택가 캠페인 반응 증가
```

## 8. 유저용 문구

### 8.1 월드맵 인양 상태

| 내부 등급 | 짧은 표기 | 설명 문구 |
|---|---|---|
| A0 | 귀환 안정 | 보급소 장비가 안정적으로 닿는 구역입니다. |
| A1 | 인양 가능 | 긴급 인양이 가능합니다. 회수품 일부를 잃을 수 있습니다. |
| A2 | 인양 불안정 | 신호가 흔들립니다. 인양 시 손실이 커질 수 있습니다. |
| A3 | 고위험 인양 | 회수선 노출 위험이 큽니다. 깊은 진입은 신중해야 합니다. |
| A4 | 신호만 감지 | 아직 인양할 수 없습니다. 중계 또는 침묵권 확장이 필요합니다. |

### 8.2 HP 0 순간 문구

P0 기본:

```text
비콘 신호 수신. 긴급 인양을 시작합니다.
```

대체 후보:

```text
등록 압력 초과. 회수선을 호출합니다.
```

```text
작전 지속 불가. 귀환 좌표를 고정합니다.
```

P0 추천:

```text
비콘 신호 수신. 긴급 인양을 시작합니다.
```

이유:

- 가장 직관적이다.
- 죽음처럼 들리지 않는다.
- 비콘과 인양 시스템을 자연스럽게 학습시킨다.

### 8.3 인양 완료 문구

성공:

```text
인양 완료. 회수품을 정산합니다.
```

일부 손실:

```text
인양 완료. 일부 회수품이 신호 중에 손실되었습니다.
```

위험 귀환:

```text
회수선이 손상되었습니다. 다음 출격 전 정비가 필요합니다.
```

추적 흔적:

```text
낯선 추적 신호가 따라붙었습니다.
```

| 상황 | 문구 |
|---|---|
| 인양 가능 | 인양 가능 구역입니다. |
| 인양 불안정 | 인양이 불안정합니다. 회수품 손실이 커질 수 있습니다. |
| 고위험 인양 | 고위험 인양 구역입니다. 장비 손상이 발생할 수 있습니다. |
| 신호만 감지 | 아직 인양 경로가 없습니다. 중계 신호가 필요합니다. |

| 역송출 장비 | 인양 장비/장비 확인 | R08-L02에서만 역송출을 조금씩 노출한다. |

| 캠페인 추적도 | 추적 신호 | 경고 문구에 적합하다. |
| 침묵권 차폐율 | 신호 안정도 | UI에서 숫자로 쓰지 않는다. |

## 10. P0 적용 세부

| 노드 | 인양 표기 | 실패 시 처리 |
|---|---|---|
| R08-L01 침묵 보급소 | 귀환 안정 | 실패 없음 |
| R08-L02 역송출 주유소 | 귀환 안정 | 실패 없음. 장비 설명만 |
| R01-L01 침묵 가장자리 | 인양 가능 | 회수품 일부 손실, 첫 인양 튜토리얼 |
| R01-L02 분양 주택 루프 | 인양 가능 | 식량태그 조각 일부 손실, 캠페인 반응 소폭 증가 |
| R01-L03 모델하우스 결절 | 인양 가능 | 보스전 재도전, 미정산 회수품 일부 손실 |

P0 금지:

- 첫 인양에서 큰 재화 손실을 주지 않는다.
- 첫 보스 실패를 장시간 반복 페널티로 만들지 않는다.
- `사망`, `부활`, `게임 오버`를 쓰지 않는다.

## 11. 연출 기준

긴급 인양 연출 순서:

```text
HP 0
-> 화면 가장자리 광고 노이즈 증가
-> 비콘음 또는 짧은 경고음
-> UI 문구 출력
-> 회수선/역송출 잡음
-> 짧은 암전 또는 신호 끊김
-> 보급소 정산 화면
```

연출 톤:

- 친절한 구조 방송처럼 들리되, 완전히 안전하게 느껴지지는 않게 한다.
- 캠페인에게 끌려가는 느낌이 아니라 보급소가 억지로 되당기는 느낌이어야 한다.
- 회수선이 닿지 않는 지역은 노이즈가 문구보다 먼저 실패를 알려야 한다.
