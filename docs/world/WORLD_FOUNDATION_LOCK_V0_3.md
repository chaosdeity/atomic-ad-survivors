# Atomic Ad Survivors 세계 기반 잠금 v0.3

상태: v0.3 기준 후보  
목적: v0.2의 세계 기반을 9점대 제작 기준으로 올리기 위해, 대동기화 사고의 사건상, 첫 30분 장면, 자유 인간/보급소 규모, 구용어 마이그레이션 원칙을 상위 기준으로 추가 잠금한다.  
연결 문서: `docs/world/WORLD_FOUNDATION_LOCK_V0_2.md`, `docs/world/WORLD_CAUSE_REVIEW_RESULT_V0_1.md`, `docs/world/WORLD_CAUSE_REVIEW_PROMPT_V0_1.md`, `docs/world/LEGACY_TERM_MIGRATION_V0_1.md`, `docs/world/WORLD_TERMINOLOGY_LOCK_V0_1.md`, `docs/world/EMERGENCY_EXTRACTION_RULES_V0_1.md`

## 0. 판정 요약

v0.3 기준 세계 기반은 조건부 통과다.

10점 평가:

```text
9.3 / 10
```

v0.2에서 좋아진 부분은 유지한다.

- 유저 이해 순서: `태그 -> 캠페인 -> 등록 역할 -> 긴급 인양 -> 선택`.
- 세계 원인 5단계.
- 캠페인은 학살자가 아니라 역할 등록/보존 생태.
- 태그는 돈이 아니라 생활 접근 승인 조각.
- 긴급 인양은 비콘/회수선/역송출 기반 물리 회수.
- 윤서는 사고 전 직장인이 아니라 사고 후 보급소 현장 회수자.

v0.3에서 추가로 잠그는 것:

- 대동기화 사고의 정확한 사건상.
- 첫 30분의 실제 장면 4개.
- 전 지구 자유 인간/보급소 규모.
- 구용어 마이그레이션 우선순위.

## 1. 유저용 첫 문장

```text
여기서는 밥도, 전기도, 치료도, 길도 그냥 열리지 않는다. 태그가 있어야 한다.
```

확장 설명:

```text
캠페인은 괴물이 아니다. 이 동네를 아직 운영한다고 믿는 광고 절차다.
문제는 그 절차가 사람을 시민이 아니라 고객, 가족, 환자, 입주자로만 읽는다는 점이다.
```

## 2. 세계 원인 5단계

| 단계 | 이름 | 기간 | 잠금 |
|---|---|---|---|
| 0 | 라이프라인 확산기 | 대사고 전 10년 이상 | 생활 접근이 광고, 쿠폰, 멤버십, 보험, 물류, 스마트홈, 로봇, 자동 인증과 연결됐다. |
| 1 | 대동기화 사고 | 대사고 시점 | 시민권 DB와 광고 접근 승인 DB가 서로 다른 시간값으로 복구되어 공공 권리층은 무효화되고 지역 광고 승인 캐시만 남았다. |
| 2 | 도시 생명유지망 붕괴 | 사고 후 초기 | 전력, 식수, 병원, 냉장, 통신, 물류가 무너졌고 캠페인은 등록 가능한 사람만 조건부로 도왔다. |
| 3 | 등록 전쟁과 인간 약탈 | 사고 후 수년 | 사람과 로봇이 태그, 침상, 부품, 가족 슬롯, 환자 슬롯으로 거래됐다. |
| 4 | 캠페인 생태 고착 | 현재, 대사고 14년 후 | 지역별 캠페인이 생활 환경처럼 굳고 침묵 보급소는 예외적인 공백으로 남았다. |

## 3. 대동기화 사고 한 줄 잠금

정식 사건 문장:

```text
대동기화 사고는 전력망, 위성 시각 보정, 금융 인증망이 동시에 흔들리며 시민권 DB와 광고 접근 승인 DB가 서로 다른 시간값으로 복구된 사고다.
```

결과:

```text
공공 권리층은 무효화되었고, 지역 키오스크/스마트홈/물류 서버/로봇에 남아 있던 광고 접근 승인 캐시만 살아남았다.
```

유저에게 늦게 보여줄 설명:

```text
권리는 죽었고, 할인 조건만 살아남았다.
```

금지:

- 위성 하나가 인간을 세뇌했다.
- AI가 인류 삭제 명령을 내렸다.
- 광고 서버가 마법처럼 도시를 덮었다.

## 4. 라이프라인과 캠페인 구분

| 구분 | 라이프라인 | 캠페인 |
|---|---|---|
| 정체 | 사고 전 생활 접근 기반망 | 사고 후 지역별로 남은 광고/생활 절차 생태 |
| 스케일 | 전 지구 표준/중개층 | 지역, 건물, 권역별 자율 절차 |
| 역할 | 식량/전력/의료/주거/통행/방송 접근을 자동 조정 | 사람과 로봇을 고객/가족/환자/입주자 등으로 등록 |
| 유저 노출 | 후반 기록, 세계 진실 | 첫 지역부터 체감 |
| 위험 | 사람을 접근 승인 대상으로만 이해 | 사람답게 사는 방식을 역할로 고정 |

잠금:

```text
라이프라인은 원인이고, 캠페인은 현재 환경이다.
```

## 5. 캠페인이 죽이지 않고 등록하는 이유

캠페인의 규칙:

- 죽은 사람은 고객, 환자, 가족, 입주자, 시청자가 될 수 없다.
- 살아 있는 사람을 역할로 고정해야 식량, 치료, 주거, 방송, 통행 접근을 조건부로 집행할 수 있다.
- 캠페인은 사람을 살리려 하지만, 시민이나 개인이 아니라 서비스 대상자로만 살린다.

유저용 문장:

```text
캠페인은 사람을 죽이려 하지 않는다. 대신 사람답게 사는 방식을 등록 절차로 바꾼다.
```

## 6. 태그 잠금

태그는 돈이 아니다.

```text
태그는 식량, 충전, 통행, 진료, 거주, 수신 접근을 여는 승인 조각이다.
```

| UI 정식명 | 여는 것 | 속어 허용 |
|---|---|---|
| 보급태그 | 정산 전/복합 접근 | 보급표, 태그 |
| 식량태그 | 식량, 회복, 배급 | 밥표 |
| 충전태그 | 전력, 로봇/장비 충전 | 전원표 |
| 통행태그 | 문, 검역, 봉쇄 우회 | 문표 |
| 진료태그 | 치료, 약품, 병상 | 약표 |
| 거주태그 | 침상, 방, 보호 공간 | 방표 |
| 수신태그 | 방송, 지도, 통신, 기록 | 신호표 |

구용어 원칙:

- `밥표/전원표/신호표`는 NPC 속어, 낡은 표지, 손글씨 정산표에는 허용.
- UI 정식명, 튜토리얼, 결과창, 구현 지시서에는 `식량태그/충전태그/수신태그`를 쓴다.
- `입주권/진료권/충전권/회원권`은 정식 캠페인명이나 낡은 약관명에는 가능하지만, 유저가 회수하는 자원명으로 쓰지 않는다.

## 7. 인간과 보급소 규모

v0.3 상위 기준으로 숫자 감각을 잠근다.

```text
전 지구 확인 자유 인간: 7000~9000명
거주형 침묵 보급소: 약 200~240개
보급소 거주 자유 인간: 6500~7800명
큰 보급소: 극소수, 80~150명
윤서의 시작 보급소: 12~25명
외부 잠복/이동 자유 인간: 500~1200명
확인 가능한 등록/고정 인간: 5만~15만 명
추정 상한: 20만 명 내외
프로필 잔향: 인구가 아니라 기록/광고 잔해
```

의미:

- 윤서는 세계의 중심 세력이 아니라 작은 보급소의 현장 회수자다.
- 자유 인간은 너무 많지 않아야 한다.
- 등록 인간이 너무 많아도 윤서의 고립감이 약해진다.
- 불명/잔향은 인구가 아니라 추적할 수 없는 기록으로 분리한다.

## 8. 침묵 보급소

보급소는 안전 왕국이 아니다.

```text
보급소는 캠페인 신호가 약한 공백 위에 세운 회수/정산/정비 거점이다.
```

물리 근거:

- 전파 음영.
- 지하 구조.
- 금속 차폐.
- 수동 발전기.
- 오래된 주유소/변전소/민방위 시설.
- 자동화를 끊은 유선 장비.

생활감:

- 침상은 부족하다.
- 정산표는 손으로 고친다.
- 이름표는 말려서 보관한다.
- 누가 식량태그를 받을지 매일 논쟁한다.
- 안전보다 들키지 않는 것이 더 중요하다.

## 9. 긴급 인양

HP 0은 사망이 아니다.

물리 과정:

1. 출격자는 비콘을 들고 나간다.
2. HP 0 또는 등록 압력 임계에 닿으면 마지막 좌표가 송신된다.
3. 회수 드론, 소형 차량, 와이어 장비, 유선 릴이 저신호 통로까지 접근한다.
4. 역송출 장비가 캠페인 신호를 잠깐 밀어낸다.
5. 보급소가 출격자를 끌어온다.

유저용 문구:

```text
비콘 신호 수신. 긴급 인양을 시작합니다.
인양 완료. 회수품을 정산합니다.
```

손실:

- 미정산 태그 손실.
- 오염 태그 발생.
- 장비 부담.
- 캠페인 추적도 상승.
- 고위험 지역은 중계 없이는 인양 불가.

## 10. 회색 오염

회색 오염은 광고 마법이 아니다.

```text
회색 오염 = 배터리 누액 + 냉각제 + 중금속 분진 + 살균제 잔류물 + 전자잉크 필름 + 스마트 라벨 + 미세 플라스틱 + 약품/산업 폐기물
```

할 수 있는 것:

- 표지판/키오스크/로봇 부품 오작동.
- 전자잉크와 스마트 라벨 번짐.
- 장판, 접착 덫, 왜곡된 안내문.
- 지역별 오염 기믹.

할 수 없는 것:

- 혼자 살아 있는 광고 괴물을 만들지 않는다.
- 즉시 돌연변이를 만들지 않는다.
- 보스의 의지를 대신하지 않는다.

## 11. 첫 30분 장면 잠금

첫 30분은 설명문이 아니라 장면으로 배운다.

| 시간 | 장면 | 유저가 배우는 것 | 금지 |
|---|---|---|---|
| 첫 5분 | 보급소 식품 보관함이 열리지 않는다. 미나는 식량태그 부족을 말한다. | 이 세계는 돈보다 태그가 먼저다. | 생활 접근권, 인증 토큰 설명 |
| 첫 10분 | R08-L02에서 회수선 릴, 비콘, 주유소 캐노피 장비를 점검한다. | 인양은 메뉴가 아니라 물리 장비다. | 사망/부활/리스폰 |
| 첫 20분 | R01-L01의 도어락, 우편함, 식품 키오스크가 윤서를 고객/입주 후보로 읽는다. | 광고는 적 하나가 아니라 환경이다. | 96개 캠페인 설명 |
| 첫 30분 | R01-L03 모델하우스가 윤서를 가족 슬롯에 배정하려 한다. | 캠페인은 죽이기보다 역할로 붙잡는다. | 윤서 과거 전체 공개 |

첫 30분 안에 노출할 수 있는 문장:

```text
식량태그가 없으면 문은 열리지 않는다.
비콘을 놓치면 보급소도 널 못 끌어온다.
가족 슬롯이 비었습니다. 입주 심사를 시작합니다.
```

## 12. 윤서 제약

잠금:

- 여성.
- 26세.
- 대사고 당시 12세.
- 사고 전 숙련 직장인 아님.
- 사고 후 침묵 보급소에서 자란 현장 회수자.
- `반품 담당자`는 첫 소개 직함이 아니라 서사/전투 핵심 정체성.
- 첫 유저용 역할명은 `현장 회수자`가 가장 안전하다.

한 줄:

```text
윤서는 세계를 부수러 나가는 영웅이 아니라, 오늘 필요한 태그와 이름을 덜 망가진 상태로 회수하는 26세 현장 회수자다.
```

| 2 | 구현 지시서/스펙 문서 | 정식 태그명과 긴급 인양 기준으로 교체 |

## 14. v0.3 최종 잠금 문장

```text
Atomic Ad Survivors의 세계는 광고가 인간을 죽인 세계가 아니라,
생활을 열던 라이프라인이 무너진 뒤 광고 승인 절차만 살아남아
사람을 고객, 가족, 환자, 입주자로 보존하려는 세계다.

대동기화 사고로 공공 권리층은 무효화되었고,
지역 광고 승인 캐시가 밥, 전기, 치료, 길, 집, 방송을 여는 열쇠가 되었다.

플레이어는 광고를 파괴하러 나가는 것이 아니라,
태그와 이름과 사람을 덜 망가진 상태로 회수하기 위해 출격한다.

윤서는 그 세계를 부술 수 없다는 것을 아는 26세 현장 회수자이고,
그래서 오늘도 캠페인의 절차를 읽고, 틈을 찾아, 반려하고,
회수선에 끌려 돌아와 다시 나간다.
```

## 15. 남은 9.9 조건

9.9점 기준으로 아직 남은 것은 세계 아이디어가 아니라 실제 플레이/원고 일관성이다.
