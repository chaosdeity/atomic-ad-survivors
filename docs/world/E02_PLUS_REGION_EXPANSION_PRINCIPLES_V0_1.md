# Atomic Ad Survivors E02 이후 광역권 확장 원칙 v0.1

상태: v0.1 기준 문서
목적: E01 이후 실제 지구 광역권을 어떤 순서, 규모, 캐릭터 연결로 열지 정한다.
연결 문서: `docs/world/WORLD_MAP_CAMPAIGN_ECOLOGY_V0_1.md`, `docs/world/GLOBAL_CAMPAIGN_CATALOG_V0_1.md`, `docs/world/GLOBAL_ADMIN_REAL_LOCATION_MAP_V0_1.md`, `docs/world/GLOBAL_CAMPAIGN_MAP_PLACEMENT_V0_1.md`, `docs/world/CHARACTER_UNLOCK_STRUCTURE_V0_1.md`, `docs/world/E01_P1_EXPANSION_DESIGN_V0_1.md`

## 0. 판정 요약

E02 이후 확장은 96개 캠페인을 전부 한꺼번에 펼치는 작업이 아니다.
E02 이후 확장은 `실제 지구 인프라 -> 권역별 2차 붕괴 -> 캐릭터 심화 -> 로컬 캠페인` 순서로 열어야 한다.

잠금:

- `E##` 코드는 출시 순서가 아니라 지구 광역권 카탈로그 코드다.
- 그래도 첫 외부 광역권 후보는 E02 일본 태평양 메갈로폴리스가 가장 안전하다.
- 새 광역권은 캐릭터가 좋아서 따라가고 싶게 열려야 한다.
- 각 광역권을 무조건 E01처럼 40개 로컬로 만들지 않는다.
- 한 시즌/확장팩은 보통 1개 광역권 안의 1~3개 정식 캠페인을 깊게 다룬다.
- 96개 정식 캠페인 밖에 새 캠페인을 즉흥 추가하지 않는다.
- 실제 지명 인식 밀도는 계속 `100명 중 10명 정도`를 목표로 한다.
- E02 이후 소형 로컬 노드도 `docs/world/GLOBAL_ADMIN_REAL_LOCATION_MAP_V0_1.md`의 부모 캠페인 현실 앵커 안에서만 만든다.

가장 큰 위험:

```text
E02~E12를 모두 E01처럼 40개 로컬로 확장하면 제작량이 폭발하고,
각 지역이 왜 필요한지보다 지도 칸 채우기가 먼저 된다.
```

따라서 이후 광역권은 `전체 권역 40개`가 아니라 `시즌 슬라이스` 단위로 연다.

## 1. 유저가 이해해야 하는 한 문장

```text
보급소가 복원한 신호를 따라가면, 같은 캠페인 계열도 지구 곳곳에서 완전히 다른 생존 규칙으로 굳어져 있다.
```

## 2. 내부 설정용 핵심 문장

```text
E02 이후 광역권은 지구 실루엣 위의 새 스테이지가 아니라,
E01에서 확인한 캐릭터 원점과 캠페인 계열을 다른 실제 인프라권에서 재검산하는 확장 단위다.
```

## 3. 확장 단위 정의

| 단위 | 설명 | 권장 로컬 수 | 사용 시점 |
|---|---|---:|---|
| 광역권 카탈로그 | E02~E12 전체 권역의 설정/인프라/캠페인 8개를 정의한 상태 | 0 | 이미 완료 |
| 시즌 슬라이스 | 한 광역권의 1~3개 정식 캠페인을 실제 플레이로 여는 단위 | 8~18 | 일반 확장 |
| 캐릭터 심화 에피소드 | 특정 캐릭터의 원점/상처/능력을 다른 광역권에서 검산하는 단위 | 5~10 | 캐릭터 업데이트 |
| 풀 시즌 광역권 | E01처럼 8개 정식 캠페인을 모두 배치하는 대형 시즌 | 32~40 | 핵심 시즌일 때만 |
| 원격 신호 | 아직 출격 불가. 지도에만 약하게 보이는 전조 | 0~3 | 후반 떡밥 |

판정:

```text
E01 = 풀 시즌 광역권 예시.
E02 이후 기본값 = 시즌 슬라이스.
```

추가 잠금:

```text
시즌 슬라이스는 작게 열어도 현실 위치가 없는 임시 캠페인으로 만들지 않는다.
모든 로컬 노드는 부모 정식 캠페인 ID와 관리자용 현실 인프라 앵커를 가진다.
```

## 4. E02 이후 해금 원칙

### 4.1 해금 조건

새 광역권은 아래 조건 중 2개 이상을 만족할 때 연다.

| 조건 | 설명 |
|---|---|
| 캐릭터 심화 | 윤서, 해방 로봇, 의료계, 브로커, 방송/정보, 검역, 침묵권 기술자 중 하나의 심화가 필요하다. |
| 캠페인 계열 변형 | E01에서 본 캠페인 계열이 다른 인프라권에서 완전히 다르게 굳은 모습을 보여준다. |
| 인양/중계 근거 | 원격 출격이 가능한 침묵권 중계, 비콘, 복원 좌표, 회수선 기준점이 있다. |
| 세계 진실 조각 | 대사고, 라이프라인, 등록 전쟁, 회색 오염 중 하나를 새로 검산한다. |
| 게임성 차이 | 적, 기믹, 보스, 이동 환경이 E01과 충분히 다르다. |

### 4.2 해금 금지

- 새 지역이 멋있다는 이유만으로 연다.
- 실제 지명 패러디가 먼저 보인다.
- 한 광역권의 8개 캠페인을 모두 같은 밀도로 펼친다.
- 캐릭터 심화 없이 보스만 추가한다.
- E01의 R01/R03/R04 구조를 지명만 바꿔 반복한다.

## 6. E02 첫 외부 확장 추천안

E02는 첫 외부 광역권으로 적합하다. 단, E02 전체 8개 캠페인을 모두 열지 않는다.

### 6.1 E02 시즌 슬라이스

추천 슬라이스명:

```text
E02 근거리 수신권
```

핵심 캠페인:

| 캠페인 | 역할 | 캐릭터 연결 |
|---|---|---|
| E02_C03 역전 수령함 회수권 | 윤서의 반품/회수 감각을 다른 생활권에서 검산 | 윤서 |
| E02_C07 지하철 환승 심사권 | 통행태그와 이동권, 막차/환승 등록 압력 | 검역/이송 후보 |
| E02_C05 재난안내 생방송권 | 방송/재난 안내가 생존 명령으로 굳은 사례 | 방송/정보 후보 |
| E02_C04 정밀부품 리콜권 | 해방 로봇과 정품/리콜 심화 | 해방 로봇 |

권장 로컬 수:

```text
12~16개
```

구성:

- P0형 도입 노드 3~4개.
- 메인 캠페인 노드 6~8개.
- 보스/결절 노드 2개.
- 원격 신호/후속 노드 1~2개.

### 6.2 E02가 가르칠 것

- 같은 반품/회수라도 주택가 택배함과 역전 수령함은 다르다.
- 같은 재난 안내라도 보급소 수신과 도시 방재 방송은 다르다.
- 같은 통행태그라도 항만 검역과 지하철 환승은 다르다.
- 같은 정품 인증이라도 산업단지 배터리와 정밀부품 리콜은 다르다.

### 6.3 E02에서 금지할 것

- 실제 지명을 바로 노출한다.
- 모든 역/편의점/지하철이 일본 관광 패러디처럼 보인다.
- E02를 E01보다 더 큰 첫 메인 시즌으로 만든다.
- 윤서의 R03 심화를 건너뛰고 E02를 먼저 연다.
- 해방 로봇 정식 U5를 E02에서 갑자기 지급한다.

## 8. 노드 규모 기준

E01은 기준 광역권이라 40개 로컬을 가진다. 그러나 이후 광역권은 아래 기준을 따른다.

| 확장 종류 | 권장 캠페인 수 | 권장 로컬 수 | 예시 |
|---|---:|---:|---|
| 캐릭터 심화 에피소드 | 1 | 5~10 | 해방 로봇 E10_C04 |
| 소형 시즌 슬라이스 | 1~2 | 8~12 | E02_C03 + E02_C07 |
| 표준 시즌 슬라이스 | 2~3 | 12~18 | E02 근거리 수신권 |
| 대형 시즌 | 4~6 | 20~30 | E10 항만-빅테크 확장 |
| 풀 시즌 광역권 | 8 | 32~40 | E01 또는 후반 E03 |

규칙:

- 한 광역권을 처음 열 때는 12~18개가 가장 안전하다.
- 40개 로컬은 "이 광역권이 시즌 전체의 주인공"일 때만 쓴다.
- 미사용 캠페인은 원격 신호나 배경 압력으로 남겨도 된다.
- 로컬 수보다 캠페인 조합의 차이가 중요하다.

## 9. 실제 지구맵 노출 기준

E02 이후에도 10% 인식 밀도를 유지한다.

| 노출 단계 | 허용 | 금지 |
|---|---|---|
| 원격 감지 | 해안선 일부, 가명, 캠페인 계열 얼룩 | 실명 도시명, 유명 랜드마크 정면 노출 |
| 부분 복원 | 철도/항만/강/산맥 방향감 | 실제 역명/공항명/병원명 |
| 출격 가능 | 로컬 기능명, 캠페인식 장소명 | 현실 장소 사고 패러디 |
| 진실 복원 | 깨진 좌표, 옛 지도 조각, 실명 일부 암시 | 전체 실명 지도 공개 |

예시:

| 실제 앵커 | 유저용 표현 |
|---|---|
| 일본 태평양 도시축 | 태평양 환승권 |
| 북미 태평양 항만-내륙 물류축 | 구독항만 내륙권 |
| 북해-라인-채널 벨트 | 보험철도 연결권 |
| 걸프 스마트시티-에너지 벨트 | 냉방도시 에너지권 |

| 조건 | 설명 |
|---|---|
| 수신 복원 | 보급소가 해당 광역권의 캠페인 신호를 해석한다. |
| 중계 기준점 | 폐역, 대피소, 산악 무전소, 항만 창고 같은 로컬 침묵점이 필요하다. |
| 회수선 기준 | 현지 비콘 또는 회수 가능한 고정 좌표가 있어야 한다. |
| 태그 호환 | 현지 캠페인 태그가 보급소 태그 체계와 충돌하지 않아야 한다. |
| 위험 비용 | 원격일수록 인양 손실, 태그 오염, 신호 노출이 증가한다. |

판정:

```text
전 지구 출격은 걷거나 비행기로 이동하는 현실 여행이 아니라,
캠페인 신호와 침묵권 중계점을 붙잡아 제한적으로 열린 회수 작전이다.
```

단, 이것도 순간이동은 아니다.

- 시작 좌표는 현지 침묵점/중계점에 묶인다.
- 인양 실패 시 귀환 지연, 회수품 손실, 현지 신호 오염이 남는다.
- 광역권 전체를 자유롭게 이동하지 못하고, 열린 로컬 주변만 출격한다.

## 11. 보스/시즌 운용

E02 이후에도 96개 보스 이름을 전부 확정하지 않는다.

| 단계 | 확정하는 것 |
|---|---|
| 광역권 원칙 | 어떤 캐릭터/캠페인 계열을 다룰지 |
| 시즌 슬라이스 | 1~3개 캠페인, 주요 로컬, 보스 슬롯 |
| 보스 세부화 | 시즌 주인공 캠페인의 보스명/패턴/선택 |
| 구현 | 전투 수치, 보상, 정산 문구 |

보스 우선순위:

1. 캐릭터 상처와 직접 연결된다.
2. 해당 광역권의 2차 붕괴 원인을 대표한다.
3. E01에서 보지 못한 등록 방식을 보여준다.
4. 후속 광역권 신호를 열 수 있다.

## 12. 금지 설정

- E02~E12를 모두 E01처럼 40개 로컬로 확정한다.
- E## 코드를 출시 순서로 오해한다.
- 실제 지명을 유저용 지도에 바로 노출한다.
- 새 지역을 캐릭터 심화 없이 단순 스테이지로 연다.
- 96개 캠페인 밖에 새 정식 캠페인을 만든다.
- 각 나라/도시를 관광 이미지나 패러디로 단순화한다.
- 인간이 많은 광역권을 자유 인간 사회처럼 보이게 한다.
- 원격 출격을 무료 순간이동처럼 처리한다.
- E03 같은 초대형 권역을 초반에 열어 스케일을 감당하지 못하게 만든다.
- E02 첫 확장에서 해방 로봇 정식 플레이어블을 갑자기 지급한다.

## 13. 논리 검증

| 항목 | 판정 |
|---|---|
| 96개 캠페인 구조를 지키는가? | 통과. 모든 확장은 기존 E02~E12, C01~C08 안에서 열린다. |
| 제작량이 감당 가능한가? | 통과. E02 이후 기본값을 12~18개 로컬 시즌 슬라이스로 제한했다. |
| 캐릭터 해금 구조와 맞는가? | 통과. 광역권은 캐릭터 원점/심화/교차 활동과 연결된다. |
| E02가 첫 외부 확장으로 설득력 있는가? | 통과. 근거리 신호, 생활 인프라 차이, 통근/편의점/방재/리콜 캠페인이 강하다. |
| 실제 지구맵 인식 밀도를 유지하는가? | 통과. 10% 인식 기준과 실루엣+가명을 유지한다. |
| 원격 출격이 순간이동처럼 보이지 않는가? | 조건부 통과. 반드시 중계점, 비콘, 회수선, 위험 비용을 UI/서사에 남겨야 한다. |
| 대사고 하나로 지역을 설명하지 않는가? | 통과. 각 광역권은 지역별 인프라와 2차 붕괴 원인을 가진다. |
| 후반 미스터리를 너무 빨리 소모하지 않는가? | 통과. 방송/검역/초대형 제조는 후반 우선으로 뒀다. |

## 14. 최종 잠금

```text
1. E## 코드는 출시 순서가 아니라 지구 광역권 카탈로그 코드다.

3. 첫 외부 광역권 후보는 E02 일본 태평양 메갈로폴리스가 가장 안전하다.

```
