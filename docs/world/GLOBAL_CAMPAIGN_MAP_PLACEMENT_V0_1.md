# Atomic Ad Survivors 전 지구 캠페인 지도 배치표 v0.1

상태: v0.1 지도 제작 전 배치 기준  
목적: 96개 정식 캠페인을 실제 지구 실루엣 위에 배치하기 위한 광역권/방향/노출 규칙을 정한다.  
연결 문서: `docs/world/WORLD_MAP_CAMPAIGN_ECOLOGY_V0_1.md`, `docs/world/GLOBAL_CAMPAIGN_CATALOG_V0_1.md`, `docs/world/E02_PLUS_REGION_EXPANSION_PRINCIPLES_V0_1.md`, `docs/world/E01_FIRST_SEASON_LOCAL_NODES_V0_1.md`, `docs/world/E01_WORLD_MAP_BLOCKOUT_V0_1.md`, `docs/world/WORLD_TERMINOLOGY_LOCK_V0_1.md`

## 0. 판정 요약

- 월드맵은 실제 지구 지도 위에 캠페인 생태를 덮은 지도다.
- 유저용 지도는 `실루엣+가명` 기반이다. 100명 중 10명 정도가 실제 장소를 알아채는 밀도를 목표로 한다.
- 실명 지명은 초반에 직접 노출하지 않는다. 해안선, 강, 산맥, 항만/공항 거리, 도시 밀도, 교통축, 랜드마크 실루엣으로 암시한다.
- 전 지구에는 12개 광역권이 있고, 각 광역권마다 8개 정식 캠페인이 있다.
- E01 한반도 서부 수도권-항만-산업 벨트가 첫 시즌 광역권이다.
- E01 이후의 광역권 해금은 캐릭터의 원점/심화 서사에 따라 열린다.
- 보스 세부화도 캠페인 목록 단계에서 전부 확정하지 않고, 캐릭터/시즌/지역 해금 시점에 확정한다.

## 1. 지도 노출 원칙

### 1.1 10% 인식 밀도

실제 지명 인식 밀도 목표:

```text
100명이 보면 10명 정도가 실제 지리 앵커를 알아챈다.
```

좋은 반응:

```text
처음 보면 가상 지역처럼 보인다.
지도와 도시 구조를 아는 유저는 어렴풋이 알아챈다.
후반 기록, 깨진 좌표, 옛 지도 조각을 보면 확신한다.
```

나쁜 반응:

- 누구나 바로 실명 도시를 알아본다.
- 실제 재난지 패러디처럼 느껴진다.
- 반대로 너무 가려서 실제 지구맵을 쓰는 의미가 사라진다.

### 1.2 실명 대신 남길 단서

| 단서 | 사용 방식 |
|---|---|
| 해안선 | 실제 윤곽은 유지하되 UI 노이즈와 캠페인 얼룩으로 일부 가린다. |
| 강줄기 | 주요 도시권의 방향감만 남긴다. |
| 산맥/고지대 | 침묵권, 대피권, 전파 음영의 근거로 쓴다. |
| 항만/공항 거리 | R07 같은 검역/이송 캠페인의 물리적 설득력으로 쓴다. |
| 도시 밀도 | 상업/의료/방송 권역의 위치 감각을 만든다. |
| 도로/철도 축 | 물류/통행/반품 캠페인의 이동축으로 쓴다. |
| 랜드마크 실루엣 | 후반 복원 단계에서만 일부 보여준다. |

### 1.3 노출 단계

| 단계 | 월드맵 표현 | 실제 지리 노출 |
|---|---|---|
| 미관측 | 지구 실루엣 위 노이즈 덩어리 | 없음 |
| 원격 감지 | 가명 광역권과 캠페인 계열만 표시 | 해안선/방향 일부 |
| 부분 복원 | 로컬 노드와 주요 축 표시 | 항만/공항/강/산맥 암시 |
| 출격 가능 | 인양 반경, 위험도, 보상 표시 | 실제 지형 구조가 더 보임 |
| 진실 복원 | 옛 지도 조각, 깨진 좌표, 라이프라인 로그 | 실명 일부 암시 가능 |

## 3. E01 첫 시즌 광역권 상세 배치

E01은 첫 시즌의 실제 플레이 지역이다. 유저에게는 실명 대신 가명과 캠페인식 이름으로 보이지만, 내부 배치는 한반도 서부 수도권-항만-산업 벨트를 기준으로 한다.

### 3.1 방향 기준

| 방향 | 권역 | 의미 |
|---|---|---|
| 중심 외곽 | R08 침묵권 군락 | 윤서의 시작 보급소, 인양 중심, 지도 기준점 |
| 동/남동 | R01 외곽 주택가 | 첫 출격, 가족/입주 캠페인, 생활형 공포 |
| 남/남서 | R03 물류 회수 벨트 | 반품/분류/회수, 윤서 중반 핵심 |
| 서/남서 | R07 항만/이송 봉쇄 구역 | 항만/공항/검역, 탈출 미스터리 |
| 남 | R04 산업 오염 지대 | 배터리/로봇/충전, 해방 로봇 서사 |
| 동/도심 | R02 의료 격리 구역 | 병원/보험/환자 등록, 구조 윤리 |
| 도심 고밀도 | R06 상업/멤버십 지구 | 인간 브로커, 태그 시장, 멤버십 폭주 |
| 북/도심 고지점 | R05 방송 중추 지구 | 수신탑, 데이터 센터, 후반 진실 |

주의: 위 방향은 실제 지도 제작용 내부 기준이다. 유저용 지도에서는 방향을 흐리고, 캠페인 농도와 수신 복원 단계로 보이게 한다.

### 3.2 인양 반경 기준

| 구역 | 인양 상태 | 설명 |
|---|---|---|
| 내부 침묵권 | 안정 | R08-L01/R08-L02. 정산, 수리, 회복 가능 |
| 근거리 출격권 | 안정~불안정 | R01 대부분. 첫 시즌 핵심 반복 출격권 |
| 중거리 회수권 | 불안정 | R03 일부, R02 일부. 비콘 소모와 회수선 노출 증가 |
| 고위험 외곽권 | 매우 불안정 | R04/R06/R07 일부. 실패 시 태그/전리품 손실 큼 |
| 원격 감지권 | 인양 불가 | R05 깊은 구역, R07 깊은 봉쇄선. 먼저 중계/침묵권 확장 필요 |

### 3.3 E01 초반 월드맵 노출

처음 보이는 것:

- R08-L01 침묵 보급소.
- R08-L02 역송출 주유소.
- R01-L01 침묵 가장자리.
- R01-L02 분양 주택 루프.
- R01-L03 모델하우스 결절의 약한 신호.

약한 신호로만 보이는 것:

- R03 물류 회수 벨트.
- R02 의료 격리 구역.
- R04 산업 오염 지대.

숨기는 것:

- R05 방송 중추 지구의 정확한 위치.
- R06 상업/멤버십 지구의 인간 브로커 상태.
- R07 항만/이송 봉쇄 구역의 탈출 정보.
- E02 이후 다른 지구 광역권.

## 4. 캐릭터 기반 광역권 해금 규칙

E01 이후의 광역권은 단순 스테이지 진행이 아니라 캐릭터의 원점/심화에 따라 열린다.

캐릭터 해금 단계와 플레이어블 전환 기준은 `docs/world/CHARACTER_UNLOCK_STRUCTURE_V0_1.md`를 따른다.
E02 이후 광역권의 추천 해금 순서와 시즌 슬라이스 규모는 `docs/world/E02_PLUS_REGION_EXPANSION_PRINCIPLES_V0_1.md`를 따른다.

### 4.1 원점과 심화

| 구분 | 설명 |
|---|---|
| 캐릭터 원점 캠페인 | 캐릭터가 합류하거나 자기 정체성이 처음 드러나는 캠페인 |
| 캐릭터 심화 캠페인 | 합류 후 다른 광역권에서 캐릭터의 진짜 갈등이 깊어지는 캠페인 |
| 교차 활동 캠페인 | 해당 캐릭터가 원점 밖에서 문제를 해결하거나 오해받는 캠페인 |

원칙:

- 캐릭터는 한 지역에만 묶이지 않는다.
- 캐릭터의 원점은 하나여도, 심화는 다른 광역권에서 열릴 수 있다.
- 새 광역권은 캐릭터가 좋아서 따라가고 싶게 열려야 한다.

### 4.2 캐릭터-광역권 해금 후보

| 캐릭터 유형 | 원점 후보 | 심화 후보 | 여는 캠페인 성격 |
|---|---|---|---|
| 윤서 | E01_C03 반품 회수 벨트 | E03_C03, E10_C03 | 반품/회수/정산의 원본과 변형 |
| 해방 로봇 캐릭터 | E01_C04 정품충전 공단 | E10_C04, E03_C04, E02_C04 | 정품 인증, 재동기화, 리콜 |
| 의료계 캐릭터 | E01_C02 백색팔찌 격리권 | E02_C02, E07_C02, E09_C02 | 치료와 퇴원 불가의 윤리 |
| 방송/정보 캐릭터 | E01_C05 마지막 수신탑 | E05_C05, E09_C05, E10_C05 | 기록, 추천, 조작된 진실 |
| 브로커 출신 캐릭터 | E01_C06 포인트 거리 | E11_C06, E06_C06 | 인간 약탈, 멤버십, 거래 윤리 |
| 이송/검역 캐릭터 | E01_C07 마지막 출항권 | E08_C07, E06_C07 | 탈출, 봉쇄, 검역, 탑승권 |
| 침묵권 기술자 | E01_C08 서부 비상보급망 | E10_C08, E12_C08 | 침묵권, 보급, 무선/전력 생존 |

## 5. 보스 슬롯 운용 규칙

96개 정식 캠페인은 지금 단계에서 모든 보스를 세부 확정하지 않는다. 대신 각 캠페인은 `보스 슬롯`만 가진다.

### 5.1 보스 슬롯 필드

| 필드 | 설명 |
|---|---|
| 캠페인 ID | 어느 정식 캠페인에 속하는가 |
| 보스 역할 | 등록 절차의 얼굴, 결절 관리자, 최종 심사 장치 등 |
| 강제 등록 방식 | 가족, 환자, 반품품, 정품 장치, 시청자 등 |
| 연결 캐릭터 | 이 보스를 통해 깊어지는 캐릭터 |
| 대표 로컬 | 실제 보스전이 벌어지는 로컬 |
| 처리 결과 | 결절 파열, 기억 추출, 배급 우회, 권한 박탈 등 |

### 5.2 확정 시점

| 시점 | 확정하는 것 |
|---|---|
| 캠페인 카탈로그 단계 | 캠페인명, 지도 앵커, 등록 방식, 보스 슬롯 |
| 캐릭터 해금 단계 | 연결 캐릭터, 감정 갈등, 보스가 건드리는 상처 |
| 시즌 진입 단계 | 보스명, 외형, 패턴, 로컬, 보스 후 선택 |
| 구현 단계 | 전투 패턴, 보상, 정산 문구, 재방문 변화 |

### 5.3 금지

- 96개 보스를 지금 전부 이름까지 확정하지 않는다.
- 보스를 단순 거대 몬스터로 만들지 않는다.
- 보스가 사람을 죽이려는 악당으로만 보이게 하지 않는다.
- 캐릭터와 연결되지 않은 보스는 우선순위를 낮춘다.
