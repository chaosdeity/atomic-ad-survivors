# Atomic Ad Survivors R01 첫 출격 레벨 연결 v0.1

상태: v0.1 제작 연결 기준  
목적: E01 P0 월드맵 노드 `R01-L01~R01-L03`을 현재 Godot R01 플레이 맵, 웨이브, 오브젝트, 인양 규칙과 연결한다.  
연결 문서: `docs/world/E01_P0_WORLD_MAP_FIRST_SCREEN_FLOW_V0_1.md`, `docs/world/E01_WORLD_MAP_BLOCKOUT_V0_1.md`, `docs/world/EMERGENCY_EXTRACTION_RULES_V0_1.md`, `docs/world/R01_L03_FIRST_BOSS_DETAIL_V0_1.md`, `docs/world/WORLD_TERMINOLOGY_LOCK_V0_1.md`  
연결 코드: `scripts/r01_layout_blockout.gd`, `scripts/r01_zone_layout.gd`, `scripts/r01_map_assembly.gd`, `scripts/main.gd`, `scripts/wave_director.gd`

## 0. 판정 요약

- 현재 R01 코드 구조는 P0 월드맵 흐름과 대체로 맞는다.
- 첫 출격은 `R01-L01 침묵 가장자리` 완료로 본다. 108초 강제 회수 때문에 첫 출격에서 R01-L03까지 가면 안 된다.
- `외곽 회수 차선`은 새 월드맵 노드가 아니라 `R01-L01` 내부 구간으로 둔다.
- `오픈하우스 거리`는 새 월드맵 노드가 아니라 `R01-L02` 안에서 `R01-L03` 신호를 여는 위험 구간으로 둔다.
- `R01-L03 모델하우스 결절`은 첫 보스 목적지다. 보스 이름/패턴은 `docs/world/R01_L03_FIRST_BOSS_DETAIL_V0_1.md` 기준으로 본다.
- 현재 코드에는 `밥표`, `신호표`, `배급권` 같은 구용어가 남아 있다. 이 문서는 `식량태그`, `수신태그`, `보급태그` 기준으로 해석하되, 코드 용어 교체는 구현 전 마이그레이션 작업으로 둔다.

가장 중요한 잠금:

```text
월드맵 R01-L01~L03은 큰 선택 노드이고,
Godot의 7개 R01 앵커는 그 안에 들어가는 실제 전투/연출 구간이다.
```

## 1. 연결 원칙

| 원칙 | 설명 |
|---|---|
| 월드맵 노드는 늘리지 않는다 | P0 첫 화면은 R08/R01의 5개 노드만 선명하게 보인다. |
| 실제 레벨 앵커는 유지한다 | 제작/QA/배치 기준에는 세부 앵커가 필요하다. |
| 첫 출격은 짧게 끝난다 | 108초 강제 회수는 죽음이 아니라 인양 튜토리얼이다. |
| 첫 보스는 처음부터 보이되 잠겨 있다 | R01-L03은 약한 신호로 보이나, 첫 출격 직후 바로 진입하지 못한다. |
| 사이드 구간은 P1로 미룬다 | 배수로와 가짜 귀환로는 R01 재방문 변형으로 쓴다. |

## 2. 월드맵 노드와 인게임 앵커 매핑

| 월드맵 노드 | 유저용 이름 | 인게임 앵커 | 코드 표시명 | P0 역할 | 판정 |
|---|---|---|---|---|---|
| R01-L01 | 침묵 가장자리 | `silence_edge_start` | 침묵 가장자리 | 첫 출격 시작점 | 유지 |
| R01-L01 | 침묵 가장자리 내부 | `outer_recovery_lane_anchor` | 외곽 회수 차선 | 첫 회수선/귀환 감각 | R01-L01 내부 구간 |
| R01-L02 | 분양 주택 루프 | `subdivision_loop_center` | 분양 주택 루프 | 기본 런 중심 | 유지 |
| R01-L02 | 분양 주택 루프 내부 | `open_house_street_anchor` | 오픈하우스 거리 | R01-L03 신호를 여는 위험 구간 | R01-L02 내부 구간 |
| R01-L03 | 모델하우스 결절 | `model_house_node_anchor` | 모델하우스 접근로 | 첫 보스 진입/결절 | 명칭 정리 필요 |
| R01-L04 | 배수로 침묵 주머니 | `drain_pocket_anchor` | 배수로 침묵 주머니 | P1 숨은 회수 | P0에서는 흐린 신호 |
| R01-L05 | 가짜 귀환로 | `fake_return_route_anchor` | 끊긴 광고 산책로 | P1 재방문 변형 | 별칭 연결 필요 |

명칭 판정:

- `model_house_node_anchor`는 월드맵/제작표에서는 `모델하우스 결절`로 둔다.
- 전투 중 구역 진입 문구는 `모델하우스 접근로`를 써도 된다. 다만 보스 노드/월드맵/정산 화면은 `모델하우스 결절`로 맞춘다.
- `끊긴 광고 산책로`는 분위기는 좋지만 월드맵의 `가짜 귀환로`와 기능이 더 명확하다. v0.1에서는 `가짜 귀환로(끊긴 광고 산책로)`로 병기하고, 최종 UI는 `가짜 귀환로` 우선이다.

## 3. P0 플레이 흐름

```text
R08-L01 침묵 보급소
-> R08-L02 역송출 주유소 확인
-> R01-L01 침묵 가장자리 첫 출격
-> 108초 강제 회수/인양
-> R01-L02 분양 주택 루프 해금
-> 오픈하우스 거리에서 결절 신호 확보
-> R01-L03 모델하우스 결절 해금
-> 첫 보스 처리
```

| 단계 | 월드맵 상태 | 실제 레벨 상태 | 결과 |
|---|---|---|---|
| S0 | R01-L01만 출격 가능 | 시작 위치가 `silence_edge_start` | 첫 위험 확인 |
| S1 | R01-L01 완료 | 108초 강제 회수 발생 | R01-L02 경로 선명화 |
| S2 | R01-L02 출격 가능 | `subdivision_loop_center` 중심 기본 런 | 식량태그/거주태그 흔적 회수 |
| S3 | R01-L03 약한 신호 강화 | `open_house_street_anchor` 체류로 신호 단계 상승 | 모델하우스 신호 해석 |
| S4 | R01-L03 출격 가능 | `model_house_node_anchor` 접근 | 첫 보스 진입 |
| S5 | P0 완료 | 보스 처리 후 정산 | P1 흐린 신호 개방 |

## 4. R01-L01 첫 출격 연결

R01-L01은 유저가 처음으로 캠페인 밖에 나간다고 느끼는 구간이다. 목적은 승리가 아니라 생존, 회수선, 캠페인 압력, 인양을 이해시키는 것이다.

| 항목 | 기준 |
|---|---|
| 시작 앵커 | `silence_edge_start` |
| 내부 구간 | `outer_recovery_lane_anchor` |
| 시간 목표 | `108초 회수까지 생존` |
| 주요 웨이브 | 0~30초 튜토리얼, 30~90초 빠른 판촉 전단, 90~108초 저가 광고 압착 초입 |
| 주요 적 | 기본 광고판, 빠른 쿠폰형 적, 탱크형 가전 초입 |
| 주요 오브젝트 | 낮은 담장, 우편함 힌트, 꺼진 간판, 회수 차선 표식, 보급소 신호 잔해 |
| 캠페인 의미 | 보급소 신호가 닿는 마지막 경계에서 주택가 캠페인이 윤서를 읽기 시작한다. |
| 회수/인양 | 70초 압력 경고, 88초 신호 붕괴, 108초 강제 회수 |
| 월드맵 결과 | R01-L02의 점선 경로가 선명해진다. |

R01-L01에서 보여주면 안 되는 것:

- 가족 적합성 심사의 정체를 모두 설명하지 않는다.
- 보스 신호를 전투 목표처럼 직접 찍지 않는다.
- 배수로/가짜 귀환로를 선택지처럼 열지 않는다.
- 인양을 무료 부활처럼 보이게 하지 않는다.

## 5. R01-L02 기본 런 연결

R01-L02는 R01의 실제 반복 플레이 중심이다. 집, 우편함, 전단, 스마트 도어, 청소/홈케어 장치가 사람을 `입주자`, `가족 구성원`, `정상 고객`으로 분류하려 한다.

| 항목 | 기준 |
|---|---|
| 중심 앵커 | `subdivision_loop_center` |
| 내부 위험 구간 | `open_house_street_anchor` |
| 주요 목표 | 반복 주택 단지를 통과해 입주/가족 신호를 추적한다. |
| 주요 오브젝트 | 반복 주택, 판매 스피커, 우편함 장치, 전단 더미, 가격/고객 태그, 추천 동선 데칼 |
| 주요 적 | 기본 광고판, 빠른 쿠폰, 스피커형 광고, 차저형 쿠폰, 탱크형 가전 |
| 태그 후보 | 식량태그, 거주태그 흔적, 수신태그 후보 |
| 게임 기능 | 기본 런, 카드/정산 학습, 광고 감사 학습 |
| 월드맵 결과 | R01-L03 잠금 문구가 `심사 신호 감지`에서 진입 가능 상태로 바뀔 수 있다. |

오픈하우스 거리의 역할:

| 체류 조건 | 현재 코드 효과 | 월드맵 해석 |
|---|---|---|
| 약 25초 | 보스 신호 `미약함` | 수신태그 후보 감지 |
| 약 50초 | 보스 신호 `감지됨` | 모델하우스 방향 해석 |
| 약 75초 | 보스 신호 `근접` | R01-L03 진입 조건에 가까워짐 |

주의:

- 오픈하우스 거리를 별도 월드맵 노드로 만들지 않는다.
- 오픈하우스는 R01-L02 안의 위험 보상 구간이다.
- 여기서 오래 머물수록 R01-L03 신호는 선명해지지만 밀도와 속도도 올라간다.

## 6. R01-L03 첫 보스 진입 연결

R01-L03은 첫 보스 노드다. 보스 세부 기준은 `docs/world/R01_L03_FIRST_BOSS_DETAIL_V0_1.md`에서 잠근다.

| 항목 | 기준 |
|---|---|
| 앵커 | `model_house_node_anchor` |
| 월드맵 이름 | 모델하우스 결절 |
| 현재 코드 전투 문구 | 모델하우스 접근로, 스마일 홈 시어머니 |
| 진입 조건 | 오픈하우스/감사/정산을 통해 신호 단서 3개 확보 |
| 현재 코드 조건 | `meta_progression.has_all_signal_clues()` 및 240초 이후 보스 시작 |
| 보스 역할 | 스마일 홈 가족심사 관리자. 전투 별칭은 스마일 홈 시어머니 |
| 보상 | 캠페인 코어 파편, 거주태그 우회권 후보, P1 신호 해석 단서 |
| P0 완료 효과 | R01-L04/R01-L05 및 R02/R03/R04 약한 신호 개방 |

보스 세부화 잠금:

- 정식 역할명은 `스마일 홈 가족심사 관리자`다.
- 전투 별칭은 `스마일 홈 시어머니`를 유지한다.
- 보스 처리는 사망이 아니라 `결절 침묵`과 `심사 우회`다.
- 보스 후 선택은 `결절 파괴`와 `기억 추출`로 나뉜다.
- 윤서의 집/가족 감정만 건드리고, 반품 담당자 핵심 과거는 R03에 남긴다.

## 7. P1로 미루는 R01 노드

| 월드맵 노드 | 인게임 앵커 | P1 역할 | P0 처리 |
|---|---|---|---|
| R01-L04 배수로 침묵 주머니 | `drain_pocket_anchor` | 침묵권의 과학적 근거, 숨은 회수물 | 보이지 않거나 흐린 점선 |
| R01-L05 가짜 귀환로 | `fake_return_route_anchor` | 캠페인이 귀환 패턴을 흉내 내는 재방문 변형 | 보이지 않거나 경고 없는 잡음 |

P1에서 열어야 하는 이유:

- P0에 열면 첫 화면 정보가 과해진다.
- 배수로는 침묵권 설명이 필요한 노드라 첫 출격 전에 열면 설명 순서가 꼬인다.
- 가짜 귀환로는 유저가 실제 인양/귀환을 한 번 겪은 뒤에야 의미가 생긴다.

## 9. 인양 튜토리얼 연결

R01-L01의 108초 강제 회수는 실패 처리가 아니라 세계 규칙의 첫 시연이다.

| 시간 | 현재 코드 이벤트 | 유저가 이해해야 하는 것 |
|---:|---|---|
| 0초 | `silence_edge_start`에서 시작 | 보급소 밖은 신호가 살아 있다. |
| 30초 | 첫 광고 감사 | 캠페인은 윤서를 고객/입주자로 읽는다. |
| 70초 | 캠페인 압력 비정상 상승 | 오래 버티면 등록 압력이 증가한다. |
| 88초 | 캠페인 신호 붕괴 | 인양 없이는 계속 버틸 수 없다. |
| 108초 | 강제 회수 | HP 0/강제 회수는 죽음이 아니라 비콘 기반 긴급 인양이다. |

정산 화면 문구 기준:

```text
인양 완료. 회수품을 정산합니다.
```

첫 출격 완료 후 월드맵 문구 후보:

```text
주택가 안쪽 경로가 고정되었습니다.
```

| 항목 | 판정 | 메모 |

## 11. 제작 잠금

- R01-L01은 첫 출격/첫 인양 노드다.
- R01-L02는 기본 반복 런 노드다.
- R01-L03은 첫 보스 노드다.
- `외곽 회수 차선`은 R01-L01 내부 구간이다.
- `오픈하우스 거리`는 R01-L02 내부 구간이다.
- `배수로 침묵 주머니`와 `가짜 귀환로`는 P1 재방문 노드다.
- 첫 출격에서 R01-L03까지 도달시키지 않는다.
- 첫 보스의 핵심은 괴물 처치가 아니라 가족/입주 심사 절차를 끊는 것이다.
