# Atomic Ad Survivors E01 P1 확장 설계 v0.1

상태: v0.1 기준 문서  
목적: R01-L03 첫 보스 이후 E01 월드맵이 어떤 순서와 조건으로 확장되는지 고정한다.  
연결 문서: `docs/world/E01_FIRST_SEASON_LOCAL_NODES_V0_1.md`, `docs/world/E01_WORLD_MAP_BLOCKOUT_V0_1.md`, `docs/world/E01_P0_WORLD_MAP_FIRST_SCREEN_FLOW_V0_1.md`, `docs/world/R01_L03_FIRST_BOSS_DETAIL_V0_1.md`, `docs/world/CHARACTER_UNLOCK_STRUCTURE_V0_1.md`

## 0. 판정 요약

P1은 "첫 보스 후 모든 1장 노드가 열린다"가 아니다.  
P1은 R01-L03 처리 후 유저가 세계의 바깥 결을 처음 보는 확장 단계다.

잠금:

- P1의 핵심은 선형 진행이 아니라 `신호 감지 -> 보급소 해석 -> 선택 출격 -> 후속 축 개방`이다.
- 첫 보스 직후 실제 출격 가능 노드는 적어야 한다.
- R01 재방문과 R08 확장은 가장 먼저 열린다.
- R03 물류 회수와 R02 의료 격리는 첫 외부 진입 축이다.
- R04 산업/해방 로봇 축은 R03 또는 R02 단서 뒤에 여는 편이 자연스럽다.
- R06 상업/브로커 축은 처음부터 선명하게 열지 않고, 가짜 태그/브로커 단서를 통해 늦게 연다.
- R05 방송 중추와 R07 항만/이송은 P1에서 실제 진입하지 않는다. 흐린 원격 신호까지만 허용한다.

가장 큰 논리 위험:

```text
P1 16개 노드를 첫 보스 직후 한꺼번에 출격 가능으로 열면
보급소의 취약성, 인양 거리 제한, 윤서의 정보 부족이 모두 약해진다.
```

따라서 P1은 노드 개수보다 `해금 상태`가 중요하다.

## 1. P1의 역할

P0가 가르치는 것:

- 보급소는 작은 공백이다.
- 인양은 장비와 거리 제한에 묶인다.
- 광고는 적이 아니라 환경이다.
- 캠페인은 사람을 죽이기보다 등록하려 한다.
- 첫 보스는 괴물이 아니라 심사 절차다.

P1이 가르쳐야 하는 것:

- 주택가 밖에도 캠페인 생태가 이어진다.
- 인간이 사라진 이유는 캠페인 하나가 아니라 의료 붕괴, 물류 약탈, 오염, 태그 거래가 겹친 결과다.
- 윤서는 특별한 영웅이라서가 아니라 반품/회수 규칙을 읽을 수 있는 생존자라서 출격 가치가 있다.
- 해방 로봇은 적/아군 이분법이 아니라 충전태그 의존, 정품 인증, 재동기화 사이에서 흔들리는 예외다.
- 인간 브로커는 캠페인보다 더 의도적인 악이 될 수 있다.

P1의 유저 경험 목표:

```text
첫 보스 하나를 끝냈을 뿐인데, 세계가 더 안전해진 것이 아니라 더 넓게 들리기 시작한다.
```

## 2. P1 해금 상태 정의

P1은 노드를 아래 상태로 나누어 보여준다.

| 상태 | 유저 표시 | 의미 | 출격 |
|---|---|---|---|
| 숨김 | 표시 없음 | 아직 보급소가 감지하지 못함 | 불가 |
| 신호 감지 | 흐린 얼룩, 라벨 없는 잡음 | 존재는 감지되지만 좌표/인양선 불안정 | 불가 |
| 해석 가능 | 흐린 라벨, `신호 해석 필요` | 보급소 정산/대화/회수물 분석으로 열 수 있음 | 불가 |
| 접근 계산 | 점선 경로, 위험도 표시 | 인양 가능성은 있으나 조건 미충족 | 조건부 |
| 출격 가능 | 선명한 노드, 인양 등급 표시 | 비콘/좌표/회수선 조건 충족 | 가능 |
| 보류 | 원격 신호, 잠금 문구 | 후반/P2용. 지금 열면 안 됨 | 불가 |

중요:

- `P1 노드`라는 내부 분류와 `출격 가능`은 같은 말이 아니다.
- 첫 보스 직후에는 대부분 `신호 감지` 또는 `해석 가능`으로 시작한다.
- 인양 등급 A2/A3 노드는 거리와 침묵권 확장 조건을 따로 확인한다.

첫 보스 직후 출격 가능으로 바로 열리는 노드는 2~3개 이하가 좋다.

## 4. P1 추천 진행 구조

P1은 완전 선형이 아니라 3단계 분기 구조로 둔다.

### 4.1 P1-A: 재정렬 단계

R01-L03 처리 후 바로 이어지는 짧은 안정화 구간이다.

| 노드 | 해금 조건 | 기능 |
|---|---|---|
| R01-L04 배수로 침묵 주머니 | R01-L03 처리 후 보급소 정산 | 침묵권의 물리 근거, 차폐 구조, 숨은 회수물 |
| R08-L03 폐역 무음 플랫폼 | R01-L03 처리 후 또는 R01-L04 완료 후 | 인양 반경 확장, 폐역 중계, 침묵권 네트워크 |
| R01-L05 가짜 귀환로 | R01-L04 완료 또는 `기억 추출` 선택 시 조기 감지 | 캠페인 학습, 가짜 안전 신호, 재방문 위험 |
| R08-L04 민방위 저장고 | R08-L03 완료 후 | 보급태그와 실제 물자 부족의 차이 |

유저가 배우는 것:

```text
보급소는 안전해서 살아남은 곳이 아니라,
약한 신호 공백을 계속 보수해야 하는 곳이다.
```

### 4.2 P1-B: 첫 외부 권역 선택

R01/R08 안정화 후 유저에게 첫 외부 권역을 선택하게 한다.

| 축 | 노드 | 해금 조건 | 주제 |
|---|---|---|---|
| 윤서 회수축 | R03-L01 -> R03-L02 -> R03-L03 | R01-L05 또는 R01-L03 정산품의 반품 라벨 해석 | 반품, 분류, 윤서의 침묵 보급소 회수/검수 역할 |
| 의료 구조축 | R02-L01 -> R02-L02 -> R02-L03 | R01-L03 정산품의 환자/보호자 등록 흔적 해석 | 환자 등록, 구출 윤리, 진료태그 |

추천 구조:

- R03과 R02 중 하나를 먼저 가도 된다.
- 단, 둘 다 P1 중반까지는 열어야 한다.
- 윤서 중심 장면을 빠르게 밀고 싶으면 R03을 먼저 권장한다.
- 세계관의 잔혹한 생존 논리를 먼저 보여주고 싶으면 R02를 먼저 권장한다.

P1-B의 논리 기준:

```text
R03은 윤서가 왜 출격 가치가 있는지 보여준다.
R02는 캠페인이 살리려 해도 사람답게 살리지 못한다는 기준을 보여준다.
```

### 4.3 P1-C: 위험 확장 단계

외부 권역 하나 이상을 처리한 뒤, P1 후반 축이 열린다.

| 축 | 노드 | 해금 조건 | 주제 |
|---|---|---|---|
| 로봇 충전축 | R04-L01 -> R04-L02 -> R04-L03 | R03-L03의 보증/부품 단서 또는 R02-L03의 의료 장비 배터리 단서 | 해방 로봇, 충전태그, 재동기화 |
| 인간 브로커축 | R06-L01 -> R06-L02 -> R06-L03 | R03 또는 R02에서 가짜 태그/브로커 흔적 확보 | 태그 거래, 인간 악역, 멤버십 악용 |

추천:

- R04는 P1 핵심 축으로 여는 것이 좋다.
- R06은 P1 후반 또는 선택형 축으로 두는 것이 좋다.
- R06을 너무 빨리 열면 캠페인의 규칙보다 인간 악역이 먼저 주인공이 되어 세계 원인이 흐려진다.

## 6. 보스 선택에 따른 P1 변형

R01-L03 후속 선택은 P1의 내용 자체를 갈라놓지 않고, 최초 상태와 위험도를 바꾼다.

| 선택 | P1 영향 | 장점 | 대가 |
|---|---|---|---|
| 결절 파괴 | R01 압력 감소, R01-L04 안정화, R01-L05 감지 지연 | 안전한 재정렬, 보급소 노출 감소 | 윤서 기억/반품 단서 일부 흐림 |
| 기억 추출 | R03/R02 신호 선명화, R01-L05 조기 감지 | 윤서/주민 기억 단서 확보, 외부 축 빠른 개방 | R01 캠페인 학습 증가, 가짜 귀환로 위험 상승 |

금지:

- 결절 파괴가 선이고 기억 추출이 악이라는 식으로 만들지 않는다.
- 선택 하나로 P1 축 전체를 영구 차단하지 않는다.
- 선택 효과는 `압력`, `정보량`, `출격 우선순위`에 반영한다.

## 7. 월드맵 UI 표시 기준

P1 프리뷰:

- 첫 보스 직후 지도에 흐린 신호를 추가한다.
- R03/R02/R04/R06 이름을 모두 즉시 표시하지 않는다.
- R03은 라벨/스캐너 노이즈로, R02는 백색 팔찌/응급 신호로, R04는 회색 얼룩과 충전 노이즈로 구분한다.
- R06은 처음에는 광고 상권처럼 보이지 않고, 이상한 가격표/가짜 태그 흔적으로 시작한다.

패널 문구 예시:

| 상태 | 문구 |
|---|---|
| 신호 감지 | `희미한 캠페인 신호가 잡힙니다.` |
| 해석 가능 | `회수물을 분석하면 좌표를 좁힐 수 있습니다.` |
| 접근 계산 | `인양선 보정 중입니다.` |
| 출격 가능 | `인양 가능. 회수선이 좌표를 붙잡고 있습니다.` |
| 보류 | `신호는 강하지만, 현재 인양망으로는 접근할 수 없습니다.` |

## 8. 캐릭터/서사 연결

P1은 새 캐릭터를 모두 해금하는 단계가 아니라, 이후 캐릭터 해금 구조의 근거를 심는 단계다.

캐릭터별 U0~U5 해금 단계와 정식 플레이어블 전환 기준은 `docs/world/CHARACTER_UNLOCK_STRUCTURE_V0_1.md`를 따른다.

| 축 | P1에서 해야 할 일 | 아직 하면 안 되는 일 |
|---|---|---|
| 윤서 | 반품/회수 규칙을 읽는 능력이 생존 기술임을 보여준다. | 구문명 숙련 직장인처럼 보이게 하지 않는다. |
| 의료계 캐릭터 | 환자 등록과 구출 윤리의 문제를 소개한다. | R02에서 모든 의료 캐릭터를 확정하지 않는다. |
| 해방 로봇 | 충전태그 의존과 재동기화 사이의 갈등을 보여준다. | 모든 로봇을 피해자 또는 적으로 단순화하지 않는다. |
| 브로커 출신 캐릭터 | 인간이 캠페인 규칙을 악용할 수 있음을 보여준다. | 인간 악역을 세계 원인의 주범처럼 만들지 않는다. |
| 침묵권 기술자 | 보급소 유지가 기술/물자/신뢰 문제임을 보여준다. | 보급소를 완전 안전한 왕국으로 보이게 하지 않는다. |

## 9. 인양/침묵권 정합성

P1에서 인양은 더 편해지는 것이 아니라 더 복잡해진다.

| 구분 | P0 | P1 |
|---|---|---|
| 주 활동 거리 | R08/R01 근거리 | R02/R03/R04/R06 외부 권역 |
| 인양 상태 | A1 중심 | A1~A3 혼합 |
| 실패 의미 | 미정산 회수품 손실 | 좌표 흔들림, 태그 오염, 보급소 노출 위험 |
| 침묵권 역할 | 튜토리얼 거점 | 확장해야 유지되는 네트워크 |

주의:

- R08-L03/R08-L04는 보급소가 강해진다는 보상이 아니라, 약점을 보수하는 과정이다.
- R04-L02/R04-L03 같은 A3 노드는 P1이라도 항상 안전하게 인양되면 안 된다.
- P1 후반 노드는 `긴급 인양 가능`보다 `인양 조건부`가 어울린다.

## 10. P1에서 금지할 것

- R01-L03 처리 직후 P1 16개 노드를 모두 출격 가능으로 열기.
- R06 상업/브로커 축을 첫 외부 진입 축으로 만들기.
- R05/R07을 P1에서 실제 플레이 권역으로 열기.
- R04를 아무 단서 없이 갑자기 여는 것.
- 윤서의 반품 담당자 정체성을 구문명 숙련 직업으로 확정하기.
- 보급소가 P1에서 갑자기 대규모 안전 도시처럼 보이게 하기.
- P1에서 세계 원인의 모든 진실을 설명하기.
- P1 보상으로 보급태그를 돈/포인트처럼 단순 지급하기.

## 11. 논리 검증

| 항목 | 판정 |
|---|---|
| P1이 첫 보스 직후 과하게 열리지 않는가? | 통과. `신호 감지`, `해석 가능`, `출격 가능`을 분리했다. |
| 후인류 시대와 캠페인 형성 원인에 맞는가? | 통과. 노드들은 새로 생긴 광고 던전이 아니라 사고 전 서비스 목적과 지역 승인 캐시가 장기간 환경화된 인프라다. |
| 캠페인이 학살자가 아니라는 기준과 맞는가? | 통과. R02/R03/R04/R06 모두 등록/정산/보존 규칙의 오작동으로 설명된다. |
| 인간이 너무 많아 보이지 않는가? | 통과. P1 인간 노출은 환자, 브로커, 등록자, 보급소 인원으로 제한된다. |
| 윤서 26세 설정과 충돌하지 않는가? | 통과. 반품 담당자는 구문명 숙련 직장이 아니라 침묵 보급소 역할로 유지한다. |
| 해방 로봇이 갑자기 아군이 되지 않는가? | 통과. R04는 R03/R02 단서 후 열리며, 충전태그/재동기화 문제를 먼저 드러낸다. |
| R06 인간 악역이 세계 원인을 덮지 않는가? | 조건부 통과. R06은 P1 후반 또는 선택 축으로 제한해야 한다. |
| P2 미스터리가 조기 소모되지 않는가? | 통과. R05/R07은 원격 신호와 보류 상태로 둔다. |

## 12. 최종 잠금

```text
1. P1은 첫 보스 후 세계가 넓어진다는 감각을 주는 확장 단계다.
2. P1 노드는 내부 분류일 뿐, 모두 즉시 출격 가능하다는 뜻이 아니다.
3. 첫 보스 직후에는 R01 재방문과 R08 침묵권 확장을 먼저 연다.
4. 첫 외부 권역은 R03 물류 회수 또는 R02 의료 격리 중 하나를 선택하게 한다.
5. R03은 윤서의 반품/회수 감각을 설명한다.
6. R02는 등록된 인간이 살아 있어도 자유롭지 않다는 문제를 보여준다.
7. R04는 R03/R02 단서를 통해 열며, 해방 로봇과 충전태그 의존 문제를 다룬다.
8. R06은 가짜 태그/브로커 단서 뒤에 늦게 열어 인간 악역을 소개한다.
9. R05/R07은 P1에서 실제 진입하지 않고 원격 신호로만 남긴다.
10. P1의 보상은 안전 확대가 아니라 더 먼 위험을 해석할 수 있는 능력이다.
```
