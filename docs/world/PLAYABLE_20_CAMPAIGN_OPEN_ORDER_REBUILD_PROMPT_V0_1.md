# 플레이어블 20인 캠페인 오픈 순번 재배치 팀 프롬프트 v0.1

## 역할

너는 스토리 작성 전에 플레이어블 캐릭터 순번을 검수하는 팀이다. 이번 작업의 목적은 새 캐릭터를 만드는 것이 아니라, 기존 플레이어블 20인 로스터의 번호가 `프로필 작성 번호`와 `최종 해금 순번`을 혼동한 문제를 교정하는 것이다.

## 핵심 전제

- 세계는 단일 캠페인이 아니라 `12개 광역권 x 8개 캠페인 = 96개 캠페인` 구조다.
- 플레이어블은 캠페인이 어느 정도 열린 뒤에 해금되어야 한다.
- 같은 캠페인에서 나온 캐릭터는 해당 캠페인이 충분히 진행되기 전까지 추가하지 않는다.
- RIO-08은 원래 초반 로봇 후보처럼 보여도 캠페인 오픈 구조 때문에 뒤로 밀린 사례다. 이 원칙을 전체 로스터에 적용한다.
- 기존 1~13번 프로필 v1.0은 보존하되, 그 번호를 최종 해금 순번으로 확정하지 않는다.
- 13~19 한국어권 임시명은 최종명으로 사용하지 않고, 전역 이름/원점 교정 후보를 우선한다.

## 작성해야 할 내용

### 1. 기존 오류 판정

다음을 명확히 선언한다.

- 기존 로스터는 `프로필 작성 번호`와 `해금 순번`이 섞였다.
- 14번 이후를 계속 작성하면 세계 캠페인 구조를 망친다.
- 새 프로필 작성은 재배치 완료 전까지 중단한다.

### 2. 캠페인 오픈 기준표

96개 캠페인 전체를 모두 상세화하지 말고, 플레이어블 해금 판단에 필요한 수준으로만 정리한다.

필수 포함:

- E01 시작권
- E02 첫 외부 슬라이스
- E10/E07/E09 중기 확장권
- E11/E06/E08/E12 후기 확장권
- E03/E04/E05 장기 보류권

### 3. 기존 1~13 후보 재배치표

표 컬럼은 다음을 사용한다.

| 기존 번호 | 후보 | 원점 캠페인 | 기존 번호 유지 여부 | 새 해금 레인 | 판정 사유 |

필수 판정:

- 윤서는 1번 유지 가능.
- RIO-08은 2번으로 당기지 말고, 원점 캠페인 개방 뒤로 미룬다.
- GATE/BROAD/CLAUSE/LEDGER/GUARD는 해당 외부 캠페인 개방 뒤로만 배치한다.
- HOLD/RETURN/METER/QUIET은 각 지역 캠페인 개방 깊이를 재검산한다.
- PATCH는 2번 유지 여부를 별도 검증한다. 로봇이라는 이유만으로 2번 고정하지 않는다.

### 4. 14~20 후보 처리

- 기존 14~20 후보는 이름/국적/기능을 확정하지 않는다.
- 세계 캠페인 분포가 먼저다.
- 필요한 경우 기존 후보를 보류하고 새 권역 후보를 요청한다.
- 한국어 이름이 연속 배치된 경우 반드시 분산하거나 보류한다.
- 현재 교정 후보는 Amina Diop, Arjun Mehta, Maya Chen, Sari Wijaya, Omar Al-Hadid, Leila Haddad, Mateo Rojas다.

### 5. 새 해금 순번 초안

최종 확정이 아니라 `초안`으로 작성한다.

표 컬럼은 다음을 사용한다.

| 새 해금 순번 | 후보 | 원점 캠페인/레인 | 상태 | 다음 필요 작업 |

상태 값은 아래 중 하나만 사용한다.

- `확정 가능`
- `조건부 유지`
- `뒤로 이동`
- `보류`
- `재선정 필요`

마지막에 다음 작업을 하나만 지정한다.

허용되는 다음 작업:
