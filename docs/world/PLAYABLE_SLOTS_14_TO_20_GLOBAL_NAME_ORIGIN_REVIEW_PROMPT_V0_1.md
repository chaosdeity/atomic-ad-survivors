# 플레이어블 14~20 글로벌 이름/원점 재검토 프롬프트 v0.1

이 문서는 플레이어블 제작 AI팀에게 `14~20번 글로벌 이름/원점 재검토`를 맡기기 위한 복붙용 프롬프트다.

현재 상태: `작성 완료 / 조건부 통과`.

이 문서는 실행 완료되었다. 기존 1~13번의 프로필 산출 번호와 최종 해금 순번 분리도 완료되었다.

전역 이름/원점 교정은 이미 `story/06_characters/playable_20_global_name_origin_correction_v0_1.md`에 반영했다.

목표는 14~20번 프로필을 쓰는 것이 아니다.
목표는 현재 레지스트리에 남아 있는 한국어권 임시명이 세계 캠페인 구조를 잠식하지 않도록, 14~20번의 이름/국적/언어권/원점 광역권을 상세 결재 전에 다시 검수하는 것이다.

핵심:

```text
캠페인은 세계적으로 연결되어 있다.
14~20번을 한국어권 임시명 그대로 밀면 안 된다.
현재 14~19번의 한국어권 이름은 최종명이 아니라 L2 방향 후보용 임시명이다.
14~20번은 E02~E12 실제 광역권, 캠페인 등록 방식, 언어권/이주/다국적 맥락에 맞춰 재판정한다.
단, 이 재판정은 캠페인 오픈 순번 재배치 완료 이후에만 진행한다.
```

## 0. 현재 문제

현재 이름 레지스트리에는 아래 L2/L1 후보가 있다.

```text
14 신이준 / TRANSIT-14 / 이준, 미탑승 승객
15 박다미 / REVIEW-15 / 다미, 리뷰 거부자
16 오진솔 / RATION-16 / 진솔, 배급 명단 밖 조리자
17 남태건 / LOCKOUT-17 / 태건, 산업 안전 미승인자
18 최미결 / NAME-18 / 미결, 이름 보관 실패자
19 나유찬 / LINE-19 / 유찬, 회수선 조정자
20 본명 보류 / OBS-20 / 시즌 관측자
```

이 표기는 방향성 슬롯을 임시로 관리하기 위한 것이었다.
하지만 캠페인이 세계적으로 연결되어 있으므로, 14~19번을 한국어권 이름으로 그대로 상세 결재하면 로스터가 E01 중심으로 다시 쏠린다.

이번 작업은 이 위험을 먼저 끊는다.

```text

4 강하람 / HOLD-04
5 한이루 / RETURN-05
6 차유건 / METER-06
7 백여울 / QUIET-07
```

판정 기준:

```text

14~19번 한국어권 임시명은 최종 한국인 카운트에 넣지 않는다.
```

중요:

```text
이 카운트는 14~20번을 한국어권으로 계속 늘리라는 뜻이 아니다.
오히려 14~20번은 세계 캠페인 분산을 회복하기 위한 재검토 대상이다.
```

## 5. 슬롯별 판정 항목

각 슬롯마다 아래를 판정한다.

```text
1. 현재 한국어권 임시명을 폐기/보류/예외 유지 중 무엇으로 처리할지.
2. 후보 원점 캠페인권을 어느 E02~E12 광역권에서 잡을지.
3. 같은 C번호라도 기존 플레이어블과 어떤 절차 차이가 있는지.
4. 본명을 지금 줄지, 코드명/역할명으로 보류할지.
5. 이름 후보가 필요하다면 어떤 언어권/다국적 맥락에서 후보를 만들지.
6. 유저 첫 표기는 본명보다 역할명/코드명을 우선해야 하는지.
7. 현실 지명/기관/민족 클리셰 노출 위험이 있는지.
8. 다음 상세 결재로 넘어갈 슬롯이 무엇인지.
```

이름 판정은 아래 다섯 단계 중 하나로 한다.

```text
폐기:
현재 한국어권 임시명이 세계 원점과 맞지 않아 버린다.

보류:
현재 임시명을 최종명으로 쓰지 않고 코드명/역할명으로 관리한다.

예외 유지:
강한 이주/다국적/서사상 이유가 있어 유지 가능하다. 단, 현실권 대표명으로 쓰지 않는다.

현지명 후보 필요:
원점 광역권 언어권에 맞는 새 후보를 다음 문서에서 뽑는다.

다국적 이름 후보 필요:
항만/공항/검역/플랫폼/관광/이송처럼 다국적 이동이 자연스러운 캠페인에서 새 후보를 뽑는다.
```

## 6. 글로벌 배치 원칙

14~20번은 한국어권 이름을 기본값으로 삼지 않는다.

배치 원칙:

```text
E01 초반 한국어권 플레이어블은 이미 충분하다.
9~12번은 E02/E07/E09/E11 외부 광역권에서 프로필 v1.0까지 완료되었다.
13번 GUARD-13은 E12_C02 원점이며 전태주 이름은 대표명 금지 조건이 붙은 예외다.
14~20번은 E02~E12에 남은 광역권과 C번호 절차를 더 넓게 분산해야 한다.
```

우선 피해야 할 것:

```text
14~19번을 모두 한국어권 이름으로 유지
E01/R01/R02/R03/R04/R08로 회귀
9번 GATE-09와 14번 TRANSIT-14를 같은 E02_C07 반복으로 처리
13번 GUARD-13을 이유로 E12에 한국어권 이름을 연속 배치
현실 문화권 이름을 장식처럼 붙이기
국적을 캐릭터 매력 태그로 쓰기
실제 도시/기관/사건 패러디
```
